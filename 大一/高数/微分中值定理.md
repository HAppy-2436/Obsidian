# 微分中值定理（Mean Value Theorems）

## 1. 罗尔定理（Rolle's Theorem）

**定理内容：**

$$
\text{若 } 
\begin{cases}
f(x) \text{ 在 } [a,b] \text{ 上连续}, \\
f(x) \text{ 在 } (a,b] \text{ 内可导}, \\
f(a)=f(b),
\end{cases}
$$

$$
\text{则存在 } \xi \in (a,b) \text{ 使得 } f'(\xi)=0.
$$


---

## 2. 拉格朗日中值定理（Lagrange Mean Value Theorem, LMVT）

**定理内容：**

$$
\text{若 } 
\begin{cases}
f(x) \text{ 在 } [a,b] \text{ 上连续}, \\
f(x) \text{ 在 } (a,b) \text{ 内可导},
\end{cases}
$$

$$
\text{则存在 } \xi \in (a,b) \text{ 使得 } 
f'(\xi)=\frac{f(b)-f(a)}{b-a}.
$$


**几何意义：**

$$
\text{存在一点切线斜率等于割线斜率。}
$$


---

## 3. 柯西中值定理（Cauchy Mean Value Theorem, CMVT）

**定理内容：**

$$
\text{若 } 
\begin{cases}
f(x), g(x) \text{ 在 } [a,b] \text{ 上连续}, \\
f(x), g(x) \text{ 在 } (a,b) \text{ 内可导}, \\
g'(x) \neq 0 \text{ 对所有 } x \in (a,b),
\end{cases}
$$

$$
\text{则存在 } \xi \in (a,b) \text{ 使得 }
\frac{f'(\xi)}{g'(\xi)} = 
\frac{f(b)-f(a)}{g(b)-g(a)}.
$$


**柯西中值定理是拉格朗日中值定理的推广：**

取 \( g(x)=x \) 即得到 LMVT。


---

## 4. 常见应用总结（考试常用）

- LMVT 用于证明：  
  - 单调性  
  - 函数不等式  
  - 控制增量 \( f(b)-f(a) \)  
  - 证明极限存在性  

- CMVT 用于：  
  - 处理两个函数的比例关系  
  - 推导洛必达法则  
  - 证明更复杂的不等式
