# 伴随矩阵

## 定义
n阶方阵A的伴随矩阵A^*定义为：(A^*)_{ij} = A_{ji}（A_{ji}是元素a_{ji}的代数余子式），即代数余子式矩阵的转置。满足重要等式：AA^* = A^*A = |A|E。伴随矩阵是求[[逆矩阵]]公式法的核心，当|A|≠0时，A^{-1} = (1/|A|)·A^*。

## 核心内容
**定义与计算**：
- (A^*)_{ij} = A_{ji}（转置！）
- A_{ji} = (-1)^{j+i}·M_{ji}
- M_{ji}是a_{ji}的余子式（划去第j行第i列）

**重要公式**：
1. **AA^* = A^*A = |A|E**
   
2. **逆矩阵公式**：
   当|A| ≠ 0时，A^{-1} = (1/|A|)·A^*

3. **伴随矩阵的性质**：
   - |A^*| = |A|^{n-1}（n阶矩阵）
   - (A^*)^* = |A|^{n-2}·A（n≥3）
   - (A^*)^T = (A^T)^*
   - (AB)^* = B^*A^*（顺序颠倒）
   - (kA)^* = k^{n-1}A^*

4. **特征值关系**：
   若λ是A的特征值，则|A|/λ是A^*的特征值

**秩的关系**：
- r(A) = n → r(A^*) = n
- r(A) = n-1 → r(A^*) = 1
- r(A) < n-1 → r(A^*) = 0

**2阶矩阵的特殊情况**：
A = [a b]     A^* = [d  -b]
    [c d]           [-c  a]

## 应用场景
理论推导中求逆矩阵公式；证明矩阵相关性质；小规模矩阵的逆矩阵计算。实际计算一般用初等变换更快。

## 注意事项
- 定义中有转置：(A^*)_{ij} = A_{ji}
- |A^*| = |A|^{n-1}，不是|A|
- (AB)^* = B^*A^*，顺序颠倒
- r(A)与r(A^*)的关系要记住
