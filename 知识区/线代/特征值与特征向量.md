# 特征值与特征向量

## 定义
对n阶方阵A，若存在数λ和非零向量x使得 Ax = λx，则称λ为A的特征值，x为对应的特征向量。几何意义：特征向量经A变换后方向不变，只伸缩λ倍。特征值是[[行列式]]|λE-A|=0的根，是矩阵最重要的数值特征之一。

## 核心内容
**求解步骤**：
1. **求特征值**：解特征方程 |λE - A| = 0
   展开得n次多项式（特征多项式）
   
2. **求特征向量**：对每个λ_i，解齐次方程组
   (λ_i E - A)x = 0
   得到的非零解即为特征向量

**重要性质**：
1. n阶矩阵有n个特征值（含重根）
2. Σλ_i = tr(A)（迹：主对角线元素之和）
3. ∏λ_i = |A|（行列式）
4. 不同特征值的特征向量线性无关
5. k重特征值最多有k个线性无关的特征向量

**特殊矩阵的特征值**：
- 三角矩阵/对角矩阵：主对角线元素
- [[正交矩阵]]：|λ| = 1
- [[正定矩阵]]：所有特征值 > 0
- 对称矩阵：特征值全为实数
- [[伴随矩阵]] A^*：
  - 若λ是A的特征值，|A|/λ 是A^*的特征值

**应用**：
1. **[[对角化]]**：
   若A有n个线性无关的特征向量，则
   A = PDP^{-1}
   D = diag(λ₁,λ₂,...,λ_n)，P的列为特征向量
   
2. **[[矩阵的幂]]**：
   A^n = PD^nP^{-1}
   D^n = diag(λ₁^n,λ₂^n,...,λ_n^n)

## 应用场景
求矩阵高次幂；微分方程组求解；主成分分析（PCA）；Google PageRank算法；量子力学中的能级计算；振动分析。

## 注意事项
- 特征向量必须非零
- 同一特征值可有多个线性无关的特征向量
- 特征向量可乘任意非零常数
- n阶矩阵可对角化 ⟺ 有n个线性无关的特征向量
