# 行列式

## 定义
n阶方阵A的行列式记作det(A)或|A|，是一个数值，表示该[[矩阵]]代表的线性变换对空间体积的缩放比例。几何意义：行向量（或列向量）张成的n维平行体的有向体积。代数意义：用于判断矩阵是否可逆、求解[[线性方程组]]等。

## 核心内容
**基本性质**：
1. 转置不变：|A^T| = |A|
2. 某行（列）为0，则行列式为0
3. 某行（列）有公因子k，可提出：k|A|
4. 互换两行（列），行列式变号
5. 某行（列）的k倍加到另一行（列），行列式不变
6. 两行（列）相同或成比例，行列式为0

**计算方法**：
1. **按行/列展开**：
   |A| = Σ a_{ij}·A_{ij}（i或j固定）
   A_{ij} = (-1)^{i+j}M_{ij}（代数余子式）

2. **化为三角形**：
   利用[[初等变换]]化为上三角，主对角线元素之积
   
3. **特殊行列式**：
   - 上/下三角：主对角线元素之积
   - 对角矩阵：对角元素之积
   - 范德蒙德：∏(x_j - x_i)

**重要公式**：
- |AB| = |A|·|B|
- |kA| = k^n|A|（n阶方阵）
- |A^{-1}| = 1/|A|（A可逆）
- |A^*| = |A|^{n-1}（伴随矩阵）

**判定定理**：
- |A| ≠ 0 ⟺ A可逆
- |A| = 0 ⟺ A的行/列向量线性相关

## 应用场景
判断矩阵是否可逆；[[克拉默法则]]求解线性方程组；求[[特征值与特征向量]]（|λE-A|=0）；几何中计算面积/体积。

## 注意事项
- |A+B| ≠ |A| + |B|（一般情况）
- 计算时灵活运用性质简化
- 化三角形时注意换行要变号
- 0行/0列可快速判断行列式为0
