# 向量空间

## 定义
满足加法和数乘封闭性的向量集合V，需满足8条运算律（加法交换律、结合律、零元素、负元素，数乘结合律、分配律等）。向量空间是线性代数的核心抽象概念，例如R^n、多项式空间、函数空间都是向量空间。子空间、基、维数是向量空间的重要概念。

## 核心内容
**基本概念**：
1. **子空间**：
   向量空间V的子集W，若对加法和数乘封闭，则W是V的子空间
   
2. **基**：
   向量空间V的线性无关向量组，且V中任意向量可由其线性表示
   
3. **维数**：
   基所含向量的个数，记作dim(V)
   
4. **坐标**：
   向量在某组基下的线性表示系数

**重要子空间**：
1. **行空间**：矩阵A的行向量张成的空间
   dim = r(A)
   
2. **列空间**：矩阵A的列向量张成的空间
   dim = r(A)
   
3. **零空间（核空间）**：Ax = 0的解空间
   dim = n - r(A)（A为m×n矩阵）
   
4. **左零空间**：A^Tx = 0的解空间

**维数定理**：
1. 有限维空间的基必含相同个数的向量
2. n维空间中n个线性无关向量构成一组基
3. 子空间的维数 ≤ 空间的维数

**子空间的运算**：
1. **和空间** V₁ + V₂：
   dim(V₁+V₂) = dim(V₁) + dim(V₂) - dim(V₁∩V₂)
   
2. **直和** V₁ ⊕ V₂：
   V₁∩V₂ = {0}
   dim(V₁⊕V₂) = dim(V₁) + dim(V₂)

**常见向量空间**：
- R^n：n维实向量空间
- P_n：次数≤n的多项式空间，dim = n+1
- C[a,b]：[a,b]上连续函数空间（无限维）

## 应用场景
抽象线性代数理论；量子力学的态空间；信号处理的信号空间；机器学习的特征空间；函数逼近理论。

## 注意事项
- 子空间必须包含零向量
- 有限维空间的基不唯一但维数唯一
- 零空间维数 = n - r(A)
- 直和要求交集只有零向量
