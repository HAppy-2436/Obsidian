# 线性变换

## 定义
从[[向量空间]]V到W的映射T满足：①T(x+y) = T(x) + T(y)；②T(kx) = kT(x)，则称T为线性变换（线性映射）。有限维空间的线性变换可用矩阵表示：T(x) = Ax。线性变换保持向量空间的线性结构，是线性代数的核心研究对象，[[矩阵乘法]]本质就是线性变换的复合。

## 核心内容
**定义**（等价条件）：
T: V → W 是线性变换 ⟺
1. T(x + y) = T(x) + T(y)（保持加法）
2. T(kx) = kT(x)（保持数乘）
⟺
T(k₁x₁ + k₂x₂) = k₁T(x₁) + k₂T(x₂)

**矩阵表示**：
若dim(V)=n，dim(W)=m
选定V的基α₁,...,α_n 和 W的基β₁,...,β_m
则T可用m×n矩阵A表示：
T(x) = Ax

A的第j列 = T(α_j)在W的基下的坐标

**核与像**：
1. **核（零空间）**：
   Ker(T) = {x∈V | T(x) = 0}
   dim(Ker(T)) = n - r(A)（nullity）

2. **像（值域）**：
   Im(T) = {T(x) | x∈V}
   dim(Im(T)) = r(A)（rank）

**秩-零化度定理**：
dim(V) = dim(Ker(T)) + dim(Im(T))
即：n = nullity + rank

**基变换**：
T在不同基下的矩阵A和B满足：
B = P^{-1}AP（[[相似矩阵]]）
P是[[基变换]]的过渡矩阵

**特殊线性变换**：
1. **恒等变换**：I(x) = x，矩阵为E
2. **零变换**：0(x) = 0，矩阵为O
3. **数乘变换**：k·I，矩阵为kE
4. **投影变换**：投影到子空间
5. **旋转变换**：绕原点旋转
6. **反射变换**：关于超平面对称

**性质**：
1. T可逆 ⟺ Ker(T) = {0} ⟺ T是单射
2. T满射 ⟺ Im(T) = W
3. (T₁∘T₂)(x) = T₁(T₂(x))，矩阵为A₁A₂

## 应用场景
几何变换（旋转、缩放、投影）；微分方程的算子理论；量子力学的态演化；计算机图形学；信号处理的滤波器；数据降维（PCA）。

## 注意事项
- 线性变换的核心是保持线性运算
- 矩阵是线性变换在特定基下的表示
- 同一变换在不同基下的矩阵相似
- 零变换的核是整个空间
