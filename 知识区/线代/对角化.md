# 对角化

## 定义
若n阶方阵A相似于对角矩阵Λ，即存在可逆矩阵P使得 P^{-1}AP = Λ = diag(λ₁, λ₂, ..., λ_n)，则称A可对角化。对角矩阵Λ的对角元素是A的[[特征值与特征向量]]，P的列向量是对应的特征向量。对角化使矩阵运算（特别是求幂）极大简化。

## 核心内容
**可对角化条件**（等价命题）：
1. A有n个线性无关的特征向量
2. A的每个特征值的几何重数 = 代数重数
3. 若A有n个互不相同的特征值，则A必可对角化

**对角化步骤**：
1. 求特征值：解|λE - A| = 0
2. 对每个特征值λ_i，求特征向量：解(λ_i E - A)x = 0
3. 将n个线性无关的特征向量作为P的列
4. 则 P^{-1}AP = Λ = diag(λ₁, λ₂, ..., λ_n)

**特殊矩阵的对角化**：
1. **对称矩阵**（实对称）：
   - 必可对角化
   - 特征值全为实数
   - 不同特征值的特征向量正交
   - 可正交对角化：Q^TAQ = Λ，Q为[[正交矩阵]]

2. **实对称矩阵正交对角化**：
   Q的列向量是单位正交的特征向量组
   先用[[施密特正交化]]，再单位化

**应用**：
1. **求矩阵的幂**：
   A = PΛP^{-1} ⟹ A^n = PΛ^nP^{-1}
   Λ^n = diag(λ₁^n, λ₂^n, ..., λ_n^n)

2. **求矩阵函数**：
   f(A) = Pf(Λ)P^{-1}
   如 e^A = Pe^ΛP^{-1}

3. **化[[二次型]]为标准形**：
   通过正交变换化简

**不可对角化的情况**：
若特征值的几何重数 < 代数重数，则不可对角化
可用若尔当(Jordan)标准形

## 应用场景
求矩阵高次幂；求矩阵函数；化二次型为标准形；微分方程组求解；马尔可夫链稳态分布；主成分分析。

## 注意事项
- 对称矩阵一定可对角化
- P的列是特征向量，顺序要与Λ的对角元素对应
- 正交对角化时要进行施密特正交化
- 不可对角化可用若尔当标准形
