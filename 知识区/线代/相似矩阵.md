# 相似矩阵

## 定义
n阶方阵A与B相似，记作A∼B，若存在可逆矩阵P使得 B = P^{-1}AP。相似关系是一种等价关系，表示A和B代表同一个[[线性变换]]在不同基下的矩阵。相似矩阵有相同的[[特征值与特征向量]]、行列式、迹、秩等不变量。[[对角化]]就是将矩阵相似于对角矩阵。

## 核心内容
**定义**：
A ∼ B ⟺ ∃可逆矩阵P，使 B = P^{-1}AP

**相似的几何意义**：
A和B表示同一线性变换在不同基下的矩阵
P是基变换矩阵

**相似关系的性质**（等价关系）：
1. 反身性：A ∼ A
2. 对称性：A ∼ B ⟹ B ∼ A
3. 传递性：A ∼ B, B ∼ C ⟹ A ∼ C

**相似不变量**：
相似矩阵有相同的：
1. **特征多项式** |λE - A|
2. **特征值**（含重数）
3. **行列式** |A|
4. **迹** tr(A) = Σλ_i
5. **秩** r(A)
6. **可逆性**
7. 特征值的几何重数

**判定方法**：
1. 有相同特征值是必要不充分条件
2. 同时对角化于同一对角矩阵是充要条件
3. 对实对称矩阵：特征值相同⟺相似

**与对角化的关系**：
- A可对角化 ⟺ A相似于对角矩阵Λ
- A ∼ Λ = diag(λ₁, ..., λ_n)
- P的列是A的特征向量

**相似标准形**：
- 可对角化：对角矩阵
- 不可对角化：若尔当(Jordan)标准形

## 应用场景
简化矩阵计算（通过对角化）；求矩阵高次幂；线性变换理论；微分方程组求解；量子力学中的表象变换。

## 注意事项
- 相似必有相同特征值，但反之不一定
- 相似矩阵有相同的迹和行列式
- A∼B不能推出A^T∼B^T（一般情况）
- 实对称矩阵：特征值相同⟺相似
