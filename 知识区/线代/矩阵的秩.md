# 矩阵的秩

## 定义
矩阵A的秩r(A)定义为A中非零子式的最高阶数，等价于A的行秩（线性无关行向量的最大个数）和列秩（线性无关列向量的最大个数）。秩反映了矩阵的"有效维度"，是判断[[线性方程组]]解的存在性和[[向量空间]]维数的关键指标。

## 核心内容
**等价定义**：
1. 非零子式的最高阶数
2. 线性无关行向量的最大个数（行秩）
3. 线性无关列向量的最大个数（列秩）
4. 行秩 = 列秩（重要定理）

**基本性质**：
1. 0 ≤ r(A) ≤ min(m, n)（m×n矩阵）
2. r(A^T) = r(A)
3. r(kA) = r(A)（k≠0）
4. r(A+B) ≤ r(A) + r(B)
5. r(AB) ≤ min{r(A), r(B)}
6. 若A可逆，r(AB) = r(B)，r(BA) = r(B)

**计算方法**：
1. **初等变换法**：
   化为阶梯形矩阵，非零行的个数即为秩
   
2. **子式法**：
   找最高阶非零子式（理论方法）

**满秩**：
- 行满秩：r(A) = m（m×n矩阵）
- 列满秩：r(A) = n
- 满秩方阵：r(A) = n ⟺ |A|≠0 ⟺ A可逆

**秩与线性方程组**：
- Ax = b有解 ⟺ r(A) = r(A|b)
- 唯一解 ⟺ r(A) = n
- 无穷多解 ⟺ r(A) < n

**重要定理**：
1. Sylvester不等式：
   r(A) + r(B) - n ≤ r(AB)
   
2. r(A) = r(AA^T) = r(A^TA)

3. 分块矩阵秩的性质：
   r([A B]) ≤ r(A) + r(B)

## 应用场景
判断线性方程组解的情况；判断向量组的线性相关性；计算向量空间的维数；判断矩阵是否可逆；数据压缩（低秩近似）。

## 注意事项
- 行秩 = 列秩（必须记住）
- r(AB) ≤ min{r(A), r(B)}
- r(A+B) ≤ r(A) + r(B)
- 初等变换不改变秩
