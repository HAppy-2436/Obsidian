# 逆矩阵

## 定义
对n阶方阵A，若存在n阶方阵B使得 AB = BA = E（单位矩阵），则称B为A的逆矩阵，记作A^{-1}。可逆的充要条件是|A|≠0（A为非奇异矩阵）。逆矩阵表示线性变换的"撤销"操作，是求解[[线性方程组]]的重要工具。

## 核心内容
**可逆条件**（等价命题）：
- |A| ≠ 0
- A的行（列）向量组线性无关
- r(A) = n
- Ax = 0只有零解
- A可表示为[[初等变换]]矩阵的乘积
- A的特征值全不为0

**求逆方法**：
1. **公式法**：
   A^{-1} = (1/|A|)·A^*
   其中A^*是[[伴随矩阵]]
   
2. **初等行变换法**：
   (A | E) → (E | A^{-1})
   对增广矩阵进行行变换
   
3. **分块矩阵法**：
   对分块对角矩阵，逆矩阵也是分块对角的

**运算性质**：
1. (A^{-1})^{-1} = A
2. (AB)^{-1} = B^{-1}A^{-1}（顺序颠倒）
3. (A^T)^{-1} = (A^{-1})^T
4. (kA)^{-1} = (1/k)A^{-1}（k≠0）
5. |A^{-1}| = 1/|A|
6. (A^n)^{-1} = (A^{-1})^n

**特殊矩阵的逆**：
- 对角矩阵：对角元素取倒数
- [[正交矩阵]]：A^{-1} = A^T
- 对称矩阵的逆仍对称
- 上（下）三角矩阵的逆仍是上（下）三角

## 应用场景
求解线性方程组 Ax=b（x = A^{-1}b）；矩阵方程求解；密码学中的加密解密；计算机图形学变换的逆操作。

## 注意事项
- 只有方阵才可能有逆
- (AB)^{-1} = B^{-1}A^{-1}（顺序颠倒！）
- A+B可逆不能推出A、B都可逆
- 初等行变换法最常用
