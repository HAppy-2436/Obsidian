# 施密特正交化

## 定义
Gram-Schmidt正交化是将线性无关向量组 α₁, α₂, ..., α_n 转化为正交向量组 β₁, β₂, ..., β_n（或标准正交向量组）的系统方法。正交化保持线性无关性和张成空间不变，是构造[[正交矩阵]]、求[[二次型]]标准形、QR分解的基础工具，在数值计算中应用广泛。

## 核心内容
**正交化过程**：
设 α₁, α₂, ..., α_n 线性无关

1. β₁ = α₁

2. β₂ = α₂ - (α₂·β₁)/(β₁·β₁)·β₁

3. β₃ = α₃ - (α₃·β₁)/(β₁·β₁)·β₁ - (α₃·β₂)/(β₂·β₂)·β₂

一般地：
β_k = α_k - Σ_{j=1}^{k-1} [(α_k·β_j)/(β_j·β_j)]·β_j

**单位化**（标准正交化）：
γ_i = β_i / ||β_i|| = β_i / √(β_i·β_i)

得到标准正交向量组：γ₁, γ₂, ..., γ_n
满足：<γ_i, γ_j> = δ_{ij}（Kronecker符号）

**几何意义**：
β_k是α_k减去它在前k-1个正交向量上的投影
即β_k垂直于span{β₁,...,β_{k-1}}

**矩阵形式**（QR分解）：
设A = [α₁ α₂ ... α_n]
则A = QR
- Q = [γ₁ γ₂ ... γ_n]（正交矩阵）
- R为上三角矩阵

**性质**：
1. β₁, ..., β_n 两两正交
2. span{α₁,...,α_k} = span{β₁,...,β_k}
3. 若进行单位化，得标准正交基
4. 正交化不改变向量组的秩

**算法稳定性**：
经典Gram-Schmidt在数值计算中可能不稳定
改进的Gram-Schmidt (MGS)更稳定

## 应用场景
构造正交矩阵；实对称矩阵的正交[[对角化]]；最小二乘法的QR分解；信号处理的正交基构造；求解线性方程组的数值方法。

## 注意事项
- 必须从第一个向量开始依次正交化
- 每次减去的是投影，不是简单相减
- 单位化不要忘记除以长度
- 数值计算中用改进的Gram-Schmidt

## 关联知识
与 [[正交矩阵]]、[[内积]]、[[对角化]]、[[二次型]]、[[QR分解]] 相关。
