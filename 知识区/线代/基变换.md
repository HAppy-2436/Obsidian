# 基变换

## 定义
[[向量空间]]V从一组基 {α₁,...,α_n} 到另一组基 {β₁,...,β_n} 的转换，通过过渡矩阵P连接：(β₁,...,β_n) = (α₁,...,α_n)P。同一向量在不同基下有不同坐标，坐标变换公式为 [x]_β = P^{-1}[x]_α。基变换是理解[[相似矩阵]]和[[线性变换]]表示矩阵变化的关键。

## 核心内容
**过渡矩阵**：
设V有两组基：
- 旧基：α₁, α₂, ..., α_n
- 新基：β₁, β₂, ..., β_n

若 β_j = Σ p_{ij}α_i（j=1,...,n）
则过渡矩阵P = (p_{ij})_{n×n}
矩阵形式：(β₁ ... β_n) = (α₁ ... α_n)P

**坐标变换公式**：
设向量v在两组基下的坐标分别为：
- 旧基下：[x]_α = (x₁,...,x_n)^T
- 新基下：[y]_β = (y₁,...,y_n)^T

则：[x]_α = P[y]_β
或：[y]_β = P^{-1}[x]_α

**直观理解**：
- 基从α变到β：β = αP
- 坐标从α变到β：y = P^{-1}x
- 基的变换矩阵与坐标的变换矩阵互逆

**线性变换的矩阵表示**：
设[[线性变换]] T:V→V 在两组基下的矩阵分别为A和B
则B = P^{-1}AP（相似变换）
即：B与A[[相似矩阵]]

**特殊情况**：
1. **正交基变换**：
   若两组基都是标准正交基
   过渡矩阵P是[[正交矩阵]]

2. **对角化**：
   从标准基到特征向量基的变换
   A相似于对角矩阵Λ

**计算步骤**：
1. 写出新基在旧基下的表示
2. 新基的坐标作为P的列向量
3. 坐标变换：y = P^{-1}x

## 应用场景
坐标系转换；线性变换矩阵的简化；图形变换；物理中的参考系变换；量子力学的表象变换；信号处理的基变换。

## 注意事项
- 基变换矩阵P：β = αP
- 坐标变换矩阵是P^{-1}
- 线性变换矩阵的相似变换：B = P^{-1}AP
- 正交基变换的过渡矩阵是正交矩阵
