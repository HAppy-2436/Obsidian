# 初等变换

## 定义
对矩阵进行的三种基本变换：换行（列）、某行（列）乘非零常数、某行（列）的倍数加到另一行（列）。初等变换不改变[[矩阵的秩]]，是求[[逆矩阵]]、解[[线性方程组]]、化简矩阵的基本工具。每次初等行（列）变换相当于左（右）乘一个初等矩阵。

## 核心内容
**三种初等变换**：
1. **互换**：互换第i、j两行（列）
   记作：r_i ⟺ r_j（或 c_i ⟺ c_j）
   
2. **倍乘**：第i行（列）乘以非零常数k
   记作：r_i × k（或 c_i × k）
   
3. **倍加**：第j行的k倍加到第i行
   记作：r_i + k·r_j（或 c_i + k·c_j）

**初等矩阵**：
对单位矩阵E进行一次初等变换得到的矩阵
- 左乘初等矩阵 = 行变换
- 右乘初等矩阵 = 列变换
- 初等矩阵都可逆，其逆也是初等矩阵

**等价标准形**：
任意m×n矩阵A经初等变换可化为：
[E_r  0]
[0    0]
其中r = r(A)

**重要性质**：
1. 初等变换不改变矩阵的秩
2. 矩阵等价 ⟺ 可经初等变换相互转化 ⟺ 秩相同且同型
3. A可逆 ⟺ A可表示为有限个初等矩阵的乘积
4. 行变换不改变列向量的线性关系

**应用方法**：
1. **求逆矩阵**：
   (A | E) →行变换→ (E | A^{-1})
   
2. **求秩**：
   化为阶梯形，数非零行数
   
3. **解线性方程组**：
   对增广矩阵 (A|b) 进行行变换
   
4. **求向量组的极大线性无关组**：
   将向量作为列，行变换化阶梯形

## 应用场景
求逆矩阵；解线性方程组（高斯消元法）；求矩阵的秩；化二次型为标准形；求向量组的极大线性无关组。

## 注意事项
- 只有行变换时才能解方程组
- 求逆只能用行变换（或只用列变换）
- 互换行要记得在行列式中变号
- 倍加变换的常数可以是任意数
