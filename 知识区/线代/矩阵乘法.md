# 矩阵乘法

## 定义
设A为m×n矩阵，B为n×p矩阵，则乘积AB为m×p矩阵，其第i行第j列元素为：(AB)_{ij} = Σ_{k=1}^n a_{ik}·b_{kj}。矩阵乘法对应线性变换的复合，AB表示先B变换再A变换。注意：矩阵乘法一般不满足交换律，即AB ≠ BA。

## 核心内容
**计算规则**：
- 行×列：A的第i行与B的第j列对应元素相乘再求和
- 前提条件：A的列数 = B的行数
- 结果维度：(m×n)·(n×p) = (m×p)

**运算律**：
1. 结合律：(AB)C = A(BC)
2. 分配律：A(B+C) = AB + AC，(A+B)C = AC + BC
3. 数乘结合：k(AB) = (kA)B = A(kB)
4. **不满足交换律**：AB ≠ BA（一般）
5. **AB=0 不能推出 A=0 或 B=0**
6. (A^T)(B^T) = (BA)^T（注意顺序）

**特殊情况**：
- 单位矩阵：EA = AE = A
- 零矩阵：0A = A0 = 0
- 对角矩阵相乘：对应对角元素相乘
- AB = BA（可交换）条件：
  - A、B都是对角矩阵
  - 一个是单位矩阵的倍数
  - A、B可同时对角化

**分块矩阵乘法**：
若分块适当，可按"块"进行乘法：
[A₁ A₂][B₁] = A₁B₁ + A₂B₂
[A₃ A₄][B₂]

**矩阵幂**：
A^n = A·A·...·A（n个）
- 若A可对角化：A = PDP^{-1}，则 A^n = PD^nP^{-1}
- D^n对角元素为特征值的n次方

## 应用场景
线性变换的复合；求解[[线性方程组]]（通过矩阵表示）；图论中邻接矩阵的幂表示路径数；马尔可夫链的状态转移。

## 注意事项
- 不满足交换律，AB ≠ BA
- AB = 0 不能推出 A=0 或 B=0
- 左乘与右乘的意义不同
- 行列式：|AB| = |A|·|B|
