# 线性方程组

## 定义
形如 Ax = b 的方程组，其中A是m×n系数矩阵，x是n×1未知向量，b是m×1常数向量。齐次方程组指b=0；非齐次指b≠0。解的存在性和唯一性由[[矩阵的秩]]决定，是线性代数的核心应用之一。

## 核心内容
**解的判定**（A为m×n矩阵）：
设r(A)=r（系数矩阵的秩），r(Ā)=r̄（增广矩阵的秩）

**非齐次 Ax = b**：
1. 无解：r̄ > r
2. 唯一解：r̄ = r = n
3. 无穷多解：r̄ = r < n（有 n-r 个自由变量）

**齐次 Ax = 0**：
1. 只有零解：r = n
2. 有非零解：r < n（解空间维数 = n-r）

**求解方法**：
1. **高斯消元法**：
   - 化增广矩阵为阶梯形
   - 回代求解
   
2. **[[克拉默法则]]**：
   当A为n阶方阵且|A|≠0时：
   x_i = |A_i| / |A|
   （A_i是将A第i列换为b得到的矩阵）

3. **矩阵求逆**：
   当A可逆时：x = A^{-1}b

**解的结构**：
- **齐次方程组**：
  - 解空间（核空间）是向量空间
  - 基础解系：解空间的一组基
  - 通解 = 基础解系的线性组合
  
- **非齐次方程组**：
  - 通解 = 特解 + 齐次方程组通解
  - x = x₀ + k₁ξ₁ + k₂ξ₂ + ... + k_{n-r}ξ_{n-r}

## 应用场景
求解实际问题中的未知量；电路分析（基尔霍夫定律）；经济学投入产出模型；计算机图形学变换；数值分析。

## 注意事项
- r(Ā) ≥ r(A)总是成立
- 齐次方程组一定有解（至少有零解）
- 求基础解系要确保线性无关且个数=n-r
- 非齐次解的结构要记住"特解+齐次通解"
