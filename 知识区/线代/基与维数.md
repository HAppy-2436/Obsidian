# 基与维数

## 定义
[[向量空间]]V的基是V的一个线性无关向量组，且V中任意向量可由其线性表示。基中向量的个数称为V的维数，记作dim(V)。基是向量空间的"坐标系"，任意向量在基下有唯一的坐标表示。维数反映了空间的"自由度"，是向量空间最重要的数值特征。

## 核心内容
**基的定义**（等价条件）：
向量组 α₁, α₂, ..., α_n 是V的基 ⟺
1. α₁, ..., α_n 线性无关
2. V中任意向量可由它们线性表示
⟺
1. α₁, ..., α_n 是V的极大线性无关组
2. V中任意向量可唯一表示为它们的线性组合

**维数**：
dim(V) = V的基所含向量个数
- 基的个数不唯一，但维数唯一
- dim(R^n) = n
- 零空间：dim({0}) = 0

**坐标**：
设 α₁, ..., α_n 是V的基
若 v = k₁α₁ + ... + k_nα_n
则 (k₁, ..., k_n)^T 称为v在该基下的坐标

**重要定理**：
1. n维空间的任意n+1个向量必线性相关
2. n维空间的任意n个线性无关向量构成一组基
3. 任意两组基所含向量个数相同
4. 若dim(V) = n，则V ≅ R^n（同构）

**子空间的维数**：
- dim(V₁ + V₂) = dim(V₁) + dim(V₂) - dim(V₁∩V₂)
- 若V₁ ⊕ V₂（直和），则dim(V₁⊕V₂) = dim(V₁) + dim(V₂)

**矩阵相关**：
- 行空间维数 = 列空间维数 = r(A)
- 零空间维数 = n - r(A)（n为列数）
- r(A) + dim(Ker(A)) = n

**标准基**：
R^n的标准基：e₁=(1,0,...,0)^T, e₂=(0,1,0,...,0)^T, ..., e_n

## 应用场景
向量空间理论；坐标变换；线性方程组解空间结构；数据降维（PCA）；函数空间的表示（傅里叶基、小波基）。

## 注意事项
- 基不唯一，但维数唯一
- n维空间恰有n个基向量
- 坐标依赖于基的选择
- 子空间的维数≤整个空间的维数
