# 正定矩阵

## 定义
实对称矩阵A称为正定矩阵，若对任意非零向量x，都有 x^TAx > 0。正定矩阵是[[二次型]]正定的充要条件，所有[[特征值与特征向量]]均为正。正定性在优化理论、统计学、控制理论中是极值点判定和稳定性分析的关键工具，正定矩阵有很多优良性质。

## 核心内容
**定义**（等价条件）：
实对称矩阵A为正定矩阵 ⟺
1. ∀ x≠0，x^TAx > 0
2. A的所有特征值 > 0
3. A的所有顺序主子式 > 0
4. 存在可逆矩阵C，使A = C^TC
5. A合同于单位矩阵E
6. A的所有主子式 > 0

**判定方法**：
1. **特征值法**：
   求A的所有特征值，若全 > 0，则正定

2. **顺序主子式法**（Sylvester判据）：
   计算A的k阶顺序主子式 D_k (k=1,...,n)
   若D₁>0, D₂>0, ..., D_n>0，则正定

3. **定义法**：
   验证 x^TAx > 0 对所有 x≠0 成立

**性质**：
1. 正定矩阵可逆，且|A| > 0
2. 正定矩阵的逆也正定
3. 两个正定矩阵的和仍正定
4. 正定矩阵的乘积一般不正定（除非可交换）
5. 若A正定，则kA正定（k>0）
6. A正定 ⟹ A^{-1}, A^T, A², A^n 都正定

**半正定、负定、半负定**：
- **半正定**：x^TAx ≥ 0，特征值 ≥ 0
- **负定**：x^TAx < 0，特征值 < 0
- **半负定**：x^TAx ≤ 0，特征值 ≤ 0
- **不定**：既有正特征值又有负特征值

**Cholesky分解**：
正定矩阵A可唯一分解为：
A = LL^T
其中L是下三角矩阵，对角元素为正

## 应用场景
多元函数极值判定（Hessian矩阵）；最优化理论（凸优化）；统计学协方差矩阵；控制理论稳定性；机器学习的核方法；数值分析的收敛性。

## 注意事项
- 正定矩阵必须是对称矩阵
- 顺序主子式全>0 ⟺ 正定
- 主子式全>0 ⟹ 正定（充分但不必要）
- 正定⟹可逆，但可逆不一定正定
