# 正交矩阵

## 定义
n阶实矩阵Q满足 Q^TQ = QQ^T = E（单位矩阵），则称Q为正交矩阵。等价定义：Q的列向量（或行向量）组成一个标准正交向量组。正交矩阵表示保持长度和角度的线性变换（旋转、反射），满足 Q^{-1} = Q^T，是线性代数中最"优美"的矩阵之一。

## 核心内容
**等价条件**：
1. Q^TQ = E
2. QQ^T = E
3. Q^{-1} = Q^T
4. Q的列向量是标准正交向量组
5. Q的行向量是标准正交向量组

**性质**：
1. |Q| = ±1
2. Q的特征值的模为1（复数情况）
   实正交矩阵的实特征值为±1
3. Q的逆也是正交矩阵：(Q^{-1})^T = Q
4. 正交矩阵的乘积仍是正交矩阵
5. Q^T也是正交矩阵

**几何意义**：
- 保持向量长度：||Qx|| = ||x||
- 保持[[内积]]：(Qx)^T(Qy) = x^Ty
- 保持角度不变
- 表示旋转、反射等刚体变换

**施密特正交化**：
将线性无关向量组 α₁, α₂, ..., α_n 变为正交向量组的方法：
1. β₁ = α₁
2. β₂ = α₂ - (α₂·β₁)/(β₁·β₁) · β₁
3. β₃ = α₃ - (α₃·β₁)/(β₁·β₁)β₁ - (α₃·β₂)/(β₂·β₂)β₂
...
再单位化：γ_i = β_i / ||β_i||

**正交变换**：
y = Qx（Q正交）
- 保持向量长度和内积
- 用于[[对角化]]对称矩阵
- 用于化[[二次型]]为标准形

**常见正交矩阵**：
- 单位矩阵E
- 旋转矩阵：[cosθ -sinθ]
            [sinθ  cosθ]
- 反射矩阵（Householder矩阵）

## 应用场景
实对称矩阵的正交对角化；二次型的标准形；QR分解；最小二乘法；计算机图形学的旋转变换；信号处理的正交变换。

## 注意事项
- Q^{-1} = Q^T（定义）
- |Q| = ±1
- 列向量（行向量）两两正交且模为1
- 正交变换保持向量长度和内积
