# 二次型

## 定义
n元二次齐次多项式f(x₁,x₂,...,x_n) = Σ Σ a_{ij}x_ix_j，可表示为矩阵形式 f(x) = x^TAx，其中A是对称矩阵。二次型通过[[对角化]]可化为标准形 λ₁y₁² + λ₂y₂² + ... + λ_ny_n²，λ_i是A的[[特征值与特征向量]]。正定性判断是二次型的重要应用。

## 核心内容
**矩阵表示**：
f(x) = x^TAx = [x₁ x₂ ... x_n] A [x₁ x₂ ... x_n]^T
其中A是对称矩阵（a_{ij} = a_{ji}）

**标准形**：
通过可逆线性变换 x = Cy，化为
f = λ₁y₁² + λ₂y₂² + ... + λ_ny_n²

**规范形**：
进一步化为
f = y₁² + ... + y_p² - y_{p+1}² - ... - y_r²
其中r = r(A)（惯性指数）

**化简方法**：
1. **配方法**：
   逐步配成完全平方和

2. **正交变换法**（主轴定理）：
   - 求A的特征值λ₁, λ₂, ..., λ_n
   - 求对应的单位正交特征向量
   - 构造正交矩阵Q
   - x = Qy，则 f = λ₁y₁² + ... + λ_ny_n²

3. **合同变换法**：
   用初等变换同时对A进行行列变换

**正定性判断**：
二次型f（或矩阵A）为：
- **正定**：所有特征值 > 0
  ⟺ 所有顺序主子式 > 0
  ⟺ 标准形系数全 > 0
  
- **负定**：所有特征值 < 0
  
- **半正定**：所有特征值 ≥ 0
  
- **不定**：既有正特征值又有负特征值

**惯性定理**：
二次型的正惯性指数p（正系数个数）、负惯性指数q（负系数个数）由矩阵唯一确定，与变换无关
r = p + q = r(A)

## 应用场景
多元函数极值判定（Hessian矩阵正定性）；优化理论；统计学的协方差矩阵；物理中的能量函数；控制理论的稳定性分析。

## 注意事项
- 二次型的矩阵必须取为对称矩阵
- 正定⟺所有特征值>0⟺所有顺序主子式>0
- 正交变换不改变二次型的正负惯性指数
- 标准形不唯一，但规范形唯一
