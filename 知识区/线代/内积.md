# 内积

## 定义
向量空间中定义的二元运算，记作<x,y>或x·y，在R^n中定义为 <x,y> = x^Ty = Σx_iy_i。内积是向量之间"相似度"的度量，由内积可定义向量的长度||x|| = √<x,x>和夹角cosθ = <x,y>/(||x||·||y||)。内积空间是线性代数和泛函分析的基础概念。

## 核心内容
**定义**（需满足的性质）：
1. **正定性**：<x,x> ≥ 0，且<x,x>=0 ⟺ x=0
2. **对称性**：<x,y> = <y,x>
3. **线性性**：
   <kx+ly, z> = k<x,z> + l<y,z>

**标准内积**（R^n）：
<x,y> = x^Ty = x₁y₁ + x₂y₂ + ... + x_ny_n

**向量长度（范数）**：
||x|| = √<x,x> = √(x₁² + x₂² + ... + x_n²)

**夹角**：
cosθ = <x,y> / (||x||·||y||)

**正交**：
<x,y> = 0 ⟺ x ⊥ y（x与y正交）

**重要不等式**：
1. **Cauchy-Schwarz不等式**：
   |<x,y>| ≤ ||x||·||y||
   等号成立 ⟺ x与y线性相关

2. **三角不等式**：
   ||x + y|| ≤ ||x|| + ||y||

3. **勾股定理**：
   若x⊥y，则||x+y||² = ||x||² + ||y||²

**正交向量组**：
向量组 α₁, ..., α_n 两两正交：<α_i, α_j> = 0 (i≠j)
- 正交向量组必线性无关
- 标准正交：既正交又||α_i|| = 1

**施密特正交化**：
将线性无关组化为正交组的方法

**投影**：
向量y在x上的投影：
proj_x(y) = (<y,x> / <x,x>)·x

## 应用场景
定义向量长度和角度；[[正交矩阵]]的定义；最小二乘法；傅里叶分析；量子力学的态向量；信号处理的相关性分析。

## 注意事项
- 内积结果是标量，不是向量
- <x,y>=0 ⟺ x⊥y
- Cauchy-Schwarz不等式非常重要
- 正交向量组必线性无关

## 关联知识
与 [[正交矩阵]]、[[施密特正交化]]、[[向量空间]]、[[范数]] 相关。
