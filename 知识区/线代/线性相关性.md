# 线性相关性

## 定义
向量组 α₁, α₂, ..., α_s 线性相关是指存在不全为零的数 k₁, k₂, ..., k_s 使得 k₁α₁ + k₂α₂ + ... + k_sα_s = 0。否则称线性无关。线性相关意味着至少有一个向量可由其余向量线性表示，是[[向量空间]]和[[矩阵的秩]]理论的核心概念。

## 核心内容
**判定方法**：
1. **定义法**：
   齐次方程组 x₁α₁ + ... + x_sα_s = 0
   - 只有零解 → 线性无关
   - 有非零解 → 线性相关

2. **秩判定法**：
   设A = [α₁, α₂, ..., α_s]
   - r(A) = s → 线性无关
   - r(A) < s → 线性相关

3. **行列式法**（n个n维向量）：
   |α₁, α₂, ..., α_n| ≠ 0 → 线性无关
   |α₁, α₂, ..., α_n| = 0 → 线性相关

**重要定理**：
1. 部分相关 → 整体相关
   整体无关 → 部分无关

2. 向量个数 > 维数 → 必线性相关

3. 线性无关组延伸后线性相关 → 延伸向量可由原向量组线性表示

4. α₁, ..., α_s 线性无关，但 α₁, ..., α_s, β 线性相关
   → β 可由 α₁, ..., α_s 唯一线性表示

5. 两个等价的线性无关向量组所含向量个数相同

**极大线性无关组**：
向量组的一个部分组，满足：
1. 该部分组线性无关
2. 加入任一其他向量都线性相关
- 个数 = [[矩阵的秩]]
- 不唯一，但个数相同

**线性表示**：
β 可由 α₁, ..., α_s 线性表示
⟺ r(α₁, ..., α_s) = r(α₁, ..., α_s, β)

## 应用场景
判断向量组的相关性；求向量空间的基；求解线性方程组；判断矩阵的秩；主成分分析中选择主成分。

## 注意事项
- 单个非零向量线性无关
- 含零向量的向量组必线性相关
- 向量个数 > 维数必相关
- 线性相关不等于成比例（多于2个向量时）
