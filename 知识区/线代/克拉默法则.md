# 克拉默法则

## 定义
对于n元[[线性方程组]] Ax = b，当系数矩阵A的[[行列式]] D = |A| ≠ 0时，方程组有唯一解：x_i = D_i / D，其中D_i是将D的第i列替换为常数项b后得到的行列式。克拉默法则提供了显式的解公式，但计算复杂度高，主要用于理论证明。

## 核心内容
**公式**：
对方程组：
a₁₁x₁ + a₁₂x₂ + ... + a₁ₙx_n = b₁
a₂₁x₁ + a₂₂x₂ + ... + a₂ₙx_n = b₂
⋮
a_n₁x₁ + a_n₂x₂ + ... + a_nₙx_n = b_n

当 D = |A| ≠ 0 时，唯一解为：
x_i = D_i / D (i = 1, 2, ..., n)

其中D_i是将系数行列式D的第i列替换为常数项列(b₁, b₂, ..., b_n)^T后得到的行列式。

**适用条件**：
1. 方程个数 = 未知数个数（n×n系统）
2. 系数行列式 |A| ≠ 0
3. 有唯一解

**齐次方程组**（b=0）：
若 |A| ≠ 0，则只有零解 x = 0
若 |A| = 0，则有非零解

**计算步骤**：
1. 计算系数行列式 D = |A|
2. 判断 D ≠ 0
3. 对每个未知数x_i，将D的第i列换为b
4. 计算D_i，求 x_i = D_i/D

**优缺点**：
- 优点：理论清晰，公式明确
- 缺点：计算量大（n!级别），实际常用高斯消元

## 应用场景
理论证明（如逆矩阵存在性）；小规模方程组（2×2、3×3）；符号计算系统。实际数值计算更常用高斯消元或LU分解。

## 注意事项
- 只适用于方阵（方程数=未知数数）
- |A| = 0 时克拉默法则不适用
- 计算复杂度O(n! · n)，不适合大规模计算
- 主要理论价值大于实际价值
