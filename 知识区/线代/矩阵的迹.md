# 矩阵的迹

## 定义
n阶方阵A的迹tr(A)定义为主对角线元素之和：tr(A) = a₁₁ + a₂₂ + ... + a_nn。迹是矩阵的重要不变量，等于所有[[特征值与特征向量]]之和（含重数）。迹在[[相似矩阵]]变换下保持不变，满足线性性质和循环性质，在矩阵分析、统计学、量子力学中应用广泛。

## 核心内容
**定义**：
tr(A) = Σ_{i=1}^n a_{ii}

**基本性质**：
1. **线性性**：
   tr(A + B) = tr(A) + tr(B)
   tr(kA) = k·tr(A)

2. **转置不变**：
   tr(A^T) = tr(A)

3. **循环性质**（重要）：
   tr(ABC) = tr(BCA) = tr(CAB)
   一般地：tr(AB) = tr(BA)
   
4. **相似不变**：
   若B = P^{-1}AP，则tr(B) = tr(A)

5. **与特征值的关系**：
   tr(A) = λ₁ + λ₂ + ... + λ_n
   （特征值之和，含重数）

**其他公式**：
1. tr(A^n) = λ₁^n + λ₂^n + ... + λ_n^n
2. tr(A^TA) = Σ Σ a_{ij}²（所有元素平方和）
3. 对实对称矩阵：tr(A²) = Σ λ_i²

**与行列式的关系**：
- tr(A) = 特征值之和
- |A| = 特征值之积
- 这两个量都是相似不变量

**判定定理**：
tr(A) = 0 不能推出 A = 0
例如：[1  0]  tr=0 但 A≠0
     [0 -1]

## 应用场景
判断矩阵相似性（必要条件）；统计学中的总方差；量子力学中的期望值；机器学习中的正则化项；矩阵微分计算。

## 注意事项
- tr(AB) = tr(BA)（循环性）
- tr(AB) ≠ tr(A)·tr(B)（一般情况）
- tr(A) = 0 不能推出 A = 0
- 迹是相似不变量
