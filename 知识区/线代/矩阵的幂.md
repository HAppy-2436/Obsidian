# 矩阵的幂

## 定义
n阶方阵A的n次幂 A^n = A·A·...·A（n个A相乘）。直接计算复杂度高，通过[[对角化]] A = PΛP^{-1} 可得 A^n = PΛ^nP^{-1}，其中Λ^n只需计算对角元素的n次幂，极大简化计算。矩阵的幂在求解递推关系、马尔可夫链、微分方程组中有重要应用。

## 核心内容
**对角化法**（主要方法）：
若A可[[对角化]]：A = PΛP^{-1}
则 A^n = PΛ^nP^{-1}

其中：
- Λ = diag(λ₁, λ₂, ..., λ_n)
- Λ^n = diag(λ₁^n, λ₂^n, ..., λ_n^n)
- P的列是[[特征值与特征向量]]

**步骤**：
1. 求A的特征值λ₁, λ₂, ..., λ_n
2. 求对应的特征向量，构成P
3. 计算 P^{-1}
4. A^n = P·diag(λ₁^n,...,λ_n^n)·P^{-1}

**特殊矩阵的幂**：
1. **对角矩阵**：
   diag(a₁,...,a_n)^n = diag(a₁^n,...,a_n^n)

2. **上三角矩阵**：
   结果仍是上三角

3. **正交矩阵** Q：
   Q^n 仍是正交矩阵

4. **幂零矩阵**：
   ∃k使A^k = 0

**性质**：
1. (A^m)^n = A^{mn}
2. A^m·A^n = A^{m+n}
3. (A^{-1})^n = (A^n)^{-1}（A可逆）
4. (kA)^n = k^n·A^n
5. 若AB = BA，则(AB)^n = A^n·B^n

**归纳法**：
若A不可对角化，可尝试：
1. 找规律：计算A², A³, A⁴...
2. 猜测通项公式
3. 数学归纳法证明

**矩阵函数**：
利用幂级数定义：
- e^A = Σ A^n/n! = Pe^ΛP^{-1}
- sin(A), cos(A)等

## 应用场景
递推数列求通项公式；马尔可夫链的n步转移概率；微分方程组 x'=Ax 的解；图论中计算n步路径数；Leslie人口模型。

## 注意事项
- A必须可对角化才能用PΛ^nP^{-1}
- 不可对角化可用若尔当标准形或归纳法
- (AB)^n ≠ A^nB^n（一般情况）
- 特征值为0会使某些幂为0
