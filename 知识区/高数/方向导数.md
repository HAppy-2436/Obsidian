# 方向导数

## 定义
函数在某点沿指定方向的变化率。设函数 z=f(x,y) 在点 P(x₀,y₀) 处沿方向 l（方向角α,β）的方向导数为：∂f/∂l = lim(ρ→0)[f(x₀+ρcosα, y₀+ρsinβ)-f(x₀,y₀)]/ρ。它推广了[[偏导数]]的概念，偏导数是沿坐标轴方向的方向导数。

## 核心内容
**计算公式**：
若 f(x,y) 可微，则方向导数为：
∂f/∂l = ∂f/∂x·cosα + ∂f/∂y·sinβ = f_x'cosα + f_y'sinβ

其中 (cosα, sinβ) 是方向 l 的单位方向向量

**三元函数**：
∂f/∂l = f_x'cosα + f_y'cosβ + f_z'cosγ
其中 (cosα, cosβ, cosγ) 是方向余弦

**与梯度的关系**：
∂f/∂l = [[梯度]] · 方向单位向量 = |gradf| · cos<grad f, l>
- 沿[[梯度]]方向，方向导数最大，值为|grad f|
- 垂直于梯度方向，方向导数为0
- 负梯度方向，下降最快

**存在条件**：
- 沿任意方向的方向导数存在 ⇏ 可微
- 可微 ⇒ 任意方向导数存在
- [[偏导数]]是特殊的方向导数

## 应用场景
优化算法（梯度下降、最速下降法）；物理中的场论（温度梯度、电场强度）；图像处理（边缘检测）；地理信息系统（坡度计算）。

## 注意事项
- 方向必须是单位向量
- 存在所有方向导数不能推出可微
- 只有可微时才能用公式计算
- [[梯度]]方向是函数增长最快的方向

## 关联知识
与 [[偏导数]]、[[梯度]]、[[多元函数微分]] 相关。
