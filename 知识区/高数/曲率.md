# 曲率

## 定义
曲线弯曲程度的量化指标，表示曲线偏离切线的快慢程度。曲率 k 定义为切线转角对弧长的变化率：k = |dα/ds|。直观理解：半径越小的圆曲率越大（越弯），直线曲率为0。曲率半径 R = 1/k，表示与曲线在该点密切的圆的半径。

## 核心内容
**曲率公式**：

1. **参数方程**：x=x(t)，y=y(t)
   k = |x'y'' - x''y'| / (x'² + y'²)^(3/2)

2. **直角坐标**：y=f(x)
   k = |y''| / (1 + y'²)^(3/2)
   这是最常用的公式

3. **极坐标**：r=r(θ)
   k = |r² + 2r'² - rr''| / (r² + r'²)^(3/2)

**曲率半径**：
R = 1/k = (1 + y'²)^(3/2) / |y''|
几何意义：曲率圆（密切圆）的半径

**曲率中心**：
曲率圆的圆心坐标：
- x_c = x - y'(1+y'²)/y''
- y_c = y + (1+y'²)/y''
曲率中心的轨迹称为渐屈线

**特殊情况**：
- 圆的曲率 k = 1/R（常数）
- 直线的曲率 k = 0
- 拐点处曲率为0

## 应用场景
道路、铁路设计（缓和曲线）；光学（透镜设计）；机械加工（刀具轨迹）；物理中的曲线运动；计算机图形学中的曲线拟合。

## 注意事项
- 曲率是非负的
- y''=0的点可能是拐点，曲率为0
- 计算曲率半径要先求曲率
- [[参数方程求导]]时注意公式的应用
