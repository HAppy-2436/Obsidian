# 全微分

## 定义
多元函数增量的线性主部。对于 z=f(x,y)，如果 Δz = f(x+Δx, y+Δy) - f(x,y) = AΔx + BΔy + o(ρ)（其中 ρ=√(Δx²+Δy²)），则称 f 在点(x,y)可微，dz = AΔx + BΔy 称为全微分。全微分是一元函数[[微分]]在多元情况的推广，描述函数的线性近似。

## 核心内容
**全微分公式**：
若 z=f(x,y) 可微，则：
dz = ∂z/∂x·dx + ∂z/∂y·dy = f_x'dx + f_y'dy

三元函数：du = ∂u/∂x·dx + ∂u/∂y·dy + ∂u/∂z·dz

**可微条件**：
1. 充分条件：[[偏导数]]连续 ⇒ 可微
2. 必要条件：可微 ⇒ 偏导存在且 dz = f_x'dx + f_y'dy
3. 注意：偏导存在 ⇏ 可微（反例：f(x,y)=xy/√(x²+y²) 在原点）

**与连续、偏导的关系**：
可微 ⇒ 连续 ⇒ 偏导存在
但反向不成立：
- 偏导存在 ⇏ 连续
- 连续 ⇏ 偏导存在
- 偏导存在 ⇏ 可微

**一阶全微分形式不变性**：
无论 u、v 是自变量还是中间变量，
dz = ∂z/∂u·du + ∂z/∂v·dv 的形式保持不变

**几何意义**：
- z=f(x,y) 的图像是曲面
- 全微分 dz 是切平面的竖坐标增量
- Δz 是曲面的实际增量
- dz 是 Δz 的线性近似

## 应用场景
近似计算、误差估计、最优化问题、物理中微小变化量计算、工程测量误差分析。

## 注意事项
- 全微分存在即可微
- 偏导连续是可微的充分条件
- 形式不变性只对一阶成立
- 与一元函数[[微分]]类比理解
