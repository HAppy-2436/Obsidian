# 导数

## 定义
函数在某点的瞬时变化率，记作 f'(x) 或 dy/dx。定义式为：f'(x) = lim(Δx→0) [f(x+Δx)-f(x)]/Δx。导数描述了函数图像在某点处的切线斜率，是研究函数局部性质的重要工具。

## 核心内容
**基本导数公式**：
- (x^n)' = nx^(n-1)
- (sinx)' = cosx，(cosx)' = -sinx
- (e^x)' = e^x，(a^x)' = a^x·lna
- (lnx)' = 1/x，(log_a x)' = 1/(x·lna)

**求导法则**：
1. 四则运算：(u±v)' = u'±v'，(uv)' = u'v+uv'，(u/v)' = (u'v-uv')/v²
2. 复合函数：链式法则 (f∘g)'(x) = f'(g(x))·g'(x)
3. [[隐函数求导]]：对方程两边同时求导
4. [[参数方程求导]]：dy/dx = (dy/dt)/(dx/dt)
5. 反函数求导：(f^(-1))'(x) = 1/f'(y)

**高阶导数**：二阶导数 f''(x) 表示函数变化率的变化率，研究函数凹凸性。

## 应用场景
求函数单调性、极值点、拐点、切线方程、最优化问题、物理中的速度加速度、经济学边际分析。

## 注意事项
- 可导必连续，但连续不一定可导
- 分段函数在分界点需用定义求导
- 复合函数求导要注意链式法则的多层嵌套
- 高阶导数计算要有规律性
