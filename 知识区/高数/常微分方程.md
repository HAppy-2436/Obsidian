# 常微分方程

## 定义
只含一个自变量的[[微分方程]]，如 dy/dx = f(x,y) 或 y'' + py' + qy = f(x)。与偏微分方程相区别，常微分方程中未知函数只依赖于一个自变量。是描述动态系统、物理过程的重要数学工具，解常微分方程就是求满足方程的函数。

## 核心内容
**基本概念**：
- 阶：方程中最高阶导数的阶数
- 通解：含有n个独立任意常数的解（n阶方程）
- 特解：满足初始条件的解
- 奇解：不能从通解中得到的解

**一阶方程类型**：
1. 可分离变量：g(y)dy = f(x)dx
2. 齐次方程：dy/dx = φ(y/x)
3. 一阶线性：dy/dx + P(x)y = Q(x)
   通解：y = e^{-∫Pdx}[∫Qe^{∫Pdx}dx + C]
4. 伯努利方程：dy/dx + P(x)y = Q(x)y^n
5. 全微分方程：Pdx + Qdy = 0，∂P/∂y = ∂Q/∂x

**高阶方程可降阶**：
1. y^(n) = f(x)：n次积分
2. y'' = f(x, y')：令p=y'，p' = f(x, p)
3. y'' = f(y, y')：令p=y'，p·dp/dy = f(y, p)

**二阶线性方程**：
y'' + p(x)y' + q(x)y = f(x)
- 解的结构：y = Y(齐次通解) + y*(特解)
- 常系数齐次：特征方程法
- 常系数非齐次：待定系数法

## 应用场景
物理中的牛顿运动定律、RC电路、振动问题、衰变问题；生物学种群模型；经济学增长模型；化学反应速率。

## 注意事项
- 通解必须含有足够的任意常数
- 初值问题要代入初始条件求常数
- 伯努利方程要先除以y^n化为线性
- 齐次与非齐次的区别很重要

## 关联知识
与 [[微分方程]]、[[导数]]、[[积分]]、[[拉普拉斯变换]] 相关。
