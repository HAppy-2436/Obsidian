# 微分

## 定义
函数增量的线性主部，记作 dy = f'(x)dx。当自变量 x 有微小增量 dx 时，函数增量 Δy = f(x+dx) - f(x) ≈ f'(x)dx = dy。微分是[[导数]]的另一种表现形式，描述了函数的局部线性近似。

## 核心内容
**基本概念**：
- 可微的充要条件：函数可微 ⟺ 函数可导
- 微分的几何意义：切线的纵坐标增量
- dy = f'(x)dx，其中 dx = Δx（自变量微分等于增量）

**微分运算法则**：
- d(u±v) = du ± dv
- d(uv) = vdu + udv
- d(u/v) = (vdu - udv)/v²
- d(f(u)) = f'(u)du（一阶微分形式不变性）

**一阶微分形式不变性**：
无论 u 是自变量还是中间变量，df(u) = f'(u)du 的形式都保持不变，这使得复合函数的微分运算非常方便。

## 应用场景
1. 近似计算：Δy ≈ dy，用于估算函数值变化
2. 误差估计：测量误差传播分析
3. 物理应用：微小变化量的计算
4. 几何应用：曲线的切线方程
5. 为[[微分方程]]提供基础

## 注意事项
- 微分 dy 是 Δy 的近似，仅在 dx 很小时精度高
- 自变量的微分等于增量：dx = Δx
- 高阶微分不具有形式不变性
- 微分与[[导数]]：dy/dx = f'(x)
