# 曲线积分

## 定义
沿曲线的积分，分为两类：第一类（对弧长）和第二类（对坐标）。第一类曲线积分 ∫_L f(x,y)ds 与路径方向无关，第二类曲线积分 ∫_L Pdx+Qdy 与路径方向有关。物理意义包括变力做功、质量、电荷分布等，是[[定积分]]在曲线上的推广。

## 核心内容
**第一类曲线积分**（对弧长）：
∫_L f(x,y)ds
- 与路径方向无关
- 计算：化为参数方程 x=φ(t), y=ψ(t)
  ∫_L f ds = ∫_α^β f[φ(t),ψ(t)]√(φ'²+ψ'²) dt
- 物理意义：曲线形构件的质量、电荷

**第二类曲线积分**（对坐标）：
∫_L Pdx + Qdy
- 与路径方向有关，反向时变号
- 计算：代入参数方程
  ∫_L Pdx+Qdy = ∫_α^β [P(φ,ψ)φ' + Q(φ,ψ)ψ'] dt
- 物理意义：变力做功、环流量

**格林公式**：
∮_L Pdx+Qdy = ∬_D (∂Q/∂x - ∂P/∂y) dxdy
条件：L是D的正向边界，P、Q在D内有连续偏导数
作用：联系曲线积分与[[二重积分]]

**积分与路径无关条件**：
在单连通区域D内，以下等价：
1. ∫_L Pdx+Qdy 与路径无关
2. ∮_L Pdx+Qdy = 0（任意闭曲线）
3. ∂Q/∂x = ∂P/∂y（全微分条件）
4. Pdx+Qdy 是某函数u的全微分

## 应用场景
物理中计算变力做功、流量、环流量；电磁学中的电场强度积分；复变函数中的柯西积分定理。

## 注意事项
- 第一类与方向无关，第二类与方向有关
- 格林公式要求区域单连通
- 补线法处理不封闭或多连通区域
- 全微分条件是路径无关的充要条件
