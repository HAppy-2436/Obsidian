# 泰勒公式

## 定义
将函数 f(x) 在点 x=a 处展开为无穷级数的方法，用多项式逼近任意函数。n阶泰勒公式为：f(x) = f(a) + f'(a)(x-a) + f''(a)(x-a)²/2! + ... + f^(n)(a)(x-a)^n/n! + R_n(x)，其中 R_n(x) 是余项。

## 核心内容
**麦克劳林公式**（a=0的特殊情况）：
- e^x = 1 + x + x²/2! + x³/3! + ...
- sinx = x - x³/3! + x⁵/5! - ...
- cosx = 1 - x²/2! + x⁴/4! - ...
- ln(1+x) = x - x²/2 + x³/3 - ...
- (1+x)^α = 1 + αx + α(α-1)x²/2! + ...

**余项形式**：
- 拉格朗日余项：R_n = f^(n+1)(ξ)(x-a)^(n+1)/(n+1)!
- 佩亚诺余项：R_n = o((x-a)^n)

## 应用场景
1. 求[[极限]]：用低阶项替代复杂函数
2. 近似计算：估算函数值（如计算器原理）
3. 证明不等式：利用余项的符号
4. 求高阶[[导数]]
5. 研究[[级数]]收敛性

## 注意事项
- 展开点的选择影响收敛速度
- 展开阶数要足够才能保证精度
- [[等价无穷小]]是泰勒展开的一阶近似
- 不同函数展开式的记忆和灵活运用很重要

## 关联知识
与 [[等价无穷小]]、[[导数]]、[[幂级数]]、[[极限]] 相关。
