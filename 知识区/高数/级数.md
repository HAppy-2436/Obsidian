# 级数

## 定义
无穷多项之和 Σ(n=1 to ∞)a_n = a_1 + a_2 + a_3 + ...，称为无穷级数。部分和 S_n = Σ(k=1 to n)a_k，若 lim(n→∞)S_n = S 存在，则称级数收敛于 S，否则称为发散。级数是研究函数展开、近似计算的重要工具。

## 核心内容
**收敛判别法**：
1. 必要条件：lim(n→∞)a_n = 0（不充分）
2. 正项级数：
   - 比值判别法：lim a_(n+1)/a_n = ρ，ρ<1收敛，ρ>1发散
   - 根值判别法：lim ⁿ√a_n = ρ，ρ<1收敛，ρ>1发散
   - 积分判别法：与[[反常积分]]比较
   - 比较判别法：与已知级数比较
3. 交错级数：莱布尼茨判别法
4. 任意项级数：绝对收敛与条件收敛

**常见级数**：
- 几何级数：Σaq^n，|q|<1时收敛于a/(1-q)
- p-级数：Σ1/n^p，p>1收敛，p≤1发散
- 调和级数：Σ1/n发散

**重要性质**：
- 收敛级数可逐项求和
- 绝对收敛级数可任意调换顺序
- 收敛级数乘常数、有限项加减仍收敛

## 应用场景
函数的[[幂级数]]展开、[[傅里叶级数]]分析、数值计算、概率论期望计算、物理中的级数解。

## 注意事项
- 必要条件不是充分条件（调和级数反例）
- 条件收敛的级数改变求和顺序可能改变和
- 判别法的选择要看级数特点
- 与[[幂级数]]的区别：级数是常数项，幂级数是函数项

## 关联知识
