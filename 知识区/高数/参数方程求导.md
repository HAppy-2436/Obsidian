# 参数方程求导

## 定义
曲线用参数方程表示为 x=φ(t)，y=ψ(t) 时，求 dy/dx 的方法。通过参数 t 建立x、y之间的联系，利用链式法则：dy/dx = (dy/dt)/(dx/dt)。参数方程在描述运动轨迹、极坐标曲线等问题时非常方便。

## 核心内容
**一阶导数**：
x=φ(t)，y=ψ(t)
dy/dx = (dy/dt)/(dx/dt) = ψ'(t)/φ'(t)（φ'(t)≠0）

**二阶导数**：
d²y/dx² = d/dx(dy/dx) = [d(dy/dx)/dt]/(dx/dt)
= [d(ψ'/φ')/dt]/φ'(t)
= (ψ''φ' - ψ'φ'')/(φ')³

**极坐标方程求导**：
r=r(θ)，转化为参数方程：
x=r(θ)cosθ，y=r(θ)sinθ
dy/dx = (dy/dθ)/(dx/dθ)

**[[曲率]]公式**：
k = |x'y'' - x''y'| / (x'² + y'²)^(3/2)
参数方程形式特别适合曲率计算

**曲线的切线与法线**：
切线斜率：k = dy/dx = ψ'/φ'
切线方程：(Y-y₀)/(ψ'(t₀)) = (X-x₀)/(φ'(t₀))
法线斜率：-φ'/ψ'

## 应用场景
摆线、心形线、渐开线等特殊曲线；物理中的质点运动轨迹；工程中的机械轨迹设计；计算机图形学中的曲线绘制。

## 注意事项
- 分母 dx/dt 不能为零
- 二阶导数要注意分母是 dx/dt 而非 dt
- 极坐标要先化为参数方程
- [[曲率]]计算用参数方程更方便
