# 定积分

## 定义
函数 f(x) 在区间 [a,b] 上的累积量，记作 ∫[a,b]f(x)dx。几何意义是曲线 y=f(x) 与 x 轴、直线 x=a、x=b 围成的面积代数和（x轴上方为正，下方为负）。物理意义是变力做功、变速运动的位移等。

## 核心内容
**计算方法**：
1. 牛顿-莱布尼茨公式：∫[a,b]f(x)dx = F(b) - F(a)，其中 F'(x) = f(x)
2. 换元积分法：注意换限
3. 分部积分法：∫[a,b]udv = [uv][a,b] - ∫[a,b]vdu

**定积分性质**：
- 线性性：∫[a,b][αf(x)+βg(x)]dx = α∫f(x)dx + β∫g(x)dx
- 区间可加性：∫[a,c] = ∫[a,b] + ∫[b,c]
- 比较性质：f≤g 则 ∫f≤∫g
- 积分中值定理：∫[a,b]f(x)dx = f(ξ)(b-a)

**特殊技巧**：
- 奇偶函数在对称区间的积分
- 周期函数的积分性质
- 华里士公式（三角函数幂次积分）
- 区间再现公式：∫[a,b]f(x)dx = ∫[a,b]f(a+b-x)dx

## 应用场景
计算平面图形面积、旋转体体积、曲线弧长、物理中的功、质心、转动惯量、概率论中的期望方差。

## 注意事项
- 换元积分要记得换限
- 被积函数要在区间上连续或只有有限个第一类间断点
- 定积分与[[不定积分]]的区别：定积分是数值，不定积分是函数族
- [[反常积分]]需要用极限定义

## 关联知识
与 [[不定积分]]、[[反常积分]]、[[导数]]、[[二重积分]] 相关。
