# 不定积分

## 定义
求原函数的过程，记作 ∫f(x)dx = F(x) + C，其中 F'(x) = f(x)，C 为任意常数。不定积分是[[导数]]的逆运算，表示所有原函数构成的函数族。几何意义是求曲线族，这些曲线的切线斜率处处相同。

## 核心内容
**基本积分公式**：
- ∫x^n dx = x^(n+1)/(n+1) + C (n≠-1)
- ∫1/x dx = ln|x| + C
- ∫e^x dx = e^x + C
- ∫a^x dx = a^x/lna + C
- ∫sinx dx = -cosx + C
- ∫cosx dx = sinx + C
- ∫1/(1+x²) dx = arctanx + C
- ∫1/√(1-x²) dx = arcsinx + C

**求解方法**：
1. 直接积分法：利用基本公式
2. 换元积分法：
   - 第一类换元（凑微分）：∫f[φ(x)]φ'(x)dx = ∫f(u)du
   - 第二类换元：令 x=φ(t)，dx=φ'(t)dt
3. 分部积分法：∫udv = uv - ∫vdu
   - 口诀："反对幂指三"（反三角、对数、幂函数、指数、三角）
4. 有理函数积分：部分分式分解

## 应用场景
求[[定积分]]的基础；求曲线围成的面积；物理中求位移、功等；微分方程求解。

## 注意事项
- 别忘记常数C
- 换元后要换回原变量
- 分部积分的u、v选择很关键
- 三角函数积分可用万能替换 t=tan(x/2)

## 关联知识
与 [[定积分]]、[[导数]]、[[微分方程]] 相关。
