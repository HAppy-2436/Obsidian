# 微分方程

## 定义
含有未知函数及其[[导数]]的方程，如 dy/dx = f(x,y)。微分方程描述了量之间的变化关系，是数学建模的重要工具。解微分方程就是求满足方程的函数。根据未知函数个数分为常微分方程和偏微分方程，根据最高阶导数分为一阶、二阶等。

## 核心内容
**一阶微分方程**：
1. 可分离变量：g(y)dy = f(x)dx，两边积分
2. 齐次方程：dy/dx = φ(y/x)，令u=y/x转化
3. 一阶线性方程：dy/dx + P(x)y = Q(x)
   - 通解公式：y = e^(-∫P dx)[∫Q e^(∫P dx)dx + C]
   - 齐次（Q=0）：y = Ce^(-∫P dx)
   - 非齐次：常数变易法

**可降阶的高阶方程**：
1. y''=f(x)：两次积分
2. y''=f(x,y')：令p=y'，降为一阶
3. y''=f(y,y')：令p=y'，dp/dx=p·dp/dy

**二阶线性微分方程**：
y'' + P(x)y' + Q(x)y = f(x)
- 解的结构：通解 = 齐次通解 + 特解
- 常系数：特征方程法
- 非齐次：待定系数法、常数变易法

## 应用场景
物理中的运动方程、衰变问题、振动问题；经济学中的增长模型；生物学中的种群模型；电路分析；工程控制系统。

## 注意事项
- 通解含任意常数，特解是满足初始条件的解
- 一阶线性方程记住通解公式
- 二阶线性方程要先求齐次通解
- 初值问题要利用初始条件确定常数
