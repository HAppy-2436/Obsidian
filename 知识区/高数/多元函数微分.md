# 多元函数微分

## 定义
对多个自变量的函数进行微分运算。二元函数 z=f(x,y) 的[[全微分]]为 dz = ∂z/∂x·dx + ∂z/∂y·dy，推广了一元函数的[[微分]]概念。多元微分学研究多元函数的局部性质、极值、最优化等问题。

## 核心内容
**偏导数**：
- ∂f/∂x：对 x 求导，y 视为常数
- ∂f/∂y：对 y 求导，x 视为常数
- 几何意义：曲面沿坐标轴方向的切线斜率
- 混合偏导数：在连续条件下，∂²f/∂x∂y = ∂²f/∂y∂x

**全微分**：
- 可微的充要条件：偏导数连续
- dz = f_x'dx + f_y'dy
- 可微一定偏导存在，反之不成立
- 几何意义：切平面的竖坐标增量

**复合函数求导**：
链式法则：z=f(u,v)，u=φ(x,y)，v=ψ(x,y)
∂z/∂x = ∂z/∂u·∂u/∂x + ∂z/∂v·∂v/∂x

**隐函数求导**：
F(x,y,z)=0 确定 z=z(x,y)
∂z/∂x = -F_x'/F_z'，∂z/∂y = -F_y'/F_z'

**极值与最值**：
- 必要条件：f_x'=0，f_y'=0（驻点）
- 充分条件：用二阶偏导数判别式 AC-B²
- 条件极值：拉格朗日乘数法

## 应用场景
求曲面的切平面和法线、优化问题（最小二乘法、极值问题）、物理中的场论、经济学边际分析、机器学习梯度下降。

## 注意事项
- 偏导存在不能推出可微
- 求偏导时其他变量视为常数
- 复合函数求导要画出结构图
- 条件极值用拉格朗日乘数法更方便
