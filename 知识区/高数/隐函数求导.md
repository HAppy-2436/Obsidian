# 隐函数求导

## 定义
由方程 F(x,y)=0 确定的函数 y=f(x)，无法显式表示为 y=f(x) 形式时的求导方法。隐函数求导避免了解出显式表达式的麻烦，直接对方程两边求导即可。扩展到多元情况，F(x,y,z)=0 可确定 z=z(x,y)。

## 核心内容
**一元隐函数求导**：
F(x,y)=0 确定 y=y(x)
方法：对方程两边对x求导，利用链式法则
F_x' + F_y'·dy/dx = 0
得：dy/dx = -F_x'/F_y'（F_y'≠0）

**二元隐函数求导**：
F(x,y,z)=0 确定 z=z(x,y)
[[偏导数]]：
- ∂z/∂x = -F_x'/F_z'
- ∂z/∂y = -F_y'/F_z'（F_z'≠0）

**隐函数存在定理**：
若 F(x₀,y₀)=0，F 在(x₀,y₀)某邻域内连续，F_y'连续且 F_y'(x₀,y₀)≠0，
则存在唯一的连续函数 y=f(x) 满足 F(x,f(x))=0

**隐函数组**：
{F(x,y,u,v)=0
{G(x,y,u,v)=0
确定 u=u(x,y)，v=v(x,y)
用雅可比行列式求解

**求导技巧**：
1. 方程两边同时对自变量求导
2. 把因变量看作自变量的函数
3. 利用链式法则展开
4. 解出所需的导数

## 应用场景
几何中的切线、法线问题；参数方程求导；经济学中的隐含关系；物理中的约束条件；最优化问题的约束条件处理。

## 注意事项
- 要保证 F_y' 或 F_z' 不为零
- 对方程两边求导时不要遗漏链式法则
- 可以验证结果（对显式函数同时用两种方法）
- 与[[参数方程求导]]区别开
