# 卡诺图

## 定义
用于化简[[逻辑代数]]表达式的图形方法，由卡诺(Karnaugh)提出。将真值表映射到二维图形，相邻格只有一个变量不同，通过圈取相邻1（或0）实现最简与或式（或或与式）。卡诺图法直观、快速，适用于4-5变量的逻辑函数化简，是数字电路设计的重要工具。

## 核心内容
**基本原理**：
利用[[逻辑代数]]吸收律：AB+AB̄=A
相邻格（相差1个变量）可合并消去该变量

**卡诺图结构**：
- 2变量：2×2格
- 3变量：2×4格
- 4变量：4×4格
- 格子排列遵循格雷码顺序（相邻格只变1位）

**化简步骤**：
1. 填卡诺图：根据真值表填1（或0）
2. **圈最大圈**：圈2^n个相邻1
   - 圈要尽可能大（包含2^n个格）
   - 圈要尽可能少
   - 每个1至少被圈一次
   - 可重复圈
   - 可跨边圈（首尾相邻）
3. 写表达式：
   - 每个圈对应一个乘积项
   - 圈内不变的变量保留
   - 变化的变量消去

**最小项与最大项**：
- **求最小项之和**：圈1
  得最简与或式（SOP）
- **求最大项之积**：圈0
  得最简或与式（POS）

**无关项**：
输入组合永不出现或输出任意
用×或d表示，可当1或0处理以简化

**多输出函数**：
可共享公共项，画多张卡诺图

**化简原则**：
1. 圈最大：每圈2^n个格（1, 2, 4, 8...）
2. 圈最少：用最少的圈覆盖所有1
3. 重复圈：为了减少总圈数可重复
4. 利用无关项：灵活处理×

## 应用场景
[[组合逻辑电路]]设计；逻辑函数化简；[[数字电路]]优化；最小化[[门电路]]数量；教学演示。

## 注意事项
- 格子排列必须按格雷码（相邻只变1位）
- 边界是相邻的（可环绕）
- 角上4个格也相邻
- 超过5变量用其他方法（Q-M法）
