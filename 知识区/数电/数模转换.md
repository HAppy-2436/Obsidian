# 数模转换

## 定义
DAC（Digital-to-Analog Converter）将数字信号（二进制码）转换为模拟信号（电压或电流）。基本原理是按[[数制和码制]]的权值将各位数字量相加得到模拟量。常用方法有权电阻网络、R-2R电阻网络、PWM方式。DAC是数字系统与模拟世界的桥梁，与[[模数转换]]ADC互为逆过程，广泛用于音频、视频、控制系统。

## 核心内容
**基本原理**：
n位二进制码 D = d_{n-1}d_{n-2}...d₁d₀
对应模拟量：V_out = V_ref × (D / 2^n)
其中V_ref是参考电压

**主要指标**：
1. **分辨率**：最小输出变化量
   n位DAC分辨率 = V_ref / 2^n
   如8位、V_ref=5V：分辨率=5/256≈19.5mV
   
2. **转换精度**：实际输出与理论值的误差
3. **转换速度**：建立时间

**权电阻网络DAC**：
- 各位用不同阻值电阻（2⁰R, 2¹R, ..., 2^{n-1}R）
- 简单直观
- 缺点：电阻阻值范围大，难制造

**R-2R电阻网络DAC**：
- 只用R和2R两种阻值
- 易于集成，精度高
- 最常用的DAC结构
- 倒T形电阻网络

**权电流型DAC**：
用电流源代替电阻
速度快，常用于高速DAC

**PWM型DAC**：
用脉宽调制产生模拟量
- 输出PWM波经低通滤波得模拟量
- 简单，成本低
- 常用于单片机系统

**典型芯片**：
- DAC0832（8位，常用）
- AD7520（12位）
- DAC1210（12位）

**性能提升**：
- 增加位数提高分辨率
- 缓冲放大器提高带载能力
- 低通滤波器平滑输出

## 应用场景
音频播放（CD、MP3）；视频显示；波形发生器；电机控制；可编程电源；测试设备输出。

## 注意事项
- 位数越多分辨率越高
- R-2R网络最常用
- 需要参考电压V_ref
- 输出需低通滤波平滑
