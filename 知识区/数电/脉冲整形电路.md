# 脉冲整形电路

## 定义
将不规则、缓变或含干扰的脉冲信号整形为标准的矩形脉冲的电路。主要功能是改善脉冲波形的上升/下降沿，消除抖动和毛刺，使脉冲符合[[时序逻辑电路]]的要求。常用器件有施密特触发器、单稳态触发器、微分电路等。脉冲整形是数字信号预处理的重要环节，提高系统抗干扰能力和工作可靠性。

## 核心内容
**需要整形的情况**：
1. 边沿缓慢（上升/下降时间长）
2. 含有毛刺和抖动
3. 幅度不稳定
4. 多次跳变
5. 输入不规则

**施密特触发器整形**：
**特点**：
- 有回差（滞后）特性
- 上限阈值V_T+，下限阈值V_T-
- V_T+ > V_T-（回差电压ΔV_T）
- 输入>V_T+→输出高
- 输入<V_T-→输出低
- V_T- < 输入 < V_T+时输出保持

**优点**：
- 抗干扰能力强（回差作用）
- 输出陡峭的矩形波
- 消除多次跳变

**应用**：
- 波形整形（正弦波→方波）
- 按键消抖
- 脉冲幅度甄别

**芯片**：74LS14（6个反相施密特触发器）

**单稳态触发器整形**：
- 输入触发产生固定宽度脉冲
- 消除输入脉宽变化
- 标准化脉冲宽度
- 可用[[555定时器]]实现

**微分电路整形**：
- RC微分电路
- 提取上升/下降沿
- 产生尖脉冲

**积分电路整形**：
- RC积分电路
- 平滑脉冲
- 消除毛刺

**其他方法**：
1. **RC滤波**：消除高频毛刺
2. **二极管限幅**：限制幅度
3. **缓冲器**：增强驱动能力

**实际应用电路**：
- 按键消抖：施密特触发器+RC滤波
- 光电检测：施密特整形+单稳态
- 时钟整形：施密特触发器

## 应用场景
按键消抖；传感器信号处理；通信信号整形；时钟信号再生；脉冲幅度甄别；波形变换。

## 注意事项
- 施密特触发器最常用于整形
- 回差电压决定抗干扰能力
- 单稳态可标准化脉宽
- 整形会产生延迟
