# 移位寄存器

## 定义
特殊的[[寄存器]]，数据可在[[触发器]]间左移或右移传输。由串联的D触发器组成，在时钟控制下数据逐级移位。可实现串并转换、并串转换、数据延时、序列发生等功能。74LS194是典型的双向移位寄存器。移位寄存器是数据传输、算术运算（移位等同乘除2）的重要电路，广泛用于通信和数据处理。

## 核心内容
**基本结构**：
n个D触发器级联
- 前一级Q输出连后一级D输入
- 同一时钟CLK控制
- 数据逐级传递

**工作模式**（74LS194）：
1. **保持**（S₁S₀=00）：输出不变
2. **右移**（S₁S₀=01）：数据向右移，串行输入DSR
3. **左移**（S₁S₀=10）：数据向左移，串行输入DSL
4. **并行加载**（S₁S₀=11）：并行输入D₀-D₃同时加载

**数据转换**：
- **串并转换**（SIPO）：
  串行输入n次，并行输出n位
  如串行通信接收
  
- **并串转换**（PISO）：
  并行加载n位，串行移出
  如串行通信发送

**功能应用**：
1. **数据传输**：
   串行传输节省线路（1根数据线）
   
2. **延时**：
   延时n个时钟周期（n级移位）
   
3. **序列发生器**：
   反馈形成特定序列
   
4. **算术运算**：
   左移1位 = ×2
   右移1位 = ÷2（无符号数）

**环形计数器**：
- 首尾相连：Q₃→D₀
- 形成循环移位
  
**扭环计数器**（约翰逊计数器）：
- Q̄₃→D₀（反馈取反）
- 状态数=2n（n个触发器）

## 应用场景
串并/并串转换；串行通信；数据延时；序列检测；算术移位；伪随机数生成；键盘扫描。

## 注意事项
- 移位方向要明确（左移/右移）
- 串行输入端要正确连接
- 移位n次需n个时钟
- 并行加载可快速置初值
