# 链表

## 定义
链表是由节点组成的动态数据结构，每个节点包含数据域和指向下一节点的[[指针]]域。通过[[结构体]]和[[动态内存分配]]实现。链表分为单链表、双链表、循环链表。与数组相比，链表插入删除快（O(1)），但访问慢（O(n)），内存不连续。链表是数据结构的基础，也是面试高频考点。

## 核心内容
**节点定义**：
```c
// 单链表节点
struct Node {
    int data;           // 数据域
    struct Node *next;  // 指针域
};
typedef struct Node Node;

// 双链表节点
struct DNode {
    int data;
    struct DNode *prev;  // 前驱指针
    struct DNode *next;  // 后继指针
};
```

**基本操作**：
1. **创建节点**：
```c
Node* createNode(int data) {
    Node *node = (Node*)malloc(sizeof(Node));
    if (node == NULL) return NULL;
    node->data = data;
    node->next = NULL;
    return node;
}
```

2. **头插法**：
```c
void insertHead(Node **head, int data) {
    Node *node = createNode(data);
    node->next = *head;
    *head = node;
}
```

3. **尾插法**：
```c
void insertTail(Node **head, int data) {
    Node *node = createNode(data);
    if (*head == NULL) {
        *head = node;
        return;
    }
    Node *p = *head;
    while (p->next != NULL) p = p->next;
    p->next = node;
}
```

4. **删除节点**：
```c
void deleteNode(Node **head, int data) {
    Node *p = *head, *prev = NULL;
    while (p != NULL && p->data != data) {
        prev = p;
        p = p->next;
    }
    if (p == NULL) return;  // 未找到
    if (prev == NULL) {
        *head = p->next;    // 删除头节点
    } else {
        prev->next = p->next;
    }
    free(p);
}
```

5. **查找**：
```c
Node* search(Node *head, int data) {
    while (head != NULL) {
        if (head->data == data) return head;
        head = head->next;
    }
    return NULL;
}
```

6. **遍历**：
```c
void traverse(Node *head) {
    while (head != NULL) {
        printf("%d -> ", head->data);
        head = head->next;
    }
    printf("NULL\n");
}
```

7. **销毁链表**：
```c
void destroy(Node **head) {
    Node *p;
    while (*head != NULL) {
        p = *head;
        *head = (*head)->next;
        free(p);
    }
}
```

**链表类型**：
1. **单链表**：只有next指针
2. **双链表**：有prev和next指针
3. **循环链表**：尾节点指向头节点
4. **带头节点的链表**：哑节点简化操作

**常见操作**：
- 反转链表
- 检测环（快慢指针）
- 找中间节点
- 合并有序链表
- 删除倒数第k个节点

## 应用场景
动态数据存储；实现栈、队列；LRU缓存；邻接表（图）；多项式运算；内存管理（空闲链表）。

## 注意事项
- 插入删除要更新指针
- 注意边界情况（空链表、单节点）
- 防止内存泄漏（及时free）
- 避免野指针和悬空指针
