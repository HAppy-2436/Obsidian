## **三、实验原理**

### **3.1 74LS151八选一数据选择器**
#### **3.1.1 功能描述**
74LS151是一种八选一数据选择器，具有8个数据输入端（$D_0$～$D_7$）、3个地址输入端（$A_2$、$A_1$、$A_0$）、一个使能端（$S$）和两个互补输出端（$Y$和$W$）。其逻辑功能如下表所示：

| $S$ | $A_2$ $A_1$ $A_0$ | 输出 $Y$ | 输出 $W$ |
|---|---------|-------|-------|
| 1 | × × × | 0 | 1 |
| 0 | 0 0 0 | $D_0$ | $\overline{D_0}$ |
| 0 | 0 0 1 | $D_1$ | $\overline{D_1}$ |
| 0 | 0 1 0 | $D_2$ | $\overline{D_2}$ |
| 0 | 0 1 1 | $D_3$ | $\overline{D_3}$ |
| 0 | 1 0 0 | $D_4$ | $\overline{D_4}$ |
| 0 | 1 0 1 | $D_5$ | $\overline{D_5}$ |
| 0 | 1 1 0 | $D_6$ | $\overline{D_6}$ |
| 0 | 1 1 1 | $D_7$ | $\overline{D_7}$ |

其逻辑表达式为：
$$Y = \sum_{i=0}^{7} m_i D_i$$
其中 $m_i$ 为地址变量对应的最小项。

#### **3.1.2 降维法原理**
对于n变量逻辑函数，若采用k变量作为地址输入（k < n），则每个数据输入端将变为剩余（n-k）个变量的函数。这种方法称为降维法或扩展法。

### **3.2 74LS138三线-八线译码器（作数据分配器）**
74LS138既可用作3线-8线译码器，也可用作数据分配器。当用作数据分配器时，将输入数据分配到8个输出端之一，具体由地址输入端决定。

### **3.3 组合逻辑函数设计方法**
本实验采用降维法实现5变量函数：
$$Q = ABC + A\overline{C}DF + \overline{B}CD + BC\overline{D}F + \overline{C}\overline{D}\overline{F} + CD\overline{F}$$

选择F、D、C作为地址变量，A、B作为数据变量，将5变量函数降维为8个2变量函数。

原函数：
$$Q = ABC + A\overline{C}DF + \overline{B}CD + BC\overline{D}F + \overline{C}\overline{D}\overline{F} + CD\overline{F}$$

选择F、D、C作为地址变量，A、B作为数据变量：
- $A_2 = F$
- $A_1 = D$
- $A_0 = C$

通过代入化简，得到各数据输入端函数：
$$
\begin{aligned}
&D_0 = 1 \\
&D_1 = AB \\
&D_2 = 0 \\
&D_3 = 1 \\
&D_4 = 0 \\
&D_5 = B \\
&D_6 = A \\
&D_7 = A + \overline{B}
\end{aligned}
$$

   - $D_5$：接B（直接连接）
   - $D_6$：接A（直接连接）

## **九、思考题**

### **9.1 如何用八选一数据选择器设计三十二选一的数据选择器？**
**答：** 可以采用4片74LS151级联实现。需要5位地址码，其中高2位用于片选，低3位用于片内选择。

电路设计方案：
1. 将5位地址码分为：$A_4A_3$（片选）和$A_2A_1A_0$（片内选择）
2. 4片74LS151的地址端并联接$A_2A_1A_0$
3. 使用2-4线译码器（如74LS139）将$A_4A_3$译码为4个片选信号
4. 每片74LS151的使能端S接相应片选信号的反相
5. 4片输出通过或门合并

逻辑表达式：
$$Y = \overline{S_0}Y_0 + \overline{S_1}Y_1 + \overline{S_2}Y_2 + \overline{S_3}Y_3$$

### **9.2 降维法中地址变量的选择原则是什么？**
**答：** 地址变量的选择应遵循以下原则：
1. **使数据输入端函数尽可能简单**：优先选择在函数中出现频率高的变量
2. **减少门电路数量**：选择能使多个数据端接固定电平（0或1）的变量
3. **简化实现**：选择能使部分数据端直接连接输入变量的组合
4. **综合考虑**：通过比较不同选择方案的门电路需求，选择最优方案
