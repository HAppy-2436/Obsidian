## 一、Ch01+2.1 系统漫游 & 信息存储（必考，约 10-12 分）

### 🔴 必背概念
- **编译系统四阶段**：预处理器(.c→.i) → 编译器(.i→.s) → **汇编器(.s→.o，输出二进制)** → **链接器(.o→可执行，输出二进制)**
- **高速缓存在 CPU 内**（SRAM 实现），**主存是 DRAM**（内存条）
- **指针字节数**：64 位机 `int** = 8`、`char* = 8`、`int = 4`
- **大小端**：小端 = 低位字节在低地址；大端 = 高位字节在低地址；**字符串按字符顺序从低地址存**（与 int 大小端无关）

### 🔴 必会计算套路

#### 计算 1：进制转换（每年必考）
| 方向 | 套路 |
|------|------|
| 十 → 二 | 除 2 取余倒序（如 `172 = 128+32+8+4 = 10101100`） |
| 二 → 十六 | **4 位一组从低位开始**（`100100100 → 0x124`） |
| 六进制乘法 | 先转十进制做完运算，**结果仍是六进制数字**（`13×12₆=120₆`，不是"156"！） |

📝 **模拟卷原题**：`13×12`（六进制）= **120₆（选项 D 以上都不是）**；`0x5C5D − 0x2E9A = 0x2DC3`（直接按 16 进制相减）；`172₁₀ = 10101100`；`100100100₂ = 0x124`。

#### 计算 2：位运算 mask
| 需求 | 表达式 | 易错点 |
|------|--------|--------|
| **最低字节不变，其他位置 1** | **`x \| 0xFF`** | 不选 `x & ~0xFF`（那会清零其它位） |
| 只保留最低字节 | `x & 0xFF` | |
| 把最低字节清零 | `x & ~0xFF` | |

#### 计算 3：逻辑 vs 位运算
- `!x`：任何非零都得 **0**；`!0x56 = 0x00`
- `~x`：**逐位翻转**（`~0x56 = 0xA9`）
- `&&` / `||`：结果是 **0 或 1**（短路）

📝 **模拟卷原题**：`!!0x52 = 0x00`（双重逻辑非最终是 0 或 1）。

#### 计算 4：大小端解读
例：`int x = 0xAC5A2EF3`，&x = 0x2100
- 地址 0x2100 是第 0 字节，0x2102 是第 2 字节
- 小端机器：低地址存低位 → `2100=F3, 2101=2E, 2102=5A, 2103=AC` → 0x2102 = **5A**
- 大端机器：低地址存高位 → `2100=AC, 2101=5A, 2102=2E, 2103=F3` → 0x2102 = **2E**

📝 **模拟卷原题**：x=0x123456，最高低字节？小端→**0x56**；大端→**0x12**（选项 C）。

## 二、Ch02 整数与浮点（必考，约 14-16 分，占比最高之一）

### 🔴 必背概念

#### 整数三大编码
| 编码 | w 位范围 | 关键特征 |
|------|----------|----------|
| **无符号** | `0 ~ 2ʷ−1` | 直接二进制 |
| **原码** | `−(2ᵂ⁻¹−1) ~ +(2ᵂ⁻¹−1)` | **正 0 负 0**，符号位占 1 位 |
| **补码** | `−2ᵂ⁻¹ ~ 2ᵂ⁻¹−1` | **只有一个 0**，最高位权重 `−2ᵂ⁻¹` |

📝 **10 位原码最小 = −511**（不是 −512）；**10 位补码 TMin = −512**。

#### TMax 加法逆元 = TMax 自身
- `TMax + TMax = (2ᵂ⁻¹−1) × 2 = 2ᵂ − 2 ≡ −2 (mod 2ᵂ) = TMax` → **TMax + TMax = TMax**

#### U2T / T2U 转换
- `U2T_w(x)`：`x < 2ᵂ⁻¹` 不变，否则减 2ᵂ
- **位模式不变，只是解读方式变**

📝 **真题**：`U2T_8(180) = 180 − 256 = −76`；`char x = −100` → `unsigned char ux` 位模式 `10011100` → ux = **156**（256−100）。

#### 关系比较陷阱（必考）
- `unsigned i = −1; i >= 0` → **true**（−1 当 UMax）
- `−1 < 1U` → **false (0)**（都按无符号比较）

### 🔴 必会计算套路

#### 计算 1：乘法优化（移位加）
把常数拆成 2 的幂之和：`52 = 64 − 16 + 4 = 2⁶ − 2⁴ + 2²`
→ **`x * 52 = (x<<6) − (x<<4) + (x<<2)`**

📝 通用套路：写出常数的二进制，从最高位的 1 开始，遇到 1 加 `x<<k`，遇到 0 减 `x<<k`。

#### 计算 2：负数除以 2ᵏ 的偏置
负数除以 2ᵏ 时，算术右移前 **加偏置 `2ᵏ−1`**：
- 除以 128 → 偏置 **127**；除以 64 → 63；除以 8 → 7

#### 计算 3：无符号加法溢出
判据 = **`x + y < x`（或 < y）**（真实结果 mod 2ᵂ）
补码正溢出：`x > 0 && y > 0 && x+y ≤ 0`；补码负溢出：`x < 0 && y < 0 && x+y ≥ 0`

#### 计算 4：补码截断
把 w 位补码截到 k 位：保留**低 k 位**，重新解读为 k 位补码。
📝 **真题**：8 位补码 33 = `00100001`，截到 6 位 = `100001` → 6 位补码 = **−31**（最高位 1 = −32）。

### 🔴 IEEE 754 浮点（必背三公式）

#### 格式
```
┌─s─┬─k 位阶码 exp─┬─n 位小数 frac─┐
```
偏置 bias = `2ᵏ⁻¹ − 1`

#### 三大公式（每年都考）
| 类型 | exp | 公式 |
|------|-----|------|
| **规格化** | 既不全 0 也不全 1 | `(−1)ˢ × 2ᵉˣᵖ⁻ᵇⁱᵃˢ × (1 + frac)` |
| **非规格化** | 全 0 | `(−1)ˢ × 2¹⁻ᵇⁱᵃˢ × (0 + frac)` |
| **最小正数**（非规格最小） | 全 0 | `2⁻ᵇⁱᵃˢ⁻ⁿ` |
| **最大非规格数** | 全 0 | `2¹⁻ᵇⁱᵃˢ × (1 − 2⁻ⁿ)` |

📝 **真题反复考**：
- 11 位（1+6+4）最大非规格 = `2⁻³⁴ × 15`
- 6 位（1+2+3）最小正数 = `2⁻⁴ = 1/16`
- 8 位（1+3+4）`01010101` = `(1 + 0.0101) × 2² = 1.3125 × 4 = 5.25`

#### 向偶数舍入（默认）
看舍入位：=0 截断；=1 看**保留位最低位**，偶不变（0），奇进位（+1）
📝 `11.011` 保留 2 位 → 看 `.1` 触发 → 保留位 `1`（奇）→ 进位 → **`11.10`**
📝 `10.1011` 保留 2 位 → 看 `.11` → 进位 → **`11.00`**

### 🔴 浮点运算必背结论
| 结论 | 真值 | 原因 |
|------|------|------|
| 浮点满足交换律 | ✅ | |
| 浮点满足结合律 | ❌ | 例：`1e20 + (−1e20 + 1.25) = 0`，但 `(1e20 + −1e20) + 1.25 = 1.25` |
| `f*f ≥ 0`（float） | ✅ **总成立** | 浮点乘法保符号 |
| `i*i ≥ 0`（int） | ❌ | `50000² = 2.5e9` 溢出变负 |
| 类型转换相等 | **只有 A 总真** | `x==(int)(double)x` ✅；其他都可能丢精度（float 只有 24 位精度） |

### 🔴 补码运算律
**只满足交换律**；不满足结合律（溢出）和分配律（溢出）。
📝 **高频考**：`x*y ≥ 0` 对 int 不一定（溢出变负）。

### 🔴 模拟卷浮点大题模板（必背）
**5 位浮点格式（1 符号 + 3 阶码 + 1 尾数）**：
- 偏置 = `2² − 1 = 3`
- 最大数 = `(1 + 0.5) × 2³ = 12`
- 最小正数 = `2¹⁻³ × 2⁻¹ = 2⁻³ = 0.125`
- 最大非规格 = `2¹⁻³ × 0.5 = 0.5`
- 1.0 = `01010`（0 符号，exp=010=2 偏置后，M=0 → `(1+0) × 2²⁻³ = 1`）

## 三、Ch03 机器级表示（汇编，必考，约 16-20 分，最大头）

### 🔴 必背概念

#### 寄存器约定
| 寄存器 | 用途 |
|--------|------|
| **%rax** | **函数返回值** |
| **%rsp** | 栈指针 |
| **%rdi, %rsi, %rdx, %rcx, %r8, %r9** | 函数参数 1~6 |
| **%rbp** | 帧基址 |
| **%rbx, %r12~%r15** | 被调者保存 |

#### 寻址公式（每年必考）
$$\text{EA} = \text{Imm} + R[R_i] + s \cdot R[R_b]$$
- 比例因子 s = 1/2/4/8（对应元素大小）

#### 关键指令速记
| 指令 | 作用 | 易错点 |
|------|------|--------|
| `leaq Imm(R_A, R_B, s), %rd` | **算地址不访存** | 不要和 mov 混淆 |
| `xorq %r, %r` | **清零** | |
| `salq $1, %rdx` | **乘 2** | 不是 `imulq`（imul 是双/三操作数） |
| `sarq $1, %rdx` | 算术右移（**除 2**） | |
| `pushq %rb` | `%rsp -= 8`，写入栈顶 | |
| `popq %ra` | 读栈顶，`%rsp += 8` | |

#### mov 规则（必考）
- ❌ 不能内存→内存
- ❌ **`movq %edx, (%rsp)` 错误**（32 位值不能直接存 64 位栈槽，要 `movslq` 先零扩展）
- ✅ `movw %si, %di`（16 位对 16 位）；`movb $22, -23(%rsi)`（立即数→内存）

#### mov ↔ C
| 汇编 | C 代码 |
|------|--------|
| `movq %rdx, (%rdi)` | **`*p = a`**（p 在 rdi，a 在 rdx） |
| `movq (%rdi), %rax` | `a = *p` |
| `movq %rax, %rdi` | `p = a` |

### 🔴 必会计算套路

#### 计算 1：寻址公式
📝 **真题反复考**：`Imm(R_A, R_B, s)` → `EA = Imm + R_A + s × R_B`
- `0x36(%rcx, %rdi, 2)`，rcx=0x100, rdi=0x18 → `0x36 + 0x100 + 2×0x18 = 0x166`
- `leaq 0x51(%rdi, %rsi, 4), %rdx`，rdi=0x310, rsi=0x06 → `0x51 + 0x310 + 4×0x06 = 0x379`
- `movl 0x60(%rdi, %rsi, 256), %eax`，rect[i][j]，rsi=i → 偏移 = `0x60 + 256×i` → j×4 = 0x60 → **j = 24**

#### 计算 2：cmp + jX 反推条件（必考）
⚠️ **cmp src, dst → dst − src**（不是 src−dst！）

| 跳转 | 条件 | 符号 |
|------|------|------|
| `jge` | **≥** | **有符号** |
| `jl` | **<** | 有符号 |
| `ja` / `jae` | > / ≥ | **无符号** |
| `jb` / `jbe` | < / ≤ | 无符号 |
| `je` / `jne` | == / != | |

| set 指令 | 条件 | 类型 |
|---------|------|------|
| `setbe %al` | **≤（无符号）** | `cmpl %edi, %esi` → esi − edi |
| `setg %al` | >（有符号） | |
| `sete` | == | |

📝 **真题**：`cmp %rdi, (%rdx); jge .L3` → 计算 `*p − a`，跳 if `*p ≥ a`（**jge 是有符号 ≥**）。
📝 `cmpq $12, %rcx; jl .L3` → if `rcx < 12`。
📝 `testq %rdx, %rdx; jne .L3` → if `rdx ≠ 0`（test 做 AND 不写回）。

#### 计算 3：call 前后 %rip（高频）
⚠️ x86 是**先取后增**，所以：
- 执行 call 指令时，**%rip 已是指向 call 自己**（call 的地址）
- call 压栈的 = **call 下一条指令地址**（返回地址）
- call 之后进入函数，**%rip = 目标函数入口**

📝 **真题**：`mov 0x5(%rdx),%rax @ 310040; callq 310020 @ 310045; add @ 310052`
- call 之前 %rip = **310045**（call 自己）
- call 压栈 = **310052**（call 下一条）
- call 之后 %rip = **310020**（目标）

📝 **真题**：汇编 `77 02 ja 4004d5; 5d pop %rbp`（`ja` 是 2 字节指令）
- 跳转目标 = 4004d5，偏移 = +2 → 第一条地址 = **4004d5 − 2 = 4004d3**

#### 计算 4：栈操作
- 栈顶 0x0f0100，`pushq %rbx` → 栈顶 = **0x0f00f8**（减 8）
- `%rax=0x108, %rsp=0x158`，`popq %rax` 后 `%rsp = 0x160`（加 8）

#### 计算 5：条件码
- **ZF = 零标志**；CF = 进位（无符号溢出）；SF = 符号；OF = 溢出（有符号）

#### 计算 6：while 循环翻译
- **while 翻译 = guarded-do**（不是 do-while）
- for / while 用 jump to middle

### 🔴 数组与结构体

#### 一维数组地址公式
| 声明 | 第 i 元素地址 |
|------|--------------|
| `int A[6]` | `a + 4i` |
| `char **C[12]` | `c + 8k`（指针 8 字节） |
| `char *s[N]` | `s + 8i` |

#### 二维数组
`int rect[M][N]` 中 `rect[i][j]` 偏移 = `i×N×sizeof(元素) + j×sizeof(元素)`
- `int rect[32][32]` 中 `rect[i][j]` 偏移 = `128i + 4j`

📝 **真题**：`movl 0x60(%rdi, %rsi, 256)`，rdi=rect, rsi=i → j = 0x60/4 = **24**

#### 结构体对齐（三大规则）
1. 成员按声明顺序排列
2. 每个成员偏移量 = **其大小的整数倍**
3. 结构体总大小 = **最大成员大小的整数倍**

📝 **真题 MEMO 结构**（`int(4)+char(1)+int(4)+char[21]+char*(8)+char(1)`）：
最优排列 = **content(8) → index(4) → ref(4) → flag(1)+title[21]+isPrivate(1)**（**大的放前**，小的扎堆）

📝 **TCP 嵌套结构**（`short+char+int+3*char+ptr`）：SYN 偏移 = **9**；总大小 = **24**

#### Union（共内存）
所有字段共享同一段内存，大小 = 最大字段大小。
📝 **小端 union 解读**：最低字节在最低位 → `dw.c[0..7] = d0..d7` → `i[0] = 0xd3d2d1d0`（i[0] 由 c[0..3] 组成，小端拼装）

### 🔴 空间局部性
- **按行读 2D 数组** → 空间局部性好（连续地址）
- **按列读** → 空间局部性差（跳步）

📝 **真题**：步长大 → 空间局部性**越差**；数组引用步长大 → 缓存命中率**越低**。

## 四、Ch06 存储器层次与高速缓存（必考，约 10-12 分）

### 🔴 必背概念

#### 存储技术速度排序
**SRAM > DRAM > SSD > Flash**
- **SRAM**：高速缓存（6 晶体管/位，贵、快）
- **DRAM**：主存/内存条（1 晶体管+电容，便宜、要刷新）
- SSD：基于 NAND Flash

#### 三大高速缓存（看每组行数 E）
| 类型 | E | S |
|------|---|---|
| **直接映射** | E = **1** | |
| **组相联** | 1 < E < C/B | |
| **全相联** | E = C/B | **S = 1** |

⚠️ **直接映射和其它的唯一区别 = E（每组行数）**

#### 三类不命中（必考，没有"热"）
| 类型 | 原因 |
|------|------|
| **冷不命中** | 第一次访问该块（不可避免） |
| **容量不命中** | 工作集 > 缓存容量 |
| **冲突不命中** | 多个块映射到同一组 |

### 🔴 必会计算套路

#### 公式四件套（每年必考）
$$S = \frac{C}{B \times E}, \quad b = \log_2 B, \quad s = \log_2 S, \quad t = m - s - b$$

📝 **真题**：`m=16, C=1024, E=4, B=4`
- S = 1024/(4×4) = **64**
- b = log₂4 = 2；s = log₂64 = 6；t = 16−6−2 = **8**

#### 地址 → 标记/组号/偏移
1. 把 m 位地址写成二进制（或 16 进制）
2. **从低位开始数 b 位 = 偏移**
3. **接下来 s 位 = 组号**
4. **剩下高位 = 标记**

📝 **真题**：m=16, C=1024, E=4, B=4，地址 0x123a
- 二进制 `0001 0010 0011 1010`
- 低 2 位 = `10`（偏移 2）
- 接下来 6 位 = `001110`（组号 14）
- 标记 = `0001001000`（72）

⚠️ **地址偏移切分必须从低到高**（先 b，再 s，剩下是 t）。

#### LRU 模拟（必考）
1. 每个地址 → 计算组号 + 标记
2. 命中：更新 LRU 顺序
3. 不命中：替换组内**最久未用**
4. 统计 hit 数和最终 tag 值

📝 **真题**：48 字节三路组相联、B=4 → S = 48/(4×3) = **4 组**
序列 `0x02, 0xb4, 0x2a, 0x28, 0x13, 0x22, 0x01, 0x30` → 命中 **2 次**（0x28, 0x22）

#### Amdahl 定律
$$\text{加速比} = \frac{1}{(1-\alpha) + \alpha/k}$$

📝 **真题**：缝制 25% + 消毒 75%，整体加速 4/3，求消毒步骤加速比 t
- `0.25 + 0.75/t = 3/4` → `0.75/t = 0.5` → **t = 1.5**

#### 磁盘访问时间
$$T_{access} = T_{avg\_seek} + T_{avg\_rotation} + T_{transfer}$$
$$T_{avg\_rotation} = \frac{1}{2} \times \frac{60s}{RPM}$$

📝 **真题**：RPM=10000, T_seek=9ms, T_access=12.02ms
- T_rotation = 3ms；T_transfer = 0.02ms；扇区数 = 3/0.02 = **150**

📝 **真题**：RPM=15000, T_seek=8ms, 扇区数=500
- T_max_rotation = 60/15000 = 4ms；T_avg_rotation = 2ms
- T_transfer = 4/500 = 0.008ms；T_access = 8 + 2 + 0.008 ≈ **10ms**

## 五、Ch07 链接（必考，约 8-10 分）

### 🔴 必背概念

#### ELF 节速记（每年都考）
| 节 | 内容 |
|----|------|
| **.text** | **程序代码** |
| **.rodata** | 只读数据（字符串、switch 表） |
| **.data** | **已初始化全局/静态变量** |
| **.bss** | **未初始化静态变量（占位不占空间）** |
| **.symtab** | **符号表** |
| **.rel.text** | 代码段的重定位条目 |
| **.rel.data** | 数据段的重定位条目 |
| **.debug** | 调试信息（-g 才有） |

⚠️ 易混点：
- **.bss 在磁盘上不占空间**（运行时 loader 给 0）
- 已初始化全局 → .data；未初始化全局 → **COMMON**（**不是 .bss！**）
- .bss 存的是已分配但未初始化的**局部静态变量**

#### 看 ELF 工具 = **readelf**

#### 符号表里有什么
- ✅ 全局变量、函数、静态变量
- ❌ **局部（auto）变量不在符号表**（运行时栈上）

#### 强弱符号规则
| 符号 | 类型 |
|------|------|
| **强符号** | 函数 + **已初始化**全局变量 |
| **弱符号** | 未初始化全局变量（存 **COMMON**） |

| 情况 | 链接器处理 |
|------|----------|
| 多强同名 | **链接错误** |
| 1 强 + N 弱 | 选强 |
| 多弱同名 | **选 size 最大** |

#### 重定位条目
| 字段 | 含义 |
|------|------|
| **offset** | 待修改引用的**节内偏移** |
| **symbol** | 被引用符号 |
| **type** | `R_X86_64_PC32`（PC 相对）或 `R_X86_64_32`（绝对） |

公式：`PC 相对 = S + A − P`；`绝对 = S + A`

#### 静态库链接顺序
- 链接器**从左到右**扫描，左边可引用右边
- **循环依赖时要在命令行重复依赖项**

📝 **真题**：`g.o → libx.a → liby.a → g.o` 且 `g.o → liby.a → libz.a → libx.a`
→ **最小命令行**：`gcc g.o libx.a liby.a libz.a libx.a`（重复 libx.a 解决 g.o 的引用）

### 🔴 静态 vs 动态库（每年必考）
| 类型 | 加载时机 | 升级 | 磁盘占用 |
|------|----------|------|----------|
| **静态库** | 链接时 | 需重新链接 | **大**（代码被拷） |
| **动态库** | 运行时 | 替换 .so 即可 | 小（共享） |

⚠️ **静态库可执行文件不需要 .a 就能运行**（代码已拷进），但**动态库必须依赖 .so** 才能运行。

#### PIC 与打桩
- `-fpic` = **生成位置无关代码**（编译共享库用）
- 链接器打桩通常条件 = **能访问可重定位目标文件 .o**
- 动态链接器输出 = **可执行目标文件**（不是 .so）

## 六、Ch08 异常控制流（必考，约 8-10 分）

### 🔴 必背概念

#### 异常四类（必背）
| 类别 | 同步性 | 行为 | 返回 |
|------|--------|------|------|
| **中断 Interrupt** | **异步** | I/O 设备信号 | 总是返回下一条 |
| **陷阱 Trap** | 同步 | 系统调用、**有意触发** | 返回下一条 |
| **故障 Fault** | 同步 | 可恢复（如缺页） | 返回**当前指令**（重试）或终止 |
| **终止 Abort** | 同步 | 不可恢复 | **从不返回** |

⚠️ **陷阱 ≠ 致命错误**（陷阱是有意的 syscall）；**中断是异步的**。

#### 进程三特性
- 独立逻辑控制流、私有地址空间、并行/并发

#### 并发判定（高频）
两个进程**时段有重叠**就是并发。
📝 **真题**：A(1-4), B(3-6), C(5-9), D(7-11) → A-B、B-C、C-D **3 对**并发

### 🔴 必会计算套路

#### fork / exec / wait
- **`fork()` 一次返回两次**：子进程得 0，父进程得子 PID
- `execve` 覆盖当前进程地址空间，**成功后不返回**
- **shell 流程 = fork → 子进程 execve**
- `wait()` **阻塞父进程**，直到任一子进程结束
- `wait(status) ≡ waitpid(-1, status, 0)`

#### fork 输出行数（必考）
📝 **真题**：
```c
fork();
if (fork() == 0) { fork(); printf("2"); exit(0); }
printf("2");
```
画进程树：P0、P0'（走 else）、P1（走 else）、P2（在 if 里）、P3
→ 输出 **4 行 "2"**

#### 僵尸 vs 孤儿（每年必考）
- **僵尸**：子结束但父没 wait → 必须父 wait 才释放
- **孤儿**：父没了 → **init 进程（PID=1）收养**

#### kill 命令
| 命令 | 效果 |
|------|------|
| `kill pid` | 发 SIGTERM（可捕获） |
| `kill -9 pid` | 发 **SIGKILL（不可捕获、不可忽略）** |
| `kill -pid` | 发信号给**进程组**（负号 = 进程组） |

📝 **真题**：3195 是 pgrp，要 kill 整个组 → `/bin/kill -9 -3195`（**负号必须**）

### 🔴 信号速记（每年必考）

| 信号 | 编号 | 来源 | 默认 |
|------|------|------|------|
| **SIGINT** | 2 | **Ctrl-C** | 终止 |
| **SIGTSTP** | 20 | **Ctrl-Z** | 停止（挂起） |
| **SIGQUIT** | 3 | Ctrl-\ | 终止 + core |
| **SIGALRM** | 14 | alarm() | 终止 |
| **SIGCHLD** | 17 | 子进程状态变化 | 忽略 |
| **SIGKILL** | 9 | **kill -9** | 终止（不可捕获） |
| **SIGSEGV** | 11 | 段错误 | 终止 + core |

⚠️ **Ctrl-Z 是 SIGTSTP 不是 SIGINT**。

#### signal + pause + exit handler
```c
signal(SIGINT, sig_handler);
printf("Wait");
pause();
printf("End");
```
按 Ctrl-C：`Wait` → handler 跳入 → `Caught` → exit(0) → **永远不打 End**
→ 输出 **2 行：Wait + Caught**（不打印 End）

#### setjmp / longjmp
- `setjmp(env)`：**调用一次，返回一次或多次**（被 longjmp 跳回时再次返回）

#### 进程调度
- 进程调度顺序 = **不确定**，与调度器实现有关

## 七、Ch09 虚拟内存 + 动态内存分配（必考，约 10-12 分）

### 🔴 必背概念

#### 虚拟内存三大能力
1. 主存当磁盘缓存（按页换入换出）
2. 独立地址空间
3. 保护（每页 R/W/X 权限）

⚠️ **虚拟内存位置** = 物理机制，但数据可放在**硬盘**；**CPU 用的是虚拟地址**（不是物理地址）。

#### 多级页表
- **第一级页表常驻主存**
- **VPN_i（1 ≤ i ≤ k-1）= 查找第 i 级页表中的 PTE 条目**
- **VPN_k = 直接拿到 PPN**
- 多级页表大小由**虚页数**决定（不是实页数）

📝 **真题易错**：误以为页表大小由"实页数"决定——**错**！是虚页数。

### 🔴 必会计算套路

#### 地址翻译（每年必考）
```
虚地址: ┌──── VPN ────┬── VPO ──┐
物地址: ┌──── PPN ────┬── PPO ──┐
```
- **PPO = VPO**（页内偏移一致）
- **TLBT = VPN 去掉 TLBI 部分**（TLBI 是 VPN 的低 log₂S 位）

📝 **真题**：页大小 32B（PPO=5位），VA=12位，PA=10位，TLB 16 条目二路相联（S=8 组，TLBI=3位）
- VA=0x3D2 = `0011 1101 0010`
- PPO = 低 5 位 = `10010` = **0x12**
- TLBI = 接下来 3 位 = `110` = 6
- TLBT = 剩余高位 = `0011` = **0x03**

#### LRU 缺页（高频）
📝 **真题**：4 帧 LRU，访问序列 `1,2,3,4,1,2,5,1,2,3,4,5`
- 标准 LRU = **6 次缺页**（题选项最接近 7）；若题要选，按模拟答案选 **A=7**

#### TLB 直接好处
- ✅ **节省页表查找时间**（不节省存储、不节省数据页时间）

### 🔴 动态内存分配

#### 分配器类型
| 类型 | 例子 |
|------|------|
| **显式分配器** | C `malloc/free` |
| **隐式分配器** | Java `new`、Python GC |

#### 性能指标权衡
- **吞吐率** vs **内存利用率** = **时间 vs 空间** 权衡

#### 隐式空闲链表（implicit free list）
- 块头存大小 + 分配位
- **块之间靠头部长度信息**链接（没有显式指针）
- **遍历整个堆**找空闲块

#### 放置策略（必背）
| 策略 | 行为 |
|------|------|
| 首次适配 First fit | 从头找第一个够大的块 |
| 下次适配 Next fit | 从上次结束继续 |
| 最佳适配 Best fit | 找最小的够大块 |
| 最差适配 Worst fit | 找最大的块（隐式不常用） |

⚠️ **没有 Mark and Sweep**（那是 GC 算法，不是放置策略）

#### 边界标记（boundary tags）
- 块**头部 + 脚部**都存大小
- 释放时便于和**前一个空闲块合并**（脚部即上一个块的边界）

📝 **真题**：边界标记便于**和前一个空闲块合并**

#### 块标记位
- 块的标记位在**最低位**（大小字段的 bit 0）
- **提取用 `& 1`**（如 `(*(int *)hp) & 1`）

#### 显式空闲块最小大小
- 头(4) + 前驱 pred(4) + 后继 succ(4) + 脚(4) = **16 字节**（双字对齐要求）

#### malloc 实际方案
- **`malloc` 实际用分离适配（segregated fit）**——按大小类分多个空闲链表

## 📚 附录：常见易错点清单

1. ❌ 误以为汇编器产生汇编代码（**编译器**产生汇编代码，**汇编器**把汇编翻译成二进制）
2. ❌ 误以为高速缓存在主存里（**在 CPU 内**）
3. ❌ 大小端记反（**小端：低位在低地址**）
4. ❌ 把六进制运算结果当十进制数（**结果仍是六进制数字**）
5. ❌ 误以为原码最小 = -2^(w-1)（**是 -（2^(w-1)-1）**）
6. ❌ 误以为补码运算满足结合律（**只有交换律**）
7. ❌ 误以为 `i*i ≥ 0` 对 int 总成立（**溢出可能变负**）
8. ❌ 误以为 `i*i ≥ 0` 对 float 不总成立（**float 总成立**）
9. ❌ 误以为 `x==(int)(float)x` 总成立（**只有 `x==(int)(double)x` 总成立**）
10. ❌ 误以为 `x<<k` 是循环移位（**普通移位，高位移出截掉**）
11. ❌ cmp 反向记成 `cmp src,dst → src − dst`（**实际是 dst − src**）
12. ❌ 误以为 `jge` 是无符号 ≥（**是有符号 ≥**；无符号是 `jae`）
13. ❌ 误以为 `setbe` 是 <（**是 ≤，无符号**）
14. ❌ 误以为 call 之前 %rip 指向 mov（**已指向 call 自己的地址**）
15. ❌ 误以为已初始化全局变量在 .bss（**在 .data；.bss 是未初始化静态变量占位**）
16. ❌ 误以为弱符号在 .bss（**在 COMMON**）
17. ❌ 误以为静态库可执行文件需要 .a 才能运行（**不需要**）
18. ❌ 误以为中断是同步异常（**异步**）
19. ❌ 误以为故障返回下一条指令（**返回当前指令重试**）
20. ❌ 误以为终止异常返回应用（**从不返回**）
21. ❌ 误以为 fork 后子进程 PID 是 0（**子进程里 fork 返回 0；父进程里 fork 返回子 PID**）
22. ❌ 误以为 Ctrl-Z 是 SIGINT（**是 SIGTSTP**）
23. ❌ 误以为 SIGKILL 可被捕获（**不可捕获、不可忽略**）
24. ❌ 误以为多级页表大小由实页数决定（**虚页数**）
25. ❌ 误以为 Mark and Sweep 是放置策略（**是 GC 算法**）
26. ❌ 误以为 malloc 用隐式空闲链表（**实际用分离适配**）
27. ❌ 误以为 LRU 缺页可以直接数（**必须画表模拟**）
28. ❌ 误以为 `S = C / B`（**是 `S = C / (B × E)`**）
