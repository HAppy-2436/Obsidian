## 一、复杂度分析（必背，填空选择年年考）

### 1.1 常见复杂度量级（从快到慢）

```
O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(n³) < O(2ⁿ) < O(n!)
```

### 1.2 复杂度分析口诀

1. **看最深层 for 循环**（初学版）
2. **看申请了多少空间**（空间复杂度）
3. **最坏情况优先**（如快排最坏 O(n²)，平时说 O(n log n) 是平均）

### 1.3 主定理（递归式 `T(n) = aT(n/b) + O(n^d)`）

| 比较 | 结果 |
|---|---|
| `d > log_b a` | `T(n) = O(n^d)` |
| `d = log_b a` | `T(n) = O(n^d log n)` |
| `d < log_b a` | `T(n) = O(n^log_b a)` |

**记忆**：归并 `T(n)=2T(n/2)+O(n)` → a=2, b=2, d=1，`log_b a=1=d` → O(n log n)

### 2.2 顺序表代码模板（Lab q165 必背）

```cpp
class MyList {
    static const int CAP = 105;
    int data[CAP];
    int sz;
public:
    MyList(): sz(0) {}
    bool full() const { return sz == CAP; }
    int  size() const { return sz; }
    bool insert(int pos, int x) {       // 在 pos 前插入
        if (full() || pos < 0 || pos > sz) return false;
        for (int i = sz; i > pos; i--) data[i] = data[i-1];
        data[pos] = x; sz++; return true;
    }
    bool remove(int pos) {
        if (pos < 0 || pos >= sz) return false;
        for (int i = pos; i < sz-1; i++) data[i] = data[i+1];
        sz--; return true;
    }
    void traverse() const {
        for (int i = 0; i < sz; i++) cout << (i?" ":"") << data[i];
        cout << "\n";
    }
};
```

### 2.4 ⭐ 链表反转（18-19 真卷编程题、Lab q156 必背）

**迭代版（三指针）**：
```cpp
Node* reverse(Node* head) {
    Node *pre = nullptr, *cur = head;
    while (cur) {
        Node* nxt = cur->next;  // ⚠️ 先存 next，否则丢链
        cur->next = pre;
        pre = cur;
        cur = nxt;
    }
    return pre;   // pre 是新头
}
```

**递归版**：
```cpp
Node* reverse(Node* head) {
    if (!head || !head->next) return head;
    Node* newHead = reverse(head->next);
    head->next->next = head;
    head->next = nullptr;
    return newHead;
}
```

**时间 O(n)，空间 O(1) 迭代 / O(n) 递归**

### 2.5 链表插入指针修改（15-16 真卷选择题）

> p, s 是指针，s 指向新节点，**插入到 p 之后**：
> ```cpp
> s->next = p->next;  // ① 先接后面
> p->next = s;        // ② 再接前面
> ```
> 答案选 **c) `s->link = p->link; p = s;`** ← **不对**，正确插入是上面两步
> 真卷选项 b) `s->link = p->link; p->link = s;` 是正确的

**记忆口诀**：先连后断；先 `s->next = p->next`，再 `p->next = s`

### 2.6 ⭐ 双向链表（Lab q169 必背）

```cpp
struct DNode { int v; DNode *prev, *next; 
               DNode(int x): v(x), prev(nullptr), next(nullptr) {} };
```

- **双向哑节点**：`head` 和 `tail` 都为哑，初始 `head->next = tail`, `tail->prev = head`
- **插入 4 个指针**：`n->next = pre->next; n->prev = pre; pre->next->prev = n; pre->next = n`
- **删除 2 个指针**：`p->prev->next = p->next; p->next->prev = p->prev`
- **优点**：可双向遍历；删除给定节点 O(1)

### 2.7 循环链表（约瑟夫 Lab q155 必背）

```cpp
struct CNode { int id; CNode* next; };
// 建环
CNode* cur = new CNode(1);
CNode* pre = cur;
for (int i = 2; i <= n; i++) {
    pre->next = new CNode(i);
    pre = pre->next;
}
pre->next = cur;   // 成环

// 报 m 出列：cur 前进 m-1 步，删除 next
void eliminate(int m) {
    for (int k = 1; k < m; k++) cur = cur->next;
    CNode* del = cur->next;   // del 报 m
    cur->next = del->next;
    if (del == cur) cur = nullptr;
    else cur = del->next;
    delete del;
}
```

**约瑟夫数学递推**：`f(1)=0, f(n)=(f(n-1)+m)%n`

### 2.8 快慢指针套路（必背）

| 应用 | 套路 |
|---|---|
| **判环** | slow +1, fast +2，相遇则有环 |
| **找环入口** | 相遇后一个从头走、一个从相遇点走，再次相遇即入口 |
| **找中点** | fast 到尾时 slow 在中点 |

### 2.10 ⭐ 大数运算（Lab q166 必背）

```cpp
class MyBigInteger {
    vector<int> d;   // 低位在前（便于进位）
public:
    MyBigInteger(const string& s) {
        for (int i = (int)s.size()-1; i >= 0; i--) d.push_back(s[i]-'0');
        trim();   // 去前导零
    }
    static MyBigInteger add(const MyBigInteger& a, const MyBigInteger& b) {
        MyBigInteger r;
        r.d.assign(max(a.d.size(), b.d.size()), 0);
        int c = 0;
        for (size_t i = 0; i < r.d.size(); i++) {
            int s = c + (i<a.d.size()?a.d[i]:0) + (i<b.d.size()?b.d[i]:0);
            r.d[i] = s % 10; c = s / 10;
        }
        if (c) r.d.push_back(c);
        return r;
    }
};
```

**记忆**：低位在前；`trim()` 去前导零；除法选做。

## 三、栈（Stack）⭐⭐⭐（每年必考）

### 3.1 核心性质

- **LIFO**（后进先出），栈顶 `top` 操作
- 所有操作 O(1)

### 3.2 ⭐ 顺序栈（Lab q23 必背）

```cpp
class Stack {
    static const int MAXN = 105;
    int data[MAXN];
    int top;          // top=-1 表示空栈；top 指向栈顶元素
public:
    Stack(): top(-1) {}
    bool empty() const { return top == -1; }
    bool full()  const { return top == MAXN-1; }
    int  size()  const { return top + 1; }
    void clear() { top = -1; }
    bool push(int x) { if (full()) return false; data[++top] = x; return true; }
    bool pop()       { if (empty()) return false; top--; return true; }
    int  peek() const { return data[top]; }
};
```

**两种约定**（别混）：
- `top = -1`：进栈 `data[++top] = x`，出栈 `data[top--]`
- `top = 0`：进栈 `data[top++] = x`，出栈 `data[--top]`

### 3.3 ⭐ 链栈（Lab q158 必背）

```cpp
struct Node { int v; Node* next; Node(int x): v(x), next(nullptr) {} };
class LinkedStack {
    Node* topNode;
    int cnt;
public:
    LinkedStack(): topNode(nullptr), cnt(0) {}
    ~LinkedStack() { clear(); }
    bool empty() const { return cnt == 0; }
    bool full()  const { return false; }   // 链式永不满
    void push(int x) {
        Node* n = new Node(x);
        n->next = topNode; topNode = n; cnt++;
    }
    void pop() {
        Node* t = topNode;
        topNode = topNode->next;
        delete t; cnt--;
    }
    int top() const { return topNode->v; }
};
```

**特点**：头插法（栈顶 = 头节点）、永不栈满、需析构释放。

### 3.4 ⭐⭐ 栈的应用（必考，每年大题）

#### (1) ⭐ 后缀表达式求值（Lab q139、真卷 15-16 必背）

**算法**：
- 扫描 token
- **数字**：入栈
- **运算符**：弹 `b = 栈顶`，弹 `a = 次栈顶`，计算 `a op b`，结果入栈
- **结束**：栈底为结果；最终栈 size ≠ 1 → 非法

```cpp

for (char c : s) {
    if (c >= '0' && c <= '9') st.push(c - '0');
    else if (c=='+'||c=='-'||c=='*'||c=='/') {
        if (st.size() < 2) { cout << "none\n"; return 0; }
        int b = st.pop();      // 后弹 = 右操作数
        int a = st.pop();
        int r = (c=='+')?a+b : (c=='-')?a-b : (c=='*')?a*b : a/b;
        st.push(r);
    }
}
cout << (st.size() == 1 ? st.peek() : -1) << "\n";
```

#### (2) ⭐ 中缀 → 后缀（调度场算法，shunting-yard）

## 四、队列（Queue）⭐⭐⭐

### 4.1 核心性质

- **FIFO**（先进先出）
- 所有操作 O(1)

### 4.2 ⭐ 顺序队列与假溢出

普通顺序队列多次入队出队后 `rear` 到末尾但前面有空位 → **假溢出**。解决：**循环队列**。

### 4.3 ⭐⭐ 循环队列（环形队列，Lab q146 必背）

```cpp
class CirQueue {
    static const int MAXN = 105;
    int data[MAXN];
    int frontIdx, rearIdx, cnt;     // 用 cnt 变量更稳
public:
    CirQueue(): frontIdx(0), rearIdx(-1), cnt(0) {}
    bool empty() const { return cnt == 0; }
    bool full()  const { return cnt == MAXN; }
    void enqueue(int x) { rearIdx = (rearIdx+1)%MAXN; data[rearIdx] = x; cnt++; }
    void dequeue()      { frontIdx = (frontIdx+1)%MAXN; cnt--; }
    int  front() const  { return data[frontIdx]; }
};
```

**两种判空判满方法**（必背）：
- **方法 1**（牺牲一个存储格）：
  - 判空：`front == rear`
  - 判满：`(rear + 1) % cap == front`
  - 长度：`(rear - front + cap) % cap`
- **方法 2**（额外 count）：
  - 判空：`cnt == 0`
  - 判满：`cnt == cap`
  - 长度：`cnt`

### 4.4 链式队列（Lab q159 必背）

```cpp
struct QNode { int v; QNode* next; QNode(int x): v(x), next(nullptr) {} };
class MyLinkedQueue {
    QNode *head, *tail;
    int cnt;
public:
    MyLinkedQueue(): head(nullptr), tail(nullptr), cnt(0) {}
    void enqueue(int x) {
        QNode* n = new QNode(x);
        if (!tail) head = tail = n;
        else { tail->next = n; tail = n; }
        cnt++;
    }
    void dequeue() {
        QNode* t = head;
        head = head->next;
        if (!head) tail = nullptr;   // ⚠️ 空链表时 tail 也要置空
        delete t; cnt--;
    }
};
```

### 5.2 ⭐ 朴素子串查找

```cpp
int find(const string& text, const string& pat) {

        int j = 0;
        while (j < m && text[i+j] == pat[j]) j++;
        if (j == m) return i;
    }
    return -1;
}
// 时间 O(n*m)
```

```cpp
}
```

## 六、树（二叉树 / BST / AVL）⭐⭐⭐（必考大题）

### 6.1 树的基本术语（必背）

- **根、父、子、兄弟**：唯一根，无父节点
- **叶子**：度 = 0 的节点
- **度**：子节点个数
- **高度 h**：从根到叶子的最长路径长度（单节点高度 = 0）
- **深度**：根到节点的路径长度

### 6.2 二叉树性质（必背，年年考）

- **第 i 层最多 `2^(i-1)` 个节点**（i ≥ 1）
- **高度 h 的二叉树最多 `2^(h+1) - 1` 个节点**（h 从 0 起）
- **⭐ 叶子数 = 度为 2 的节点数 + 1**（`n₀ = n₂ + 1`）
- **总节点数** = `n₀ + n₁ + n₂`
- **完全二叉树高度**：`h = ⌊log₂ n⌋`
- **满二叉树节点数**：`2^(h+1) - 1`（15-16 真卷考过）

### 6.3 二叉树存储

**顺序存储**（完全二叉树）：
- 0-indexed：父 `(i-1)/2`，左子 `2i+1`，右子 `2i+2`
- 1-indexed：父 `i/2`，左子 `2i`，右子 `2i+1`

**链式存储**：
```cpp
struct Node { T data; Node *left, *right; 
              Node(T v): data(v), left(nullptr), right(nullptr) {} };
```

### 6.4 ⭐⭐ 二叉树遍历（每年必考）

#### 递归版（必背）

```cpp
void preorder(Node* p)  { if (!p) return; visit(p); preorder(p->left); preorder(p->right); }
void inorder(Node* p)   { if (!p) return; inorder(p->left); visit(p); inorder(p->right); }
void postorder(Node* p) { if (!p) return; postorder(p->left); postorder(p->right); visit(p); }
```

**记忆**：前 N L R（**根**在前）、中 L N R（**根**在中）、后 L R N（**根**在后）

#### 非递归版（必背）

```cpp
// 前序迭代
vector<int> preorder(Node* root) {
    vector<int> res;
    stack<Node*> st;
    st.push(root);
    while (!st.empty()) {
        Node* p = st.top(); st.pop();
        if (!p) continue;
        res.push_back(p->val);
        st.push(p->right);   // 先右后左，左先出栈
        st.push(p->left);
    }
    return res;
}

// 中序迭代（最常考）
vector<int> inorder(Node* root) {
    vector<int> res;
    stack<Node*> st;
    Node* cur = root;
    while (cur || !st.empty()) {
        while (cur) { st.push(cur); cur = cur->left; }  // 一路向左压栈
        cur = st.top(); st.pop();
        res.push_back(cur->val);   // 出栈才访问
        cur = cur->right;
    }
    return res;
}

// 后序迭代（前序改造：根右左 → 反转）
vector<int> postorder(Node* root) {
    vector<int> res;
    stack<Node*> st;
    st.push(root);
    while (!st.empty()) {
        Node* p = st.top(); st.pop();
        if (!p) continue;
        res.push_back(p->val);
        st.push(p->left);    // 先左后右
        st.push(p->right);
    }
    reverse(res.begin(), res.end());
    return res;
}
```

#### 层序遍历（BFS，必背）

```cpp
void levelOrder(Node* root) {
    if (!root) return;
    queue<Node*> q;
    q.push(root);
    while (!q.empty()) {
        Node* p = q.front(); q.pop();
        visit(p);
        if (p->left)  q.push(p->left);
        if (p->right) q.push(p->right);

}
```

### 6.5 ⭐⭐ 由遍历序列构造二叉树（15-16 真卷考过）

- **中序 + 前序** → 唯一确定
- **中序 + 后序** → 唯一确定
- **前序 + 后序** → ❌ 不能唯一（除非满二叉树）

**思路**：前序/后序找根 → 中序分左右 → 递归

```cpp
Node* buildFromPreIn(vector<int>& pre, vector<int>& in,
                     int pl, int pr, int il, int ir) {
    if (pl > pr) return nullptr;
    Node* root = new Node(pre[pl]);   // 前序首 = 根
    int k = il;
    while (k <= ir && in[k] != pre[pl]) k++;   // 在中序找根
    int leftSize = k - il;
    root->left  = buildFromPreIn(pre, in, pl+1, pl+leftSize, il, k-1);
    root->right = buildFromPreIn(pre, in, pl+leftSize+1, pr, k+1, ir);
    return root;
}
```

### 6.6 ⭐⭐ 二叉搜索树 BST（Lab q209 必背）

**性质**：左 ≤ 根 ≤ 右；**中序遍历得升序**

| 操作 | 平均 | 最坏 |
|---|---|---|
| 查找 | O(log n) | O(n) |
| 插入 | O(log n) | O(n) |
| 删除 | O(log n) | O(n) |

#### ⭐ BST 删除 3 种情况（必背，每年大题）

1. **叶子节点**：直接删
2. **只有一个子树**：子树顶替
3. **两个子树**：用**前驱（左子树最大）** 或 **后继（右子树最小）** 顶替，再删前驱/后继

```cpp
// 用后继（右子树最小）顶替版
Node* findMin(Node* n) { while (n->left) n = n->left; return n; }
Node* deleteNode(Node* root, int key) {
    if (!root) return nullptr;
    if (key < root->val) root->left  = deleteNode(root->left, key);
    else if (key > root->val) root->right = deleteNode(root->right, key);
    else {
        if (!root->left)  return root->right;
        if (!root->right) return root->left;
        Node* succ = findMin(root->right);   // 后继
        root->val = succ->val;
        root->right = deleteNode(root->right, succ->val);
    }
    return root;
}
```

#### ⭐⭐ BST 的 LCA（Lab q216 必背）— **利用有序性**

```cpp
Node* LCA(Node* root, int p, int q) {
    int a = min(p, q), b = max(p, q);
    while (root) {
        if (b < root->val) root = root->left;
        else if (a > root->val) root = root->right;
        else return root;   // a ≤ root ≤ b，root 即 LCA
    }
    return nullptr;
}
```

#### 路径和（Lab q215 必背）

```cpp
bool hasPathSum(Node* root, int sum) {
    if (!root) return false;
    if (!root->left && !root->right) return root->val == sum;
    return hasPathSum(root->left, sum - root->val) ||
           hasPathSum(root->right, sum - root->val);
}
```

### 6.7 ⭐⭐ AVL 树（15-16/18-19 真卷考过插入删除）

**定义**：BST + 左右子树高度差 ≤ 1
**平衡因子 BF** = h(左) - h(右) ∈ {-1, 0, 1}

#### ⭐⭐⭐ 4 种失衡与旋转（必背）

| 失衡 | 插入点位置 | 调整 |
|---|---|---|
| **LL** | 失衡节点的**左孩子的左子树** | **右旋**（单旋）|
| **RR** | 失衡节点的**右孩子的右子树** | **左旋**（单旋）|
| **LR** | 失衡节点的**左孩子的右子树** | **先左后右**（双旋）|
| **RL** | 失衡节点的**右孩子的左子树** | **先右后左**（双旋）|

**记忆口诀**：左左右右分不清？**看新节点在失衡节点的哪一侧**

**右旋**（LL）：
```cpp
Node* rotateRight(Node* y) {
    Node* x = y->left;
    Node* T2 = x->right;
    x->right = y;  y->left = T2;
    updateHeight(y); updateHeight(x);
    return x;
}
```

**左旋**（RR）：对称。

**LR = 先 `root->left = rotateLeft(root->left);` 再 `rotateRight(root)`**

#### 时间复杂度：所有操作 **O(log n)**

### 6.8 红黑树 5 大性质（必背）

1. 节点红/黑
2. 根黑
3. 叶（NIL）黑
4. 红节点的子节点必须黑（不能连续红）
5. 从任一节点到叶的所有路径黑节点数相同

**保证**：树高 ≤ 2·log(n+1)
**应用**：C++ `map`/`set`、Java `TreeMap`、Linux CFS
**vs AVL**：AVL 查找更快；红黑树插入/删除旋转少（更适合频繁修改）

### 6.9 伸展树（Splay Tree）

- 思想：刚访问的节点 → 旋转到根
- 操作：Zig / Zig-Zig / Zig-Zag
- 摊还时间 O(log n)，单次最坏 O(n)

### 6.10 ⭐ Huffman 树（哈夫曼树）

### 7.2 ⭐ B 树（m 阶，18-19 / 15-16 真卷考过插入）

**5 大性质**：
1. 每个节点**最多 m 个孩子**
2. 除根外，每个节点**至少 ⌈m/2⌉ 个孩子**
3. 有 k 个孩子的节点有 k-1 个 key
4. **所有叶节点在同一层**
5. 节点内 key 升序

**高度**：`h ≈ O(log_m N)`，m 越大越矮

#### ⭐ 插入（关键：分裂）

1. 找到应插入的叶子节点
2. 插入关键字（保持有序）
3. **如果节点关键字数 > m-1**：
   - 取中间关键字（⌈n/2⌉）**上提到父节点**
   - 节点分裂为两个：左 [1..⌈n/2⌉-1]，右 [⌈n/2⌉+1..n]
   - 父节点关键字数 +1
   - **如果父节点也超了，递归上提**

**示例（3 阶 B 树，最少 1 key，最多 2 key）**：
```
插入 1, 3, 7, 10, 11, 13, 14：
插入 1, 3:   [1, 3]
插入 7：节点满 → 分裂：3 上提，   [3]
                                       /    \
                                   [1]       [7]
插入 10:                  [3]
                          /    \
                      [1]       [7, 10]
插入 11:                  [3]
                          /    \
                      [1]       [7, 10, 11]  ← 满
插入 13：插入后满 → 分裂：10 上提，  [3, 10]
                                  /    |    \
                              [1]    [7]   [11, 13]
```

#### 删除（关键：合并/借位）

1. 删叶子节点：删除后 ≥ ⌈m/2⌉-1 → 直接删；不够 → 借位或合并
2. 删非叶子节点：用前驱或后继替换，再递归删
3. 借位：父节点关键字下移，兄弟上移
4. 合并：父节点下移 + 兄弟合并 → 父节点关键字数 -1

### 7.3 ⭐ B+ 树 vs B 树（必背）

| | B 树 | B+ 树 |
|---|---|---|
| 数据存储 | 每个节点都存 | **只在叶节点** |
| 叶子节点 | 独立 | **用链表相连** |
| 内部节点关键字 | 数据 | 子树 key 的副本 |
| 范围查询 | 中序遍历 | **链表直接扫** |

**B+ 树优势**：范围查询快、磁盘 IO 少、查找稳定
**应用**：MySQL InnoDB 主键索引

## 八、堆（Heap）⭐⭐⭐

### 8.1 堆的定义

- **完全二叉树** + 父节点 ≥（大根堆）/ ≤（小根堆）子节点
- 大根堆：根 = 最大；小根堆：根 = 最小

### 8.2 数组存储（必背两种约定）

**0-indexed**：
- 父 `(i-1)/2`，左子 `2i+1`，右子 `2i+2`

**1-indexed**（Lab q191/q192 用）：
- 父 `i/2`，左子 `2i`，右子 `2i+1`
- `h[0]` 不用

### 8.3 ⭐⭐ 堆的核心操作（Lab q191/q192 必背）

**siftDown**（下沉，删除/建堆用）：
```cpp
void siftDown(vector<int>& h, int n, int i) {    // 1-indexed, 范围 [1, n]
    while (2*i <= n) {
        int j = 2*i;
        if (j+1 <= n && h[j+1] > h[j]) j++;   // 取较大子
        if (h[i] >= h[j]) break;
        swap(h[i], h[j]);
        i = j;
    }
}
```

**siftUp**（上浮，插入用）：
```cpp
void siftUp(vector<int>& h, int i) {
    while (i > 1 && h[i] > h[i/2]) {
        swap(h[i], h[i/2]);
        i /= 2;
    }
}
```

**buildHeap**（O(n)）：
```cpp
for (int i = n/2; i >= 1; i--) siftDown(h, n, i);
```

### 8.4 ⭐⭐ 堆排序（Lab q191 必背）

```cpp
void heapSort(vector<int>& h) {    // 1-indexed
    int n = (int)h.size() - 1;
    // buildHeap
    for (int i = n/2; i >= 1; i--) siftDown(h, n, i);
    // 输出初始大根堆（Lab 必输出）

    cout << "\n";
    // 排序

        swap(h[1], h[i]);
        siftDown(h, i-1, 1);
    }
    // 输出最终升序

    cout << "\n";
}
```

### 8.5 ⭐ 优先队列（Lab q192 必背）

```cpp
class MaxHeap {
    vector<int> h;   // 1-indexed
    int n;
public:
    MaxHeap(): n(0) { h.push_back(0); }
    void buildHeap(vector<int> a) {
        h.clear(); h.push_back(0);
        for (int x : a) h.push_back(x);
        n = a.size();
        for (int i = n/2; i >= 1; i--) siftDown(h, n, i);   // ⚠️ 注意范围仍是 n
    }
    int popMax() {
        int v = h[1];
        h[1] = h[n--];
        siftDown(h, n, 1);
        return v;
    }
    void insert(int x) { h.push_back(x); n++; siftUp(h, n); }   // 插入用 siftUp
};
```

## 九、哈希表（Hash Table）⭐⭐⭐（必考大题）

### 9.1 基本概念（必背）

- **哈希函数 H(k)**：key → [0, m) 的映射
- **冲突（Collision）**：不同 key → 同一地址
- **同义词（Synonym）**：哈希值相同的不同 key
- **装填因子 α = n/m**（数据数 / 表大小）
- α 越小越好；α > 0.7 时通常扩容

### 9.2 ⭐⭐ 哈希函数（必背）

| 方法 | 公式 | 适用 |
|---|---|---|
| 直接定址法 | `H(k) = a·k + b` | key 分布连续 |
| **除留余数法** | `H(k) = k mod p`（**p 是质数**）| **整数 key，最常用** |
| 数字分析法 | 取 key 的若干位 | 已知 key 分布 |
| 平方取中法 | 取 k² 中间几位 | 关键字位数少 |
| 折叠法 | 分段求和 | 关键字位数多 |
| **BKDRHash** | `h = h*seed + ch`（**seed=31**）| **字符串 key，Lab 必背** |

**为什么 p 必须是质数**：避免 `k ≡ 0 (mod a)` 的所有 key 冲突到 {0, a, 2a...}

**BKDRHash 模板（必背）**：
```cpp
unsigned int BKDR(const string& s, unsigned int size) {
    unsigned int h = 0, seed = 31;   // 31, 131, 1313, 13131...
    for (char c : s) h = (h * seed + (unsigned char)c) % size;
    return h;
}
// 经典版本（题目常用）：
unsigned int BKDRHash(char *str, unsigned int Hash_Size) {
    unsigned int seed = 31, hash = 0;
    while (*str) hash = (hash * seed + (*str++)) % Hash_Size;
    return hash;
}
```

### 9.3 ⭐⭐⭐ 冲突解决方法（必背，每年大题）

#### (1) 链地址法（拉链法，Lab q203 必背）

- 每个桶是链表；插入头插、查找遍历链表
- **优点**：简单、不会"装满"、删除简单
- **缺点**：指针开销、链长 → 性能退化
- 优化：Java 8+ 链表长 > 8 转红黑树

**链地址法 HashMap 模板**：
```cpp
class HashMap {
    static const int M = 1009;   // 质数
    vector<list<pair<string,string>>> buckets;
public:
    HashMap() : buckets(M) {}
    void put(string k, string v) {
        int h = BKDR(k, M);
        for (auto& p : buckets[h])
            if (p.first == k) { p.second = v; return; }
        buckets[h].push_front({k, v});
    }
    string get(string k) {
        int h = BKDR(k, M);
        for (auto& p : buckets[h])
            if (p.first == k) return p.second;
        return "-1";
    }
    void remove(string k) {
        int h = BKDR(k, M);
        auto& lst = buckets[h];
        for (auto it = lst.begin(); it != lst.end(); ++it)
            if (it->first == k) { lst.erase(it); return; }
    }
};
```

#### (2) ⭐ 开放定址法（Lab q202 必背，15-16 真卷考过线性探测）

**线性探测**：`H_i = (H + i) mod m`
- 优点：简单
- 缺点：**堆积（clustering）** 严重

**平方探测**：`H_i = (H + i²) mod m`（**常用 +i²**，探测 H, H+1, H+4, H+9, ...）
- 优点：缓解堆积
- 缺点：不能探遍所有位置（**m = 4k+3 质数才行**）

**平方探测代码模板（Lab q202 必背）**：
```cpp
// key % n 哈希 + 平方探测
int n, m;
vector<int> key(n, -1), val(n, -1);   // -1 = 空

// 插入（重复 key 不覆盖）
for (int i = 0; i < m; i++) {
    int k, v; cin >> k >> v;
    int h = k % n;
    int j = 0, idx;
    for (; j < n; j++) {
        idx = (h + j*j) % n;
        if (key[idx] == -1 || key[idx] == k) break;
    }
    if (key[idx] == -1) { key[idx] = k; val[idx] = v; }
}

// 查询（输出探测路径 + 值；找不到输出 NULL）
auto query = [&](int k) {
    int h = k % n;
    vector<int> path;
    for (int j = 0; j < n; j++) {
        int idx = (h + j*j) % n;
        path.push_back(idx);
        if (key[idx] == -1) return make_pair(path, -1);
        if (key[idx] == k) return make_pair(path, val[idx]);
    }
    return make_pair(path, -1);
};
```

#### (3) 双散列：`H_i = (H + i × H₂) mod m`（最均匀但复杂）

#### (4) 再哈希法 / 公共溢出区（较少用）

### 9.4 ⭐⭐ 哈希表删除问题（必背）

- **链地址法**：直接删节点，O(1)
- **开放定址法**：**不能直接置空**（否则查找链断）→ 用**墓碑（tombstone）**：
  - 状态：`EMPTY` / `OCCUPIED` / `DELETED`
  - 删除：设为 `DELETED`

```cpp
int partition(vector<int>& a, int lo, int hi) {
    int pivot = a[hi];
    int i = lo, j = hi;
    while (i < j) {
        while (i < j && a[i] < pivot) i++;    // 找 ≥ pivot
        while (i < j && a[j] >= pivot) j--;   // 找 < pivot
        if (i < j) swap(a[i], a[j]);
    }
    swap(a[i], a[hi]);    // 轴点就位
    return i;
}

void quicksort(vector<int>& a, int lo, int hi) {
    if (lo >= hi) return;
    int p = partition(a, lo, hi);
    quicksort(a, lo, p-1);
    quicksort(a, p+1, hi);
}
```

### 10.7 ⭐⭐ 归并排序（Lab q189 必背）

```cpp
void mergeSort(vector<int>& a, int l, int r) {
    if (l >= r) return;
    int m = (l + r) / 2;
    mergeSort(a, l, m);
    mergeSort(a, m+1, r);
    merge(a, l, m, r);
}
void merge(vector<int>& a, int l, int m, int r) {
    vector<int> tmp(r-l+1);
    int i = l, j = m+1, k = 0;
    while (i <= m && j <= r) tmp[k++] = (a[i] <= a[j]) ? a[i++] : a[j++];
    while (i <= m) tmp[k++] = a[i++];
    while (j <= r) tmp[k++] = a[j++];
    for (int p = 0; p < k; p++) a[l+p] = tmp[p];
}
```

```cpp
void bfs(int s) {
```

- 行宽 60，位置 1..60（**1-indexed**）
- n=0 输出初始行
- 每代用新数组（双 buffer）

```cpp
const int N = 60;
vector<int> cur(N+5, 0), nxt(N+5, 0);
int p;
while (cin >> p && p != -1) cur[p] = 1;
int g; cin >> g;
for (int step = 1; step <= g; step++) {
    for (int i = 1; i <= N; i++) {
        int nb = cur[i-2] + cur[i-1] + cur[i+1] + cur[i+2];   // 4 个邻居
        if (cur[i] == 1)
            nxt[i] = (nb == 2 || nb == 4) ? 1 : 0;
        else
            nxt[i] = (nb == 2 || nb == 3) ? 1 : 0;
    }
    cur = nxt;
}
for (int i = 1; i <= N; i++) cout << (cur[i] ? '*' : '-');
```

### 14.2 二维 Life（Lab q134）

**规则 B3/S23**：
- 活细胞：邻居 0/1 → 死；2/3 → 活；≥4 → 死
- 死细胞：邻居 = 3 → 活

**关键**：
- 网格 20×60
- 8 邻居（**8 方向**）
- 死 → `-`，活 → `*`

```cpp
const int R = 20, C = 60;
vector<vector<int>> cur(R, vector<int>(C, 0));
int x, y;
while (cin >> x >> y) {
    if (x == -1 && y == -1) break;
    cur[x][y] = 1;
}
int g; cin >> g;
for (int step = 0; step < g; step++) {
    vector<vector<int>> nxt(R, vector<int>(C, 0));
    for (int i = 0; i < R; i++)
        for (int j = 0; j < C; j++) {
            int nb = 0;
            for (int di = -1; di <= 1; di++)
                for (int dj = -1; dj <= 1; dj++) {
                    if (di == 0 && dj == 0) continue;
                    int ni = i+di, nj = j+dj;
                    if (ni>=0 && ni<R && nj>=0 && nj<C)
                        nb += cur[ni][nj];
                }
            if (cur[i][j]) nxt[i][j] = (nb==2 || nb==3);
            else nxt[i][j] = (nb==3);
        }
    cur = nxt;
}
```

### 14.3 ⭐ Magic Square（Lab q135）

```cpp
int n; cin >> n;
vector<vector<int>> a(n, vector<int>(n, 0));
int r = 0, c = n/2;        // 1 放 (0, n/2)
a[r][c] = 1;
for (int k = 2; k <= n*n; k++) {
    int nr = (r - 1 + n) % n;   // 向上
    int nc = (c + 1) % n;       // 向右
    if (a[nr][nc] != 0) {       // 撞占位 → 正下方
        nr = (r + 1) % n;
        nc = c;
    }
    a[nr][nc] = k;
    r = nr; c = nc;
}
// 输出：数字之间两个空格，行末无空格
for (int i = 0; i < n; i++) {
    for (int j = 0; j < n; j++) {
        if (j) cout << "  ";
        cout << a[i][j];
    }
    cout << "\n";
}
```

### 16.1 公式速查

### 16.2 易错点 Top 10（必看）

1. **二分边界**：`while (l <= r)` vs `while (l < r)` 用错 → 选错就死循环或漏解
2. **环形队列判空判满**：用 count 变量最稳；牺牲存储格易混
3. **链表反转**：必须先存 `nxt = cur->next`，否则丢链
4. **BST 删除双子树**：用前驱（左子树最大）或后继（右子树最小）顶替，再删前驱/后继
5. **中缀表达式负数**：开头 / `(` 后 / 运算符后 → 补 0（如 `-x` 视为 `0-x`）
6. **输出格式**：行内空格分隔，行末无空格；中间结果在**特定长度**时输出
7. **输入 `-1` 结束**：循环里 break，**别忘了 cin.ignore()**
8. **堆 1-indexed vs 0-indexed**：父子关系公式差 1（1-indexed：父 `i/2`；0-indexed：父 `(i-1)/2`）
9. **哈希表删除**：链地址法直接删；开放定址要墓碑（`DELETED`）
10. **AVL 旋转**：看新节点在失衡节点的**哪一侧**（左左→右旋；左右→先左后右）
11. **Dijkstra 不能有负权边**（但 Bellman-Ford / SPFA 可以）
12. **排序稳定性**：选择/希尔/快速/堆 都不稳定；其余稳定

### 16.3 Lab 输入输出坑位速查

| 坑 | 避坑 |
|---|---|
| `mid*mid` 溢出 int | 用 `long long` |
| 环形队列 front==rear | 用 count 变量更稳 |
| 链表反转没保存 next | 一定要先存 `nxt = cur->next` |
| BST 删除双子树 | 用前驱或后继，再删前驱/后继 |
| 中缀表达式负数 | 补 0；浮点保留 3 位 |
| 输出空格 | 行内空格分隔，行末不空格 |
| 输入 `-1` 结束 | 一定要 break |
| 排序中间结果 | 在插入/选择次数 = n/2 时输出 |
| 堆 1-indexed vs 0-indexed | 父子公式差 1 |
| 哈希表删除 | 链地址直接删；开放定址墓碑 |
| 哈希表取模 | 质数 p |
| Floyd 二维数组 | `dist[i][j] = min(dist[i][j], dist[i][k]+dist[k][j])` |
| DFS 递归深度 | 树深 ≤ 10⁴，递归可能栈溢出（Lab 不涉及超大）|
| AVL 旋转 | 失衡点在祖先，回溯到第一个失衡的调整 |
