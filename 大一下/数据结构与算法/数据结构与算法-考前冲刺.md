### 1.2 Lab01-题目1：One-Dimensional Life（q136, 中等, 50 分）

**题意**：一维格子，每个活细胞有 4 个邻居（左右各 1、2）。规则：
- 死细胞：邻居数 = 2 或 3 → 复活
- 活细胞：邻居数 = 0/1/3 → 死
- 活细胞：邻居数 = 2 或 4 → 活
- 总数 < 60。

**输入**：活细胞位置（1..60），以 `-1` 结束；n 代。
**输出**：第 n 代行（`-` 空 / `*` 活）。

**关键点**：
- 行宽 60，位置 1..60（不是 0..59）
- n=0 输出初始行
- 每代用新数组存，邻居 = 自身 + 左右各 1、2（共 4 个）

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    const int N = 60;
    vector<int> cur(N+5, 0), nxt(N+5, 0);
    int p;
    while (cin >> p && p != -1) cur[p] = 1;   // 1-indexed
    int g; cin >> g;
    for (int step = 1; step <= g; step++) {
        for (int i = 1; i <= N; i++) {
            int nb = cur[i-2] + cur[i-1] + cur[i+1] + cur[i+2];
            if (cur[i] == 1)
                nxt[i] = (nb == 2 || nb == 4) ? 1 : 0;
            else
                nxt[i] = (nb == 2 || nb == 3) ? 1 : 0;
        }
        cur = nxt;
    }
    for (int i = 1; i <= N; i++) cout << (cur[i] ? '*' : '-');
    cout << '\n';
    return 0;
}
```

### 1.3 Lab01-题目2：Magic Square（q135, 中等, 50 分）

**题意**：奇数 n（1..101）生成 n×n 魔方阵，方法：1 放顶行中间；后续数字沿右上对角走；撞顶/撞右则环绕；撞到已占位则正下方一格。

**关键点**：
- 行 = `(row - 1 + n) % n`，列 = `(col + 1) % n`
- 撞占位 → 行 = `(row + 1) % n`，列不变
- 输出：数字之间**两个空格**，行末无空格

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n; cin >> n;
    vector<vector<int>> a(n, vector<int>(n, 0));
    int r = 0, c = n/2;        // 1 放 (0, n/2)
    a[r][c] = 1;
    for (int k = 2; k <= n*n; k++) {
        int nr = (r - 1 + n) % n;
        int nc = (c + 1) % n;
        if (a[nr][nc] != 0) {  // 撞占位
            nr = (r + 1) % n;
            nc = c;
        }
        a[nr][nc] = k;
        r = nr; c = nc;
    }
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            if (j) cout << "  ";
            cout << a[i][j];
        }
        cout << '\n';
    }
    return 0;
}
```

### 1.4 Lab21：二维 Life Game（q134, 中等, 100 分）

**题意**：20×60 棋盘；8 邻居；规则 B3/S23。
- 活细胞：邻居 0/1 → 死；2/3 → 活；≥4 → 死
- 死细胞：邻居 = 3 → 活

**关键点**：
- 输出范围 20×60 全网格
- 死 → `-`，活 → `*`
- 每代双 buffer 切换

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    const int R = 20, C = 60;
    vector<vector<int>> cur(R, vector<int>(C, 0));
    int x, y;
    while (cin >> x >> y) {
        if (x == -1 && y == -1) break;
        cur[x][y] = 1;   // 题目说 (x, y) 行列都从 1 开始
    }
    int g; cin >> g;
    for (int step = 0; step < g; step++) {
        vector<vector<int>> nxt(R, vector<int>(C, 0));
        for (int i = 0; i < R; i++) {
            for (int j = 0; j < C; j++) {
                int nb = 0;
                for (int di = -1; di <= 1; di++)
                    for (int dj = -1; dj <= 1; dj++) {
                        if (di == 0 && dj == 0) continue;
                        int ni = i+di, nj = j+dj;
                        if (ni>=0 && ni<R && nj>=0 && nj<C)
                            nb += cur[ni][nj];
                    }
                if (cur[i][j]) nxt[i][j] = (nb==2 || nb==3);
                else nxt[i][j] = (nb==3);
            }
        }
        cur = nxt;
    }
    for (int i = 0; i < R; i++) {
        for (int j = 0; j < C; j++)
            cout << (cur[i][j] ? '*' : '-');
        cout << '\n';
    }
    return 0;
}
```

## 第 2 章 栈

### 2.1 理论速记

- **栈（Stack）**：LIFO（后进先出），只在**一端**做插入和删除。
- **基本操作**：
  - `push(x)` 入栈
  - `pop()` 出栈（不返回）
  - `top()` / `peek()` 访问栈顶
  - `size()` / `empty()` / `clear()` / `full()`（顺序栈有容量限制时）
- **顺序栈**（数组实现）：
  - 用数组 + `top` 指针（指向栈顶元素）
  - 初始化 `top = -1`
  - `push`：`data[++top] = x`
  - `pop`：`return data[top--]`
  - `full`：`top == capacity-1`
  - 时间 O(1)，空间 O(n)
- **链栈**（单链表实现）：
  - 头插法（push 头部 O(1)，pop 头部 O(1)）
  - 不需要判满（除非系统内存耗尽）
  - 头节点 `next` 即栈顶
- **栈的应用**：
  - 括号匹配
  - 表达式求值（中缀 → 后缀 / 波兰式）
  - 递归调用栈（递归本质）
  - 深度优先搜索（DFS）
  - 函数调用 / 撤销操作

### 2.2 Lab23：栈的顺序实现（q137, 困难, 100 分）

**题意**：实现 Stack（顺序存储），支持 pop/push/top/size/clear/full；用其将输入整数序列**逆序输出**。

**关键点**：
- 容量设为 100（题目说 <100）
- `top` 初始 -1
- 输入 `-1` 结束

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

class Stack {
    static const int MAXN = 105;
    int data[MAXN];
    int top;
public:
    Stack(): top(-1) {}
    bool empty() const { return top == -1; }
    bool full()  const { return top == MAXN-1; }
    int  size()  const { return top + 1; }
    void clear() { top = -1; }
    bool push(int x) { if (full()) return false; data[++top] = x; return true; }
    bool pop()        { if (empty()) return false; top--; return true; }
    int  peek() const { return data[top]; }
};

int main() {
    Stack s;
    int x; 
    while (cin >> x && x != -1) s.push(x);
    bool first = true;
    while (!s.empty()) {
        if (!first) cout << ' ';
        first = false;
        cout << s.peek();
        s.pop();
    }
    cout << '\n';
    return 0;
}
```

### 2.3 Lab05-链式存储的栈（q158, 困难, 100 分）

**题意**：实现 `MLinkedStack`（链栈），支持 pop/push/top/size/clear/full；逆序输出。

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct Node { int val; Node* next; Node(int v): val(v), next(nullptr) {} };

class MLinkedStack {
    Node* topNode;
    int   cnt;
public:
    MLinkedStack(): topNode(nullptr), cnt(0) {}
    ~MLinkedStack() { clear(); }
    bool empty() const { return cnt == 0; }
    bool full()  const { return false; }    // 链式永不满
    int  size()  const { return cnt; }
    void clear() { while (topNode) { Node* t = topNode; topNode = topNode->next; delete t; } cnt = 0; }
    void push(int x) { Node* n = new Node(x); n->next = topNode; topNode = n; cnt++; }
    void pop()        { Node* t = topNode; topNode = topNode->next; delete t; cnt--; }
    int  top() const  { return topNode->val; }
};

int main() {
    MLinkedStack s;
    int x; while (cin >> x && x != -1) s.push(x);
    bool first = true;
    while (!s.empty()) {
        if (!first) cout << ' ';
        first = false;
        cout << s.top();
        s.pop();
    }
    cout << '\n';
}
```

### 2.4 Lab02-题目1：10 以内的后序波兰式求解（q139, 中等, 33 分）

**题意**：后缀表达式（操作数在前），数字 0..9，遇运算符弹两个数计算再压回。
- 输入 `23+6*` → `(2+3)*6 = 30`
- 非法 → 输出 `none`

**关键点**：
- 表达式是**一整行**，先 `getline` 再处理
- 数字字符 `'0'..'9'` 压栈
- 运算符 `+ - * /`：弹 b（后弹的右操作数），再弹 a，结果 `a op b` 压回
- 最终栈 size==1 才合法

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    string s; getline(cin, s);
    vector<int> st;
    for (char c : s) {
        if (c >= '0' && c <= '9') st.push_back(c - '0');
        else if (c=='+'||c=='-'||c=='*'||c=='/') {
            if (st.size() < 2) { cout << "none\n"; return 0; }
            int b = st.back(); st.pop_back();
            int a = st.back(); st.pop_back();
            int r = 0;
            if (c=='+') r = a+b;
            else if (c=='-') r = a-b;
            else if (c=='*') r = a*b;
            else r = a/b;                 // 整数除
            st.push_back(r);
        }
    }
    if (st.size() == 1) cout << st.back() << "\n";
    else                cout << "none\n";
}
```

### 2.5 Lab02-题目2：出入栈序列判断（q141, 中等, 33 分）

**题意**：I 入栈，O 出栈；初态终态都空。判断 n 个序列是否合法。
- 思路：模拟一遍，维护栈大小 `sz`。
  - I → `sz++`
  - O → `sz--`（出栈前 `sz` 必须 > 0）
  - 结束：合法当且仅当 `sz == 0`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    int n; cin >> n;
    while (n--) {
        string s; cin >> s;
        int sz = 0; bool ok = true;
        for (char c : s) {
            if (c == 'I') sz++;
            else { if (sz == 0) { ok = false; break; } sz--; }
        }
        if (sz != 0) ok = false;
        cout << (ok ? "true" : "false") << "\n";
    }
}
```

### 2.6 Lab02-题目3：程序代码中的括号匹配（q142, 中等, 34 分）

**题意**：判断三类括号 `() [] {}` 是否匹配；**注释不参与**。
- 多行注释 `/* ... */`
- 单行注释 `// ...`

**关键点**：
- 维护一个 bool `inBlock`（多行注释内）和 `inLine`（单行注释内）
- 遇到 `/*` 且不在注释中 → `inBlock = true`
- 遇到 `*/` 且 `inBlock` → `inBlock = false`
- `//` 触发 `inLine = true` 直到换行
- 字符计数 = 各类括号总数，匹配 = 栈最终空

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    string all, line;
    // 读若干行直到 EOF
    string chunk;
    bool first = true;
    while (getline(cin, line)) {
        if (!first) all += '\n';
        all += line;
        first = false;
    }

    vector<char> st;
    int cnt = 0;
    bool inBlock = false, inLine = false;
    bool ok = true;
    for (size_t i = 0; i < all.size(); i++) {
        char c = all[i], nxt = (i+1<all.size()? all[i+1]:'\0');
        if (inLine) { if (c=='\n') inLine = false; continue; }
        if (inBlock) { if (c=='*' && nxt=='/') { inBlock = false; i++; } continue; }
        if (c=='/' && nxt=='/') { inLine = true; i++; continue; }
        if (c=='/' && nxt=='*') { inBlock = true; i++; continue; }
        if (c=='('||c=='['||c=='{') { st.push_back(c); cnt++; }
        else if (c==')'||c==']'||c=='}') {
            cnt++;
            char m = (c==')')?'(':(c==']')?'[':'{';
            if (st.empty() || st.back()!=m) ok = false;
            else st.pop_back();
        }
    }
    if (!st.empty()) ok = false;
    cout << cnt << ' ' << (ok ? "yes" : "no") << "\n";
}
```

### 2.7 Lab20-题目1：中序波兰式求解（q143, 简单, 100 分）

**题意**：中缀表达式求值，操作符 `+ - * /`，含括号；可能含浮点、负数。整数直接输出；浮点保留 3 位小数。

**关键点**：
- **双栈法**（操作数栈 + 操作符栈）是最稳的写法
- 优先级：`(` < `+ -` < `* /` < `+ -(一元)`？——课程题不需要一元运算符
- 遇到 `(` 入栈；遇到 `)` 弹到 `(`；数字（含负数、小数）压栈
- 优先级比较：当前 op ≤ 栈顶 op 时，先计算栈顶

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int pri(char c) {
    if (c == '+' || c == '-') return 1;
    if (c == '*' || c == '/') return 2;
    return 0;
}
void calc(stack<double>& num, char op) {
    double b = num.top(); num.pop();
    double a = num.top(); num.pop();
    double r = 0;
    if (op=='+') r=a+b; else if (op=='-') r=a-b;
    else if (op=='*') r=a*b; else r=a/b;
    num.push(r);
}
int main() {
    string s; getline(cin, s);
    stack<double> num;
    stack<char> op;
    int n = s.size();
    for (int i = 0; i < n; ) {
        char c = s[i];
        if (c==' ' || c=='\t') { i++; continue; }
        if (c=='(') { op.push(c); i++; }
        else if (c==')') {
            while (!op.empty() && op.top()!='(') calc(num, op.top()), op.pop();
            if (!op.empty()) op.pop();   // 弹出 (
            i++;
        }
        else if (c=='+'||c=='-'||c=='*'||c=='/') {
            // 处理一元负号：表达式开头 或 上一个是 ( 或 运算符
            if (c=='-' && (i==0 || s[i-1]=='(' || strchr("+-*/", s[i-1]))) {
                // 负数: 直接读入 -<数字>
                // 简化：把 -x 当 0-x
                num.push(0);
            }
            while (!op.empty() && op.top()!='(' && pri(op.top()) >= pri(c))
                calc(num, op.top()), op.pop();
            op.push(c); i++;
        }
        else { // 数字
            int j = i;
            while (j<n && (isdigit(s[j]) || s[j]=='.')) j++;
            num.push(stod(s.substr(i, j-i)));
            i = j;
        }
    }
    while (!op.empty()) calc(num, op.top()), op.pop();
    double ans = num.top();
    if (ans == (long long)ans) cout << (long long)ans << "\n";
    else                        printf("%.3f\n", ans);
}
```

## 第 3 章 队列

### 3.1 理论速记

- **队列（Queue）**：FIFO（先进先出），一端入（rear/tail）一端出（front/head）。
- **基本操作**：
  - `enqueue(x)` / `push(x)` 入队（rear）
  - `dequeue()` / `pop()` 出队（front）
  - `front()` / `peek()` 访问队首
  - `rear()` / `back()` 访问队尾
  - `size()` / `empty()` / `full()` / `clear()`
- **顺序队列的假溢出**：普通数组实现的队列多次入队出队后，front 往前走，rear 触底但前面空着。
- **环形队列（循环队列）**：用模运算 `idx % capacity` 复用空间。
  - 判空：`front == rear`
  - 判满：`(rear+1) % capacity == front`（**牺牲一个存储单元**）；或用 `count` 变量记数。
  - 入队：`rear = (rear+1) % cap; data[rear] = x;`
  - 出队：`front = (front+1) % cap;`
- **链式队列**：单链表 + 头尾指针（头出尾入），O(1) 操作，**永不满**。
- **双端队列（Deque）**：两端都能入出。
  - 既是栈也是队列的泛化。
- **队列的应用**：
  - BFS（广度优先搜索）
  - 任务调度 / 消息队列
  - 缓冲管理
  - 操作系统进程就绪队列

### 3.2 Lab24：环形队列（q146, 中等, 100 分）

**题意**：实现 `CirQueue`（顺序环形），用其将输入整数**顺序输出**。

**关键点**：
- 用 `count` 变量记录元素个数（比牺牲存储格更直观）
- 容量 105

**C++ 模板**：

```cpp
#include <bits/stdc++.h>
using namespace std;

class CirQueue {
    static const int MAXN = 105;
    int data[MAXN];
    int frontIdx, rearIdx, cnt;
public:
    CirQueue(): frontIdx(0), rearIdx(-1), cnt(0) {}
    bool empty() const { return cnt == 0; }
    bool full()  const { return cnt == MAXN; }
    int  size()  const { return cnt; }

    int  front() const  { return data[frontIdx]; }
```

### 4.2 Lab06-题目1：链式存储的队列（q159, 困难, 50 分）

**题意**：实现 `MyLinkedQueue`（链式），用其将输入整数**顺序输出**。

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct QNode { int val; QNode* next; QNode(int v): val(v), next(nullptr) {} };

class MyLinkedQueue {
    QNode *head, *tail;     // head 出队端，tail 入队端
    int cnt;
public:
    MyLinkedQueue(): head(nullptr), tail(nullptr), cnt(0) {}
    ~MyLinkedQueue() { while (head) { QNode* t = head; head = head->next; delete t; } }
    bool empty() const { return cnt == 0; }
    bool full()  const { return false; }
    int  size()  const { return cnt; }
    void clear() { this->~MyLinkedQueue(); head=tail=nullptr; cnt=0; }
    void enqueue(int x) {
        QNode* n = new QNode(x);
        if (!tail) head = tail = n;
        else { tail->next = n; tail = n; }
        cnt++;
    }
    void dequeue() {
        QNode* t = head; head = head->next;
        if (!head) tail = nullptr;
        delete t; cnt--;
    }
    int front() const { return head->val; }
};

int main() {
    MyLinkedQueue q;
    int x; while (cin >> x && x != -1) q.enqueue(x);
    bool first = true;
    while (!q.empty()) {
        if (!first) cout << ' ';
        first = false;
        cout << q.front();
        q.dequeue();
    }
    cout << '\n';
}
```

### 4.3 Lab04-题目3：约瑟夫问题（q155, 中等, 20 分）

**题意**：n 个人（2..60）围圈，从 1 报数到 m 出列；n=8, m=3 → 输出 7。

**关键点**：
- 循环单链表（尾→头）
- `cur` 每次前进 m-1 步后删除 next
- 终止条件：链表剩 1 个节点

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct CNode { int id; CNode* next; CNode(int i): id(i), next(nullptr) {} };

int main() {
    int n, m; cin >> n >> m;
    CNode* head = new CNode(1);
    CNode* pre = head;
    for (int i = 2; i <= n; i++) { pre->next = new CNode(i); pre = pre->next; }
    pre->next = head;   // 成环

    CNode* cur = head;
    while (cur->next != cur) {       // 剩 1 个
        for (int k = 1; k < m; k++) cur = cur->next;
        CNode* del = cur->next;       // cur 报 m-1，del 报 m
        cur->next = del->next;
        delete del;
    }
    cout << cur->id << "\n";
}
```

### 4.4 Lab06-题目2：多项式的运算（q160, 中等, 50 分）

**题意**：多项式按指数降序输入，系数 + 指数成对，遇 `0 0` 结束。运算符 `+ - *`（乘法选做）。
- 输出 `5X^2` 这种格式，常数项写 `5`

**关键点**：
- 数据结构：链表（每项 `{coef, exp, next}`），按指数**降序**
- 加/减：双指针归并（指数大者先入）；相等指数则合并系数（减法取负）；系数为 0 则跳过
- 乘：每个 A 项 × 整个 B → 多项式，再加到结果中（外层加法用前面那个函数）
- 输出注意：
  - 第一项不带 `+` 号
  - 系数为 ±1 且指数 >0 时省略 1
  - 指数 = 0 不写 `X^0`；指数 = 1 写 `X`；否则 `X^e`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct Term { int coef, exp; Term* next; Term(int c,int e): coef(c),exp(e),next(nullptr) {} };

// 释放
void freePoly(Term* p) { while (p) { Term* t = p; p = p->next; delete t; } }

// 加法：a + sign*b（sign=+1 或 -1）
Term* polyAdd(Term* a, Term* b, int sign = 1) {
    Term dummy(0,0); Term* tail = &dummy; dummy.next = nullptr;
    while (a && b) {
        if (a->exp > b->exp) {
            tail->next = new Term(a->coef, a->exp); tail = tail->next; a = a->next;
        } else if (a->exp < b->exp) {
            tail->next = new Term(b->coef * sign, b->exp); tail = tail->next; b = b->next;
        } else {
            int c = a->coef + b->coef * sign;
            if (c != 0) { tail->next = new Term(c, a->exp); tail = tail->next; }
            a = a->next; b = b->next;
        }
    }
    while (a) { tail->next = new Term(a->coef, a->exp); tail = tail->next; a = a->next; }
    while (b) { tail->next = new Term(b->coef * sign, b->exp); tail = tail->next; b = b->next; }
    tail->next = nullptr;
    return dummy.next;
}

Term* polyMul(Term* a, Term* b) {
    Term* res = nullptr;
    for (Term* pa = a; pa; pa = pa->next) {
        Term* row = nullptr;
        for (Term* pb = b; pb; pb = pb->next) {
            Term* n = new Term(pa->coef * pb->coef, pa->exp + pb->exp);
            n->next = row; row = n;        // 头插，无所谓顺序
        }
        Term* tmp = polyAdd(res, row, 1);
        freePoly(res); freePoly(row);
        res = tmp;
    }
    return res;
}

// 读入
Term* readPoly() {
    Term dummy(0,0); Term* tail = &dummy;
    while (true) {
        int c, e; cin >> c >> e;
        if (c == 0 && e == 0) break;
        tail->next = new Term(c, e); tail = tail->next;
    }
    tail->next = nullptr;
    return dummy.next;
}

void printPoly(Term* p) {
    if (!p) { cout << "0\n"; return; }
    bool first = true;
    while (p) {
        int c = p->coef, e = p->exp;
        if (!first) cout << (c >= 0 ? "+" : "");
        if (e == 0) cout << c;
        else {
            if (c == 1) cout << "X";
            else if (c == -1) cout << "-X";
            else cout << c << "X";
            if (e != 1) cout << "^" << e;
        }
        first = false;
        p = p->next;
    }
    cout << "\n";
}

int main() {
    char op; cin >> op;
    Term *p1 = readPoly(), *p2 = readPoly();
    Term* res = nullptr;
    if (op == '+') res = polyAdd(p1, p2, 1);
    else if (op == '-') res = polyAdd(p1, p2, -1);
    else if (op == '*') res = polyMul(p1, p2);
    printPoly(res);
}
```

## 第 5 章 递归

### 5.1 理论速记

- **递归三要素**：
  1. **终止条件（base case）**：不能递归到底的最小问题。
  2. **递归假设**：假设子问题已解。
  3. **递归方程**：用子问题构造原问题解。
- **递归 vs 迭代**：
  - 递归代码短、可读性强。
  - 递归有函数调用开销（栈），可能栈溢出。
  - 尾递归可被编译器优化为循环。
- **递归复杂度分析**（主定理）：
  - `T(n) = aT(n/b) + O(n^d)`
  - `d > log_b a` → `O(n^d)`
  - `d = log_b a` → `O(n^d log n)`
  - `d < log_b a` → `O(n^log_b a)`
- **经典递归问题**：
  - 阶乘 / 斐波那契 / 整数划分
  - 全排列（n!）
  - 组合数
  - 汉诺塔（Hanoi）
  - 八皇后 / N 皇后
  - 迷宫 / 单词拆分
  - 树的遍历（前/中/后/层）
- **递归设计步骤**：
  1. 找重复（子问题）
  2. 找变化（参数）
  3. 找边界（终止）

### 5.5 Lab07-题目2：A Maze Problem（q164, 中等, 50 分）

**题意**：12×12 迷宫，O/W/E，3 个起点，问到出口的最短路径长度（起终点都算入），不可达输出 -1。

**关键点**：
- **BFS 求最短路**（不是 DFS，DFS 求的是任意路径）
- `dirs = {(1,0),(-1,0),(0,1),(0,-1)}`
- `O=0, W=1, E=2`；E 是终点
- BFS 起点 = (x, y)（注意题目说 x 是纵坐标，y 是横坐标）
- 距离 = `dist[x][y]`，终点 `dist[ex][ey]`
- 起始结束都算：长度 = 步数 + 1

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    const int N = 12;
    int grid[N][N];
    for (int i = 0; i < N; i++)
        for (int j = 0; j < N; j++) {
            char c; cin >> c;
            grid[i][j] = (c=='W'?1:(c=='E'?2:0));
        }
    // 找 E
    int ex=-1, ey=-1;
    for (int i=0;i<N;i++) for (int j=0;j<N;j++) if (grid[i][j]==2){ ex=i; ey=j; }

    // 3 个起点
    for (int k = 0; k < 3; k++) {
        int x, y; cin >> x >> y;     // (x,y) = (row, col)
        if (k) cout << ' ';
        if (grid[x][y] == 1) { cout << -1; continue; }

        int dist[N][N];
        for (int i=0;i<N;i++) for (int j=0;j<N;j++) dist[i][j] = -1;
        queue<pair<int,int>> q;
        dist[x][y] = 0; q.push({x,y});
        int d4[4][2] = {{-1,0},{1,0},{0,-1},{0,1}};
        while (!q.empty()) {
            auto [i,j] = q.front(); q.pop();
            for (auto& d : d4) {
                int ni = i+d[0], nj = j+d[1];
                if (ni<0||ni>=N||nj<0||nj>=N) continue;
                if (grid[ni][nj]==1) continue;
                if (dist[ni][nj]!=-1) continue;
                dist[ni][nj] = dist[i][j] + 1;
                q.push({ni,nj});
            }
        }
        if (dist[ex][ey] == -1) cout << -1;
        else cout << dist[ex][ey] + 1;     // 起终点都算
    }
    cout << "\n";
}
```

## 第 6 章 列表 List

### 6.1 理论速记

- **List ADT**：
  - 顺序表（数组实现）：查找 O(1)，插入/删除 O(n)
  - 单链表：插入/删除 O(1)（已知位置），查找 O(n)
  - 双链表：可双向遍历
  - 循环链表：尾指头
- **基本操作**（ADT 接口）：
  - `Insert(pos, x)` 在 pos 前插入
  - `Remove(pos)` 删除 pos 处
  - `Retrieve(pos)` 取 pos 处
  - `Replace(pos, x)` 替换
  - `Traverse()` 遍历
  - `Size()` / `Full()` / `Empty()`

### 6.8 Lab08-题目2：List 的单链表实现（q168, 困难, 50 分）

**题意**：单链表实现通用 List（含最后访问位置记录），N 次插入、M 次删除、P 次替换；输出最终序列。

**关键点**：
- 哑节点 + 头插/尾插
- 最后访问位置 = 索引 → 节点指针 + 索引（更新时机：每次 insert/remove/replace 后更新）
- 输入：4 行；操作均以 `-1` 或 `-1 -1` 结束

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct LNode { int v; LNode* next; LNode(int x): v(x), next(nullptr) {} };

class MyList {
    LNode* head;      // dummy
    int sz;
    int lastPos;      // 最后访问位置（0-indexed）
    LNode* lastPtr;   // 指向 lastPos 节点
    void updateLast(int pos) {
        lastPos = pos;
        LNode* p = head->next;
        for (int i = 0; i < pos; i++) p = p->next;
        lastPtr = p;
    }
public:
    MyList(): head(new LNode(0)), sz(0), lastPos(-1), lastPtr(nullptr) {}
    ~MyList() { while (head) { LNode* t = head; head = head->next; delete t; } }
    int size() const { return sz; }
    bool insert(int pos, int x) {
        if (pos < 0 || pos > sz) return false;
        LNode* pre = head;
        for (int i = 0; i < pos; i++) pre = pre->next;
        LNode* n = new LNode(x);
        n->next = pre->next; pre->next = n; sz++;
        updateLast(pos);
        return true;
    }
    bool remove(int pos) {
        if (pos < 0 || pos >= sz) return false;
        LNode* pre = head;
        for (int i = 0; i < pos; i++) pre = pre->next;
        LNode* t = pre->next; pre->next = t->next; delete t; sz--;
        if (pos < sz) updateLast(pos); else { lastPos = -1; lastPtr = nullptr; }
        return true;
    }
    bool replace(int pos, int x) {
        if (pos < 0 || pos >= sz) return false;
        LNode* p = head->next;
        for (int i = 0; i < pos; i++) p = p->next;
        p->v = x; updateLast(pos); return true;
    }
    void traverse() const {
        LNode* p = head->next;
        bool first = true;
        while (p) {
            if (!first) cout << ' ';
            first = false; cout << p->v;
            p = p->next;
        }
        cout << "\n";
    }
};

int main() {
    MyList L;
    int x;
    // 第 1 行：原序列
    while (cin >> x && x != -1) L.insert(L.size(), x);
    // 吃掉换行（如果有）
    // 第 2 行：插入
    while (cin >> x) {
        int a = x, b; cin >> b;
        if (a == -1 && b == -1) break;
        L.insert(a, b);
    }
    // 第 3 行：删除
    while (cin >> x) {
        if (x == -1) break;
        L.remove(x);
    }
    // 第 4 行：替换
    while (cin >> x) {
        int d = x, e; cin >> e;
        if (d == -1 && e == -1) break;
        L.replace(d, e);
    }
    L.traverse();
}
```

### 6.9 Lab09-题目1：List 的双链表实现（q169, 困难, 33 分）

**题意**：双链表实现 List，记录最后访问位置，输出正序和逆序。

**关键点**：
- 双链表每个节点 `{prev, next, val}`
- 维护 `head`（哑）和 `tail`
- Insert：在 pos 前插入；同步维护 `tail`
- Remove：删 pos 节点；tail 要更新
- 输出正序 + 逆序

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct DNode { int v; DNode *prev, *next; DNode(int x): v(x), prev(nullptr), next(nullptr) {} };

class MyList {
    DNode *head, *tail;     // 双向哑节点简化
    int sz;
    void link(DNode* a, DNode* b) { if (a) a->next = b; if (b) b->prev = a; }
public:
    MyList() {
        head = new DNode(0); tail = new DNode(0);
        link(head, tail); sz = 0;
    }
    ~MyList() {
        DNode* p = head;
        while (p) { DNode* t = p; p = p->next; delete t; }
    }
    int size() const { return sz; }
    bool insert(int pos, int x) {
        if (pos < 0 || pos > sz) return false;
        DNode* pre = head;
        for (int i = 0; i < pos; i++) pre = pre->next;
        DNode* n = new DNode(x);
        link(n, pre->next); link(pre, n); sz++; return true;
    }
    bool remove(int pos) {
        if (pos < 0 || pos >= sz) return false;
        DNode* p = head->next;
        for (int i = 0; i < pos; i++) p = p->next;
        link(p->prev, p->next); delete p; sz--; return true;
    }
    bool replace(int pos, int x) {
        if (pos < 0 || pos >= sz) return false;
        DNode* p = head->next;
        for (int i = 0; i < pos; i++) p = p->next;
        p->v = x; return true;
    }
    void traverse() const {
        DNode* p = head->next;
        bool first = true;
        while (p != tail) {
            if (!first) cout << ' ';
            first = false; cout << p->v;
            p = p->next;
        }
        cout << "\n";
    }
    void traverseRev() const {
        DNode* p = tail->prev;
        bool first = true;
        while (p != head) {
            if (!first) cout << ' ';
            first = false; cout << p->v;
            p = p->prev;
        }
        cout << "\n";
    }
};

int main() {
    MyList L;
    int x;
    while (cin >> x && x != -1) L.insert(L.size(), x);
    while (cin >> x) { int a=x,b; cin>>b; if (a==-1&&b==-1) break; L.insert(a,b); }
    while (cin >> x) { if (x==-1) break; L.remove(x); }
    while (cin >> x) { int d=x,e; cin>>e; if (d==-1&&e==-1) break; L.replace(d,e); }
    L.traverse();
    L.traverseRev();
}
```

### 6.11 Lab09-题目3：大数运算（q166, 简单, 34 分）

**题意**：实现 `MyBigInteger` 支持 `+ - *`（除法选做）。位数 ≤ 200。

**关键点**：
- 用 `vector<int>` 存十进制位，**低位在前**
- 加：从低位加，进位 carry
- 减：先比较长度（被减数 ≥ 减数），否则取负
- 乘：每位 × 另一个数 + 进位，O(n²) 够用
- 输出：去前导 0；若为 0 输出 `0`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct BigInt {
    vector<int> d;            // 低位在前
    BigInt(long long x = 0) { while (x) { d.push_back(x%10); x/=10; } }
    BigInt(const string& s) { d.clear(); for (int i=(int)s.size()-1; i>=0; i--) d.push_back(s[i]-'0'); trim(); }
    void trim() { while (d.size()>1 && d.back()==0) d.pop_back(); }
    bool isZero() const { return d.size()==1 && d[0]==0; }
    bool operator<(const BigInt& o) const {
        if (d.size()!=o.d.size()) return d.size()<o.d.size();
        for (int i=d.size()-1;i>=0;i--) if (d[i]!=o.d[i]) return d[i]<o.d[i];
        return false;
    }
    string toString() const {
        if (isZero()) return "0";
        string s; for (int i=d.size()-1; i>=0; i--) s.push_back('0'+d[i]);
        return s;
    }
};

BigInt add(const BigInt& a, const BigInt& b) {
    BigInt r; r.d.resize(max(a.d.size(),b.d.size()),0); int c=0;
    for (size_t i=0; i<r.d.size(); i++) {
        int s = c + (i<a.d.size()?a.d[i]:0) + (i<b.d.size()?b.d[i]:0);
        r.d[i] = s%10; c = s/10;
    }
    if (c) r.d.push_back(c);
    return r;
}
BigInt sub(BigInt a, const BigInt& b) {     // a>=b
    int c = 0;
    for (size_t i=0; i<a.d.size(); i++) {
        int s = a.d[i] - (i<b.d.size()?b.d[i]:0) - c;
        if (s<0) { s+=10; c=1; } else c=0;
        a.d[i] = s;
    }
    a.trim(); return a;
}
BigInt mul(const BigInt& a, const BigInt& b) {
    BigInt r; r.d.assign(a.d.size()+b.d.size(), 0);
    for (size_t i=0; i<a.d.size(); i++)
        for (size_t j=0; j<b.d.size(); j++) {
            r.d[i+j] += a.d[i]*b.d[j];
        }
    for (size_t i=0; i<r.d.size(); i++) {
        if (r.d[i] >= 10) { r.d[i+1] += r.d[i]/10; r.d[i] %= 10; }
    }
    r.trim(); return r;
}

int main() {
    string s1, s2; char op;
    cin >> s1 >> s2 >> op;
    BigInt A(s1), B(s2), R;
    if (op=='+') R = add(A,B);
    else if (op=='-') {
        if (A<B) { cout << "-"; R = sub(B,A); }
        else R = sub(A,B);
    }
    else if (op=='*') R = mul(A,B);
    cout << R.toString() << "\n";
}
```

**题意**：用 `List<char>` 内部存储实现 String，支持比较、连接、子串查找、取子串；N 个串，输出子串查询结果 + 字典序排序。

**关键点**：
- 内部用 `vector<char>` 装字符
- 算长度、`==` `<` 比较：长度不等直接定，否则逐字符比
- 子串查找：朴素 O(nm) 够用
- 字典序：直接用 `<` 比较，sort

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

class String {
    vector<char> s;
public:
    String() {}
    String(const char* p) { while (*p) s.push_back(*p++); }
    String(const string& t) : s(t.begin(), t.end()) {}
    int length() const { return (int)s.size(); }
    const char* c_str() const {
        static char buf[100005];   // 简化：非线程安全，Lab 够用
        for (size_t i = 0; i < s.size(); i++) buf[i] = s[i];
        buf[s.size()] = '\0';
        return buf;
    }
    String substr(int l, int r) const {   // [l, r)
        String t;
        for (int i = l; i < r; i++) t.s.push_back(s[i]);
        return t;
    }
    int find(const String& sub) const {
        int n = length(), m = sub.length();
        if (m == 0) return 0;
        for (int i = 0; i + m <= n; i++) {
            int j = 0;
            while (j < m && s[i+j] == sub.s[j]) j++;
            if (j == m) return i;
        }
        return -1;
    }
    bool operator==(const String& o) const { return s == o.s; }
    bool operator<(const String& o) const {
        size_t n = min(s.size(), o.s.size());
        for (size_t i = 0; i < n; i++) if (s[i] != o.s[i]) return s[i] < o.s[i];
        return s.size() < o.s.size();
    }
};

int main() {
    int N; cin >> N;
    vector<String> arr(N);
    for (int i = 0; i < N; i++) {
        string t; cin >> t; arr[i] = String(t);
    }
    // 第 3 行：查询（a b 对，判断 arr[a] 是否是 arr[b] 的子串）
    vector<int> ans;
    while (true) {
        int a, b; cin >> a >> b;
        if (a == -1 && b == -1) break;
        if (a < 0 || a >= N || b < 0 || b >= N) ans.push_back(-1);
        else ans.push_back(arr[b].find(arr[a]));
    }
    for (int i = 0; i < (int)ans.size(); i++) {
        if (i) cout << ' ';
        cout << ans[i];
    }
    cout << "\n";
    // 字典序
    vector<String> sorted = arr;
    sort(sorted.begin(), sorted.end());
    for (int i = 0; i < N; i++) {
        if (i) cout << ' ';
        cout << sorted[i].c_str();
    }
    cout << "\n";
}
```

## 第 7 章 查找

### 7.1 理论速记

- **查找表**：由同一类型数据元素组成的集合。
- **静态查找表 vs 动态查找表**：
  - 静态：只查不改（顺序/二分/索引）
  - 动态：可插入删除（BST/AVL/哈希）
- **顺序查找（线性查找）**：O(n)，无序数据可用。
- **二分查找**：前提是**有序**，O(log n)。
  - 关键：循环不变式 `lo <= hi`
  - 找第一个 ≥ target：`lo = mid+1` if `a[mid] < target`
  - 找最后一个 ≤ target：`hi = mid-1` if `a[mid] > target`
  - **左边界 / 右边界** 两个变体要分清
- **有序表**：插入 O(n)，查找 O(log n)。
- **二叉查找树（BST）**：详见第 10 章。
- **哈希查找**：详见第 9 章。
- **平均查找长度 ASL**：查成功的平均比较次数。

### 7.2 Lab11-题目1：前向和后向的最近邻值（q172, 中等, 33 分）

**题意**：双链表节点值均为正整数。
- **前向最近邻值**：节点 i 之前 < i 的最大值（不存在 → 0）
- **后向最近邻值**：节点 i 之后 > i 的最小值（不存在 → 0）

**关键点**：
- 暴力 O(n²) 即可（m < 1000，n = m）
- 前向：j 从 i-1 往 0 走，取 < a[i] 的最大值
- 后向：j 从 i+1 往 m-1 走，取 > a[i] 的最小值
- 注意：找"最大值"用 `>`，找"最小值"用 `<`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    vector<int> a;
    int x; while (cin >> x && x != -1) a.push_back(x);
    int n = a.size();
    // 前向最近邻值
    for (int i = 0; i < n; i++) {
        int best = 0;
        for (int j = 0; j < i; j++) if (a[j] < a[i] && a[j] > best) best = a[j];
        if (i) cout << ' ';
        cout << best;
    }
    cout << "\n";
    // 后向最近邻值
```

## 第 8 章 排序

### 8.1 理论速记

#### 8.1.1 排序总览

| 排序 | 平均 | 最坏 | 空间 | 稳定 | 思路 |
|---|---|---|---|---|---|
| 冒泡 | O(n²) | O(n²) | O(1) | ✅ | 相邻交换 |
| **插入** | O(n²) | O(n²) | O(1) | ✅ | 已排序区+新元素 |
| **希尔** | O(n^1.3) | O(n²) | O(1) | ❌ | 插入排序的分组改进 |
| **选择** | O(n²) | O(n²) | O(1) | ❌ | 每轮选最小 |
| 堆（堆排序）| O(n log n) | O(n log n) | O(1) | ❌ | 选最大 → 调堆 |
| **归并** | O(n log n) | O(n log n) | O(n) | ✅ | 分治 + 合并 |
| **快排** | O(n log n) | O(n²) | O(log n) | ❌ | 分治 + 划分 |
| 基数 | O(d·n) | O(d·n) | O(n+k) | ✅ | 按位分桶 |
| 计数 | O(n+k) | O(n+k) | O(k) | ✅ | 频次累加 |
| 桶 | O(n+k) | O(n²) | O(n+k) | ✅ | 哈希分桶 |

#### 8.1.2 关键概念

- **稳定排序**：相等元素的相对顺序保持不变（归并/插入/冒泡/基数/计数 ✅）
- **内排序 vs 外排序**：数据是否全部在内存
- **原地排序**：O(1) 辅助空间（快排、堆排、希尔、插入、选择）
- **比较排序下界**：Ω(n log n)
- **分治法思想**（快排/归并）：
  1. 分解（Divide）
  2. 解决（Conquer）
  3. 合并（Combine）

#### 8.1.3 堆

- **大根堆**：父 ≥ 子；根是最大
- **小根堆**：父 ≤ 子；根是最小
- **数组存储完全二叉树**：节点 i 的父 = (i-1)/2，左子 = 2i+1，右子 = 2i+2（0-indexed）；或父 = i/2，左子 = 2i，右子 = 2i+1（1-indexed）
- **`siftDown(i, n)`**：把 i 向下调整到正确位置
- **`siftUp(i)`**：把 i 向上调整
- **`buildHeap`**：从最后一个非叶节点 `(n-1)/2` 倒序 siftDown，O(n)
- **`heapSort`**：建堆 → 反复把堆顶与末尾交换 + siftDown，O(n log n)
- **堆支持的操作**：`insert` (siftUp)、`popMax` (siftDown)、`peekMax`

#### 8.1.4 排序相关概念

- **轴点（pivot）**：快排中用来划分的元素
- **划分（partition）**：Hoare 版本 / Lomuto 版本 / 课本"末元素为轴"版本
- **归并的合并**：两个有序子序列合成一个

### 8.6 Lab15-题目1：归并排序的链式实现（q189, 中等, 20 分）

**题意**：链表归并排序；输出**归并前的两个子序列**和**最终结果**。

**关键点**：
- 自顶向下：递归地把链表对半拆
- 拆到剩 1 个节点时返回
- 合并：双指针归并
- 输出：合并**之前**打印左右子序列

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;
struct LNode { int v; LNode* next; LNode(int x): v(x), next(nullptr) {} };

void print(LNode* h) {
    bool f = true;
    while (h) { if (!f) cout << ' '; f=false; cout << h->v; h = h->next; }
    cout << "\n";
}

// 找中点（slow 指向左半最后一个）
LNode* midAndSplit(LNode* head) {
    LNode *slow = head, *fast = head->next;
    while (fast && fast->next) { slow = slow->next; fast = fast->next->next; }
    LNode* mid = slow->next;
    slow->next = nullptr;
    return mid;
}

LNode* merge(LNode* a, LNode* b) {
    LNode dummy(0), *tail = &dummy;
    while (a && b) {
        if (a->v <= b->v) { tail->next = a; a = a->next; }
        else              { tail->next = b; b = b->next; }
        tail = tail->next;
    }
    tail->next = a ? a : b;
    return dummy.next;
}

LNode* mergeSort(LNode* head) {
    if (!head || !head->next) return head;
    LNode* mid = midAndSplit(head);
    LNode* left  = mergeSort(head);
    LNode* right = mergeSort(mid);
    // 题目要求：归并前打印子序列
    print(left);
    print(right);
    return merge(left, right);
}

int main() {
    int n; cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    LNode* head = nullptr; LNode* tail = nullptr;
    for (int i = 0; i < n; i++) {
        LNode* p = new LNode(a[i]);
        if (!head) head = tail = p;
        else { tail->next = p; tail = p; }
    }
    LNode* res = mergeSort(head);
    print(res);
    // 释放
    while (res) { LNode* t = res; res = res->next; delete t; }
}
```

### 8.7 Lab15-题目2：快速排序的顺序实现（q190, 中等, 20 分）

**题意**：顺序 List 快排，**末元素为轴点**；输出第一轮划分结果 + 最终结果。

**关键点**：
- 课本的"末元素为轴"方法（Hoare 变种）：
  - 设轴 = a[hi]
  - i = lo, j = hi
  - 从左找到第一个 > 轴的；从右找到第一个 < 轴的；交换
  - 直到 i ≥ j；最后把 a[i] 和 a[hi] 交换
- **一轮结束**：轴点左边的都 ≤ 轴，右边的都 ≥ 轴
- 递归 quicksort(lo, pivot-1)、quicksort(pivot+1, hi)
- 第一轮结束立刻打印数组

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int partition(vector<int>& a, int lo, int hi) {
    int pivot = a[hi];
    int i = lo, j = hi;
    while (i < j) {
        while (i < j && a[i] < pivot) i++;
        while (i < j && a[j] >= pivot) j--;
        if (i < j) swap(a[i], a[j]);
    }
    swap(a[i], a[hi]);
    return i;
}

void quicksort(vector<int>& a, int lo, int hi) {
    if (lo >= hi) return;
    int p = partition(a, lo, hi);
    quicksort(a, lo, p-1);
    quicksort(a, p+1, hi);
}

void print(const vector<int>& a) {
    for (int i = 0; i < (int)a.size(); i++) cout << (i?" ":"") << a[i];
    cout << "\n";
}

int main() {
    int n; cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    // 手动做第一轮
    int p = partition(a, 0, n-1);
    print(a);
    quicksort(a, 0, p-1);
    quicksort(a, p+1, n-1);
    print(a);
}
```

### 8.9 Lab15-题目4：优先级队列（q192, 中等, 20 分）

**题意**：用堆（顺序存储）实现优先级队列（数字大优先）。要求：
- 初始序列 buildHeap 建堆
- 删除 m 个最大
- 插入 k 个新元素（**自下而上 siftUp**）
- 输出：初始堆 → 删除后 → 插入后

**关键点**：
- 1-indexed 数组，`parent = i/2`, `left = 2i`, `right = 2i+1`
- `siftDown`：取子节点中较大者，必要时上浮
- `siftUp`：与父节点比，必要时下沉（这里插入用 siftUp）
- 题目要求"删除 m 个" → 反复 popMax

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

class MaxHeap {
    vector<int> h;       // 1-indexed
    int n;
public:
    MaxHeap(): n(0) { h.push_back(0); }
    void siftDown(int i) {
        while (2*i <= n) {
            int j = 2*i;
            if (j+1 <= n && h[j+1] > h[j]) j++;
            if (h[i] >= h[j]) break;
            swap(h[i], h[j]);
            i = j;
        }
    }
    void siftUp(int i) {
        while (i > 1 && h[i] > h[i/2]) {
            swap(h[i], h[i/2]);
            i /= 2;
        }
    }
    void buildHeap(vector<int> a) {    // 数组 a 元素
        h.clear(); h.push_back(0);
        for (int x : a) h.push_back(x);
        n = a.size();
        for (int i = n/2; i >= 1; i--) siftDown(i);
    }
    int popMax() {
        int v = h[1];
        h[1] = h[n--];
        siftDown(1);
        return v;
    }
    void insert(int x) { h.push_back(x); n++; siftUp(n); }
    void print() const {
        for (int i = 1; i <= n; i++) cout << (i==1?"":" ") << h[i];
        cout << "\n";
    }
};

int main() {
    MaxHeap pq;
    vector<int> init;
    int x;
    while (cin >> x && x != -1) init.push_back(x);
    pq.buildHeap(init);
    pq.print();
    int m; cin >> m;
    for (int i = 0; i < m; i++) pq.popMax();
    pq.print();
    while (cin >> x && x != -1) pq.insert(x);
    pq.print();
}
```

### 8.10 Lab15-题目5：堆排序的顺序实现（q191, 中等, 20 分）

**题意**：对顺序 List 做堆排序；输出**初始大根堆** + **最终升序**。

**关键点**：
- 1-indexed：buildHeap（O(n)）+ 反复交换 h[1] 与 h[i]，n--，siftDown(1)
- **输出顺序**：初始堆是 buildHeap 后的数组；最终是排好序的（升序）

```cpp
#include <bits/stdc++.h>
using namespace std;

void siftDown(vector<int>& h, int n, int i) {
    while (2*i <= n) {
        int j = 2*i;
        if (j+1 <= n && h[j+1] > h[j]) j++;
        if (h[i] >= h[j]) break;
        swap(h[i], h[j]);
        i = j;
    }
}

void heapSort(vector<int>& h) {   // 1-indexed
    int n = (int)h.size() - 1;
    for (int i = n/2; i >= 1; i--) siftDown(h, n, i);
    // 第一阶段完成：初始大根堆

        swap(h[1], h[i]);
        siftDown(h, i-1, 1);
    }
    for (int k = 1; k <= n; k++) cout << (k==1?"":" ") << h[k];
    cout << "\n";
}

int main() {
    int n; cin >> n;
    vector<int> h(1);
    for (int i = 0; i < n; i++) { int x; cin >> x; h.push_back(x); }
```

## 第 9 章 哈希表

### 9.1 理论速记

#### 9.1.1 概念

- **哈希函数（Hash Function）**：把 key 映射到 [0, m) 的桶下标
- **冲突（Collision）**：不同 key → 同一桶下标
- **装填因子 α = n/m**：数据数 / 表大小
- **理想情况**：α 越小越好（< 0.5）

#### 9.1.2 常见哈希函数

| 方法 | 公式 | 适用 |
|---|---|---|
| 除留余数法 | `H(k) = k mod p`（p 最好是质数） | 整数 key |
| 直接定址法 | `H(k) = a*k + b` | key 分布连续 |
| 数字分析法 | 取 key 的若干位 | 已知 key 分布 |
| 平方取中法 | 取 k² 的中间几位 | 关键字位数不多 |
| 折叠法 | 分段求和 | 关键字位数多 |
| **BKDRHash** | `hash = hash*seed + ch` | **字符串 key（Lab 必背）** |

**BKDRHash 模板**：
```cpp
unsigned int BKDRHash(const char* str, unsigned int size) {
    unsigned int seed = 31;
    unsigned int h = 0;
    while (*str) h = (h * seed + (unsigned char)(*str++)) % size;
    return h;
}
```

#### 9.1.3 冲突解决方法

**(1) 链地址法（Separate Chaining）**
- 每个桶是一个链表
- 插入：查到链表头部
- 查找/删除：遍历链表
- 优点：简单、不会"装满"
- 缺点：指针开销

**(2) 开放定址法（Open Addressing）**
- 冲突时按规则探测下一个空位
- **线性探测**：`H_i = (H + i) mod m`（易**堆积**）
- **平方探测**：`H_i = (H + i²) mod m` 或 `H_i = (H ± i²) mod m`
- **双散列**：`H_i = (H + i * H2) mod m`
- 删除时需要**墓碑标记**（不能直接置空），否则查找链会断

#### 9.1.4 哈希表性能

| 解决方式 | 成功查找 | 失败查找 |
|---|---|---|
| 链地址法 | `1 + α/2` | `α` |
| 线性探测 | `(1 + 1/(1-α))/2` | `(1 + 1/(1-α)²)/2` |
| 平方探测 | `-ln(1-α)/α` | `1/(1-α)` |

### 9.2 Lab17-题目1：Hash Table 的顺序实现（q202, 中等, 100 分）

**题意**：平方探测 + 取余法（key % n）。要求：
- 重复 key 只存第一个
- 探测次数上限 = 课本定义（一般是 `n/2` 或"上溢"）
- 输出：探测到 key1 的过程 + key1 的 value；探测到 key2 的过程；找不到输出 `NULL`

**关键点**：
- 表大小 n 选质数
- 平方探测：`for (i=0; i<n; i++) idx = (H + i*i) % n`（只 +i²），或 ±i²
- 课本惯例：探测 `H, H+1, H+4, H+9, ...`
- 探测失败：`idx` 第一次为空时（未占用）就停
- 探测到 key1 → 输出所有走过的 idx + "key1 value"
- 找不到 → 输出走过的 idx + `NULL`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, m; cin >> n >> m;
    vector<int> key(n, -1), val(n, -1);    // -1 表示空

    for (int i = 0; i < m; i++) {
        int k, v; cin >> k >> v;
        int h = k % n;
        int j = 0, idx;
        for (; j < n; j++) {
            idx = (h + j*j) % n;
            if (key[idx] == -1 || key[idx] == k) break;
        }
        if (key[idx] == -1) { key[idx] = k; val[idx] = v; }
        // 重复不覆盖
    }
    // 查询
    auto query = [&](int k) {
        int h = k % n;
        vector<int> path;
        for (int j = 0; j < n; j++) {
            int idx = (h + j*j) % n;
            path.push_back(idx);
            if (key[idx] == -1) return make_pair(path, -1);   // NULL
            if (key[idx] == k) return make_pair(path, val[idx]);
        }
        return make_pair(path, -1);
    };
    int k1, k2; cin >> k1 >> k2;
    for (int q : {k1, k2}) {
        auto [path, v] = query(q);
        for (size_t i = 0; i < path.size(); i++) cout << (i?" ":"") << path[i];
        if (v == -1) cout << " NULL";
        else         cout << " " << q << " " << v;
        cout << "\n";
    }
}
```

### 9.3 Lab16-题目1：HashMap 的实现（q203, 中等, 50 分）

**题意**：用**链地址法**实现 HashMap，支持 `put` / `get` / `remove`。字符串 key 用 **BKDRHash**。要求：
- put 重复 key → 更新 value
- get 不存在 → 返回 -1
- 输出：插入某 key 的地址 + 删除某 key 的过程（找不到输出 NULL）+ 查询某 key 的过程（NULL 输出 NULL）

**关键点**：
- 数组 `vector<list<pair<string,string>>>` 存桶
- BKDRHash 把字符串映射到 [0, m)
- 输出格式：先输出 key 的地址码，然后是过程（"key1 key2"）

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

unsigned int BKDR(const string& s, unsigned int size) {
    unsigned int h = 0, seed = 31;
    for (char c : s) h = (h * seed + (unsigned char)c) % size;
    return h;
}

int main() {
    int n, m; cin >> n >> m;
    vector<list<pair<string,string>>> bucket(n);
    for (int i = 0; i < m; i++) {
        string k, v; cin >> k >> v;
        unsigned int idx = BKDR(k, n);
        bool found = false;
        for (auto& p : bucket[idx]) if (p.first == k) { p.second = v; found = true; break; }
        if (!found) bucket[idx].push_front({k, v});
    }
    // 第 3 行：插入（输出地址码）
    string k3, v3; cin >> k3 >> v3;
    unsigned int idx3 = BKDR(k3, n);
    bool found = false;
    for (auto& p : bucket[idx3]) if (p.first == k3) { p.second = v3; found = true; break; }
    if (!found) bucket[idx3].push_front({k3, v3});
    cout << idx3 << "\n";

    // 第 4 行：删除（输出地址 + "key" + "NULL" 如果失败）
    string k4; cin >> k4;
    unsigned int idx4 = BKDR(k4, n);
    bool removed = false;
    for (auto it = bucket[idx4].begin(); it != bucket[idx4].end(); ++it) {
        if (it->first == k4) { bucket[idx4].erase(it); removed = true; break; }
    }
    cout << idx4 << ' ' << k4 << ' ' << (removed ? "" : "NULL") << "\n";
    // 注意：题目样例 "6 dog NULL" 实际是：地址 dog 删除结果(NULL)
    // 严格按样例输出格式

    // 第 5 行：查询
    string k5; cin >> k5;
    unsigned int idx5 = BKDR(k5, n);
    cout << idx5 << ' ';
    for (auto& p : bucket[idx5]) cout << p.first << (p.first==k5?"":" ");  // 简化
    cout << " " << k5 << " " << (查询到的value) << "\n";
}
```

### 9.4 Lab16-题目2：Longest Substring Without Repeating Characters（q205, 中等, 50 分）

**题意**：字符串中无重复字符的最长子串长度。

**关键点**：**滑动窗口 + 哈希表记录上次出现位置**。
- `last[ch]` = ch 上次出现的位置（-1 表示没出现）
- 遍历 i = 0..n-1
- 维护左边界 `lo`：`lo = max(lo, last[s[i]] + 1)`（如果 ch 出现过）
- 更新答案：`ans = max(ans, i - lo + 1)`
- 更新 `last[s[i]] = i`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    string s; cin >> s;
    vector<int> last(256, -1);
    int lo = 0, ans = 0;
    for (int i = 0; i < (int)s.size(); i++) {
        lo = max(lo, last[s[i]] + 1);
        ans = max(ans, i - lo + 1);
        last[s[i]] = i;
    }
    cout << ans << "\n";
}
```

## 第 10 章 二叉树

### 10.1 理论速记

#### 10.1.1 概念

- **二叉树**：每个节点 ≤ 2 个子树（左、右）
- **满二叉树**：所有层满
- **完全二叉树**：除最后一层外全满，最后一层从左到右排列
- **二叉查找树（BST）**：左 ≤ 根 ≤ 右（严格：左 < 根 < 右）
- **平衡二叉树**：左右子树高度差 ≤ 1（AVL 是代表）

#### 10.1.2 存储方式

- **顺序存储**（数组）：完全二叉树最合适；一般二叉树浪费空间
- **链式存储**：节点 `{val, left, right}`，用指针

#### 10.1.3 遍历

| 遍历 | 顺序 |
|---|---|
| 前序 | 根 → 左 → 右 |
| 中序 | 左 → 根 → 右 |
| 后序 | 左 → 右 → 根 |
| 层次 | 一层一层（用队列） |

- **中序 + 前序** → 唯一确定二叉树
- **中序 + 后序** → 唯一确定二叉树
- **前序 + 后序** → 不能唯一确定（除非是满二叉树）

#### 10.1.4 BST 操作（O(h)，h = 树高）

- **查找**：从根开始比较
- **插入**：找到空位插入
- **删除**：
  - 叶节点：直接删
  - 单子树：子树顶替
  - 双子树：用**前驱**（左子树最大）或**后继**（右子树最小）替代，再删前驱/后继
- **最小/最大**：一直向左/右

#### 10.1.5 AVL 树

- **平衡因子 BF = h(left) - h(right)**
- 失衡四型：**LL、RR、LR、RL**
- 调整：单旋（LL/RR）、双旋（LR/RL）
- 插入/删除后从插入点向上回溯，更新 BF 并旋转

#### 10.1.6 Splay 树（伸展树）

- 思想：刚访问的节点"伸展"到根
- 三种操作：**Zig / Zig-Zig / Zig-Zag**
- 自适应：访问频繁的节点靠近根
- 摊还时间 O(log n)

#### 10.1.7 重要性质

- **n 个节点的二叉树**：n-1 条边
- **完全二叉树高度**：`⌊log₂ n⌋` 或 `⌈log₂(n+1)⌉ - 1`
- **节点数 = 2^(h+1) - 1**（满二叉树）
- **第 i 层最多 2^(i-1) 个节点**

### 10.2 Lab19-题目1：二叉树遍历（q27, 中等, 100 分）

**题意**：把输入序列构造成**完全二叉树**，输出：
- 第 1 行：树高度（仅 1 个节点 = 0）
- 第 2 行：前序
- 第 3 行：中序
- 第 4 行：后序

**关键点**：
- 完全二叉树的数组下标规则：i 节点的左子 = 2i+1，右子 = 2i+2（0-indexed）
- 树高 = `⌊log₂ n⌋`（当 n = 2^k - 1 时等于 k-1；通用 `floor(log2(n))`）
- 递归遍历

```cpp
#include <bits/stdc++.h>

int n;
vector<int> a;

// 完全二叉树：i 的左子 2i+1，右子 2i+2（0-indexed）
int height(int i) {
    if (i >= n) return -1;
    return 1 + max(height(2*i+1), height(2*i+2));
}
void pre(int i)  { if (i>=n) return; cout << a[i] << " "; pre(2*i+1); pre(2*i+2); }

int main() {
    cin >> n;
    a.resize(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    cout << height(0) << "\n";
    pre(0);  cout << "\n";

    post(0); cout << "\n";
}
```

### 10.3 Lab18-题目1：二叉查找树（q209, 中等, 33 分）

**题意**：BST 实现插入、删除（**用前驱**）、查找；输入单词（key + 中文），输出查询路径 + 内容；不存在输出 `NULL`。

**关键点**：
- BST 节点：`{string key, string other; Node *l, *r;}`
- 插入：标准 BST 插入
- 删除：双子树用**前驱**（左子树最大）
- 查找：从根开始记录路径，输出路径上每个节点的 key
- 找不到时路径继续到 NULL，输出 `NULL`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct TNode {
    string key, other;
    TNode *l, *r;
    TNode(string k, string o): key(k), other(o), l(nullptr), r(nullptr) {}
};

TNode* insert(TNode* root, string k, string o) {
    if (!root) return new TNode(k, o);
    if (k < root->key) root->l = insert(root->l, k, o);
    else if (k > root->key) root->r = insert(root->r, k, o);
    return root;
}

// 找左子树最大
TNode* findMax(TNode* p) { while (p->r) p = p->r; return p; }

TNode* remove(TNode* root, string k) {
    if (!root) return nullptr;
    if (k < root->key) root->l = remove(root->l, k);
    else if (k > root->key) root->r = remove(root->r, k);
    else {
        if (!root->l) { TNode* t = root->r; delete root; return t; }
        if (!root->r) { TNode* t = root->l; delete root; return t; }
        // 双子树：用前驱（左子树最大）替代
        TNode* pre = findMax(root->l);
        root->key = pre->key; root->other = pre->other;
        root->l = remove(root->l, pre->key);
    }
    return root;
}

void query(TNode* root, string k) {
    TNode* p = root;
    bool first = true;
    while (p) {
        if (!first) cout << " ";
        first = false;
        cout << p->key;
        if (k == p->key) {
            cout << " " << p->key << " " << p->other << "\n";
            return;
        }
        p = (k < p->key) ? p->l : p->r;
    }
    cout << " NULL\n";
}

int main() {
    int n; cin >> n;
    TNode* root = nullptr;
    for (int i = 0; i < n; i++) {
        string k, o; cin >> k >> o;
        root = insert(root, k, o);
    }
    int d; cin >> d;
    for (int i = 0; i < d; i++) {
        string k; cin >> k;
        root = remove(root, k);
    }
    string t1, t2; cin >> t1 >> t2;
    query(root, t1);
    query(root, t2);
}
```

### 10.4 Lab18-题目2：The Sum of Path（q215, 困难, 33 分）

**题意**：给定插入序列生成**BST**，判断是否存在根→叶路径节点和 = k。同时输出树的高度（仅 1 节点 = 0）。

**关键点**：
- 按 BST 插入方式建树（数值序列就是插入顺序）
- 高度 = `floor(log2(n))` 对**完全**BST？不一定，**一般树高**用递归算
- 路径和 = k：DFS，传入当前和，递归到叶节点判断
- 数值**不要求不重复**（样例里有重复），BST 重复节点按"≥"放右子树（约定）

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct TNode {
    int v;
    TNode *l, *r;
    TNode(int x): v(x), l(nullptr), r(nullptr) {}
};

TNode* insert(TNode* p, int x) {
    if (!p) return new TNode(x);
    if (x < p->v) p->l = insert(p->l, x);
    else p->r = insert(p->r, x);
    return p;
}
int height(TNode* p) { return p ? 1 + max(height(p->l), height(p->r)) : -1; }
bool hasPath(TNode* p, int sum) {
    if (!p) return false;
    if (!p->l && !p->r) return p->v == sum;
    return hasPath(p->l, sum - p->v) || hasPath(p->r, sum - p->v);
}

int main() {
    int n; cin >> n;
    TNode* root = nullptr;
    for (int i = 0; i < n; i++) { int x; cin >> x; root = insert(root, x); }
    int k; cin >> k;
    cout << height(root) << ' ' << (hasPath(root, k) ? "true" : "false") << "\n";
}
```

### 10.5 Lab18-题目3：二分查找树的最矮公共祖先（q216, 中等, 34 分）

**题意**：BST 找 p 和 q 的 LCA（最低公共祖先）。

**关键点**：**BST 的 LCA 利用有序性**：
- 从根开始
- 若 p、q 都 < 根 → 递归左
- 若 p、q 都 > 根 → 递归右
- 否则（p ≤ 根 ≤ q 或反过来）→ 根就是 LCA

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct TNode {
    int v;
    TNode *l, *r;
    TNode(int x): v(x), l(nullptr), r(nullptr) {}
};

TNode* insert(TNode* p, int x) {
    if (!p) return new TNode(x);
    if (x < p->v) p->l = insert(p->l, x);
    else p->r = insert(p->r, x);
    return p;
}
int lca(TNode* root, int p, int q) {
    int a = min(p, q), b = max(p, q);
    while (root) {
        if (b < root->v) root = root->l;
        else if (a > root->v) root = root->r;
        else return root->v;
    }
    return -1;
}

int main() {
    int n; cin >> n;
    TNode* root = nullptr;
    for (int i = 0; i < n; i++) { int x; cin >> x; root = insert(root, x); }
    int p, q; cin >> p >> q;
    cout << lca(root, p, q) << "\n";
}
```

## 第 11 章 多叉树（理论速记）

> ℹ️ **本章 Lab 不涉及**，仅考概念题，掌握以下速记即可。

### 11.1 Trie 树（字典树 / 前缀树）

- **结构**：每个节点有多个孩子指针（按字符数 26 / 52 / 64）
- **节点字段**：`children[26]`、`isEnd`（标记是否单词结尾）
- **应用**：
  - 字符串集合存储
  - 前缀匹配（自动补全）
  - 词频统计
  - 拼写检查
- **时间复杂度**：插入 / 查找 O(L)，L = 字符串长度
- **空间复杂度**：O(总字符数 × 字符集大小)
- **优点**：查找前缀效率高
- **缺点**：空间浪费

### 11.2 B 树 / B+ 树

- **动机**：磁盘 I/O 优化（减少访问次数）
- **B 树定义**（m 阶 B 树）：
  - 每个节点最多 m 个孩子
  - 除根外，每个节点 ≥ ⌈m/2⌉ 个孩子
  - 根节点 ≥ 2 个孩子（除非只有 1 个 key）
  - 有 k 个孩子的节点有 k-1 个 key
  - 所有叶节点在同一层
- **B+ 树 vs B 树**：
  - B+ 树数据**只在叶节点**
  - B+ 树叶节点**链表**连接（范围查询快）
  - B+ 树内部节点的 key 是子树 key 的副本
- **应用**：
  - 数据库索引（MySQL InnoDB 用 B+ 树）
  - 文件系统（NTFS、ext4）
- **阶数 m 与磁盘块大小**：
  - 树高 h = `O(log_m n)`
  - m 越大，h 越小，I/O 越少

### 11.3 多叉树比较

| 结构 | 用途 | 时间复杂度 | 空间 |
|---|---|---|---|
| Trie | 前缀匹配 | O(L) | 字符集 × 节点数 |
| B 树 | 外存索引 | O(log_m n) | 紧凑 |
| B+ 树 | 数据库 | O(log_m n) | 更紧凑 |
| 红黑树 | 内存平衡 | O(log n) | 节点 2-3 倍开销 |

| 邻接多重表 | O(V+E) | 无向图 |

### 12.3 图的遍历

- **DFS（深度优先）**：栈（递归）；访问 - 标记 - 递归邻居
- **BFS（广度优先）**：队列；访问 - 入队 - 标记
- **时间复杂度**：O(V+E)（邻接表）/ O(V²)（邻接矩阵）
- **应用**：
  - 连通性
  - 路径查找
  - 拓扑序
  - 连通分量

### 12.4 拓扑排序

- **适用**：DAG（有向无环图）
- **Kahn 算法**：
  1. 选入度 = 0 的顶点
  2. 删除该顶点 + 出边
  3. 重复
- **DFS 法**：后序输出，再反转
- **时间**：O(V+E)
- **检测环**：拓扑序列 < V 个 → 有环

### 12.5 最短路径

| 算法 | 适用 | 边权 | 时间 |
|---|---|---|---|
| BFS | 无权 | - | O(V+E) |
| **Dijkstra** | 单源、非负权 | ≥ 0 | O((V+E) log V)（堆优化） |
| Bellman-Ford | 单源、可负权 | 任意 | O(VE) |
| **Floyd-Warshall** | 全对 | 任意 | O(V³) |
| SPFA | 单源、负权 | 任意 | 平均 O(E) |

**Dijkstra 步骤**：
1. 初始化 dist[s] = 0，其他 = ∞
2. 选 dist 最小的未访问顶点 u
3. 更新 u 邻居 v 的 dist：`dist[v] = min(dist[v], dist[u] + w(u,v))`
4. 标记 u 已访问
5. 重复直到所有顶点访问完

### 12.6 最小生成树（MST）

- **适用**：连通**无向加权**图
- **性质**：
  - MST 有 V-1 条边
  - 不唯一（边权相同时）
  - 切割性质：跨任意切割的最短边必在某个 MST 中

| 算法 | 思路 | 时间 |
|---|---|---|
| **Prim** | 从一个顶点出发，长边扩展 | O(E log V) |
| **Kruskal** | 按边权排序，并查集 | O(E log E) |

**Kruskal 步骤**：
1. 边按权值升序
2. 依次考虑：若两端点不连通（并查集）→ 加入 MST
3. 加到 V-1 条边为止

### 12.7 图的应用

| 场景 | 经典算法 |
|---|---|
| 工程调度（依赖关系） | 拓扑排序 |
| 地图导航 | Dijkstra / Floyd |
| 网络布线 | 最小生成树 |
| 社交关系 | BFS/DFS（连通分量） |
| 任务依赖 | 关键路径（AOE 网） |
| 城市间最短 | Floyd |
| 单源最短路（含负权） | Bellman-Ford / SPFA |
