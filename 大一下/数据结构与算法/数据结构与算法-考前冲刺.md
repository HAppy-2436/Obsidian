## 第 1 章 课程介绍

### 1.1 理论全掌握

#### 1.1.1 数据结构基本概念

- **数据（Data）**：客观事物的符号表示，是所有能输入到计算机并被程序处理的符号的总称。
- **数据元素（Data Element）**：数据的基本单位，通常作为一个整体考虑和处理（如一个学生、一条记录）。
- **数据项（Data Item）**：数据元素的不可分割的最小单位（如学生的学号、姓名）。
- **数据对象（Data Object）**：性质相同的数据元素的集合（如所有学生）。
- **数据结构（Data Structure）**：**相互之间存在一种或多种特定关系的数据元素的集合**。包括三方面：
  1. **数据的逻辑结构**：数据元素之间的逻辑关系（线性、树、图、集合）。
  2. **数据的存储结构（物理结构）**：数据在计算机中的存储方式（顺序、链式、索引、散列）。
  3. **数据的运算**：对数据施加的操作（增删改查、排序等）。

#### 1.1.2 逻辑结构四大类

| 类型 | 关系特征 | 典型 |
|---|---|---|
| **集合** | 同属一个集合，无其他关系 | 集合 |
| **线性** | 一对一 | 数组、链表、栈、队列 |
| **树** | 一对多 | 二叉树、堆 |
| **图** | 多对多 | 社交网络、地图 |

#### 1.1.3 存储结构四种

- **顺序存储**：逻辑上相邻的元素物理上也相邻（数组）。优点：随机访问 O(1)。缺点：插入删除 O(n)，需要连续空间。
- **链式存储**：元素任意存储，靠指针维系关系（链表）。优点：插入删除 O(1)。缺点：不能随机访问，额外指针开销。
- **索引存储**：建立索引表（关键字 → 地址）。优点：查找快。缺点：索引占空间、维护索引开销。
- **散列（哈希）存储**：用哈希函数直接算出地址。优点：期望 O(1) 查找。缺点：冲突处理复杂、不支持顺序访问。

#### 1.1.4 抽象数据类型（ADT）

- **定义**：一个数学模型 + 定义在该模型上的一组操作。**只关心"做什么"，不关心"怎么做"**。
- **三要素**：
  1. **数据对象（Data Object）**
  2. **数据关系（Data Relation）**
  3. **基本操作（Basic Operations）**
- **表示格式**：
  ```
  ADT 抽象数据类型名
  Data
      数据对象的定义
  Relation
      数据关系的定义
  Operation
      操作 1：初始条件 / 操作结果
      操作 2：...
  endADT
  ```
- **实现**：ADT 通过具体的存储结构和算法实现（C++ 的 class / struct）。

#### 1.1.5 算法五大特性

1. **有穷性**：必须在有限步后终止。
2. **确定性**：每一步骤无歧义。
3. **可行性**：每一步都能执行。
4. **输入**：零个或多个输入。
5. **输出**：一个或多个输出。

#### 1.1.6 算法评价指标

- **正确性**：满足问题需求。
- **可读性**：清晰易理解。
- **健壮性**：对非法输入能处理而非崩溃。
- **效率**：
  - **时间复杂度 T(n)**：基本操作次数随 n 的增长趋势（用大 O 表示）。
  - **空间复杂度 S(n)**：算法占用的额外存储空间。

#### 1.1.7 时间复杂度分析

- **常见量级**（从快到慢）：
  `O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(n³) < O(2ⁿ) < O(n!)`
- **计算方法**：
  - 顺序语句：加法
  - 选择语句：取 max
  - 循环语句：乘积
  - 递归：递推方程（主定理）
- **最好 / 最坏 / 平均情况**：
  - 最好：最佳输入
  - 最坏：最差输入
  - 平均：所有输入等概率时的期望
- **摊还时间**：一系列操作的总成本平均到每次（splay 树摊还 O(log n)）。

#### 1.1.8 主定理（Master Theorem）

递归式 `T(n) = aT(n/b) + O(n^d)`：
- `d > log_b a` → `T(n) = O(n^d)`
- `d = log_b a` → `T(n) = O(n^d log n)`
- `d < log_b a` → `T(n) = O(n^log_b a)`

**例**：归并排序 `T(n) = 2T(n/2) + O(n)`，a=2, b=2, d=1，`log_b a = 1 = d` → O(n log n)。

#### 1.1.9 C++ 类（Lab 必写）

```cpp
class MyClass {
public:
    MyClass();                                  // 默认构造
    MyClass(const MyClass& other);              // 拷贝构造
    MyClass(MyClass&& other);                   // 移动构造（C++11）
    MyClass& operator=(const MyClass& other);   // 拷贝赋值
    MyClass& operator=(MyClass&& other);        // 移动赋值
    ~MyClass();                                 // 析构
    // 业务方法
    void push(int x);
    int pop();
    int top() const;                            // const 成员函数不修改对象
    int size() const;
    bool empty() const;
private:
    int* data;
    int sz;
    int cap;
};
```

- **Rule of Three**：如果定义了析构、拷贝构造、拷贝赋值中任何一个，通常三个都要定义（资源管理）。
- **Rule of Five**（C++11）：再加移动构造、移动赋值。
- **const 正确性**：不修改对象的成员函数加 `const`。
- **RAII**：资源获取即初始化（智能指针 `unique_ptr` / `shared_ptr`）。

#### 1.1.10 数组

- **一维数组**：连续存储，类型相同。下标 `0..n-1`。
- **二维数组**：行优先 / 列优先存储。多维 → 一维映射：`a[i][j]` = `*(base + i*col + j)`。
- **时间复杂度**：
  - 随机访问 O(1)
  - 查找（无序）O(n)
  - 插入/删除 O(n)
  - 遍历 O(n)
- **特殊矩阵压缩**：
  - 对称矩阵：存一半（下三角）
  - 三角矩阵
  - 稀疏矩阵：三元组 / 十字链表

#### 1.1.11 指针与内存

- **指针**：存地址的变量。`int* p = &x;`
- **空指针**：`nullptr`（C++11 起，替代 `NULL`）
- **指针运算**：`p + 1` 跳过一个类型大小
- **动态内存**：`new` / `delete`（C++）或 `malloc` / `free`（C）
- **内存泄漏**：申请了没释放
- **悬挂指针**：指向已释放的内存
- **栈内存**：函数局部变量、自动管理
- **堆内存**：`new` 出来的、手动管理（智能指针更好）

#### 1.1.12 Life Game 通用思路

- **规则核心**：每个 cell 根据邻居状态决定下一代的生死。
- **编程套路**：
  1. 维护**两份网格**（当前 + 下一代），避免原地修改
  2. 每代遍历所有 cell，统计邻居数
  3. 按规则更新下一代
  4. 交换指针 / 数组，重复
- **关键技巧**：
  - 边界处理（无限网格 vs 有限网格）
  - 邻居方向（4 邻 / 8 邻 / 自定义）
  - 输出格式（`-` / `*` / `0` / `1`）

### 1.2 Lab01-题目1：One-Dimensional Life（q136, 中等, 50 分）

**题意**：一维格子，每个活细胞有 4 个邻居（左右各 1、2）。规则：
- 死细胞：邻居数 = 2 或 3 → 复活
- 活细胞：邻居数 = 0/1/3 → 死
- 活细胞：邻居数 = 2 或 4 → 活
- 总数 < 60。

**输入**：活细胞位置（1..60），以 `-1` 结束；n 代。
**输出**：第 n 代行（`-` 空 / `*` 活）。

**关键点**：
- 行宽 60，位置 1..60（不是 0..59）
- n=0 输出初始行
- 每代用新数组存，邻居 = 自身 + 左右各 1、2（共 4 个）

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    const int N = 60;
    vector<int> cur(N+5, 0), nxt(N+5, 0);
    int p;
    while (cin >> p && p != -1) cur[p] = 1;   // 1-indexed
    int g; cin >> g;
    for (int step = 1; step <= g; step++) {
        for (int i = 1; i <= N; i++) {
            int nb = cur[i-2] + cur[i-1] + cur[i+1] + cur[i+2];
            if (cur[i] == 1)
                nxt[i] = (nb == 2 || nb == 4) ? 1 : 0;
            else
                nxt[i] = (nb == 2 || nb == 3) ? 1 : 0;
        }
        cur = nxt;
    }
    for (int i = 1; i <= N; i++) cout << (cur[i] ? '*' : '-');
    cout << '\n';
    return 0;
}
```

### 1.3 Lab01-题目2：Magic Square（q135, 中等, 50 分）

**题意**：奇数 n（1..101）生成 n×n 魔方阵，方法：1 放顶行中间；后续数字沿右上对角走；撞顶/撞右则环绕；撞到已占位则正下方一格。

**关键点**：
- 行 = `(row - 1 + n) % n`，列 = `(col + 1) % n`
- 撞占位 → 行 = `(row + 1) % n`，列不变
- 输出：数字之间**两个空格**，行末无空格

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n; cin >> n;
    vector<vector<int>> a(n, vector<int>(n, 0));
    int r = 0, c = n/2;        // 1 放 (0, n/2)
    a[r][c] = 1;
    for (int k = 2; k <= n*n; k++) {
        int nr = (r - 1 + n) % n;
        int nc = (c + 1) % n;
        if (a[nr][nc] != 0) {  // 撞占位
            nr = (r + 1) % n;
            nc = c;
        }
        a[nr][nc] = k;
        r = nr; c = nc;
    }
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            if (j) cout << "  ";
            cout << a[i][j];
        }
        cout << '\n';
    }
    return 0;
}
```

### 1.4 Lab21：二维 Life Game（q134, 中等, 100 分）

**题意**：20×60 棋盘；8 邻居；规则 B3/S23。
- 活细胞：邻居 0/1 → 死；2/3 → 活；≥4 → 死
- 死细胞：邻居 = 3 → 活

**关键点**：
- 输出范围 20×60 全网格
- 死 → `-`，活 → `*`
- 每代双 buffer 切换

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    const int R = 20, C = 60;
    vector<vector<int>> cur(R, vector<int>(C, 0));
    int x, y;
    while (cin >> x >> y) {
        if (x == -1 && y == -1) break;
        cur[x][y] = 1;   // 题目说 (x, y) 行列都从 1 开始
    }
    int g; cin >> g;
    for (int step = 0; step < g; step++) {
        vector<vector<int>> nxt(R, vector<int>(C, 0));
        for (int i = 0; i < R; i++) {
            for (int j = 0; j < C; j++) {
                int nb = 0;
                for (int di = -1; di <= 1; di++)
                    for (int dj = -1; dj <= 1; dj++) {
                        if (di == 0 && dj == 0) continue;
                        int ni = i+di, nj = j+dj;
                        if (ni>=0 && ni<R && nj>=0 && nj<C)
                            nb += cur[ni][nj];
                    }
                if (cur[i][j]) nxt[i][j] = (nb==2 || nb==3);
                else nxt[i][j] = (nb==3);
            }
        }
        cur = nxt;
    }
    for (int i = 0; i < R; i++) {
        for (int j = 0; j < C; j++)
            cout << (cur[i][j] ? '*' : '-');
        cout << '\n';
    }
    return 0;
}
```

## 第 2 章 栈

### 2.1 理论全掌握

#### 2.1.1 栈的定义与性质

- **栈（Stack）**：限定仅在**表尾**进行插入和删除操作的线性表。
- **LIFO**（Last In First Out）：后进先出。
- **栈顶（Top）**：允许操作的一端。
- **栈底（Bottom）**：固定不动的一端。
- **关键术语**：
  - **入栈 / 压栈（Push）**：栈顶插入元素
  - **出栈 / 弹栈（Pop）**：栈顶删除元素
  - **读栈顶（Top/Peek）**：看栈顶元素但不删除
  - **栈空（Empty）**、**栈满（Full）**（仅顺序栈）
- **ADT 接口**：
  ```
  ADT Stack
  Data
      一维数组 + 栈顶指针
  Operation
      InitStack(&S)
      DestroyStack(&S)
      ClearStack(&S)
      StackEmpty(S)
      StackLength(S)
      GetTop(S, &e)
      Push(&S, e)
      Pop(&S, &e)
  ```

#### 2.1.2 顺序栈（顺序存储）

- **结构**：数组 + 栈顶指针 `top`。
- **约定**：
  - `top = -1` 表示空栈
  - `top` 指向**栈顶元素**（不是下一个空位）
  - 进栈：`data[++top] = x`
  - 出栈：`return data[top--]`
  - 栈满：`top == capacity - 1`
- **另一种约定**（栈顶指向下一个空位）：
  - `top = 0` 表示空栈
  - 进栈：`data[top++] = x`
  - 出栈：`return data[--top]`
  - 栈满：`top == capacity`
- **时间复杂度**：所有操作 O(1)
- **空间复杂度**：O(capacity)
- **优缺点**：
  - ✅ 简单、随机访问、CPU 缓存友好
  - ❌ 容量固定（浪费 or 溢出）

**C++ 实现**：
```cpp
class Stack {
    static const int CAP = 105;
    int data[CAP];
    int top;
public:
    Stack(): top(-1) {}
    bool empty() const { return top == -1; }
    bool full()  const { return top == CAP-1; }
    int  size()  const { return top + 1; }
    void clear() { top = -1; }
    bool push(int x) { if (full()) return false; data[++top] = x; return true; }
    bool pop()        { if (empty()) return false; top--; return true; }
    int  peek() const { return data[top]; }
};
```

#### 2.1.3 两栈共享空间

- **场景**：两个**同类型**栈，需求此消彼长（如一个栈大时另一个小）。
- **方法**：一个数组从两端向中间生长。
  - 栈 1 从左端：`top1 = -1` 初始
  - 栈 2 从右端：`top2 = n` 初始
  - 栈满条件：`top1 + 1 == top2`
- **优点**：节省空间
- **缺点**：只能两栈共享

#### 2.1.4 链栈（链式存储）

- **结构**：单链表，**头插法**（头节点处即栈顶）。
- **优点**：
  - 不需要预先分配空间
  - 不存在栈满（除非系统内存耗尽）
  - 长度动态
- **缺点**：
  - 每个节点多一个指针
  - 不可随机访问
- **时间复杂度**：所有操作 O(1)
- **C++ 实现**：
```cpp
struct Node { int val; Node* next; Node(int v): val(v), next(nullptr) {} };
class LinkedStack {
    Node* topNode;
    int cnt;
public:
    LinkedStack(): topNode(nullptr), cnt(0) {}
    ~LinkedStack() { clear(); }
    bool empty() const { return cnt == 0; }
    int  size()  const { return cnt; }
    void clear() { while (topNode) { Node* t = topNode; topNode = topNode->next; delete t; } cnt = 0; }
    void push(int x) { Node* n = new Node(x); n->next = topNode; topNode = n; cnt++; }
    void pop()        { Node* t = topNode; topNode = topNode->next; delete t; cnt--; }
    int  top() const  { return topNode->val; }
};
```

#### 2.1.5 顺序栈 vs 链栈

| 维度 | 顺序栈 | 链栈 |
|---|---|---|
| 存储 | 数组 | 链表 |
| 容量 | 固定 | 动态 |
| 空间利用率 | 可能浪费 | 灵活 |
| 访问速度 | 快（连续） | 慢（指针） |
| 判满 | 需要 | 不需要 |
| 实现难度 | 简单 | 中等 |

#### 2.1.6 栈的应用（核心）

**(1) 括号匹配**
- 思路：扫描字符串，遇左括号入栈，遇右括号弹栈顶匹配
- 时间 O(n)，空间 O(n)
- 关键：处理嵌套、注释、字符串字面量

**(2) 表达式求值**
- **中缀表达式**（人写）：`2+3*4`
- **前缀表达式 / 波兰式**（运算符前）：`+ 2 * 3 4`
- **后缀表达式 / 逆波兰式**（运算符后）：`2 3 4 * +`
- **后缀求值算法**：
  - 扫描 token
  - 数字：入栈
  - 运算符：弹两个数（b = 栈顶, a = 次栈顶），计算 `a op b`，结果入栈
  - 结束：栈底为结果
- **中缀 → 后缀（调度场算法，Shunting-yard）**：
  - 数字：直接输出
  - `(`：入栈
  - `)`：弹栈到 `(` 为止
  - 运算符：弹出栈顶优先级 ≥ 当前的所有运算符，再把当前入栈
  - 结束：弹出栈中剩余运算符

**(3) 递归**
- **递归调用栈**：每次递归调用都把现场（参数、局部变量、返回地址）压栈
- **递归 = 栈的应用**（隐式）
- **递归转非递归**：用显式栈模拟

**(4) 深度优先搜索（DFS）**
- 用栈实现非递归 DFS

**(5) 函数调用 / 撤销操作（undo）**
- 撤销栈：每次操作前入栈当前状态，撤销时弹栈恢复

**(6) 浏览器前进/后退**
- 两个栈：back 栈和 forward 栈

**(7) 中缀表达式求值的双栈法**
- 一个**操作数栈**，一个**操作符栈**
- 数字入操作数栈
- 运算符按优先级比较，决定计算还是入栈

#### 2.1.7 常见考点

- 后缀表达式求值（必考）
- 栈的入栈出栈合法序列判断（Catalan 数：`C_n = (2n)! / ((n+1)! n!)`）
- 栈与递归的等价性
- 双端栈 / 共享栈
- 中缀→后缀转换

### 2.7 Lab20-题目1：中序波兰式求解（q143, 简单, 100 分）

**题意**：中缀表达式求值，操作符 `+ - * /`，含括号；可能含浮点、负数。整数直接输出；浮点保留 3 位小数。

**关键点**：
- **双栈法**（操作数栈 + 操作符栈）是最稳的写法
- 优先级：`(` < `+ -` < `* /`
- 遇到 `(` 入栈；遇到 `)` 弹到 `(`；数字（含负数、小数）压栈
- 优先级比较：当前 op ≤ 栈顶 op 时，先计算栈顶

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int pri(char c) {
    if (c == '+' || c == '-') return 1;
    if (c == '*' || c == '/') return 2;
    return 0;
}
void calc(stack<double>& num, char op) {
    double b = num.top(); num.pop();
    double a = num.top(); num.pop();
    double r = 0;
    if (op=='+') r=a+b; else if (op=='-') r=a-b;
    else if (op=='*') r=a*b; else r=a/b;
    num.push(r);
}
int main() {
    string s; getline(cin, s);
    stack<double> num;
    stack<char> op;
    int n = s.size();
    for (int i = 0; i < n; ) {
        char c = s[i];
        if (c==' ' || c=='\t') { i++; continue; }
        if (c=='(') { op.push(c); i++; }
        else if (c==')') {
            while (!op.empty() && op.top()!='(') calc(num, op.top()), op.pop();
            if (!op.empty()) op.pop();   // 弹出 (
            i++;
        }
        else if (c=='+'||c=='-'||c=='*'||c=='/') {
            // 处理一元负号：表达式开头 或 上一个是 ( 或 运算符
            if (c=='-' && (i==0 || s[i-1]=='(' || strchr("+-*/", s[i-1]))) {
                // 负数: 直接读入 -<数字>
                // 简化：把 -x 当 0-x
                num.push(0);
            }
            while (!op.empty() && op.top()!='(' && pri(op.top()) >= pri(c))
                calc(num, op.top()), op.pop();
            op.push(c); i++;
        }
        else { // 数字
            int j = i;
            while (j<n && (isdigit(s[j]) || s[j]=='.')) j++;
            num.push(stod(s.substr(i, j-i)));
            i = j;
        }
    }
    while (!op.empty()) calc(num, op.top()), op.pop();
    double ans = num.top();
    if (ans == (long long)ans) cout << (long long)ans << "\n";
    else                        printf("%.3f\n", ans);
}
```

## 第 3 章 队列

### 3.1 理论全掌握

#### 3.1.1 队列的定义与性质

- **队列（Queue）**：只允许在**一端**（队尾）插入、在**另一端**（队首）删除的线性表。
- **FIFO**（First In First Out）：先进先出。
- **队首（Front/Head）**：允许删除的一端。
- **队尾（Rear/Tail）**：允许插入的一端。
- **关键术语**：
  - **入队（Enqueue）**：队尾插入
  - **出队（Dequeue）**：队首删除
  - **读队首（Front/Peek）**：看队首元素
  - **队空（Empty）**、**队满（Full）**（仅顺序队列）
- **ADT 接口**：
  ```
  ADT Queue
  Data
      一维数组 + 队首、队尾指针
  Operation
      InitQueue(&Q)
      DestroyQueue(&Q)
      ClearQueue(&Q)
      QueueEmpty(Q)
      QueueLength(Q)
      GetHead(Q, &e)
      EnQueue(&Q, e)
      DeQueue(&Q, &e)
  ```

#### 3.1.2 顺序队列与假溢出

- **普通顺序队列**：数组 + `front` + `rear`
  - 初始：`front = rear = 0`（指向 0 号位置或 0 号前）
  - 入队：`data[rear++] = x`
  - 出队：`return data[front++]`
  - 判空：`front == rear`
  - 判满：`rear == MAXSIZE`（这是**假溢出**问题）
- **假溢出**：经过多次入队出队后，`rear` 到了数组末尾但前面有空位
- **解决**：循环队列（环形队列）

#### 3.1.3 循环队列（环形队列）—— 重点

- **核心思想**：用**模运算**把数组首尾相接。
- **结构**：数组 + `front` + `rear`，都从 0 增长，到 `MAXSIZE` 模回 0。
- **入队**：`rear = (rear + 1) % MAXSIZE; data[rear] = x;`
- **出队**：`front = (front + 1) % MAXSIZE;`
- **判空**：`front == rear`
- **判满**（两种方法）：
  - **方法 1（牺牲一个存储格）**：`(rear + 1) % MAXSIZE == front`
    - 容量实际可用 = MAXSIZE - 1
  - **方法 2（额外 count 变量）**：`count == MAXSIZE`
    - 容量实际可用 = MAXSIZE
- **队列长度**：
  - 方法 1：`(rear - front + MAXSIZE) % MAXSIZE`
  - 方法 2：直接 `count`
- **时间复杂度**：所有操作 O(1)
- **C++ 实现**（方法 2，更直观）：
```cpp
class CirQueue {
    static const int MAXN = 105;
    int data[MAXN];
    int frontIdx, rearIdx, cnt;
public:
    CirQueue(): frontIdx(0), rearIdx(-1), cnt(0) {}
    bool empty() const { return cnt == 0; }
    bool full()  const { return cnt == MAXN; }
    int  size()  const { return cnt; }
    void clear() { frontIdx=0; rearIdx=-1; cnt=0; }
    void enqueue(int x) { rearIdx = (rearIdx+1)%MAXN; data[rearIdx] = x; cnt++; }
    void dequeue()      { frontIdx = (frontIdx+1)%MAXN; cnt--; }
    int  front() const  { return data[frontIdx]; }
};
```

#### 3.1.4 链式队列

- **结构**：单链表 + 头指针（队首）+ 尾指针（队尾）。
- **优点**：
  - 长度动态，不存在假溢出
  - 永不满（除内存耗尽）
- **缺点**：
  - 每个节点多一个指针
  - 不可随机访问
- **关键操作**：
  - 入队：从尾部插入，`tail->next = new; tail = new;`
  - 出队：从头部删除，`head = head->next;`（空链表时 tail 也要置空）
- **时间复杂度**：所有操作 O(1)

#### 3.1.5 双端队列（Deque）

- **定义**：两端都能入队和出队的队列。
- **操作**：
  - `push_front` / `pop_front` / `front()`
  - `push_back` / `pop_back` / `back()`
- **应用**：
  - 既是栈也是队列的泛化
  - 滑动窗口（单调队列）
  - 撤销/重做（双栈效果）
- **C++ STL**：`std::deque<T>`（双端队列）

#### 3.1.6 优先队列（Priority Queue）

- **定义**：元素带优先级，出队时优先级最高者先出。
- **实现**：
  - **堆（最常用）**：插入 O(log n)，删除最值 O(log n)
  - **有序数组**：插入 O(n)，删除 O(1)
  - **无序数组**：插入 O(1)，删除 O(n)
- **应用**：
  - 任务调度（操作系统）
  - Dijkstra 算法的核心
  - Huffman 编码
  - 数据流的中位数 / Top K 问题
- **C++ STL**：`std::priority_queue<T>`（默认大根堆）

#### 3.1.7 队列的应用

- **BFS（广度优先搜索）**：用队列实现，层序遍历
- **操作系统进程就绪队列**
- **消息队列（Message Queue）**：进程间通信
- **缓冲区管理**：生产者-消费者模型
- **打印机任务队列**
- **滑动窗口 / 单调队列**：求最值
- **树的层序遍历**
- **图的 BFS 最短路（无权图）**

#### 3.1.8 顺序队列 vs 链式队列

| 维度 | 顺序（循环） | 链式 |
|---|---|---|
| 存储 | 数组 | 链表 |
| 容量 | 固定 | 动态 |
| 假溢出 | 用循环解决 | 不存在 |
| 判满 | 需要 | 不需要 |
| 额外空间 | 0 | 每节点 1 指针 |
| 访问速度 | 快 | 慢 |
| 实现难度 | 模运算易错 | 中等 |

#### 3.1.9 常见考点

- 循环队列的判空判满（必考，注意**牺牲存储格**约定）
- 队列与栈的相互实现（两个栈实现队列 / 两个队列实现栈）
- BFS 用队列
- 优先队列的实现
- 双端队列的滑动窗口

## 第 4 章 链式存储的栈与队列（含循环链表）

### 4.1 理论全掌握

#### 4.1.1 链栈（详见 §2.1.4）

- 单链表头插法实现
- 优点：动态长度、永不栈满
- 缺点：指针开销、不能随机访问
- 适用：栈深度不可预知

#### 4.1.2 链式队列（详见 §3.1.4）

- 单链表 + 头尾指针
- 优点：动态长度、无假溢出
- 关键：**尾指针维护**（不然入队要 O(n) 遍历）

#### 4.1.3 循环链表

- **单循环链表**：尾节点 `next` 指向头
- **双循环链表**：头节点 `prev` 指向尾，尾 `next` 指向头
- **优点**：
  - 从任意节点出发可遍历全表
  - 尾插 / 头插 O(1)
  - 适合"绕圈"操作（如约瑟夫）
- **判空**：
  - 单循环：`head == head->next`（头节点同时指向自己）
  - 双循环：`head->next == head && head->prev == head`
- **判尾**：
  - 单循环：`p->next == head`
  - 双循环：`p->next == head`
- **约瑟夫问题**：用循环单链表，O(n·m)

#### 4.1.4 链表常见操作

- **头插**：O(1)
- **尾插**（带尾指针）：O(1)
- **按位查找**：O(n)
- **按值查找**：O(n)
- **中间插入**（已知前驱）：O(1)
- **删除**（已知前驱）：O(1)
- **链表反转**：三指针迭代 O(n) / 递归 O(n)
- **快慢指针找中点**：slow 走一步、fast 走两步
- **判环**：fast 与 slow 相遇则有环

#### 4.1.5 链式结构的应用

**(1) 约瑟夫问题（Josephus）**
- n 个人围圈，从 1 报数到 m 出列，循环往复
- 用循环单链表实现
- 时间复杂度：O(n·m) 朴素 / O(n log n) 数学公式

**(2) 多项式运算**
- 节点 `{coef, exp, next}`，按指数排序
- 加减：双指针归并
- 乘法：每个 A 项 × 整个 B 累加

**(3) 大整数运算**
- 节点存一位（十进制）或一长块（万进制）
- 加减：从低位加起 + 进位
- 乘法：每位 × 整个数 + 进位

**(4) 字符串的链式存储**
- 一般用顺序存储（`char[]` 或 `string`）即可
- 链式适合频繁插入/删除字符

#### 4.1.6 顺序 vs 链式 综合对比

| 操作 | 顺序（数组） | 链式（链表） |
|---|---|---|
| 访问第 i 个 | O(1) | O(n) |
| 头部插入/删除 | O(n) | O(1) |
| 尾部插入/删除 | O(1) | O(1)（带尾指针） |
| 中间插入/删除 | O(n) | O(1)（已知前驱） |
| 查找（无序） | O(n) | O(n) |
| 空间 | 预先固定 | 动态 |
| 额外开销 | 0 | 指针 |
| 适用场景 | 长度已知、查询多 | 长度变化、增删多 |

#### 4.1.7 常见考点

- 循环链表的建立与判空判尾
- 约瑟夫问题
- 链表反转（一趟 / 递归）
- 快慢指针（找中点、判环、找倒数第 k 个）
- 哑节点（dummy head）的好处

### 4.2 Lab06-题目1：链式存储的队列（q159, 困难, 50 分）

**题意**：实现 `MyLinkedQueue`（链式），用其将输入整数**顺序输出**。

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct QNode { int val; QNode* next; QNode(int v): val(v), next(nullptr) {} };

class MyLinkedQueue {
    QNode *head, *tail;     // head 出队端，tail 入队端
    int cnt;
public:
    MyLinkedQueue(): head(nullptr), tail(nullptr), cnt(0) {}
    ~MyLinkedQueue() { while (head) { QNode* t = head; head = head->next; delete t; } }
    bool empty() const { return cnt == 0; }
    bool full()  const { return false; }
    int  size()  const { return cnt; }
    void clear() { this->~MyLinkedQueue(); head=tail=nullptr; cnt=0; }
    void enqueue(int x) {
        QNode* n = new QNode(x);
        if (!tail) head = tail = n;
        else { tail->next = n; tail = n; }
        cnt++;
    }
    void dequeue() {
        QNode* t = head; head = head->next;
        if (!head) tail = nullptr;
        delete t; cnt--;
    }
    int front() const { return head->val; }
};

int main() {
    MyLinkedQueue q;
    int x; while (cin >> x && x != -1) q.enqueue(x);
    bool first = true;
    while (!q.empty()) {
        if (!first) cout << ' ';
        first = false;
        cout << q.front();
        q.dequeue();
    }
    cout << '\n';
}
```

### 4.3 Lab04-题目3：约瑟夫问题（q155, 中等, 20 分）

**题意**：n 个人（2..60）围圈，从 1 报数到 m 出列；n=8, m=3 → 输出 7。

**关键点**：
- 循环单链表（尾→头）
- `cur` 每次前进 m-1 步后删除 next
- 终止条件：链表剩 1 个节点

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct CNode { int id; CNode* next; CNode(int i): id(i), next(nullptr) {} };

int main() {
    int n, m; cin >> n >> m;
    CNode* head = new CNode(1);
    CNode* pre = head;
    for (int i = 2; i <= n; i++) { pre->next = new CNode(i); pre = pre->next; }
    pre->next = head;   // 成环

    CNode* cur = head;
    while (cur->next != cur) {       // 剩 1 个
        for (int k = 1; k < m; k++) cur = cur->next;
        CNode* del = cur->next;       // cur 报 m-1，del 报 m
        cur->next = del->next;
        delete del;
    }
    cout << cur->id << "\n";
}
```

### 4.4 Lab06-题目2：多项式的运算（q160, 中等, 50 分）

**题意**：多项式按指数降序输入，系数 + 指数成对，遇 `0 0` 结束。运算符 `+ - *`（乘法选做）。
- 输出 `5X^2` 这种格式，常数项写 `5`

**关键点**：
- 数据结构：链表（每项 `{coef, exp, next}`），按指数**降序**
- 加/减：双指针归并（指数大者先入）；相等指数则合并系数（减法取负）；系数为 0 则跳过
- 乘：每个 A 项 × 整个 B → 多项式，再加到结果中（外层加法用前面那个函数）
- 输出注意：
  - 第一项不带 `+` 号
  - 系数为 ±1 且指数 >0 时省略 1
  - 指数 = 0 不写 `X^0`；指数 = 1 写 `X`；否则 `X^e`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct Term { int coef, exp; Term* next; Term(int c,int e): coef(c),exp(e),next(nullptr) {} };

// 释放
void freePoly(Term* p) { while (p) { Term* t = p; p = p->next; delete t; } }

// 加法：a + sign*b（sign=+1 或 -1）
Term* polyAdd(Term* a, Term* b, int sign = 1) {
    Term dummy(0,0); Term* tail = &dummy; dummy.next = nullptr;
    while (a && b) {
        if (a->exp > b->exp) {
            tail->next = new Term(a->coef, a->exp); tail = tail->next; a = a->next;
        } else if (a->exp < b->exp) {
            tail->next = new Term(b->coef * sign, b->exp); tail = tail->next; b = b->next;
        } else {
            int c = a->coef + b->coef * sign;
            if (c != 0) { tail->next = new Term(c, a->exp); tail = tail->next; }
            a = a->next; b = b->next;
        }
    }
    while (a) { tail->next = new Term(a->coef, a->exp); tail = tail->next; a = a->next; }
    while (b) { tail->next = new Term(b->coef * sign, b->exp); tail = tail->next; b = b->next; }
    tail->next = nullptr;
    return dummy.next;
}

Term* polyMul(Term* a, Term* b) {
    Term* res = nullptr;
    for (Term* pa = a; pa; pa = pa->next) {
        Term* row = nullptr;
        for (Term* pb = b; pb; pb = pb->next) {
            Term* n = new Term(pa->coef * pb->coef, pa->exp + pb->exp);
            n->next = row; row = n;        // 头插，无所谓顺序
        }
        Term* tmp = polyAdd(res, row, 1);
        freePoly(res); freePoly(row);
        res = tmp;
    }
    return res;
}

// 读入
Term* readPoly() {
    Term dummy(0,0); Term* tail = &dummy;
    while (true) {
        int c, e; cin >> c >> e;
        if (c == 0 && e == 0) break;
        tail->next = new Term(c, e); tail = tail->next;
    }
    tail->next = nullptr;
    return dummy.next;
}

void printPoly(Term* p) {
    if (!p) { cout << "0\n"; return; }
    bool first = true;
    while (p) {
        int c = p->coef, e = p->exp;
        if (!first) cout << (c >= 0 ? "+" : "");
        if (e == 0) cout << c;
        else {
            if (c == 1) cout << "X";
            else if (c == -1) cout << "-X";
            else cout << c << "X";
            if (e != 1) cout << "^" << e;
        }
        first = false;
        p = p->next;
    }
    cout << "\n";
}

int main() {
    char op; cin >> op;
    Term *p1 = readPoly(), *p2 = readPoly();
    Term* res = nullptr;
    if (op == '+') res = polyAdd(p1, p2, 1);
    else if (op == '-') res = polyAdd(p1, p2, -1);
    else if (op == '*') res = polyMul(p1, p2);
    printPoly(res);
}
```

## 第 5 章 递归

### 5.1 理论全掌握

#### 5.1.1 递归的定义

- **递归（Recursion）**：一个函数**直接或间接**调用自己。
- **递归的本质**：把大问题分解为**同类子问题**。
- **直接递归**：`f() { f(); }`
- **间接递归**：`f() { g(); } g() { f(); }`
- **数学基础**：数学归纳法。

#### 5.1.2 递归三要素

1. **递归终止条件（Base Case / 出口）**：最小问题的解，不再递归
2. **递归假设**：假设子问题已经解决
3. **递归方程（递推式）**：用子问题构造原问题

**例子：阶乘**
```cpp
int fact(int n) {
    if (n <= 1) return 1;          // 终止
    return n * fact(n - 1);        // 假设 fact(n-1) 已求
}
```

#### 5.1.3 递归 vs 迭代

| 维度 | 递归 | 迭代 |
|---|---|---|
| 思想 | 自顶向下分解 | 自底向上递推 |
| 代码 | 简洁 | 繁琐 |
| 性能 | 函数调用开销 | 无额外开销 |
| 栈空间 | O(深度) | O(1) |
| 适用 | 问题天然递归 | 简单循环 |
| 风险 | 栈溢出 | 几乎无 |

**能递归一定能迭代，迭代更高效**。

#### 5.1.4 递归调用栈（Call Stack）

- 每次递归调用都把**现场**压栈：
  - 参数值
  - 局部变量
  - 返回地址
- 函数返回时弹栈恢复
- **栈深度 = 递归深度**；深度过大会**栈溢出**（stack overflow）
- C++ 默认栈 8MB 左右；递归 10⁵ 层可能溢出

#### 5.1.5 递归的两种视角

- **自顶向下（Top-down）**：从原问题出发，分解到子问题
- **自底向上（Bottom-up）**：从最小子问题出发，构造原问题
- 递归算法是**自顶向下**写出来的
- 迭代 / 动态规划是**自底向上**实现

#### 5.1.6 尾递归

- **定义**：递归调用是函数的**最后一步操作**
- **特点**：编译器可以优化为循环（栈帧复用）
- **C++ 不保证尾递归优化**（GCC/O2 通常会优化）
- **例子**：
  ```cpp
  // 尾递归
  int fact(int n, int acc) {
      if (n <= 1) return acc;
      return fact(n - 1, n * acc);   // 最后一步是递归调用
  }
  ```

#### 5.1.7 递归时间复杂度分析

**展开法**（代入法、递归树）：

例：`T(n) = 2T(n/2) + O(n)`（归并排序）
- 第 0 层：1 个 O(n) = O(n)
- 第 1 层：2 个 O(n/2) = O(n)
- ...
- 第 log n 层：n 个 O(1) = O(n)
- 总计：`O(n log n)`

**主定理**：`T(n) = aT(n/b) + O(n^d)`
- 若 `d > log_b a`：`T(n) = O(n^d)`
- 若 `d = log_b a`：`T(n) = O(n^d log n)`
- 若 `d < log_b a`：`T(n) = O(n^log_b a)`

#### 5.1.8 经典递归问题

| 问题 | 终止 | 递推 | 时间 | 空间 |
|---|---|---|---|---|
| 阶乘 | n=1 → 1 | n * fact(n-1) | O(n) | O(n) |
| 斐波那契 | f(0)=0, f(1)=1 | f(n-1) + f(n-2) | O(2ⁿ) 朴素 | O(n) |
| 整数划分 | n=0 → 1, m=0 → 0（n>0）| 多种 | O(?) | O(?) |
| 全排列 | 长度到 n | 选未用元素 | O(n!) | O(n) |
| 组合 | 选够 k 个 | 选/不选 | O(C(n,k)) | O(n) |
| 汉诺塔 | n=1 移动一次 | 三步递归 | O(2ⁿ) | O(n) |
| N 皇后 | 放完 / 冲突 | 逐行放 | O(n!) | O(n²) |
| 迷宫 DFS | 到出口 / 死路 | 4 方向 | O(4^n) | O(n²) |

#### 5.1.9 递归设计步骤

1. **找重复**：子问题是什么？
2. **找变化**：哪些参数随递归变化？
3. **找边界**：什么条件下不再递归？

**例子：二叉树遍历**
```cpp
void preorder(Node* p) {
    if (!p) return;            // 边界：空树
    visit(p);                  // 访问根
    preorder(p->left);         // 重复 1：左子树
    preorder(p->right);        // 重复 2：右子树
}
```

#### 5.1.10 递归 vs 回溯

- **回溯（Backtracking）**：递归的"剪枝版"
- 在递归基础上：
  - 走不通时**撤销选择**（状态恢复）
  - 剪去明显不合法的分支
- 经典：八皇后、数独、组合/排列、子集和

```cpp
void backtrack(vector<int>& path, vector<bool>& used) {
    if (path.size() == n) {  // 终止
        // 记录答案
        return;
    }
    for (int i = 0; i < n; i++) {
        if (used[i]) continue;   // 剪枝
        used[i] = true;          // 做选择
        path.push_back(i);
        backtrack(path, used);   // 递归
        path.pop_back();         // 撤销选择
        used[i] = false;
    }
}
```

#### 5.1.11 递归的应用

- **数学**：阶乘、斐波那契、组合数、Catalan 数
- **数据结构**：树的遍历、链表反转、图的 DFS
- **分治**：归并、快排、二分查找
- **回溯**：排列、子集、八皇后
- **动态规划**（递归+记忆化）
- **编译原理**：语法分析（递归下降）

#### 5.1.12 常见考点

- 写递归求阶乘/斐波那契/全排列
- 写递归求二叉树高度/遍历
- 写递归判断回文 / 字符串反转
- 分析递归的时间空间复杂度
- 递归转非递归（用栈模拟）
- 八皇后、迷宫

### 5.2 Lab25-题目1：递归求最小求和（q103, 困难, 50 分）

**题意**：用递归实现 `Min(arr, n)` 和 `Sum(arr, n)`。

**关键点**：
- `Sum(arr, n) = Sum(arr, n-1) + arr[n-1]`；`Sum(arr, 1) = arr[0]`
- `Min(arr, n) = min(Min(arr, n-1), arr[n-1])`；`Min(arr, 1) = arr[0]`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int Sum(int arr[], int n) { return n == 1 ? arr[0] : Sum(arr, n-1) + arr[n-1]; }
int Min(int arr[], int n) { return n == 1 ? arr[0] : min(Min(arr, n-1), arr[n-1]); }

int main() {
    int n; cin >> n;
    int* a = new int[n];
    for (int i = 0; i < n; i++) cin >> a[i];
    cout << Min(a, n) << ' ' << Sum(a, n) << "\n";
    delete[] a;
}
```

### 5.3 Lab25-题目2：二进制转十进制（q161, 中等, 50 分）

#include <bits/stdc++.h>

### 5.4 Lab07-题目1：n 个数的全排列（q163, 中等, 50 分）

**题意**：n ≤ 20，输出 1..n 的全排列（超过 10 个只输出前 10），并输出总排列数。

**关键点**：
- 回溯 + `used[]` 标记
- `dfs(k)`：已选 k 个，再选第 k+1 个
- `n!` 超过 10 之后停止输出（但仍可继续计算）

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int n, printed = 0;
vector<int> a;
vector<bool> used;

void dfs(int k) {
    if (k == n) {
        if (printed < 10) {
            cout << (printed+1) << ":";
            for (int i = 0; i < n; i++) cout << (i?" ":"") << a[i];
            cout << "\n";
            printed++;
        }
        return;
    }
    for (int i = 1; i <= n; i++) {
        if (!used[i]) {
            used[i] = true; a.push_back(i);
            dfs(k+1);
            used[i] = false; a.pop_back();
        }
    }
}

int main() {
    cin >> n;
    a.assign(n, 0);
    used.assign(n+1, false);
    dfs(0);
    long long total = 1;
    for (int i = 2; i <= n; i++) total *= i;
    cout << total << "\n";
}
```

### 5.5 Lab07-题目2：A Maze Problem（q164, 中等, 50 分）

**题意**：12×12 迷宫，O/W/E，3 个起点，问到出口的最短路径长度（起终点都算入），不可达输出 -1。

**关键点**：
- **BFS 求最短路**（不是 DFS，DFS 求的是任意路径）
- `dirs = {(1,0),(-1,0),(0,1),(0,-1)}`
- `O=0, W=1, E=2`；E 是终点
- BFS 起点 = (x, y)（注意题目说 x 是纵坐标，y 是横坐标）
- 距离 = `dist[x][y]`，终点 `dist[ex][ey]`
- 起始结束都算：长度 = 步数 + 1

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    const int N = 12;
    int grid[N][N];
    for (int i = 0; i < N; i++)
        for (int j = 0; j < N; j++) {
            char c; cin >> c;
            grid[i][j] = (c=='W'?1:(c=='E'?2:0));
        }
    // 找 E
    int ex=-1, ey=-1;
    for (int i=0;i<N;i++) for (int j=0;j<N;j++) if (grid[i][j]==2){ ex=i; ey=j; }

    // 3 个起点
    for (int k = 0; k < 3; k++) {
        int x, y; cin >> x >> y;     // (x,y) = (row, col)
        if (k) cout << ' ';
        if (grid[x][y] == 1) { cout << -1; continue; }

        int dist[N][N];
        for (int i=0;i<N;i++) for (int j=0;j<N;j++) dist[i][j] = -1;
        queue<pair<int,int>> q;
        dist[x][y] = 0; q.push({x,y});
        int d4[4][2] = {{-1,0},{1,0},{0,-1},{0,1}};
        while (!q.empty()) {
            auto [i,j] = q.front(); q.pop();
            for (auto& d : d4) {
                int ni = i+d[0], nj = j+d[1];
                if (ni<0||ni>=N||nj<0||nj>=N) continue;
                if (grid[ni][nj]==1) continue;
                if (dist[ni][nj]!=-1) continue;
                dist[ni][nj] = dist[i][j] + 1;
                q.push({ni,nj});
            }
        }
        if (dist[ex][ey] == -1) cout << -1;
        else cout << dist[ex][ey] + 1;     // 起终点都算
    }
    cout << "\n";
}
```
