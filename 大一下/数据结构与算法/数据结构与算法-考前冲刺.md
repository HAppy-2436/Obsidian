## 第 1 章 课程介绍

### 1.1 理论全掌握

#### 1.1.1 数据结构基本概念

- **数据（Data）**：客观事物的符号表示，是所有能输入到计算机并被程序处理的符号的总称。
- **数据元素（Data Element）**：数据的基本单位，通常作为一个整体考虑和处理（如一个学生、一条记录）。
- **数据项（Data Item）**：数据元素的不可分割的最小单位（如学生的学号、姓名）。
- **数据对象（Data Object）**：性质相同的数据元素的集合（如所有学生）。
- **数据结构（Data Structure）**：**相互之间存在一种或多种特定关系的数据元素的集合**。包括三方面：
  1. **数据的逻辑结构**：数据元素之间的逻辑关系（线性、树、图、集合）。
  2. **数据的存储结构（物理结构）**：数据在计算机中的存储方式（顺序、链式、索引、散列）。
  3. **数据的运算**：对数据施加的操作（增删改查、排序等）。

#### 1.1.2 逻辑结构四大类

| 类型 | 关系特征 | 典型 |
|---|---|---|
| **集合** | 同属一个集合，无其他关系 | 集合 |
| **线性** | 一对一 | 数组、链表、栈、队列 |
| **树** | 一对多 | 二叉树、堆 |
| **图** | 多对多 | 社交网络、地图 |

#### 1.1.3 存储结构四种

- **顺序存储**：逻辑上相邻的元素物理上也相邻（数组）。优点：随机访问 O(1)。缺点：插入删除 O(n)，需要连续空间。
- **链式存储**：元素任意存储，靠指针维系关系（链表）。优点：插入删除 O(1)。缺点：不能随机访问，额外指针开销。
- **索引存储**：建立索引表（关键字 → 地址）。优点：查找快。缺点：索引占空间、维护索引开销。
- **散列（哈希）存储**：用哈希函数直接算出地址。优点：期望 O(1) 查找。缺点：冲突处理复杂、不支持顺序访问。

#### 1.1.4 抽象数据类型（ADT）

- **定义**：一个数学模型 + 定义在该模型上的一组操作。**只关心"做什么"，不关心"怎么做"**。
- **三要素**：
  1. **数据对象（Data Object）**
  2. **数据关系（Data Relation）**
  3. **基本操作（Basic Operations）**
- **表示格式**：
  ```
  ADT 抽象数据类型名
  Data
      数据对象的定义
  Relation
      数据关系的定义
  Operation
      操作 1：初始条件 / 操作结果
      操作 2：...
  endADT
  ```
- **实现**：ADT 通过具体的存储结构和算法实现（C++ 的 class / struct）。

#### 1.1.5 算法五大特性

1. **有穷性**：必须在有限步后终止。
2. **确定性**：每一步骤无歧义。
3. **可行性**：每一步都能执行。
4. **输入**：零个或多个输入。
5. **输出**：一个或多个输出。

#### 1.1.6 算法评价指标

- **正确性**：满足问题需求。
- **可读性**：清晰易理解。
- **健壮性**：对非法输入能处理而非崩溃。
- **效率**：
  - **时间复杂度 T(n)**：基本操作次数随 n 的增长趋势（用大 O 表示）。
  - **空间复杂度 S(n)**：算法占用的额外存储空间。

#### 1.1.7 时间复杂度分析

- **常见量级**（从快到慢）：
  `O(1) < O(log n) < O(n) < O(n log n) < O(n²) < O(n³) < O(2ⁿ) < O(n!)`
- **计算方法**：
  - 顺序语句：加法
  - 选择语句：取 max
  - 循环语句：乘积
  - 递归：递推方程（主定理）
- **最好 / 最坏 / 平均情况**：
  - 最好：最佳输入
  - 最坏：最差输入
  - 平均：所有输入等概率时的期望
- **摊还时间**：一系列操作的总成本平均到每次（splay 树摊还 O(log n)）。

#### 1.1.8 主定理（Master Theorem）

递归式 `T(n) = aT(n/b) + O(n^d)`：
- `d > log_b a` → `T(n) = O(n^d)`
- `d = log_b a` → `T(n) = O(n^d log n)`
- `d < log_b a` → `T(n) = O(n^log_b a)`

**例**：归并排序 `T(n) = 2T(n/2) + O(n)`，a=2, b=2, d=1，`log_b a = 1 = d` → O(n log n)。

#### 1.1.9 C++ 类（Lab 必写）

```cpp
class MyClass {
public:
    MyClass();                                  // 默认构造
    MyClass(const MyClass& other);              // 拷贝构造
    MyClass(MyClass&& other);                   // 移动构造（C++11）
    MyClass& operator=(const MyClass& other);   // 拷贝赋值
    MyClass& operator=(MyClass&& other);        // 移动赋值
    ~MyClass();                                 // 析构
    // 业务方法
    void push(int x);
    int pop();
    int top() const;                            // const 成员函数不修改对象
    int size() const;
    bool empty() const;
private:
    int* data;
    int sz;
    int cap;
};
```

- **Rule of Three**：如果定义了析构、拷贝构造、拷贝赋值中任何一个，通常三个都要定义（资源管理）。
- **Rule of Five**（C++11）：再加移动构造、移动赋值。
- **const 正确性**：不修改对象的成员函数加 `const`。
- **RAII**：资源获取即初始化（智能指针 `unique_ptr` / `shared_ptr`）。

#### 1.1.10 数组

- **一维数组**：连续存储，类型相同。下标 `0..n-1`。
- **二维数组**：行优先 / 列优先存储。多维 → 一维映射：`a[i][j]` = `*(base + i*col + j)`。
- **时间复杂度**：
  - 随机访问 O(1)
  - 查找（无序）O(n)
  - 插入/删除 O(n)
  - 遍历 O(n)
- **特殊矩阵压缩**：
  - 对称矩阵：存一半（下三角）
  - 三角矩阵
  - 稀疏矩阵：三元组 / 十字链表

#### 1.1.11 指针与内存

- **指针**：存地址的变量。`int* p = &x;`
- **空指针**：`nullptr`（C++11 起，替代 `NULL`）
- **指针运算**：`p + 1` 跳过一个类型大小
- **动态内存**：`new` / `delete`（C++）或 `malloc` / `free`（C）
- **内存泄漏**：申请了没释放
- **悬挂指针**：指向已释放的内存
- **栈内存**：函数局部变量、自动管理
- **堆内存**：`new` 出来的、手动管理（智能指针更好）

#### 1.1.12 Life Game 通用思路

- **规则核心**：每个 cell 根据邻居状态决定下一代的生死。
- **编程套路**：
  1. 维护**两份网格**（当前 + 下一代），避免原地修改
  2. 每代遍历所有 cell，统计邻居数
  3. 按规则更新下一代
  4. 交换指针 / 数组，重复
- **关键技巧**：
  - 边界处理（无限网格 vs 有限网格）
  - 邻居方向（4 邻 / 8 邻 / 自定义）
  - 输出格式（`-` / `*` / `0` / `1`）

### 1.2 Lab01-题目1：One-Dimensional Life（q136, 中等, 50 分）

**题意**：一维格子，每个活细胞有 4 个邻居（左右各 1、2）。规则：
- 死细胞：邻居数 = 2 或 3 → 复活
- 活细胞：邻居数 = 0/1/3 → 死
- 活细胞：邻居数 = 2 或 4 → 活
- 总数 < 60。

**输入**：活细胞位置（1..60），以 `-1` 结束；n 代。
**输出**：第 n 代行（`-` 空 / `*` 活）。

**关键点**：
- 行宽 60，位置 1..60（不是 0..59）
- n=0 输出初始行
- 每代用新数组存，邻居 = 自身 + 左右各 1、2（共 4 个）

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    const int N = 60;
    vector<int> cur(N+5, 0), nxt(N+5, 0);
    int p;
    while (cin >> p && p != -1) cur[p] = 1;   // 1-indexed
    int g; cin >> g;
    for (int step = 1; step <= g; step++) {
        for (int i = 1; i <= N; i++) {
            int nb = cur[i-2] + cur[i-1] + cur[i+1] + cur[i+2];
            if (cur[i] == 1)
                nxt[i] = (nb == 2 || nb == 4) ? 1 : 0;
            else
                nxt[i] = (nb == 2 || nb == 3) ? 1 : 0;
        }
        cur = nxt;
    }
    for (int i = 1; i <= N; i++) cout << (cur[i] ? '*' : '-');
    cout << '\n';
    return 0;
}
```

### 1.3 Lab01-题目2：Magic Square（q135, 中等, 50 分）

**题意**：奇数 n（1..101）生成 n×n 魔方阵，方法：1 放顶行中间；后续数字沿右上对角走；撞顶/撞右则环绕；撞到已占位则正下方一格。

**关键点**：
- 行 = `(row - 1 + n) % n`，列 = `(col + 1) % n`
- 撞占位 → 行 = `(row + 1) % n`，列不变
- 输出：数字之间**两个空格**，行末无空格

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n; cin >> n;
    vector<vector<int>> a(n, vector<int>(n, 0));
    int r = 0, c = n/2;        // 1 放 (0, n/2)
    a[r][c] = 1;
    for (int k = 2; k <= n*n; k++) {
        int nr = (r - 1 + n) % n;
        int nc = (c + 1) % n;
        if (a[nr][nc] != 0) {  // 撞占位
            nr = (r + 1) % n;
            nc = c;
        }
        a[nr][nc] = k;
        r = nr; c = nc;
    }
    for (int i = 0; i < n; i++) {
        for (int j = 0; j < n; j++) {
            if (j) cout << "  ";
            cout << a[i][j];
        }
        cout << '\n';
    }
    return 0;
}
```

### 1.4 Lab21：二维 Life Game（q134, 中等, 100 分）

**题意**：20×60 棋盘；8 邻居；规则 B3/S23。
- 活细胞：邻居 0/1 → 死；2/3 → 活；≥4 → 死
- 死细胞：邻居 = 3 → 活

**关键点**：
- 输出范围 20×60 全网格
- 死 → `-`，活 → `*`
- 每代双 buffer 切换

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    const int R = 20, C = 60;
    vector<vector<int>> cur(R, vector<int>(C, 0));
    int x, y;
    while (cin >> x >> y) {
        if (x == -1 && y == -1) break;
        cur[x][y] = 1;   // 题目说 (x, y) 行列都从 1 开始
    }
    int g; cin >> g;
    for (int step = 0; step < g; step++) {
        vector<vector<int>> nxt(R, vector<int>(C, 0));
        for (int i = 0; i < R; i++) {
            for (int j = 0; j < C; j++) {
                int nb = 0;
                for (int di = -1; di <= 1; di++)
                    for (int dj = -1; dj <= 1; dj++) {
                        if (di == 0 && dj == 0) continue;
                        int ni = i+di, nj = j+dj;
                        if (ni>=0 && ni<R && nj>=0 && nj<C)
                            nb += cur[ni][nj];
                    }
                if (cur[i][j]) nxt[i][j] = (nb==2 || nb==3);
                else nxt[i][j] = (nb==3);
            }
        }
        cur = nxt;
    }
    for (int i = 0; i < R; i++) {
        for (int j = 0; j < C; j++)
            cout << (cur[i][j] ? '*' : '-');
        cout << '\n';
    }
    return 0;
}
```

## 第 2 章 栈

### 2.1 理论全掌握

#### 2.1.1 栈的定义与性质

- **栈（Stack）**：限定仅在**表尾**进行插入和删除操作的线性表。
- **LIFO**（Last In First Out）：后进先出。
- **栈顶（Top）**：允许操作的一端。
- **栈底（Bottom）**：固定不动的一端。
- **关键术语**：
  - **入栈 / 压栈（Push）**：栈顶插入元素
  - **出栈 / 弹栈（Pop）**：栈顶删除元素
  - **读栈顶（Top/Peek）**：看栈顶元素但不删除
  - **栈空（Empty）**、**栈满（Full）**（仅顺序栈）
- **ADT 接口**：
  ```
  ADT Stack
  Data
      一维数组 + 栈顶指针
  Operation
      InitStack(&S)
      DestroyStack(&S)
      ClearStack(&S)
      StackEmpty(S)
      StackLength(S)
      GetTop(S, &e)
      Push(&S, e)
      Pop(&S, &e)
  ```

#### 2.1.2 顺序栈（顺序存储）

- **结构**：数组 + 栈顶指针 `top`。
- **约定**：
  - `top = -1` 表示空栈
  - `top` 指向**栈顶元素**（不是下一个空位）
  - 进栈：`data[++top] = x`
  - 出栈：`return data[top--]`
  - 栈满：`top == capacity - 1`
- **另一种约定**（栈顶指向下一个空位）：
  - `top = 0` 表示空栈
  - 进栈：`data[top++] = x`
  - 出栈：`return data[--top]`
  - 栈满：`top == capacity`
- **时间复杂度**：所有操作 O(1)
- **空间复杂度**：O(capacity)
- **优缺点**：
  - ✅ 简单、随机访问、CPU 缓存友好
  - ❌ 容量固定（浪费 or 溢出）

**C++ 实现**：
```cpp
class Stack {
    static const int CAP = 105;
    int data[CAP];
    int top;
public:
    Stack(): top(-1) {}
    bool empty() const { return top == -1; }
    bool full()  const { return top == CAP-1; }
    int  size()  const { return top + 1; }
    void clear() { top = -1; }
    bool push(int x) { if (full()) return false; data[++top] = x; return true; }
    bool pop()        { if (empty()) return false; top--; return true; }
    int  peek() const { return data[top]; }
};
```

#### 2.1.3 两栈共享空间

- **场景**：两个**同类型**栈，需求此消彼长（如一个栈大时另一个小）。
- **方法**：一个数组从两端向中间生长。
  - 栈 1 从左端：`top1 = -1` 初始
  - 栈 2 从右端：`top2 = n` 初始
  - 栈满条件：`top1 + 1 == top2`
- **优点**：节省空间
- **缺点**：只能两栈共享

#### 2.1.4 链栈（链式存储）

- **结构**：单链表，**头插法**（头节点处即栈顶）。
- **优点**：
  - 不需要预先分配空间
  - 不存在栈满（除非系统内存耗尽）
  - 长度动态
- **缺点**：
  - 每个节点多一个指针
  - 不可随机访问
- **时间复杂度**：所有操作 O(1)
- **C++ 实现**：
```cpp
struct Node { int val; Node* next; Node(int v): val(v), next(nullptr) {} };
class LinkedStack {
    Node* topNode;
    int cnt;
public:
    LinkedStack(): topNode(nullptr), cnt(0) {}
    ~LinkedStack() { clear(); }
    bool empty() const { return cnt == 0; }
    int  size()  const { return cnt; }
    void clear() { while (topNode) { Node* t = topNode; topNode = topNode->next; delete t; } cnt = 0; }
    void push(int x) { Node* n = new Node(x); n->next = topNode; topNode = n; cnt++; }
    void pop()        { Node* t = topNode; topNode = topNode->next; delete t; cnt--; }
    int  top() const  { return topNode->val; }
};
```

#### 2.1.5 顺序栈 vs 链栈

| 维度 | 顺序栈 | 链栈 |
|---|---|---|
| 存储 | 数组 | 链表 |
| 容量 | 固定 | 动态 |
| 空间利用率 | 可能浪费 | 灵活 |
| 访问速度 | 快（连续） | 慢（指针） |
| 判满 | 需要 | 不需要 |
| 实现难度 | 简单 | 中等 |

#### 2.1.6 栈的应用（核心）

**(1) 括号匹配**
- 思路：扫描字符串，遇左括号入栈，遇右括号弹栈顶匹配
- 时间 O(n)，空间 O(n)
- 关键：处理嵌套、注释、字符串字面量

**(2) 表达式求值**
- **中缀表达式**（人写）：`2+3*4`
- **前缀表达式 / 波兰式**（运算符前）：`+ 2 * 3 4`
- **后缀表达式 / 逆波兰式**（运算符后）：`2 3 4 * +`
- **后缀求值算法**：
  - 扫描 token
  - 数字：入栈
  - 运算符：弹两个数（b = 栈顶, a = 次栈顶），计算 `a op b`，结果入栈
  - 结束：栈底为结果
- **中缀 → 后缀（调度场算法，Shunting-yard）**：
  - 数字：直接输出
  - `(`：入栈
  - `)`：弹栈到 `(` 为止
  - 运算符：弹出栈顶优先级 ≥ 当前的所有运算符，再把当前入栈
  - 结束：弹出栈中剩余运算符

**(3) 递归**
- **递归调用栈**：每次递归调用都把现场（参数、局部变量、返回地址）压栈
- **递归 = 栈的应用**（隐式）
- **递归转非递归**：用显式栈模拟

**(4) 深度优先搜索（DFS）**
- 用栈实现非递归 DFS

**(5) 函数调用 / 撤销操作（undo）**
- 撤销栈：每次操作前入栈当前状态，撤销时弹栈恢复

**(6) 浏览器前进/后退**
- 两个栈：back 栈和 forward 栈

**(7) 中缀表达式求值的双栈法**
- 一个**操作数栈**，一个**操作符栈**
- 数字入操作数栈
- 运算符按优先级比较，决定计算还是入栈

#### 2.1.7 常见考点

- 后缀表达式求值（必考）
- 栈的入栈出栈合法序列判断（Catalan 数：`C_n = (2n)! / ((n+1)! n!)`）
- 栈与递归的等价性
- 双端栈 / 共享栈
- 中缀→后缀转换

### 2.4 Lab02-题目1：10 以内的后序波兰式求解（q139, 中等, 33 分）

- 输入 `23+6*` → `(2+3)*6 = 30`

- 表达式是**一整行**，先 `getline` 再处理
- 数字字符 `'0'..'9'` 压栈
- 运算符 `+ - * /`：弹 b（后弹的右操作数），再弹 a，结果 `a op b` 压回
- 最终栈 size==1 才合法

**C++ 模板**：
```cpp
#include <bits/stdc++.h>

class MyStack {
    static const int MAXN = 105;
    int data[MAXN];
    int top;
public:
    MyStack(): top(-1) {}
    void push(int x) { data[++top] = x; }
    int pop()        { return data[top--]; }

};

int main() {
    string s; getline(cin, s);
    MyStack st;       // 用 class MyStack，而不是 vector
    for (char c : s) {
        if (c >= '0' && c <= '9') st.push(c - '0');
        else if (c=='+'||c=='-'||c=='*'||c=='/') {
            if (st.size() < 2) { cout << "none\n"; return 0; }

            int r = 0;
            if (c=='+') r = a + b;
            else if (c=='-') r = a - b;
            else if (c=='*') r = a * b;
            else r = a / b;               // 整数除
            st.push(r);
        }
    }
    if (st.size() == 1) cout << st.peek() << "\n";
    else                cout << "none\n";
```

### 2.6 Lab02-题目3：程序代码中的括号匹配（q142, 中等, 34 分）

**题意**：判断三类括号 `() [] {}` 是否匹配；**注释不参与**。
- 多行注释 `/* ... */`
- 单行注释 `// ...`

**关键点**：
- 维护一个 bool `inBlock`（多行注释内）和 `inLine`（单行注释内）
- 遇到 `/*` 且不在注释中 → `inBlock = true`
- 遇到 `*/` 且 `inBlock` → `inBlock = false`
- `//` 触发 `inLine = true` 直到换行
- 字符计数 = 各类括号总数，匹配 = 栈最终空

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

class MyStack {
    static const int MAXN = 2048;
    char data[MAXN];
    int top;
public:
    MyStack(): top(-1) {}
    void push(char c) { data[++top] = c; }
    char pop() { return data[top--]; }
    char peek() const { return data[top]; }
    bool empty() const { return top == -1; }
    bool nonEmpty() const { return top != -1; }
};

int main() {
    string all, line;
    // 读若干行直到 EOF
    bool first = true;
    while (getline(cin, line)) {
        if (!first) all += '\n';
        all += line;
        first = false;
    }

    MyStack st;       // 用 class MyStack
    int cnt = 0;
    bool inBlock = false, inLine = false;
    bool ok = true;
    for (size_t i = 0; i < all.size(); i++) {
        char c = all[i], nxt = (i+1<all.size()? all[i+1]:'\0');
        if (inLine) { if (c=='\n') inLine = false; continue; }
        if (inBlock) { if (c=='*' && nxt=='/') { inBlock = false; i++; } continue; }
        if (c=='/' && nxt=='/') { inLine = true; i++; continue; }
        if (c=='/' && nxt=='*') { inBlock = true; i++; continue; }
        if (c=='('||c=='['||c=='{') { st.push(c); cnt++; }
        else if (c==')'||c==']'||c=='}') {
            cnt++;
            char m = (c==')')?'(':(c==']')?'[':'{';
            if (st.empty() || st.pop() != m) ok = false;
        }
    }
    if (st.nonEmpty()) ok = false;
    cout << cnt << ' ' << (ok ? "yes" : "no") << "\n";

```

### 2.7 Lab20-题目1：中序波兰式求解（q143, 简单, 100 分）

**题意**：中缀表达式求值，操作符 `+ - * /`，含括号；可能含浮点、负数。整数直接输出；浮点保留 3 位小数。

**关键点**：
- **双栈法**（操作数栈 + 操作符栈）是最稳的写法
- 优先级：`(` < `+ -` < `* /`
- 遇到 `(` 入栈；遇到 `)` 弹到 `(`；数字（含负数、小数）压栈
- 优先级比较：当前 op ≤ 栈顶 op 时，先计算栈顶

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

// 自定义栈模板
template <class T>
class MyStack {
    static const int MAXN = 1024;
    T data[MAXN];
    int top;
public:
    MyStack(): top(-1) {}
    void push(T x) { data[++top] = x; }
    T pop()        { return data[top--]; }
    T peek() const { return data[top]; }
    bool empty() const { return top == -1; }
    int size() const { return top + 1; }
};

int pri(char c) {
    if (c == '+' || c == '-') return 1;
    if (c == '*' || c == '/') return 2;
    return 0;
}

void calc(MyStack<double>& num, char op) {
    double b = num.pop();
    double a = num.pop();
    double r = 0;
    if (op=='+') r=a+b; else if (op=='-') r=a-b;
    else if (op=='*') r=a*b; else r=a/b;
    num.push(r);
}

int main() {
    string s; getline(cin, s);
    MyStack<double> num;       // 操作数栈
    MyStack<char>   op;        // 操作符栈
    int n = s.size();
    for (int i = 0; i < n; ) {
        char c = s[i];
        if (c==' ' || c=='\t') { i++; continue; }
        if (c=='(') { op.push(c); i++; }
        else if (c==')') {
            while (!op.empty() && op.peek()!='(') calc(num, op.pop());
            if (!op.empty()) op.pop();   // 弹出 (
            i++;
        }
        else if (c=='+'||c=='-'||c=='*'||c=='/') {
            // 一元负号：表达式开头 / 前一个 ( / 运算符后
            if (c=='-' && (i==0 || s[i-1]=='(' || strchr("+-*/", s[i-1]))) {
                num.push(0);   // 把 -x 看成 0-x
            }
            while (!op.empty() && op.peek()!='(' && pri(op.peek()) >= pri(c))
                calc(num, op.pop());
            op.push(c); i++;
        }
        else { // 数字
            int j = i;
            while (j<n && (isdigit(s[j]) || s[j]=='.')) j++;
            num.push(stod(s.substr(i, j-i)));
            i = j;
        }
    }
    while (!op.empty()) calc(num, op.pop());
    double ans = num.peek();
    if (ans == (long long)ans) cout << (long long)ans << "\n";
    else                        printf("%.3f\n", ans);
}
```

## 第 3 章 队列

### 3.1 理论全掌握

#### 3.1.1 队列的定义与性质

- **队列（Queue）**：只允许在**一端**（队尾）插入、在**另一端**（队首）删除的线性表。
- **FIFO**（First In First Out）：先进先出。
- **队首（Front/Head）**：允许删除的一端。
- **队尾（Rear/Tail）**：允许插入的一端。
- **关键术语**：
  - **入队（Enqueue）**：队尾插入
  - **出队（Dequeue）**：队首删除
  - **读队首（Front/Peek）**：看队首元素
  - **队空（Empty）**、**队满（Full）**（仅顺序队列）
- **ADT 接口**：
  ```
  ADT Queue
  Data
      一维数组 + 队首、队尾指针
  Operation
      InitQueue(&Q)
      DestroyQueue(&Q)
      ClearQueue(&Q)
      QueueEmpty(Q)
      QueueLength(Q)
      GetHead(Q, &e)
      EnQueue(&Q, e)
      DeQueue(&Q, &e)
  ```

#### 3.1.2 顺序队列与假溢出

- **普通顺序队列**：数组 + `front` + `rear`
  - 初始：`front = rear = 0`（指向 0 号位置或 0 号前）
  - 入队：`data[rear++] = x`
  - 出队：`return data[front++]`
  - 判空：`front == rear`
  - 判满：`rear == MAXSIZE`（这是**假溢出**问题）
- **假溢出**：经过多次入队出队后，`rear` 到了数组末尾但前面有空位
- **解决**：循环队列（环形队列）

#### 3.1.3 循环队列（环形队列）—— 重点

- **核心思想**：用**模运算**把数组首尾相接。
- **结构**：数组 + `front` + `rear`，都从 0 增长，到 `MAXSIZE` 模回 0。
- **入队**：`rear = (rear + 1) % MAXSIZE; data[rear] = x;`
- **出队**：`front = (front + 1) % MAXSIZE;`
- **判空**：`front == rear`
- **判满**（两种方法）：
  - **方法 1（牺牲一个存储格）**：`(rear + 1) % MAXSIZE == front`
    - 容量实际可用 = MAXSIZE - 1
  - **方法 2（额外 count 变量）**：`count == MAXSIZE`
    - 容量实际可用 = MAXSIZE
- **队列长度**：
  - 方法 1：`(rear - front + MAXSIZE) % MAXSIZE`
  - 方法 2：直接 `count`
- **时间复杂度**：所有操作 O(1)
- **C++ 实现**（方法 2，更直观）：
```cpp
class CirQueue {
    static const int MAXN = 105;
    int data[MAXN];
    int frontIdx, rearIdx, cnt;
public:
    CirQueue(): frontIdx(0), rearIdx(-1), cnt(0) {}
    bool empty() const { return cnt == 0; }
    bool full()  const { return cnt == MAXN; }
    int  size()  const { return cnt; }
    void clear() { frontIdx=0; rearIdx=-1; cnt=0; }
    void enqueue(int x) { rearIdx = (rearIdx+1)%MAXN; data[rearIdx] = x; cnt++; }
    void dequeue()      { frontIdx = (frontIdx+1)%MAXN; cnt--; }
    int  front() const  { return data[frontIdx]; }
};
```

#### 3.1.4 链式队列

- **结构**：单链表 + 头指针（队首）+ 尾指针（队尾）。
- **优点**：
  - 长度动态，不存在假溢出
  - 永不满（除内存耗尽）
- **缺点**：
  - 每个节点多一个指针
  - 不可随机访问
- **关键操作**：
  - 入队：从尾部插入，`tail->next = new; tail = new;`
  - 出队：从头部删除，`head = head->next;`（空链表时 tail 也要置空）
- **时间复杂度**：所有操作 O(1)

#### 3.1.5 双端队列（Deque）

- **定义**：两端都能入队和出队的队列。
- **操作**：
  - `push_front` / `pop_front` / `front()`
  - `push_back` / `pop_back` / `back()`
- **应用**：
  - 既是栈也是队列的泛化
  - 滑动窗口（单调队列）
  - 撤销/重做（双栈效果）
- **C++ STL**：`std::deque<T>`（双端队列）

#### 3.1.6 优先队列（Priority Queue）

- **定义**：元素带优先级，出队时优先级最高者先出。
- **实现**：
  - **堆（最常用）**：插入 O(log n)，删除最值 O(log n)
  - **有序数组**：插入 O(n)，删除 O(1)
  - **无序数组**：插入 O(1)，删除 O(n)
- **应用**：
  - 任务调度（操作系统）
  - Dijkstra 算法的核心
  - Huffman 编码
  - 数据流的中位数 / Top K 问题
- **C++ STL**：`std::priority_queue<T>`（默认大根堆）

#### 3.1.7 队列的应用

- **BFS（广度优先搜索）**：用队列实现，层序遍历
- **操作系统进程就绪队列**
- **消息队列（Message Queue）**：进程间通信
- **缓冲区管理**：生产者-消费者模型
- **打印机任务队列**
- **滑动窗口 / 单调队列**：求最值
- **树的层序遍历**
- **图的 BFS 最短路（无权图）**

#### 3.1.8 顺序队列 vs 链式队列

| 维度 | 顺序（循环） | 链式 |
|---|---|---|
| 存储 | 数组 | 链表 |
| 容量 | 固定 | 动态 |
| 假溢出 | 用循环解决 | 不存在 |
| 判满 | 需要 | 不需要 |
| 额外空间 | 0 | 每节点 1 指针 |
| 访问速度 | 快 | 慢 |
| 实现难度 | 模运算易错 | 中等 |

#### 3.1.9 常见考点

- 循环队列的判空判满（必考，注意**牺牲存储格**约定）
- 队列与栈的相互实现（两个栈实现队列 / 两个队列实现栈）
- BFS 用队列
- 优先队列的实现
- 双端队列的滑动窗口

```cpp
    int  size()  const { return cnt; }
    void clear() { frontIdx=0; rearIdx=-1; cnt=0; }
    void enqueue(int x) { rearIdx = (rearIdx+1)%MAXN; data[rearIdx] = x; cnt++; }
    void dequeue()      { frontIdx = (frontIdx+1)%MAXN; cnt--; }
    int  front() const  { return data[frontIdx]; }
};

int main() {
    CirQueue q;
    int x; while (cin >> x && x != -1) q.enqueue(x);
    bool first = true;
    while (!q.empty()) {
        if (!first) cout << ' ';
        first = false;
        cout << q.front();
        q.dequeue();
    }
    cout << '\n';
}
```

### 3.4 Lab03-题目2：双端队列（q147, 中等, 33 分）

**题意**：实现 Deque，存最近 n 条操作记录（n=1..50），按"由近到远"输出（即倒序）。
- 历史上传超 n 条 → 只留最近 n 条
- 总数小于 n → 全输出

**关键点**：
- 维护 `vector<char>` 容器
- 入队（队尾）：`push_back`；满了就 `erase(begin())` 删最老的；或者用循环 buffer
- 输出：从 `back()` 倒着遍历到 `front()`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

// 自定义双端队列（顺序存储 + 模运算）
class MyDeque {
    static const int MAXN = 105;
    char data[MAXN];
    int head, tail, cnt;       // head 出队端，tail 入队端
public:
    MyDeque(): head(0), tail(-1), cnt(0) {}
    bool empty() const { return cnt == 0; }
    bool full()  const { return cnt == MAXN; }
    int  size()  const { return cnt; }
    void push_back(char c) { tail = (tail+1)%MAXN; data[tail] = c; cnt++; }
    void pop_front()      { head = (head+1)%MAXN; cnt--; }
    char front() const { return data[head]; }
    char back()  const { return data[tail]; }
};

int main() {
    int n; cin >> n;
    MyDeque dq;             // 用 class MyDeque
    char c;
    while (cin >> c) dq.push_back(c);
    while (dq.size() > n) dq.pop_front();   // 只留最近 n 条
    // 把内容存到数组再倒序输出（环形队列不便于直接倒序遍历）
    char buf[105];
    int len = 0;
    while (!dq.empty()) { buf[len++] = dq.front(); dq.pop_front(); }
    bool first = true;
    for (int i = len-1; i >= 0; i--) {
        if (!first) cout << ' ';
        first = false; cout << buf[i];
    }
    cout << '\n';
}
```

### 3.5 Lab03-题目3：团队队列（q150, 中等, 34 分）

**题意**：ENQUEUE x y（团队 x，编号 y）入队；若有队友在队，新人站到该团队**最后一名**之后；否则站到总队尾。DEQUEUE 出队首。STOP 结束。

**关键点**：
- 数据结构：外层是"团队顺序队列"，内层是"每个团队的成员队列"
- 维护一个 `member_to_team` 哈希（或数组）把编号映射回团队
- 新人到来：
  - 在 `member_to_team` 查 y 所属团队 t（题目输入即给出 t，简化）
  - 团队 t 不在团队顺序中 → 加到队尾
  - 把 y 加到团队 t 自己的队列尾
- DEQUEUE：
  - 取团队顺序首团队 t，弹出其队首成员 m
  - 团队 t 空了 → 从团队顺序里删掉 t
- 输出 m

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

// 自定义队列（顺序循环）
class MyQueue {
    static const int MAXN = 1024;
    int data[MAXN];
    int head, tail, cnt;
public:
    MyQueue(): head(0), tail(-1), cnt(0) {}
    bool empty() const { return cnt == 0; }
    void push(int x) { tail = (tail+1)%MAXN; data[tail] = x; cnt++; }
    int front() const { return data[head]; }
    void pop() { head = (head+1)%MAXN; cnt--; }
};

// 团队队列 = 外层团队顺序 + 内层每个团队的成员队列
class TeamQueue {
    MyQueue teamOrder;                              // 团队进入顺序
    unordered_map<int, MyQueue> memberQueue;        // 团队 -> 成员队列
public:
    void enqueue(int team, int id) {
        if (memberQueue[team].empty()) teamOrder.push(team);
        memberQueue[team].push(id);
    }
    int dequeue() {
        int t = teamOrder.front();
        int m = memberQueue[t].front();
        memberQueue[t].pop();
        if (memberQueue[t].empty()) teamOrder.pop();
        return m;
    }
};

int main() {
    int t, kase = 1;
    while (cin >> t && t) {
        cout << "Scenario #" << kase++ << "\n";
        TeamQueue tq;
        string op;
        while (cin >> op) {
            if (op == "STOP") break;
            if (op == "ENQUEUE") {
                int x, y; cin >> x >> y;
                tq.enqueue(x, y);
            } else if (op == "DEQUEUE") {
                cout << tq.dequeue() << "\n";
            }
        }
        cout << "\n";
    }
}
```

## 第 4 章 链式存储的栈与队列（含循环链表）

### 4.1 理论全掌握

#### 4.1.1 链栈（详见 §2.1.4）

- 单链表头插法实现
- 优点：动态长度、永不栈满
- 缺点：指针开销、不能随机访问
- 适用：栈深度不可预知

#### 4.1.2 链式队列（详见 §3.1.4）

- 单链表 + 头尾指针
- 优点：动态长度、无假溢出
- 关键：**尾指针维护**（不然入队要 O(n) 遍历）

#### 4.1.3 循环链表

- **单循环链表**：尾节点 `next` 指向头
- **双循环链表**：头节点 `prev` 指向尾，尾 `next` 指向头
- **优点**：
  - 从任意节点出发可遍历全表
  - 尾插 / 头插 O(1)
  - 适合"绕圈"操作（如约瑟夫）
- **判空**：
  - 单循环：`head == head->next`（头节点同时指向自己）
  - 双循环：`head->next == head && head->prev == head`
- **判尾**：
  - 单循环：`p->next == head`
  - 双循环：`p->next == head`
- **约瑟夫问题**：用循环单链表，O(n·m)

#### 4.1.4 链表常见操作

- **头插**：O(1)
- **尾插**（带尾指针）：O(1)
- **按位查找**：O(n)
- **按值查找**：O(n)
- **中间插入**（已知前驱）：O(1)
- **删除**（已知前驱）：O(1)
- **链表反转**：三指针迭代 O(n) / 递归 O(n)
- **快慢指针找中点**：slow 走一步、fast 走两步
- **判环**：fast 与 slow 相遇则有环

#### 4.1.5 链式结构的应用

**(1) 约瑟夫问题（Josephus）**
- n 个人围圈，从 1 报数到 m 出列，循环往复
- 用循环单链表实现
- 时间复杂度：O(n·m) 朴素 / O(n log n) 数学公式

**(2) 多项式运算**
- 节点 `{coef, exp, next}`，按指数排序
- 加减：双指针归并
- 乘法：每个 A 项 × 整个 B 累加

**(3) 大整数运算**
- 节点存一位（十进制）或一长块（万进制）
- 加减：从低位加起 + 进位
- 乘法：每位 × 整个数 + 进位

**(4) 字符串的链式存储**
- 一般用顺序存储（`char[]` 或 `string`）即可
- 链式适合频繁插入/删除字符

#### 4.1.6 顺序 vs 链式 综合对比

| 操作 | 顺序（数组） | 链式（链表） |
|---|---|---|
| 访问第 i 个 | O(1) | O(n) |
| 头部插入/删除 | O(n) | O(1) |
| 尾部插入/删除 | O(1) | O(1)（带尾指针） |
| 中间插入/删除 | O(n) | O(1)（已知前驱） |
| 查找（无序） | O(n) | O(n) |
| 空间 | 预先固定 | 动态 |
| 额外开销 | 0 | 指针 |
| 适用场景 | 长度已知、查询多 | 长度变化、增删多 |

#### 4.1.7 常见考点

- 循环链表的建立与判空判尾
- 约瑟夫问题
- 链表反转（一趟 / 递归）
- 快慢指针（找中点、判环、找倒数第 k 个）
- 哑节点（dummy head）的好处

### 4.2 Lab06-题目1：链式存储的队列（q159, 困难, 50 分）

**题意**：实现 `MyLinkedQueue`（链式），用其将输入整数**顺序输出**。

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct QNode { int val; QNode* next; QNode(int v): val(v), next(nullptr) {} };

class MyLinkedQueue {
    QNode *head, *tail;     // head 出队端，tail 入队端
    int cnt;
public:
    MyLinkedQueue(): head(nullptr), tail(nullptr), cnt(0) {}
    ~MyLinkedQueue() { while (head) { QNode* t = head; head = head->next; delete t; } }
    bool empty() const { return cnt == 0; }

    int  size()  const { return cnt; }

    void enqueue(int x) {
        QNode* n = new QNode(x);
        if (!tail) head = tail = n;
        else { tail->next = n; tail = n; }
        cnt++;
    }
    void dequeue() {
        QNode* t = head; head = head->next;
        if (!head) tail = nullptr;
        delete t; cnt--;
    }
    int front() const { return head->val; }
};

int main() {

        cout << q.front();
        q.dequeue();
    }
```

### 4.3 Lab04-题目3：约瑟夫问题（q155, 中等, 20 分）

**题意**：n 个人（2..60）围圈，从 1 报数到 m 出列；n=8, m=3 → 输出 7。

**关键点**：
- 循环单链表（尾→头）
- `cur` 每次前进 m-1 步后删除 next
- 终止条件：链表剩 1 个节点

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

// 循环链表节点
class CNode {
public:
    int id;
    CNode* next;
    CNode(int i): id(i), next(nullptr) {}
};

// 循环链表封装约瑟夫问题
class JosephusCircle {
    CNode* cur;       // 当前报数者
public:
    JosephusCircle(int n) {
        cur = new CNode(1);
        CNode* pre = cur;
        for (int i = 2; i <= n; i++) {
            pre->next = new CNode(i);
            pre = pre->next;
        }
        pre->next = cur;      // 成环
    }
    ~JosephusCircle() {
        // 释放
        CNode* p = cur->next;
        while (p != cur) { CNode* t = p; p = p->next; delete t; }
        delete cur;
    }
    // 报 m 出列，cur 前进 m-1 步，删除 next
    void eliminate(int m) {
        for (int k = 1; k < m; k++) cur = cur->next;
        CNode* del = cur->next;   // cur 报 m-1，del 报 m（出列）
        cur->next = del->next;
        if (del == cur) cur = nullptr;   // 删完只剩 cur 自己
        else            cur = del->next;
        delete del;
    }
    bool single() const { return cur && cur->next == cur; }
    int  winner() const { return cur ? cur->id : -1; }
};

int main() {
    int n, m; cin >> n >> m;
    JosephusCircle jc(n);
    while (!jc.single()) {
        jc.eliminate(m);
    }
    cout << jc.winner() << "\n";
}
```

### 4.4 Lab06-题目2：多项式的运算（q160, 中等, 50 分）

**题意**：多项式按指数降序输入，系数 + 指数成对，遇 `0 0` 结束。运算符 `+ - *`（乘法选做）。
- 输出 `5X^2` 这种格式，常数项写 `5`

**关键点**：
- 数据结构：链表（每项 `{coef, exp, next}`），按指数**降序**
- 加/减：双指针归并（指数大者先入）；相等指数则合并系数（减法取负）；系数为 0 则跳过
- 乘：每个 A 项 × 整个 B → 多项式，再加到结果中（外层加法用前面那个函数）
- 输出注意：
  - 第一项不带 `+` 号
  - 系数为 ±1 且指数 >0 时省略 1
  - 指数 = 0 不写 `X^0`；指数 = 1 写 `X`；否则 `X^e`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

// 多项式项（链表节点）
class Term {
public:
    int coef, exp;
    Term* next;
    Term(int c, int e): coef(c), exp(e), next(nullptr) {}
};

// 多项式（按指数降序）
class Polynomial {
    Term* head;       // 哑节点
    void copyFrom(const Polynomial& other);
public:
    Polynomial(): head(new Term(0,0)) { head->next = nullptr; }
    Polynomial(const Polynomial& other) { copyFrom(other); }
    ~Polynomial() {
        while (head) { Term* t = head; head = head->next; delete t; }
    }
    Polynomial& operator=(const Polynomial& other) {
        if (this != &other) {
            while (head) { Term* t = head; head = head->next; delete t; }
            head = new Term(0,0); head->next = nullptr;
            copyFrom(other);
        }
        return *this;
    }
    void read() {
        Term* tail = head;
        while (true) {
            int c, e; cin >> c >> e;
            if (c == 0 && e == 0) break;
            tail->next = new Term(c, e); tail = tail->next;
        }
        tail->next = nullptr;
    }
    void print() const {
        if (!head->next) { cout << "0\n"; return; }
        bool first = true;
        for (Term* p = head->next; p; p = p->next) {
            int c = p->coef, e = p->exp;
            if (!first) cout << (c >= 0 ? "+" : "");
            if (e == 0) cout << c;
            else {
                if (c == 1) cout << "X";
                else if (c == -1) cout << "-X";
                else cout << c << "X";
                if (e != 1) cout << "^" << e;
            }
            first = false;
        }
        cout << "\n";
    }
    // 加法：a + sign*b（sign=+1 加 / -1 减）
    static Polynomial add(const Polynomial& a, const Polynomial& b, int sign = 1) {
        Polynomial r;
        Term *ta = a.head->next, *tb = b.head->next, *tail = r.head;
        while (ta && tb) {
            if (ta->exp > tb->exp) {
                tail->next = new Term(ta->coef, ta->exp); tail = tail->next; ta = ta->next;
            } else if (ta->exp < tb->exp) {
                tail->next = new Term(tb->coef * sign, tb->exp); tail = tail->next; tb = tb->next;
            } else {
                int c = ta->coef + tb->coef * sign;
                if (c != 0) { tail->next = new Term(c, ta->exp); tail = tail->next; }
                ta = ta->next; tb = tb->next;
            }
        }
        while (ta) { tail->next = new Term(ta->coef, ta->exp); tail = tail->next; ta = ta->next; }
        while (tb) { tail->next = new Term(tb->coef * sign, tb->exp); tail = tail->next; tb = tb->next; }
        tail->next = nullptr;
        return r;
    }
    // 乘法：每个 A 项 × 整个 B 累加
    static Polynomial mul(const Polynomial& a, const Polynomial& b) {
        Polynomial res;
        for (Term* pa = a.head->next; pa; pa = pa->next) {
            Polynomial row;
            Term* tail = row.head;
            for (Term* pb = b.head->next; pb; pb = pb->next) {
                tail->next = new Term(pa->coef * pb->coef, pa->exp + pb->exp);
                tail = tail->next;
            }
            tail->next = nullptr;
            Polynomial tmp = add(res, row, 1);
            res = tmp;
        }
        return res;
    }
};

void Polynomial::copyFrom(const Polynomial& other) {
    head = new Term(0,0); head->next = nullptr;
    Term* tail = head;
    for (Term* p = other.head->next; p; p = p->next) {
        tail->next = new Term(p->coef, p->exp);
        tail = tail->next;
    }
    tail->next = nullptr;
}

int main() {
    char op; cin >> op;
    Polynomial p1, p2;
    p1.read(); p2.read();
    Polynomial res;
    if (op == '+')      res = Polynomial::add(p1, p2,  1);
    else if (op == '-') res = Polynomial::add(p1, p2, -1);
    else if (op == '*') res = Polynomial::mul(p1, p2);
    res.print();
}
```

## 第 5 章 递归

### 5.1 理论全掌握

#### 5.1.1 递归的定义

- **递归（Recursion）**：一个函数**直接或间接**调用自己。
- **递归的本质**：把大问题分解为**同类子问题**。
- **直接递归**：`f() { f(); }`
- **间接递归**：`f() { g(); } g() { f(); }`
- **数学基础**：数学归纳法。

#### 5.1.2 递归三要素

1. **递归终止条件（Base Case / 出口）**：最小问题的解，不再递归
2. **递归假设**：假设子问题已经解决
3. **递归方程（递推式）**：用子问题构造原问题

**例子：阶乘**
```cpp
int fact(int n) {
    if (n <= 1) return 1;          // 终止
    return n * fact(n - 1);        // 假设 fact(n-1) 已求
}
```

#### 5.1.3 递归 vs 迭代

| 维度 | 递归 | 迭代 |
|---|---|---|
| 思想 | 自顶向下分解 | 自底向上递推 |
| 代码 | 简洁 | 繁琐 |
| 性能 | 函数调用开销 | 无额外开销 |
| 栈空间 | O(深度) | O(1) |
| 适用 | 问题天然递归 | 简单循环 |
| 风险 | 栈溢出 | 几乎无 |

**能递归一定能迭代，迭代更高效**。

#### 5.1.4 递归调用栈（Call Stack）

- 每次递归调用都把**现场**压栈：
  - 参数值
  - 局部变量
  - 返回地址
- 函数返回时弹栈恢复
- **栈深度 = 递归深度**；深度过大会**栈溢出**（stack overflow）
- C++ 默认栈 8MB 左右；递归 10⁵ 层可能溢出

#### 5.1.5 递归的两种视角

- **自顶向下（Top-down）**：从原问题出发，分解到子问题
- **自底向上（Bottom-up）**：从最小子问题出发，构造原问题
- 递归算法是**自顶向下**写出来的
- 迭代 / 动态规划是**自底向上**实现

#### 5.1.6 尾递归

- **定义**：递归调用是函数的**最后一步操作**
- **特点**：编译器可以优化为循环（栈帧复用）
- **C++ 不保证尾递归优化**（GCC/O2 通常会优化）
- **例子**：
  ```cpp
  // 尾递归
  int fact(int n, int acc) {
      if (n <= 1) return acc;
      return fact(n - 1, n * acc);   // 最后一步是递归调用
  }
  ```

#### 5.1.7 递归时间复杂度分析

**展开法**（代入法、递归树）：

例：`T(n) = 2T(n/2) + O(n)`（归并排序）
- 第 0 层：1 个 O(n) = O(n)
- 第 1 层：2 个 O(n/2) = O(n)
- ...
- 第 log n 层：n 个 O(1) = O(n)
- 总计：`O(n log n)`

**主定理**：`T(n) = aT(n/b) + O(n^d)`
- 若 `d > log_b a`：`T(n) = O(n^d)`
- 若 `d = log_b a`：`T(n) = O(n^d log n)`
- 若 `d < log_b a`：`T(n) = O(n^log_b a)`

#### 5.1.8 经典递归问题

| 问题 | 终止 | 递推 | 时间 | 空间 |
|---|---|---|---|---|
| 阶乘 | n=1 → 1 | n * fact(n-1) | O(n) | O(n) |
| 斐波那契 | f(0)=0, f(1)=1 | f(n-1) + f(n-2) | O(2ⁿ) 朴素 | O(n) |
| 整数划分 | n=0 → 1, m=0 → 0（n>0）| 多种 | O(?) | O(?) |
| 全排列 | 长度到 n | 选未用元素 | O(n!) | O(n) |
| 组合 | 选够 k 个 | 选/不选 | O(C(n,k)) | O(n) |
| 汉诺塔 | n=1 移动一次 | 三步递归 | O(2ⁿ) | O(n) |
| N 皇后 | 放完 / 冲突 | 逐行放 | O(n!) | O(n²) |
| 迷宫 DFS | 到出口 / 死路 | 4 方向 | O(4^n) | O(n²) |

#### 5.1.9 递归设计步骤

1. **找重复**：子问题是什么？
2. **找变化**：哪些参数随递归变化？
3. **找边界**：什么条件下不再递归？

**例子：二叉树遍历**
```cpp
void preorder(Node* p) {
    if (!p) return;            // 边界：空树
    visit(p);                  // 访问根
    preorder(p->left);         // 重复 1：左子树
    preorder(p->right);        // 重复 2：右子树
}
```

#### 5.1.10 递归 vs 回溯

- **回溯（Backtracking）**：递归的"剪枝版"
- 在递归基础上：
  - 走不通时**撤销选择**（状态恢复）
  - 剪去明显不合法的分支
- 经典：八皇后、数独、组合/排列、子集和

```cpp
void backtrack(vector<int>& path, vector<bool>& used) {
    if (path.size() == n) {  // 终止
        // 记录答案
        return;
    }
    for (int i = 0; i < n; i++) {
        if (used[i]) continue;   // 剪枝
        used[i] = true;          // 做选择
        path.push_back(i);
        backtrack(path, used);   // 递归
        path.pop_back();         // 撤销选择
        used[i] = false;
    }
}
```

#### 5.1.11 递归的应用

- **数学**：阶乘、斐波那契、组合数、Catalan 数
- **数据结构**：树的遍历、链表反转、图的 DFS
- **分治**：归并、快排、二分查找
- **回溯**：排列、子集、八皇后
- **动态规划**（递归+记忆化）
- **编译原理**：语法分析（递归下降）

### 5.5 Lab07-题目2：A Maze Problem（q164, 中等, 50 分）

**题意**：12×12 迷宫，O/W/E，3 个起点，问到出口的最短路径长度（起终点都算入），不可达输出 -1。

**关键点**：
- **BFS 求最短路**（不是 DFS，DFS 求的是任意路径）
- `dirs = {(1,0),(-1,0),(0,1),(0,-1)}`
- `O=0, W=1, E=2`；E 是终点
- BFS 起点 = (x, y)（注意题目说 x 是纵坐标，y 是横坐标）
- 距离 = `dist[x][y]`，终点 `dist[ex][ey]`
- 起始结束都算：长度 = 步数 + 1

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    const int N = 12;
    int grid[N][N];
    for (int i = 0; i < N; i++)
        for (int j = 0; j < N; j++) {
            char c; cin >> c;
            grid[i][j] = (c=='W'?1:(c=='E'?2:0));
        }
    // 找 E
    int ex=-1, ey=-1;
    for (int i=0;i<N;i++) for (int j=0;j<N;j++) if (grid[i][j]==2){ ex=i; ey=j; }

    // 3 个起点
    for (int k = 0; k < 3; k++) {
        int x, y; cin >> x >> y;     // (x,y) = (row, col)
        if (k) cout << ' ';
        if (grid[x][y] == 1) { cout << -1; continue; }

        int dist[N][N];
        for (int i=0;i<N;i++) for (int j=0;j<N;j++) dist[i][j] = -1;
        queue<pair<int,int>> q;
        dist[x][y] = 0; q.push({x,y});
        int d4[4][2] = {{-1,0},{1,0},{0,-1},{0,1}};
        while (!q.empty()) {
            auto [i,j] = q.front(); q.pop();
            for (auto& d : d4) {
                int ni = i+d[0], nj = j+d[1];
                if (ni<0||ni>=N||nj<0||nj>=N) continue;
                if (grid[ni][nj]==1) continue;
                if (dist[ni][nj]!=-1) continue;
                dist[ni][nj] = dist[i][j] + 1;
                q.push({ni,nj});
            }
        }
        if (dist[ex][ey] == -1) cout << -1;
        else cout << dist[ex][ey] + 1;     // 起终点都算
    }
    cout << "\n";
}
```

### 6.1 理论全掌握

#### 6.1.1 List 的定义

- **List（线性表）**：n 个数据元素的**有限序列**。
- 元素之间有**顺序**（线性关系）。
- 一般操作：增、删、改、查、遍历、求长度。
- **ADT 接口**：
  ```
  ADT List
  Data
      a[0..n-1] 线性序列
  Operation
      InitList(&L)
      DestroyList(&L)
      ClearList(&L)
      ListEmpty(L)
      ListLength(L)
      GetElem(L, i, &e)
      LocateElem(L, e, compare)        // 查找
      PriorElem(L, cur_e, &pre_e)
      NextElem(L, cur_e, &next_e)
      ListInsert(&L, i, e)
      ListDelete(&L, i, &e)
      ListTraverse(L, visit)
  ```

#### 6.1.2 顺序表（顺序存储）

- **结构**：数组 + 长度变量
- **特点**：
  - 逻辑相邻 = 物理相邻
  - 随机访问 O(1)
  - 插入/删除 O(n)
- **实现要点**：
  - 插入位置 i（0..n）：从后往前移
  - 删除位置 i（0..n-1）：从前往后移
  - **判满**：`size == capacity`
  - **判空**：`size == 0`
- **顺序表的操作时间复杂度**：
  | 操作 | 时间 |
  |---|---|
  | 初始化 | O(1) |
  | 销毁 | O(1) |
  | 清空 | O(1) |
  | 求长度 | O(1) |
  | 判空 | O(1) |
  | 读第 i 个 | O(1) |
  | 查找 | O(n) |
  | 插入 | O(n) |
  | 删除 | O(n) |
  | 遍历 | O(n) |

#### 6.1.3 链表（链式存储）

- **结构**：节点 `{data, *next}`，通过指针连接
- **种类**：
  - **单链表**：每个节点一个 next
  - **双链表**：每个节点 prev + next
  - **循环链表**：尾节点 next 回头
- **特点**：
  - 动态长度，无需预先分配
  - 插入/删除（已知前驱）O(1)
  - 查找 O(n)
  - 不能随机访问
- **关键概念**：
  - **头节点**：第一个节点（可存数据或不存）
  - **哑节点（dummy head）**：在头节点前的辅助节点，统一插入删除逻辑
  - **尾指针**：指向最后一个节点，便于 O(1) 尾插
  - **最后访问位置**：Lab 要求的优化，记录上次访问的节点和索引

#### 6.1.4 顺序表 vs 链表 详细对比

| 维度 | 顺序表 | 链表 |
|---|---|---|
| 存储空间 | 连续 | 离散 |
| 空间大小 | 固定 | 动态 |
| 随机访问 | O(1) | O(n) |
| 顺序访问 | O(n) | O(n) |
| 头部插入/删除 | O(n) | O(1) |
| 尾部插入/删除 | O(1) | O(1)（带尾指针）|
| 中间插入/删除 | O(n) | O(1)（已知前驱）|
| 空间开销 | 0 | 指针 |
| 缓存友好 | ✅ | ❌ |
| 适用场景 | 长度已知、查询多 | 长度变化、增删多 |

#### 6.1.5 单链表的改进

- **头节点 + 哑节点**：所有操作统一为"找到前驱"
- **循环链表**：尾插 O(1)，适合约瑟夫
- **双链表**：可双向遍历，删除给定节点 O(1)
- **静态链表**：用数组模拟链表（无指针的语言）
- **跳跃表（Skip List）**：多级索引的链表，O(log n) 查找
- **最后访问位置**：记录访问过的节点，下次访问就近

#### 6.1.6 链表反转

- **迭代版（三指针）**：
  ```cpp
  Node* reverse(Node* head) {
      Node *pre = nullptr, *cur = head;
      while (cur) {
          Node* nxt = cur->next;
          cur->next = pre;
          pre = cur;
          cur = nxt;
      }
      return pre;
  }
  ```
- **递归版**：
  ```cpp
  Node* reverse(Node* head) {
      if (!head || !head->next) return head;
      Node* newHead = reverse(head->next);
      head->next->next = head;
      head->next = nullptr;
      return newHead;
  }
  ```
- **时间 O(n)，空间 O(1) 迭代 / O(n) 递归**

#### 6.1.7 快慢指针

- **找中点**：slow 走一步，fast 走两步，fast 到尾时 slow 在中点
- **判环**：fast 与 slow 相遇则有环
- **找环入口**：相遇后一个从起点走，一个从相遇点走，再次相遇处即为入口
- **找倒数第 k 个**：fast 先走 k 步，然后 slow 跟上，fast 到尾时 slow 即是

#### 6.1.8 大数运算

- **存储**：`vector<int>` 存十进制位，**低位在前**（便于进位）
- **加法**：从低位加，处理进位
- **减法**：被减数 ≥ 减数（否则取负），借位处理
- **乘法**：每位 × 整个数 + 进位，O(n²)
- **除法**：试商法（Lab 选做）
- **去除前导零**：`while (d.size() > 1 && d.back() == 0) pop_back();`
- **零的判断**：`d.size() == 1 && d[0] == 0`
- **优化**：万进制或亿进制（每位存 4-8 位十进制）减少节点数

#### 6.1.9 字符串（String）

- **C++ STL**：`std::string`（封装好，自动管理）

- **自实现 String**：

```cpp
class Node {
```

**题意**：实现通用顺序线性表 `MyList`，支持 Insert/Remove/Retrieve/Replace/Traverse/Size/Full；用其将输入序列顺序输出。

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

class MyList {
    static const int CAP = 105;
    int data[CAP];
    int sz;
public:
    MyList(): sz(0) {}
    bool full() const { return sz == CAP; }
    int  size() const { return sz; }
    bool insert(int pos, int x) {       // 0 <= pos <= sz，在 pos 前插入
        if (full() || pos < 0 || pos > sz) return false;
        for (int i = sz; i > pos; i--) data[i] = data[i-1];
        data[pos] = x; sz++; return true;
    }
    bool remove(int pos) {              // 0 <= pos < sz
        if (pos < 0 || pos >= sz) return false;
        for (int i = pos; i < sz-1; i++) data[i] = data[i+1];
        sz--; return true;
    }
    bool retrieve(int pos, int& x) const {
        if (pos < 0 || pos >= sz) return false;
        x = data[pos]; return true;
    }
    bool replace(int pos, int x) {
        if (pos < 0 || pos >= sz) return false;
        data[pos] = x; return true;
    }
    void traverse() const {
        for (int i = 0; i < sz; i++) cout << (i?" ":"") << data[i];
        cout << "\n";
    }
};
```

### 6.8 Lab08-题目2：List 的单链表实现（q168, 困难, 50 分）

**题意**：单链表实现通用 List（含最后访问位置记录），N 次插入、M 次删除、P 次替换；输出最终序列。

**关键点**：
- 哑节点 + 头插/尾插
- 最后访问位置 = 索引 → 节点指针 + 索引（更新时机：每次 insert/remove/replace 后更新）
- 输入：4 行；操作均以 `-1` 或 `-1 -1` 结束

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct LNode { int v; LNode* next; LNode(int x): v(x), next(nullptr) {} };

class MyList {
    LNode* head;      // dummy
    int sz;
    int lastPos;      // 最后访问位置（0-indexed）
    LNode* lastPtr;   // 指向 lastPos 节点
    void updateLast(int pos) {
        lastPos = pos;
        LNode* p = head->next;
        for (int i = 0; i < pos; i++) p = p->next;
        lastPtr = p;
    }
public:
    MyList(): head(new LNode(0)), sz(0), lastPos(-1), lastPtr(nullptr) {}
    ~MyList() { while (head) { LNode* t = head; head = head->next; delete t; } }
    int size() const { return sz; }
    bool insert(int pos, int x) {
        if (pos < 0 || pos > sz) return false;
        LNode* pre = head;
        for (int i = 0; i < pos; i++) pre = pre->next;
        LNode* n = new LNode(x);
        n->next = pre->next; pre->next = n; sz++;
        updateLast(pos);
        return true;
    }
    bool remove(int pos) {
        if (pos < 0 || pos >= sz) return false;
        LNode* pre = head;
        for (int i = 0; i < pos; i++) pre = pre->next;
        LNode* t = pre->next; pre->next = t->next; delete t; sz--;
        if (pos < sz) updateLast(pos); else { lastPos = -1; lastPtr = nullptr; }
        return true;
    }
    bool replace(int pos, int x) {
        if (pos < 0 || pos >= sz) return false;
        LNode* p = head->next;
        for (int i = 0; i < pos; i++) p = p->next;
        p->v = x; updateLast(pos); return true;
    }
    void traverse() const {
        LNode* p = head->next;
        bool first = true;
        while (p) {
            if (!first) cout << ' ';
            first = false; cout << p->v;
            p = p->next;
        }
        cout << "\n";
    }
};

int main() {
    MyList L;
    int x;
    // 第 1 行：原序列
    while (cin >> x && x != -1) L.insert(L.size(), x);
    // 吃掉换行（如果有）
    // 第 2 行：插入
    while (cin >> x) {
        int a = x, b; cin >> b;
        if (a == -1 && b == -1) break;
        L.insert(a, b);
    }
    // 第 3 行：删除
    while (cin >> x) {
        if (x == -1) break;
        L.remove(x);
    }
    // 第 4 行：替换
    while (cin >> x) {
        int d = x, e; cin >> e;
        if (d == -1 && e == -1) break;
        L.replace(d, e);
    }
    L.traverse();
}
```

### 6.9 Lab09-题目1：List 的双链表实现（q169, 困难, 33 分）

**题意**：双链表实现 List，记录最后访问位置，输出正序和逆序。

**关键点**：
- 双链表每个节点 `{prev, next, val}`
- 维护 `head`（哑）和 `tail`
- Insert：在 pos 前插入；同步维护 `tail`
- Remove：删 pos 节点；tail 要更新
- 输出正序 + 逆序

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

struct DNode { int v; DNode *prev, *next; DNode(int x): v(x), prev(nullptr), next(nullptr) {} };

class MyList {
    DNode *head, *tail;     // 双向哑节点简化
    int sz;
    void link(DNode* a, DNode* b) { if (a) a->next = b; if (b) b->prev = a; }
public:
    MyList() {
        head = new DNode(0); tail = new DNode(0);
        link(head, tail); sz = 0;
    }
    ~MyList() {
        DNode* p = head;
        while (p) { DNode* t = p; p = p->next; delete t; }
    }
    int size() const { return sz; }
    bool insert(int pos, int x) {
        if (pos < 0 || pos > sz) return false;
        DNode* pre = head;
        for (int i = 0; i < pos; i++) pre = pre->next;
        DNode* n = new DNode(x);
        link(n, pre->next); link(pre, n); sz++; return true;
    }
    bool remove(int pos) {
        if (pos < 0 || pos >= sz) return false;
        DNode* p = head->next;
        for (int i = 0; i < pos; i++) p = p->next;
        link(p->prev, p->next); delete p; sz--; return true;
    }
    bool replace(int pos, int x) {
        if (pos < 0 || pos >= sz) return false;
        DNode* p = head->next;
        for (int i = 0; i < pos; i++) p = p->next;
        p->v = x; return true;
    }
    void traverse() const {
        DNode* p = head->next;
        bool first = true;
        while (p != tail) {
            if (!first) cout << ' ';
            first = false; cout << p->v;
            p = p->next;
        }
        cout << "\n";
    }
    void traverseRev() const {
        DNode* p = tail->prev;
        bool first = true;
        while (p != head) {
            if (!first) cout << ' ';
            first = false; cout << p->v;
            p = p->prev;
        }
        cout << "\n";
    }
};

int main() {
    MyList L;
    int x;
    while (cin >> x && x != -1) L.insert(L.size(), x);
    while (cin >> x) { int a=x,b; cin>>b; if (a==-1&&b==-1) break; L.insert(a,b); }
    while (cin >> x) { if (x==-1) break; L.remove(x); }
    while (cin >> x) { int d=x,e; cin>>e; if (d==-1&&e==-1) break; L.replace(d,e); }
    L.traverse();
    L.traverseRev();
}
```

### 6.11 Lab09-题目3：大数运算（q166, 简单, 34 分）

**题意**：实现 `MyBigInteger` 支持 `+ - *`（除法选做）。位数 ≤ 200。

**关键点**：
- 用 `vector<int>` 存十进制位，**低位在前**
- 加：从低位加，进位 carry
- 减：先比较长度（被减数 ≥ 减数），否则取负
- 乘：每位 × 另一个数 + 进位，O(n²) 够用
- 输出：去前导 0；若为 0 输出 `0`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

// 大整数类（十进制位，低位在前）
class MyBigInteger {
    vector<int> d;            // 低位在前
    void trim() { while (d.size()>1 && d.back()==0) d.pop_back(); }
public:
    MyBigInteger(long long x = 0) { while (x) { d.push_back(x%10); x/=10; } if (d.empty()) d.push_back(0); }
    MyBigInteger(const string& s) {
        d.clear();
        for (int i=(int)s.size()-1; i>=0; i--) d.push_back(s[i]-'0');
        trim();
    }
    bool isZero() const { return d.size()==1 && d[0]==0; }
    bool operator<(const MyBigInteger& o) const {
        if (d.size()!=o.d.size()) return d.size()<o.d.size();
        for (int i=(int)d.size()-1;i>=0;i--) if (d[i]!=o.d[i]) return d[i]<o.d[i];
        return false;
    }
    string toString() const {
        if (isZero()) return "0";
        string s; for (int i=(int)d.size()-1; i>=0; i--) s.push_back('0'+d[i]);
        return s;
    }
    // 静态成员：加
    static MyBigInteger add(const MyBigInteger& a, const MyBigInteger& b) {
        MyBigInteger r;
        r.d.assign(max(a.d.size(),b.d.size()), 0);
        int c = 0;
        for (size_t i = 0; i < r.d.size(); i++) {
            int s = c + (i<a.d.size()?a.d[i]:0) + (i<b.d.size()?b.d[i]:0);
            r.d[i] = s % 10; c = s / 10;
        }
        if (c) r.d.push_back(c);
        return r;
    }
    // 静态成员：减（要求 a >= b）
    static MyBigInteger sub(const MyBigInteger& a, const MyBigInteger& b) {
        MyBigInteger r = a;
        int c = 0;
        for (size_t i = 0; i < r.d.size(); i++) {
            int s = r.d[i] - (i<b.d.size()?b.d[i]:0) - c;
            if (s < 0) { s += 10; c = 1; } else c = 0;
            r.d[i] = s;
        }
        r.trim();
        return r;
    }
    // 静态成员：乘
    static MyBigInteger mul(const MyBigInteger& a, const MyBigInteger& b) {
        MyBigInteger r;
        r.d.assign(a.d.size()+b.d.size(), 0);
        for (size_t i=0; i<a.d.size(); i++)
            for (size_t j=0; j<b.d.size(); j++)
                r.d[i+j] += a.d[i]*b.d[j];
        for (size_t i=0; i<r.d.size(); i++) {
            if (r.d[i] >= 10) { r.d[i+1] += r.d[i]/10; r.d[i] %= 10; }
        }
        r.trim();
        return r;
    }
};

int main() {
    string s1, s2; char op;
    cin >> s1 >> s2 >> op;
    MyBigInteger A(s1), B(s2), R;
    if (op=='+') R = MyBigInteger::add(A,B);
    else if (op=='-') {
        if (A < B) { cout << "-"; R = MyBigInteger::sub(B, A); }
        else        R = MyBigInteger::sub(A, B);
    }
    else if (op=='*') R = MyBigInteger::mul(A, B);
    cout << R.toString() << "\n";
}
```

### 6.12 Lab10-题目1：String 的实现（q170, 中等, 100 分）

**题意**：用 `List<char>` 内部存储实现 String，支持比较、连接、子串查找、取子串；N 个串，输出子串查询结果 + 字典序排序。

**关键点**：
- 内部用 `vector<char>` 装字符
- 算长度、`==` `<` 比较：长度不等直接定，否则逐字符比
- 子串查找：朴素 O(nm) 够用
- 字典序：直接用 `<` 比较，sort

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

class String {
    vector<char> s;
public:
    String() {}
    String(const char* p) { while (*p) s.push_back(*p++); }
    String(const string& t) : s(t.begin(), t.end()) {}
    int length() const { return (int)s.size(); }
    const char* c_str() const {
        static char buf[100005];   // 简化：非线程安全，Lab 够用
        for (size_t i = 0; i < s.size(); i++) buf[i] = s[i];
        buf[s.size()] = '\0';
        return buf;
    }
    String substr(int l, int r) const {   // [l, r)
        String t;
        for (int i = l; i < r; i++) t.s.push_back(s[i]);
        return t;
    }
    int find(const String& sub) const {
        int n = length(), m = sub.length();
        if (m == 0) return 0;
        for (int i = 0; i + m <= n; i++) {
            int j = 0;
            while (j < m && s[i+j] == sub.s[j]) j++;
            if (j == m) return i;
        }
        return -1;
    }
    bool operator==(const String& o) const { return s == o.s; }
    bool operator<(const String& o) const {
        size_t n = min(s.size(), o.s.size());
        for (size_t i = 0; i < n; i++) if (s[i] != o.s[i]) return s[i] < o.s[i];
        return s.size() < o.s.size();
    }
};

int main() {
    int N; cin >> N;
    vector<String> arr(N);
    for (int i = 0; i < N; i++) {
        string t; cin >> t; arr[i] = String(t);
    }
    // 第 3 行：查询（a b 对，判断 arr[a] 是否是 arr[b] 的子串）
    vector<int> ans;
    while (true) {
        int a, b; cin >> a >> b;
        if (a == -1 && b == -1) break;
        if (a < 0 || a >= N || b < 0 || b >= N) ans.push_back(-1);
        else ans.push_back(arr[b].find(arr[a]));
    }
    for (int i = 0; i < (int)ans.size(); i++) {
        if (i) cout << ' ';
        cout << ans[i];
    }
    cout << "\n";
    // 字典序
    vector<String> sorted = arr;
    sort(sorted.begin(), sorted.end());
    for (int i = 0; i < N; i++) {
        if (i) cout << ' ';
        cout << sorted[i].c_str();
    }
    cout << "\n";
}
```

## 第 7 章 查找

### 7.1 理论全掌握

#### 7.1.1 查找的基本概念

- **查找表（Search Table）**：由同一类型的数据元素（或记录）组成的集合。
- **关键字（Key）**：数据元素中某个数据项的值，用以标识一个数据元素。
  - **主关键字**：唯一标识
  - **次关键字**：不唯一
- **查找（Searching）**：根据关键字，确定查找表中是否存在该记录，以及其位置。
- **查找表分类**：
  - **静态查找表**：只查不改（顺序/二分/索引）
  - **动态查找表**：可插入删除（BST/AVL/哈希）

#### 7.1.2 查找的评价指标

- **平均查找长度 ASL（Average Search Length）**：查找过程中关键字比较次数的期望。
  - 成功 ASL：`Σ(查找次数 × 概率) / n`
  - 失败 ASL：基于失败情况的平均
- **时间复杂度**：T(n) 随 n 增长
- **空间复杂度**：S(n)

#### 7.1.3 顺序查找（线性查找）

- **思想**：从表的一端开始，逐个比较
- **时间复杂度**：O(n)
- **空间复杂度**：O(1)
- **优点**：对数据无要求（有序无序都行）
- **缺点**：效率低
- **改进**：
  - 设哨兵：从尾部开始放哨兵，省去边界检查
  - 概率优化：高频数据放前面
  - 分块查找：每块内顺序、块间索引

#### 7.1.4 折半查找（二分查找）—— 重点

- **前提**：顺序存储 + **有序**
- **思想**：每次与中间元素比较，缩小一半范围
- **核心循环**：
  ```cpp
  int lo = 0, hi = n - 1;
  while (lo <= hi) {
      int mid = lo + (hi - lo) / 2;   // 防溢出
      if (a[mid] == target) return mid;
      else if (a[mid] < target) lo = mid + 1;
      else hi = mid - 1;
  }
  return -1;
  ```
- **时间复杂度**：O(log n)
- **空间复杂度**：O(1) 迭代 / O(log n) 递归
- **判定树**：高度 ⌈log₂(n+1)⌉，平衡二叉树

#### 7.1.5 二分查找的 6 大变体（必背）

**1. 标准查找**：找任意一个等于 target 的位置
```cpp
while (lo <= hi) {
    int mid = (lo+hi)/2;
    if (a[mid] == target) return mid;
    else if (a[mid] < target) lo = mid+1;
    else hi = mid-1;
}
```

**2. 找第一个 ≥ target 的位置（lower_bound）**
```cpp
int lo=0, hi=n;
while (lo < hi) {
    int mid = (lo+hi)/2;
    if (a[mid] < target) lo = mid+1;
    else hi = mid;
}
return lo;   // 可能 == n
```

**3. 找第一个 > target 的位置（upper_bound）**
```cpp

while (lo < hi) {
    int mid = (lo+hi)/2;
    if (a[mid] <= target) lo = mid+1;
    else hi = mid;
}
return lo;
```

**4. 找最后一个 ≤ target 的位置**
```cpp
// = upper_bound(target) - 1
```

**5. 找最后一个 < target 的位置**
```cpp
// = lower_bound(target) - 1
```

**6. 找第一个 = target（不存在则 -1）**
```cpp
int p = lower_bound(a, target);
if (p < n && a[p] == target) return p;
return -1;
```

#### 7.1.6 插值查找

- **思想**：按 key 的分布估计位置（类似字典查词）
- **公式**：`mid = lo + (target - a[lo]) * (hi - lo) / (a[hi] - a[lo])`
- **适用**：数据分布均匀
- **时间复杂度**：O(log log n) 均匀分布 / O(n) 最坏

#### 7.1.7 斐波那契查找

- **思想**：用斐波那契数列分割
- **优点**：避免除法
- **时间复杂度**：O(log n)

#### 7.1.8 分块查找（索引顺序查找）

- **思想**：把表分成若干块，块内无序、块间有序；建索引表
- **步骤**：
  1. 在索引表（有序）里二分找块
  2. 在块内顺序找
- **时间复杂度**：O(√n) 平均

### 7.2 Lab11-题目1：前向和后向的最近邻值（q172, 中等, 33 分）

**题意**：双链表节点值均为正整数。
- **前向最近邻值**：节点 i 之前 < i 的最大值（不存在 → 0）

- 后向：j 从 i+1 往 m-1 走，取 > a[i] 的最小值

```cpp
#include <bits/stdc++.h>
```

### 7.4 Lab11-题目3：Start and End Position（q175, 中等, 34 分）

**题意**：升序数组找 target 第一次和最后一次出现的位置；不存在返回 `[-1,-1]`；要求 O(log n)。

- 两次二分：找左边界、找右边界
- 左边界：找第一个 ≥ target
- 右边界：找第一个 > target，再 -1

```cpp
#include <bits/stdc++.h>

    int lo=0, hi=a.size();
    while (lo < hi) {
        int mid = (lo+hi)/2;
        if (a[mid] < target) lo = mid+1;
        else hi = mid;
    }
    return lo;
}

int main() {
    int target; cin >> target;
    vector<int> a; int x;
    while (cin >> x && x != -1) a.push_back(x);
    int lo = lowerBound(a, target);
    if (lo == (int)a.size() || a[lo] != target) {
        cout << "[-1,-1]\n";
    } else {
        int hi = lowerBound(a, target+1) - 1;
```

## 第 8 章 排序

### 8.1 理论全掌握

#### 8.1.1 排序的基本概念

- **排序（Sorting）**：将一组数据按关键字递增/递减的顺序重新排列。
- **稳定性**：相等关键字排序前后相对位置不变。
- **内排序 vs 外排序**：
  - **内排序**：数据全部在内存（O(n²) ~ O(n log n)）
  - **外排序**：数据在外存（归并思想，复杂度以磁盘 I/O 衡量）
- **原地排序**：辅助空间 O(1)
- **比较排序下界**：Ω(n log n)（决策树证明）

#### 8.1.2 排序总览（必背）

| 排序 | 平均 | 最坏 | 空间 | 稳定 | 类别 |
|---|---|---|---|---|---|
| **冒泡** | O(n²) | O(n²) | O(1) | ✅ | 交换 |
| **插入** | O(n²) | O(n²) | O(1) | ✅ | 插入 |
| **希尔** | O(n^1.3) | O(n²) | O(1) | ❌ | 插入改进 |
| **选择** | O(n²) | O(n²) | O(1) | ❌ | 选择 |
| **堆排** | O(n log n) | O(n log n) | O(1) | ❌ | 选择 |
| **归并** | O(n log n) | O(n log n) | O(n) | ✅ | 分治 |
| **快排** | O(n log n) | O(n²) | O(log n) | ❌ | 分治 |
| **计数** | O(n+k) | O(n+k) | O(k) | ✅ | 分配 |
| **基数** | O(d·n) | O(d·n) | O(n+k) | ✅ | 分配 |
| **桶** | O(n+k) | O(n²) | O(n+k) | ✅ | 分配 |

#### 8.1.3 冒泡排序（Bubble Sort）

- **思想**：相邻元素两两比较，逆序则交换；每轮将最大元素"冒"到末尾
- **步骤**：
  - 外层 i = 0..n-1
  - 内层 j = 0..n-i-1
  - `if (a[j] > a[j+1]) swap(a[j], a[j+1])`
- **优化**：加 `swapped` 标志，本轮无交换则已排好
- **时间**：O(n²) 平均/最坏 / O(n) 已有序
- **空间**：O(1)
- **稳定**：✅

#### 8.1.4 插入排序（Insertion Sort）—— 重点

- **思想**：把数组分为已排序区（左侧）和未排序区；依次将未排序元素插入已排序区
- **步骤**：
  - `for i = 1..n-1`：
    - `key = a[i]`
    - `j = i - 1`
    - `while j >= 0 && a[j] > key: a[j+1] = a[j]; j--`
    - `a[j+1] = key`
- **时间**：O(n²) 平均/最坏 / O(n) 已有序
- **空间**：O(1)
- **稳定**：✅
- **应用**：数据基本有序时效率高；小规模数据；其他排序（如希尔、归并）的子过程

#### 8.1.5 希尔排序（Shell Sort）

- **思想**：插入排序的改进；按一定**增量**分组，对每组做插入排序；增量逐步缩小直到 1
- **增量序列**（影响性能）：
  - Shell 原始：n/2, n/4, ..., 1
  - Hibbard：1, 3, 7, 15, ...
  - Sedgewick：复杂序列
- **时间**：
  - Shell 增量：O(n²) 最坏
  - Hibbard 增量：O(n^1.5) 最坏
  - 平均：O(n^1.3)
- **空间**：O(1)
- **稳定**：❌
- **意义**：第一个突破 O(n²) 的排序

#### 8.1.6 选择排序（Selection Sort）

- **思想**：每轮从未排序区找最小元素，与未排序区第一个元素交换
- **步骤**：
  - `for i = 0..n-1`：
    - `mi = i`
    - `for j = i+1..n-1: if a[j] < a[mi] mi = j`
    - `swap(a[i], a[mi])`
- **时间**：O(n²) 永远（无论数据如何）
- **空间**：O(1)
- **稳定**：❌（交换破坏顺序）
- **优点**：交换次数少（最多 n-1 次）

#### 8.1.7 堆排序（Heap Sort）—— 重点

- **堆的定义**：
  - **大根堆**：父 ≥ 子，根 = 最大
  - **小根堆**：父 ≤ 子，根 = 最小
- **数组存储完全二叉树**：
  - 0-indexed：父 `(i-1)/2`，左子 `2i+1`，右子 `2i+2`
  - 1-indexed：父 `i/2`，左子 `2i`，右子 `2i+1`
- **`siftDown(i, n)`**：把 i 向下调整（与较大子节点交换）
- **`siftUp(i)`**：把 i 向上调整（与父节点比较）
- **`buildHeap`**：从最后一个非叶节点 `(n-1)/2` 倒序 siftDown，O(n)
- **`heapSort`**：
  1. buildHeap（大根堆）
  2. 反复 `swap(0/1, n-1)`，n--，siftDown
- **时间**：O(n log n) 永远
- **空间**：O(1)
- **稳定**：❌
- **应用**：Top K、堆、优先队列

#### 8.1.8 归并排序（Merge Sort）—— 重点

- **思想**：分治——把数组对半拆分，递归排序，合并两个有序子数组
- **步骤**：
  1. `mergeSort(lo, hi)`：`mid = (lo+hi)/2`
  2. `mergeSort(lo, mid); mergeSort(mid+1, hi)`
  3. `merge(lo, mid, hi)`：双指针合并
- **合并**：
  - 临时数组存合并结果
  - 双指针 i, j 指向两个子数组
  - 每次取较小者放入临时数组
  - 最后把剩余部分复制回
- **时间**：O(n log n) 永远
- **空间**：O(n)（合并需要临时数组）
- **稳定**：✅
- **应用**：外排序（多路归并）

#### 8.1.9 快速排序（Quick Sort）—— 重点

- **思想**：分治——选轴点（pivot），划分（partition）为左右两半，递归排序
- **轴点选择**：
  - 第一个 / 最后一个 / 中间
  - 随机选
  - 三数取中（首/中/尾取中位数）
- **划分算法**：
  - **Lomuto**（单指针）
  - **Hoare**（双指针）
  - **课本"末元素为轴"**（Lab 用）
- **课本版本步骤**（Lab15 用）：
  - 设轴 `pivot = a[hi]`
  - `i = lo, j = hi`
  - 从左找到第一个 > 轴；从右找到第一个 < 轴
  - 若 i < j，交换；继续
  - 结束交换 `a[i]` 与 `a[hi]`
  - 返回 i 作为新轴点位置
- **递归**：`quickSort(lo, p-1); quickSort(p+1, hi)`
- **时间**：
  - 平均 O(n log n)
  - 最坏 O(n²)（已排序 / 逆序 / 全相同）
- **空间**：O(log n) 递归栈
- **稳定**：❌
- **优化**：
  - 随机轴点
  - 小规模用插入排序
  - 三数取中
  - 尾递归优化

#### 8.1.10 计数排序（Counting Sort）

- **思想**：统计每个值出现的次数，再按顺序输出
- **步骤**：
  1. 找 max
  2. 建 count[0..max]
  3. 扫描 a 统计 count[a[i]]++
  4. 累加 count 得到位置
  5. 反向填充（保持稳定）
- **时间**：O(n + k)
- **空间**：O(k)
- **稳定**：✅
- **适用**：数据范围小（k << n）

#### 8.1.11 基数排序（Radix Sort）

- **思想**：按位（个位、十位、百位…）依次排序（用稳定的子排序）
- **LSD（Least Significant Digit）**：从低位到高位
- **MSD**：从高位到低位
- **子排序**：计数排序
- **时间**：O(d·n)
- **空间**：O(n + k)
- **稳定**：✅
- **适用**：整数、字符串（按字符位）

#### 8.1.12 桶排序（Bucket Sort）

- **思想**：分桶（按范围）→ 各桶内排序 → 合并
- **时间**：O(n + k) 平均
- **空间**：O(n + k)
- **适用**：均匀分布的浮点数

#### 8.1.13 优先队列（Priority Queue）

- **定义**：元素带优先级，出队时优先级最高者先出
- **实现**：
  - **堆（最常用）**
  - 有序数组
  - 无序数组
  - 二项堆 / 斐波那契堆
- **基本操作**：
  - `insert(x)`：插入元素
  - `findMax()` / `findMin()`：取最值
  - `deleteMax()` / `deleteMin()`：删除最值
- **时间（堆实现）**：
  - 插入 O(log n)
  - 取最值 O(1) 或 O(log n)
  - 删除最值 O(log n)
  - 建堆 O(n)

#### 8.1.14 各种排序的选用指南

| 场景 | 推荐 |
|---|---|
| 一般情况 | 快排 |
| 要求稳定 | 归并 |
| 空间有限 | 堆排 |
| 数据基本有序 | 插入 |
| 大数据 + 内存不够 | 归并（外排序）|
| 范围小 | 计数 |
| 整数且位数少 | 基数 |
| Top K 问题 | 堆 |

#### 8.1.15 常见考点

- 各种排序的算法步骤（必会手写）
- 时间/空间复杂度
- 稳定性判断
- 适用场景对比
- 排序下界（决策树证明）
- 二叉树/链表上的排序

### 8.6 Lab15-题目1：归并排序的链式实现（q189, 中等, 20 分）

**题意**：链表归并排序；输出**归并前的两个子序列**和**最终结果**。

**关键点**：
- 自顶向下：递归地把链表对半拆
- 拆到剩 1 个节点时返回
- 合并：双指针归并
- 输出：合并**之前**打印左右子序列

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;
struct LNode { int v; LNode* next; LNode(int x): v(x), next(nullptr) {} };

void print(LNode* h) {
    bool f = true;
    while (h) { if (!f) cout << ' '; f=false; cout << h->v; h = h->next; }
    cout << "\n";
}

// 找中点（slow 指向左半最后一个）
LNode* midAndSplit(LNode* head) {
    LNode *slow = head, *fast = head->next;
    while (fast && fast->next) { slow = slow->next; fast = fast->next->next; }
    LNode* mid = slow->next;
    slow->next = nullptr;
    return mid;
}

LNode* merge(LNode* a, LNode* b) {
    LNode dummy(0), *tail = &dummy;
    while (a && b) {
        if (a->v <= b->v) { tail->next = a; a = a->next; }
        else              { tail->next = b; b = b->next; }
        tail = tail->next;
    }
    tail->next = a ? a : b;
    return dummy.next;
}

LNode* mergeSort(LNode* head) {
    if (!head || !head->next) return head;
    LNode* mid = midAndSplit(head);
    LNode* left  = mergeSort(head);
    LNode* right = mergeSort(mid);
    // 题目要求：归并前打印子序列
    print(left);
    print(right);
    return merge(left, right);
}

int main() {
    int n; cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    LNode* head = nullptr; LNode* tail = nullptr;
    for (int i = 0; i < n; i++) {
        LNode* p = new LNode(a[i]);
        if (!head) head = tail = p;
        else { tail->next = p; tail = p; }
    }
    LNode* res = mergeSort(head);
    print(res);
    // 释放
    while (res) { LNode* t = res; res = res->next; delete t; }
}
```

### 8.7 Lab15-题目2：快速排序的顺序实现（q190, 中等, 20 分）

**题意**：顺序 List 快排，**末元素为轴点**；输出第一轮划分结果 + 最终结果。

**关键点**：
- 课本的"末元素为轴"方法（Hoare 变种）：
  - 设轴 = a[hi]
  - i = lo, j = hi
  - 从左找到第一个 > 轴的；从右找到第一个 < 轴的；交换
  - 直到 i ≥ j；最后把 a[i] 和 a[hi] 交换
- **一轮结束**：轴点左边的都 ≤ 轴，右边的都 ≥ 轴
- 递归 quicksort(lo, pivot-1)、quicksort(pivot+1, hi)
- 第一轮结束立刻打印数组

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int partition(vector<int>& a, int lo, int hi) {
    int pivot = a[hi];
    int i = lo, j = hi;
    while (i < j) {
        while (i < j && a[i] < pivot) i++;
        while (i < j && a[j] >= pivot) j--;
        if (i < j) swap(a[i], a[j]);
    }
    swap(a[i], a[hi]);
    return i;
}

void quicksort(vector<int>& a, int lo, int hi) {
    if (lo >= hi) return;
    int p = partition(a, lo, hi);
    quicksort(a, lo, p-1);
    quicksort(a, p+1, hi);
}

void print(const vector<int>& a) {
    for (int i = 0; i < (int)a.size(); i++) cout << (i?" ":"") << a[i];
    cout << "\n";
}

int main() {
    int n; cin >> n;
    vector<int> a(n);
    for (int i = 0; i < n; i++) cin >> a[i];
    // 手动做第一轮
    int p = partition(a, 0, n-1);
    print(a);
    quicksort(a, 0, p-1);
    quicksort(a, p+1, n-1);
    print(a);
}
```

### 8.8 Lab15-题目3：颜色排序 Sort Colors（q186, 中等, 20 分）

**题意**：0/1/2 排序，要求 O(n) 时间和 O(1) 空间。

**关键点**：**三向切分 / 荷兰国旗问题**。
- 三指针：left（下一个 0 的位置）、i（当前）、right（下一个 2 的位置）
- `a[i] == 0`：交换 a[i], a[left]，left++, i++
- `a[i] == 1`：i++
- `a[i] == 2`：交换 a[i], a[right]，right--，i 不动（因为换过来的还要检查）

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    vector<int> a; int x;
    while (cin >> x && x != -1) a.push_back(x);
    int n = a.size();
    int l = 0, r = n - 1, i = 0;
    while (i <= r) {
        if (a[i] == 0) swap(a[i++], a[l++]);
        else if (a[i] == 1) i++;
        else swap(a[i], a[r--]);
    }
    for (int k = 0; k < n; k++) cout << (k?" ":"") << a[k];
    cout << "\n";
}
```

### 8.9 Lab15-题目4：优先级队列（q192, 中等, 20 分）

**题意**：用堆（顺序存储）实现优先级队列（数字大优先）。要求：
- 初始序列 buildHeap 建堆
- 删除 m 个最大
- 插入 k 个新元素（**自下而上 siftUp**）
- 输出：初始堆 → 删除后 → 插入后

**关键点**：
- 1-indexed 数组，`parent = i/2`, `left = 2i`, `right = 2i+1`
- `siftDown`：取子节点中较大者，必要时上浮
- `siftUp`：与父节点比，必要时下沉（这里插入用 siftUp）
- 题目要求"删除 m 个" → 反复 popMax

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

class MaxHeap {
    vector<int> h;       // 1-indexed
    int n;
public:
    MaxHeap(): n(0) { h.push_back(0); }
    void siftDown(int i) {
        while (2*i <= n) {
            int j = 2*i;
            if (j+1 <= n && h[j+1] > h[j]) j++;
            if (h[i] >= h[j]) break;
            swap(h[i], h[j]);
            i = j;
        }
    }
    void siftUp(int i) {
        while (i > 1 && h[i] > h[i/2]) {
            swap(h[i], h[i/2]);
            i /= 2;
        }
    }
    void buildHeap(vector<int> a) {    // 数组 a 元素
        h.clear(); h.push_back(0);
        for (int x : a) h.push_back(x);
        n = a.size();
        for (int i = n/2; i >= 1; i--) siftDown(i);
    }
    int popMax() {
        int v = h[1];
        h[1] = h[n--];
        siftDown(1);
        return v;
    }
    void insert(int x) { h.push_back(x); n++; siftUp(n); }
    void print() const {
        for (int i = 1; i <= n; i++) cout << (i==1?"":" ") << h[i];
        cout << "\n";
    }
};

int main() {
    MaxHeap pq;
    vector<int> init;
    int x;
    while (cin >> x && x != -1) init.push_back(x);
    pq.buildHeap(init);
    pq.print();
    int m; cin >> m;
    for (int i = 0; i < m; i++) pq.popMax();
    pq.print();
    while (cin >> x && x != -1) pq.insert(x);
    pq.print();
}
```

### 8.10 Lab15-题目5：堆排序的顺序实现（q191, 中等, 20 分）

**题意**：对顺序 List 做堆排序；输出**初始大根堆** + **最终升序**。

**关键点**：
- 1-indexed：buildHeap（O(n)）+ 反复交换 h[1] 与 h[i]，n--，siftDown(1)
- **输出顺序**：初始堆是 buildHeap 后的数组；最终是排好序的（升序）

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

void siftDown(vector<int>& h, int n, int i) {
    while (2*i <= n) {
        int j = 2*i;
        if (j+1 <= n && h[j+1] > h[j]) j++;
        if (h[i] >= h[j]) break;
        swap(h[i], h[j]);
        i = j;
    }
}

void heapSort(vector<int>& h) {   // 1-indexed
    int n = (int)h.size() - 1;
    for (int i = n/2; i >= 1; i--) siftDown(h, n, i);
    // 第一阶段完成：初始大根堆
    for (int k = 1; k <= n; k++) cout << (k==1?"":" ") << h[k];
    cout << "\n";
    // 第二阶段
    for (int i = n; i > 1; i--) {
        swap(h[1], h[i]);
        siftDown(h, i-1, 1);
    }
    for (int k = 1; k <= n; k++) cout << (k==1?"":" ") << h[k];
    cout << "\n";
}

int main() {
    int n; cin >> n;
    vector<int> h(1);
    for (int i = 0; i < n; i++) { int x; cin >> x; h.push_back(x); }
    heapSort(h);
}
```

## 第 9 章 哈希表

### 9.1 理论全掌握

#### 9.1.1 哈希表的基本概念

- **哈希表（Hash Table）**：根据关键码值（Key）直接访问的数据结构。通过**哈希函数**将 key 映射到数组下标。
- **优点**：期望 O(1) 的查找、插入、删除
- **缺点**：冲突处理复杂、不支持顺序访问、空间利用率不高
- **关键概念**：
  - **哈希函数 H(k)**：key → [0, m) 的映射
  - **冲突（Collision）**：不同 key → 同一地址
  - **同义词（Synonym）**：哈希值相同的不同 key
  - **装填因子 α = n/m**：数据数 / 表大小（α 越小越好）

#### 9.1.2 哈希函数设计要求

1. **简单**：计算快
2. **均匀**：地址分布均匀，少冲突
3. **一致**：相同 key → 相同地址
4. **满射**：尽量用满所有地址

#### 9.1.3 常见哈希函数

| 方法 | 公式 | 适用 | 评价 |
|---|---|---|---|
| **直接定址法** | `H(k) = a·k + b` | key 分布连续 | 简单但不通用 |
| **除留余数法** | `H(k) = k mod p`（p 最好是质数） | 整数 key | **最常用** |
| **数字分析法** | 取 key 的若干位 | 已知 key 分布 | 特殊场景 |
| **平方取中法** | 取 k² 的中间几位 | 关键字位数不多 | 均匀 |
| **折叠法** | 分段求和 | 关键字位数多 | 简单 |
| **旋转法** | 末位移到首位 | 字符串 | 特定 |
| **BKDRHash** | `h = h*seed + ch` | **字符串 key** | **Lab 必背** |
| **DJB2** | `h = h*33 + ch` | 字符串 | 简单 |
| **SDBM** | `h = ch + h<<6 + h<<16 - h` | 字符串 | 较均匀 |
| **FNV-1** | `h = (h ^ ch) * FNV_prime` | 字符串 | 较均匀 |

**BKDRHash 模板（C++ 必背）**：
```cpp
unsigned int BKDRHash(const char* str, unsigned int size) {
    unsigned int seed = 31;        // 31, 131, 1313, 13131...
    unsigned int h = 0;
    while (*str) h = (h * seed + (unsigned char)(*str++)) % size;
    return h;
}
// 或 std::string 版本
unsigned int BKDR(const string& s, unsigned int size) {
    unsigned int h = 0, seed = 31;
    for (char c : s) h = (h * seed + (unsigned char)c) % size;
    return h;
}
```

#### 9.1.4 冲突解决方法

**(1) 链地址法（Separate Chaining）**
- 每个桶是一个**链表**
- 插入：查到链表（头插法）
- 查找/删除：遍历链表
- 优点：简单、不会"装满"、删除简单
- 缺点：指针开销、链表长 → 性能退化
- 优化：链表 → 红黑树 / 跳表（Java HashMap 8 后）

**(2) 开放定址法（Open Addressing）**
- 冲突时按规则探测下一个空位
- **三种探测**：
  - **线性探测**：`H_i = (H + i) mod m`
    - 优点：简单
    - 缺点：**堆积（clustering）** 严重
  - **平方探测（Quadratic Probing）**：`H_i = (H + i²) mod m` 或 `H_i = (H ± i²) mod m`
    - 优点：缓解堆积
    - 缺点：不能探遍所有位置
  - **双散列（Double Hashing）**：`H_i = (H + i * H₂) mod m`
    - 优点：均匀
    - 缺点：需要第二个哈希函数

**(3) 再哈希法（Rehashing）**
- 冲突时用另一个哈希函数
- 较少用

**(4) 建立公共溢出区**
- 基本表 + 溢出表
- 较少用

#### 9.1.5 哈希表删除问题

- **链地址法**：直接删节点，O(1)
- **开放定址法**：**不能直接置空**（否则查找链断），需用**墓碑（tombstone）标记**：
  - 状态：`EMPTY` / `OCCUPIED` / `DELETED`
  - 删除时设为 `DELETED`
  - 查找时遇到 `DELETED` 继续
  - 插入时遇到 `DELETED` 可复用
- 缺点：墓碑积累影响性能 → 定期 rebuild

#### 9.1.6 装填因子与性能

| 解决方式 | 成功 ASL | 失败 ASL |
|---|---|---|
| 链地址法 | `1 + α/2` | `α` |
| 线性探测 | `(1 + 1/(1-α))/2` | `(1 + 1/(1-α)²)/2` |
| 平方探测 | `-ln(1-α)/α` | `1/(1-α)` |

- α 越接近 1，性能越差
- α > 0.7 时通常需要**扩容（rehash）**
- 一般保持 α < 0.5 ~ 0.75

#### 9.1.7 动态哈希表

- **扩容策略**：当 α 超过阈值（如 0.75）时：
  1. 开新表（容量 2 倍）
  2. 重新计算所有 key 的哈希
  3. 重新插入
- **摊还成本**：每次插入 O(1) 摊还
- **C++ STL `unordered_map`**：自动扩容，负载因子 1.0

#### 9.1.8 一致性哈希（进阶）

- 用于**分布式系统**的缓存/数据库分片
- 把哈希空间组织成环（0 ~ 2³²）
- 节点和数据都映射到环上
- 数据归属：顺时针第一个节点
- **虚拟节点**：解决数据倾斜

#### 9.1.9 哈希表的应用

- 编译器符号表
- 数据库索引（哈希索引）
- 缓存（Memcached / Redis）
- 集合 / 字典
- 字符串去重 / 计数
- LRU 缓存（哈希 + 双向链表）

#### 9.1.10 常见考点

- 各种哈希函数的应用场景
- 冲突解决方法对比
- 装填因子对性能的影响
- 平方探测序列的计算
- 链地址法的链表操作
- 字符串 hash（BKDR / DJB2）
- 哈希扩容的摊还分析

### 9.2 Lab17-题目1：Hash Table 的顺序实现（q202, 中等, 100 分）

**题意**：平方探测 + 取余法（key % n）。要求：
- 重复 key 只存第一个
- 探测次数上限 = 课本定义（一般是 `n/2` 或"上溢"）
- 输出：探测到 key1 的过程 + key1 的 value；探测到 key2 的过程；找不到输出 `NULL`

**关键点**：
- 表大小 n 选质数
- 平方探测：`for (i=0; i<n; i++) idx = (H + i*i) % n`（只 +i²），或 ±i²
- 课本惯例：探测 `H, H+1, H+4, H+9, ...`
- 探测失败：`idx` 第一次为空时（未占用）就停
- 探测到 key1 → 输出所有走过的 idx + "key1 value"
- 找不到 → 输出走过的 idx + `NULL`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

int main() {
    int n, m; cin >> n >> m;
    vector<int> key(n, -1), val(n, -1);    // -1 表示空

    for (int i = 0; i < m; i++) {
        int k, v; cin >> k >> v;
        int h = k % n;
        int j = 0, idx;
        for (; j < n; j++) {
            idx = (h + j*j) % n;
            if (key[idx] == -1 || key[idx] == k) break;
        }
        if (key[idx] == -1) { key[idx] = k; val[idx] = v; }
        // 重复不覆盖
    }
    // 查询
    auto query = [&](int k) {
        int h = k % n;
        vector<int> path;
        for (int j = 0; j < n; j++) {
            int idx = (h + j*j) % n;
            path.push_back(idx);
            if (key[idx] == -1) return make_pair(path, -1);   // NULL
            if (key[idx] == k) return make_pair(path, val[idx]);
        }
        return make_pair(path, -1);
    };
    int k1, k2; cin >> k1 >> k2;
    for (int q : {k1, k2}) {
        auto [path, v] = query(q);
        for (size_t i = 0; i < path.size(); i++) cout << (i?" ":"") << path[i];
        if (v == -1) cout << " NULL";
        else         cout << " " << q << " " << v;
        cout << "\n";
    }
}
```

### 9.3 Lab16-题目1：HashMap 的实现（q203, 中等, 50 分）

**题意**：用**链地址法**实现 HashMap，支持 `put` / `get` / `remove`。字符串 key 用 **BKDRHash**。要求：
- put 重复 key → 更新 value
- get 不存在 → 返回 -1
- 输出：插入某 key 的地址 + 删除某 key 的过程（找不到输出 NULL）+ 查询某 key 的过程（NULL 输出 NULL）

**关键点**：
- 数组 `vector<list<pair<string,string>>>` 存桶
- BKDRHash 把字符串映射到 [0, m)
- 输出格式：先输出 key 的地址码，然后是过程（"key1 key2"）

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

unsigned int BKDR(const string& s, unsigned int size) {
    unsigned int h = 0, seed = 31;
    for (char c : s) h = (h * seed + (unsigned char)c) % size;
    return h;
}

int main() {
    int n, m; cin >> n >> m;
    vector<list<pair<string,string>>> bucket(n);
    for (int i = 0; i < m; i++) {
        string k, v; cin >> k >> v;
        unsigned int idx = BKDR(k, n);
        bool found = false;
        for (auto& p : bucket[idx]) if (p.first == k) { p.second = v; found = true; break; }
        if (!found) bucket[idx].push_front({k, v});
    }
    // 第 3 行：插入（输出地址码）
    string k3, v3; cin >> k3 >> v3;
    unsigned int idx3 = BKDR(k3, n);
    bool found = false;
    for (auto& p : bucket[idx3]) if (p.first == k3) { p.second = v3; found = true; break; }
    if (!found) bucket[idx3].push_front({k3, v3});
    cout << idx3 << "\n";

    // 第 4 行：删除（输出地址 + "key" + "NULL" 如果失败）
    string k4; cin >> k4;
    unsigned int idx4 = BKDR(k4, n);
    bool removed = false;
    for (auto it = bucket[idx4].begin(); it != bucket[idx4].end(); ++it) {
        if (it->first == k4) { bucket[idx4].erase(it); removed = true; break; }
    }
    cout << idx4 << ' ' << k4 << ' ' << (removed ? "" : "NULL") << "\n";
    // 注意：题目样例 "6 dog NULL" 实际是：地址 dog 删除结果(NULL)
    // 严格按样例输出格式

    // 第 5 行：查询
    string k5; cin >> k5;
    unsigned int idx5 = BKDR(k5, n);
    cout << idx5 << ' ';
    for (auto& p : bucket[idx5]) cout << p.first << (p.first==k5?"":" ");  // 简化
    cout << " " << k5 << " " << (查询到的value) << "\n";
}
```

### 9.4 Lab16-题目2：Longest Substring Without Repeating Characters（q205, 中等, 50 分）

**题意**：字符串中无重复字符的最长子串长度。

**关键点**：**滑动窗口 + 哈希表记录上次出现位置**。
- `last[ch]` = ch 上次出现的位置（-1 表示没出现）
- 遍历 i = 0..n-1
- 维护左边界 `lo`：`lo = max(lo, last[s[i]] + 1)`（如果 ch 出现过）
- 更新答案：`ans = max(ans, i - lo + 1)`
- 更新 `last[s[i]] = i`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;
int main() {
    string s; cin >> s;
    vector<int> last(256, -1);
    int lo = 0, ans = 0;
    for (int i = 0; i < (int)s.size(); i++) {
        lo = max(lo, last[s[i]] + 1);
        ans = max(ans, i - lo + 1);
        last[s[i]] = i;
    }
    cout << ans << "\n";
}
```

## 第 10 章 二叉树

### 10.1 理论全掌握

#### 10.1.1 树的术语

- **节点（Node）**：树的基本单位
- **边（Edge）**：节点之间的连线
- **根（Root）**：唯一，没有父节点
- **父节点 / 子节点 / 兄弟节点**
- **度（Degree）**：节点的子树个数
- **叶子（Leaf）**：度为 0 的节点
- **内部节点**：度 ≠ 0
- **层次（Level）**：根为第 1 层
- **深度（Depth）**：根到节点的路径长度
- **高度（Height）**：节点到叶子的最长路径
- **森林（Forest）**：m 棵互不相交树的集合

#### 10.1.2 二叉树的定义

- **二叉树**：每个节点 ≤ 2 个子树（左、右），子树有顺序。
- **5 种形态**：
  - 空树
  - 只有根
  - 只有左子树
  - 只有右子树
  - 完整二叉树（左 + 右）

#### 10.1.3 特殊二叉树

- **满二叉树**：所有层都满。节点数 = `2^(h+1) - 1`（h 从 0 开始）。
- **完全二叉树**：除最后一层外全满，最后一层从左到右排列。
  - 数组存储最优
  - 高度 h = `⌊log₂ n⌋`
- **平衡二叉树**：左右子树高度差 ≤ 1
- **二叉查找树（BST）**：左 ≤ 根 ≤ 右
- **线索二叉树**：空指针指向前驱/后继
- **Huffman 树**：带权路径长度最小的二叉树

#### 10.1.4 二叉树的性质（必背）

- **第 i 层最多 2^(i-1) 个节点**（i ≥ 1）
- **高度 h 的二叉树最多 2^(h+1) - 1 个节点**（h ≥ 0，根高度 = 0）
- **叶子数 = 度为 2 的节点数 + 1**（`n₀ = n₂ + 1`）
- **总节点数 = n₀ + n₁ + n₂**
- **完全二叉树**：节点 i（0-indexed）的父 = (i-1)/2，左子 = 2i+1，右子 = 2i+2
- **完全二叉树高度**：`h = ⌊log₂ n⌋`（n 节点）

#### 10.1.5 二叉树的存储

**(1) 顺序存储**
- 用数组存（完全二叉树最合适）
- 节点 i 的左子 = 2i+1，右子 = 2i+2（0-indexed）
- 父 = (i-1)/2
- 一般二叉树会浪费空间（空位用特殊值）

**(2) 链式存储**
```cpp
struct Node {
    T data;
    Node *left, *right;
    Node(T v): data(v), left(nullptr), right(nullptr) {}
};
```
- 优点：节省空间、灵活
- 缺点：不能随机访问
- **三叉链表**：再加 `parent` 指针

#### 10.1.6 二叉树的遍历（必背）

**(1) 深度优先（DFS）**
- **前序（Pre-order）**：根 → 左 → 右
- **中序（In-order）**：左 → 根 → 右
- **后序（Post-order）**：左 → 右 → 根

**递归版**：
```cpp
void preorder(Node* p)  { if (!p) return; visit(p); preorder(p->left); preorder(p->right); }
void inorder(Node* p)   { if (!p) return; inorder(p->left); visit(p); inorder(p->right); }
void postorder(Node* p) { if (!p) return; postorder(p->left); postorder(p->right); visit(p); }
```

**非递归版**（用栈）：
```cpp
// 前序
vector<int> preorder(Node* root) {
    vector<int> res;
    stack<Node*> st;
    st.push(root);
    while (!st.empty()) {
        Node* p = st.top(); st.pop();
        if (!p) continue;
        res.push_back(p->val);
        st.push(p->right);   // 先右后左
        st.push(p->left);
    }
    return res;
}
// 中序
vector<int> inorder(Node* root) {
    vector<int> res;
    stack<Node*> st;
    Node* cur = root;
    while (cur || !st.empty()) {
        while (cur) { st.push(cur); cur = cur->left; }
        cur = st.top(); st.pop();
        res.push_back(cur->val);
        cur = cur->right;
    }
    return res;
}
// 后序（前序改造，根右左，再反转）
vector<int> postorder(Node* root) {
    vector<int> res;
    stack<Node*> st;
    st.push(root);
    while (!st.empty()) {
        Node* p = st.top(); st.pop();
        if (!p) continue;
        res.push_back(p->val);
        st.push(p->left);
        st.push(p->right);
    }
    reverse(res.begin(), res.end());
    return res;
}
```

**(2) 广度优先（BFS / 层序）**
- 用队列
- 一层一层遍历
```cpp
void levelOrder(Node* root) {
    if (!root) return;
    queue<Node*> q;
    q.push(root);
    while (!q.empty()) {
        Node* p = q.front(); q.pop();
        visit(p);
        if (p->left)  q.push(p->left);
        if (p->right) q.push(p->right);
    }
}
```

#### 10.1.7 由遍历序列构造二叉树

- **中序 + 前序** → 唯一确定
- **中序 + 后序** → 唯一确定
- **前序 + 后序** → 不能唯一（除非满二叉树）
- 思路：前序/后序找根，中序分左右，递归

#### 10.1.8 二叉查找树（BST）

- **定义**：左子树所有节点 < 根 < 右子树所有节点
- **中序遍历得到升序序列**
- **操作时间复杂度**：O(h)，h = 树高
  - 平均 O(log n)
  - 最坏 O(n)（退化成链表）

**基本操作**：
- **查找**：从根开始，小左大右
- **插入**：找到空位插入
- **删除**（最复杂）：
  - **叶节点**：直接删
  - **单子树**：子树顶替
  - **双子树**：用**前驱（左子树最大）** 或**后继（右子树最小）** 替代，再删前驱/后继
- **最小/最大**：一直向左/右

#### 10.1.9 平衡二叉树（AVL）—— 重点

- **定义**：BST + 左右子树高度差 ≤ 1
- **平衡因子 BF**：BF = h(left) - h(right)
- **失衡四种情况**：
  - **LL**：插入点在失衡节点的左孩子的左子树
  - **RR**：插入点在失衡节点的右孩子的右子树
  - **LR**：插入点在失衡节点的左孩子的右子树
  - **RL**：插入点在失衡节点的右孩子的左子树
- **调整**：
  - **LL → 右单旋**
  - **RR → 左单旋**
  - **LR → 先左后右双旋**
  - **RL → 先右后左双旋**
- **插入/删除后**：从插入/删除点向上回溯，更新 BF，发现失衡立刻调整
- **时间复杂度**：所有操作 O(log n)
- **空间**：每个节点多存一个 BF

#### 10.1.10 红黑树（Red-Black Tree）

- **5 大性质**：
  1. 每个节点是红或黑
  2. 根是黑
  3. 叶（NIL）是黑
  4. 红节点的子节点必须黑（不能连续红）
  5. 从任一节点到叶的所有路径黑节点数相同
- **保证**：树高 ≤ 2·log(n+1)
- **操作**：旋转 + 变色
- **应用**：C++ `std::map` / `std::set`、Java `TreeMap`、Linux CFS 调度
- **比 AVL 优势**：插入/删除旋转次数少（最多 3 次）

#### 10.1.11 伸展树（Splay Tree）

- **思想**：刚访问的节点"伸展"到根
- **三种操作**：
  - **Zig**：单次旋转（一字型 / 之字型尽头）
  - **Zig-Zig**：一字型连续两次旋转
  - **Zig-Zag**：之字型两次旋转
- **自适应性**：访问频繁的节点靠近根
- **摊还时间**：O(log n)
- **应用**：缓存、文本编辑器撤销

#### 10.1.12 堆（Heap）—— 见第 8 章

#### 10.1.13 Huffman 树（哈夫曼树）

- **带权路径长度 WPL**：`Σ wᵢ · lᵢ`（wᵢ = 权值，lᵢ = 路径长）
- **构造**：每次取两个最小权值的节点合并，O(n log n)
- **应用**：Huffman 编码（数据压缩）
- **性质**：WPL 最小的二叉树

#### 10.1.14 树的应用

- **文件系统**：目录树
- **数据库索引**：B 树 / B+ 树
- **表达式求值**：表达式树
- **决策树**：机器学习
- **编译器**：语法树 / AST
- **网络路由**：路由树
- **组织结构**：公司树

#### 10.1.15 常见考点

- 各种遍历（递归 + 非递归）必会
- 由遍历序列构造树
- BST 的查找/插入/删除（特别是**删除双子树**用前驱/后继）
- AVL 失衡的判断与调整（四种情况）
- 红黑树的 5 大性质
- 树的性质公式
- 各种树的对比（AVL / 红黑 / 伸展）

### 10.3 Lab18-题目1：二叉查找树（q209, 中等, 33 分）

**题意**：BST 实现插入、删除（**用前驱**）、查找；输入单词（key + 中文），输出查询路径 + 内容；不存在输出 `NULL`。

**关键点**：
- BST 节点：`{string key, string other; Node *l, *r;}`
- 插入：标准 BST 插入
- 删除：双子树用**前驱**（左子树最大）
- 查找：从根开始记录路径，输出路径上每个节点的 key
- 找不到时路径继续到 NULL，输出 `NULL`

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

// BST 节点
class TNode {
public:
    string key, other;
    TNode *l, *r;
    TNode(string k, string o): key(k), other(o), l(nullptr), r(nullptr) {}
};

// BST 封装
class BST {
    TNode* root;
    TNode* insertRec(TNode* p, string k, string o) {
        if (!p) return new TNode(k, o);
        if (k < p->key)      p->l = insertRec(p->l, k, o);
        else if (k > p->key) p->r = insertRec(p->r, k, o);
        return p;
    }
    TNode* findMax(TNode* p) { while (p->r) p = p->r; return p; }
    TNode* removeRec(TNode* p, string k) {
        if (!p) return nullptr;
        if (k < p->key)      p->l = removeRec(p->l, k);
        else if (k > p->key) p->r = removeRec(p->r, k);
        else {
            if (!p->l) { TNode* t = p->r; delete p; return t; }
            if (!p->r) { TNode* t = p->l; delete p; return t; }
            TNode* pre = findMax(p->l);          // 用前驱
            p->key = pre->key; p->other = pre->other;
            p->l = removeRec(p->l, pre->key);
        }
        return p;
    }
    void clear(TNode* p) { if (!p) return; clear(p->l); clear(p->r); delete p; }
public:
    BST(): root(nullptr) {}
    ~BST() { clear(root); }
    void insert(string k, string o) { root = insertRec(root, k, o); }
    void remove(string k)            { root = removeRec(root, k); }
    void query(string k) const {
        TNode* p = root;
        bool first = true;
        while (p) {
            if (!first) cout << " ";
            first = false;
            cout << p->key;
            if (k == p->key) { cout << " " << p->key << " " << p->other << "\n"; return; }
            p = (k < p->key) ? p->l : p->r;
        }
        cout << " NULL\n";
    }
};

int main() {
    int n; cin >> n;
    BST tree;
    for (int i = 0; i < n; i++) { string k, o; cin >> k >> o; tree.insert(k, o); }
    int d; cin >> d;
    for (int i = 0; i < d; i++) { string k; cin >> k; tree.remove(k); }
    string t1, t2; cin >> t1 >> t2;
    tree.query(t1);
    tree.query(t2);
}
```

### 10.4 Lab18-题目2：The Sum of Path（q215, 困难, 33 分）

**题意**：给定插入序列生成**BST**，判断是否存在根→叶路径节点和 = k。同时输出树的高度（仅 1 节点 = 0）。

**关键点**：
- 按 BST 插入方式建树（数值序列就是插入顺序）
- 高度 = `floor(log2(n))` 对**完全**BST？不一定，**一般树高**用递归算
- 路径和 = k：DFS，传入当前和，递归到叶节点判断
- 数值**不要求不重复**（样例里有重复），BST 重复节点按"≥"放右子树（约定）

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

// BST 节点
class TNode {
public:
    int v;
    TNode *l, *r;
    TNode(int x): v(x), l(nullptr), r(nullptr) {}
};

// BST 封装
class BST {
    TNode* root;
    TNode* insertRec(TNode* p, int x) {
        if (!p) return new TNode(x);
        if (x < p->v) p->l = insertRec(p->l, x);
        else          p->r = insertRec(p->r, x);
        return p;
    }
    int heightRec(TNode* p) const { return p ? 1 + max(heightRec(p->l), heightRec(p->r)) : -1; }
    bool hasPathRec(TNode* p, int sum) const {
        if (!p) return false;
        if (!p->l && !p->r) return p->v == sum;
        return hasPathRec(p->l, sum - p->v) || hasPathRec(p->r, sum - p->v);
    }
public:
    BST(): root(nullptr) {}
    void insert(int x) { root = insertRec(root, x); }
    int  height() const { return heightRec(root); }
    bool hasPathSum(int k) const { return hasPathRec(root, k); }
};

int main() {
    int n; cin >> n;
    BST tree;
    for (int i = 0; i < n; i++) { int x; cin >> x; tree.insert(x); }
    int k; cin >> k;
    cout << tree.height() << ' ' << (tree.hasPathSum(k) ? "true" : "false") << "\n";
}
```

### 10.5 Lab18-题目3：二分查找树的最矮公共祖先（q216, 中等, 34 分）

**题意**：BST 找 p 和 q 的 LCA（最低公共祖先）。

**关键点**：**BST 的 LCA 利用有序性**：
- 从根开始
- 若 p、q 都 < 根 → 递归左
- 若 p、q 都 > 根 → 递归右
- 否则（p ≤ 根 ≤ q 或反过来）→ 根就是 LCA

**C++ 模板**：
```cpp
#include <bits/stdc++.h>
using namespace std;

// BST 节点
class TNode {
public:
    int v;
    TNode *l, *r;
    TNode(int x): v(x), l(nullptr), r(nullptr) {}
};

// BST 封装
class BST {
    TNode* root;
    TNode* insertRec(TNode* p, int x) {
        if (!p) return new TNode(x);
        if (x < p->v) p->l = insertRec(p->l, x);
        else          p->r = insertRec(p->r, x);
        return p;
    }
    // 利用 BST 有序性求 LCA
    int lcaRec(TNode* p, int a, int b) const {
        while (p) {
            if (b < p->v) p = p->l;
            else if (a > p->v) p = p->r;
            else return p->v;
        }
        return -1;
    }
public:
    BST(): root(nullptr) {}
    void insert(int x) { root = insertRec(root, x); }
    int lca(int p, int q) const {
        int a = min(p, q), b = max(p, q);
        return lcaRec(root, a, b);
    }
};

int main() {
    int n; cin >> n;
    BST tree;
    for (int i = 0; i < n; i++) { int x; cin >> x; tree.insert(x); }
    int p, q; cin >> p >> q;
    cout << tree.lca(p, q) << "\n";
}
```

## 第 11 章 多叉树

### 11.1 理论全掌握

#### 11.1.1 树的存储（一般树 → 二叉树）

- **孩子表示法**：每个节点的孩子用链表连
- **双亲表示法**：每个节点存父节点下标
- **孩子兄弟表示法（最常用）**：
  - 节点 = `{data, firstChild, nextSibling}`
  - 把一般树**转化为二叉树**

#### 11.1.2 Trie 树（字典树 / 前缀树）—— 重点

- **定义**：每个节点有多个孩子指针（按字符数 26/52/64），专门为字符串集合设计
- **结构**：
  ```cpp
  struct TrieNode {
      TrieNode* children[26] = {nullptr};
      bool isEnd = false;     // 是否单词结尾
  };
  ```
- **操作**：
  - **插入**：从根开始，按字符走，没孩子就建；结尾标 `isEnd`
  - **查找**：按字符走，能走通 + 结尾是 `isEnd` 才算存在
  - **前缀匹配**：能走通即可（不管 `isEnd`）
- **时间复杂度**：O(L)，L = 字符串长度
- **空间复杂度**：O(总字符数 × 字符集大小)
- **应用**：
  - 字符串集合存储
  - 前缀匹配（自动补全、搜索建议）
  - 词频统计
  - 拼写检查
  - IP 路由（最长前缀匹配）
- **优点**：查找前缀效率高
- **缺点**：空间浪费（特别是大量无共同前缀的字符串）
- **压缩 Trie**：合并只有一个孩子的链
- **后缀树 / 后缀数组**：字符串子串问题

#### 11.1.3 B 树（B-Tree）—— 重点

- **动机**：磁盘 I/O 优化（减少访问次数）
- **m 阶 B 树定义**：
  - 每个节点最多 m 个孩子
  - 除根外，每个节点 ≥ ⌈m/2⌉ 个孩子
  - 根节点 ≥ 2 个孩子（除非只有 1 个 key）
  - 有 k 个孩子的节点有 k-1 个 key
  - 所有叶节点在同一层
  - 节点内 key 升序
- **高度**：`h = O(log_m n)`
- **m 越大 → 树高越小 → I/O 越少**
- **操作**：
  - 查找：从根开始，二分（节点内有序）
  - 插入：找到叶节点插入；满了就**分裂**
  - 删除：合并或借位
- **应用**：
  - 数据库索引
  - 文件系统
  - 外存数据结构

#### 11.1.4 B+ 树（B+ Tree）

- **vs B 树**：
  - B+ 树的**数据只在叶节点**
  - B+ 树的叶节点用**链表**相连
  - B+ 树内部节点的 key 是子树 key 的副本（不是数据）
- **优点**：
  - 范围查询快（链表连接）
  - 非叶节点只存索引，能存更多 key → 树更矮
  - 全树遍历方便（直接走链表）
- **应用**：
  - **MySQL InnoDB** 的主键索引
  - 关系数据库
- **聚簇索引 vs 非聚簇索引**：
  - 聚簇：数据按主键顺序存储（InnoDB）
  - 非聚簇：数据单独存，索引只存指针（MyISAM）

#### 11.1.5 多叉树比较

| 结构 | 用途 | 时间复杂度 | 空间 |
|---|---|---|---|
| Trie | 前缀匹配 | O(L) | 字符集 × 节点数 |
| B 树 | 外存索引 | O(log_m n) | 紧凑 |
| B+ 树 | 数据库 | O(log_m n) | 更紧凑 |
| 红黑树 | 内存平衡 | O(log n) | 节点 2-3 倍开销 |
| AVL | 内存平衡 | O(log n) | 节点 BF |

#### 11.1.6 常见考点

- Trie 树的插入/查找
- B 树的阶、节点数、高度
- B 树 vs B+ 树的区别
- 数据库索引原理
- 磁盘 I/O 与树高的关系

### 12.1 理论全掌握

#### 12.1.1 图的定义

- **图 G = (V, E)**：顶点集 V + 边集 E
  - V：顶点（vertex）的非空有限集
  - E：边（edge）的有限集
- **有向图 / 无向图**
- **加权图 / 无权图**
- **简单图 vs 多重图**：是否有重边/自环
- **稀疏图 / 稠密图**：边数远小于 V² / 接近 V²

#### 12.1.2 图的术语

- **度（Degree）**：
  - 无向图：`deg(v)` = 与 v 关联的边数
  - 有向图：`入度 in(v)` + `出度 out(v)` = `deg(v)`
- **邻接（Adjacent）**：两点有边相连
- **路径（Path）**：顶点序列，相邻顶点有边
- **简单路径**：顶点不重复
- **回路 / 环（Cycle）**：起点 = 终点
- **连通 / 强连通**：
  - 无向图：连通（任何两点有路径）
  - 有向图：强连通（任何两点双向可达）
- **连通分量 / 强连通分量**
- **生成树**：连通图含所有顶点的极小连通子图（V-1 条边）
- **完全图**：每对顶点都有边，n 顶点的无向完全图 Kₙ 有 n(n-1)/2 条边
- **子图 / 生成子图**

#### 12.1.3 握手定理

- 无向图：`Σ deg(v) = 2|E|`
- 有向图：`Σ in(v) = Σ out(v) = |E|`
- 度为奇数的顶点个数是偶数

#### 12.1.4 图的存储

**(1) 邻接矩阵**
- 适合**稠密图**
- 空间 O(V²)
- 判边存在 O(1)
- 遍历邻居 O(V)
```cpp
vector<vector<int>> adj(V, vector<int>(V, 0));
// 加边：adj[u][v] = w;
```

**(2) 邻接表**
- 适合**稀疏图**
- 空间 O(V+E)
- 判边存在 O(度数)
- 遍历邻居 O(度数)
```cpp
vector<vector<pair<int,int>>> adj(V);   // (邻居, 权值)
// 加边：adj[u].push_back({v, w});
```

**(3) 十字链表**（有向图）
- 节点 = `{data, firstIn, firstOut}`
- 边节点 = `{tail, head, hLink, tLink, weight}`
- 空间 O(V+E)

**(4) 邻接多重表**（无向图）
- 一条边只存一次
- 节省空间

#### 12.1.5 图的遍历

**(1) DFS（深度优先搜索）**
- 用**栈**（或递归）
- 步骤：从起点开始，访问 → 标记 → 递归访问未访问的邻居
- **时间复杂度**：O(V+E)（邻接表）/ O(V²)（邻接矩阵）
- **应用**：
  - 连通性
  - 路径查找
  - 拓扑排序（后序）
  - 连通分量
  - 强连通分量
```cpp
void dfs(int u) {
    visited[u] = true;
    for (int v : adj[u]) {
        if (!visited[v]) dfs(v);
    }
}
```

**(2) BFS（广度优先搜索）**
- 用**队列**
- 步骤：起点入队 → 弹出 → 访问 → 未访问邻居入队
- **时间复杂度**：O(V+E)
- **应用**：
  - 无权图最短路径
  - 层次遍历
  - 连通分量
```cpp
void bfs(int s) {
    queue<int> q;
    q.push(s); visited[s] = true; dist[s] = 0;
    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (int v : adj[u]) {
            if (!visited[v]) {
                visited[v] = true;
                dist[v] = dist[u] + 1;
                q.push(v);
            }
        }
    }
}
```

#### 12.1.6 拓扑排序

- **适用**：DAG（有向无环图）
- **Kahn 算法**（BFS）：
  1. 选入度 = 0 的顶点
  2. 删除该顶点 + 其出边
  3. 重复
- **DFS 法**：后序遍历，再反转
- **时间**：O(V+E)
- **检测环**：拓扑序列个数 < V → 有环
- **应用**：
  - 课程排课
  - 编译依赖
  - 任务调度

```cpp
vector<int> topo(int n) {
    vector<int> indeg(n, 0);
    for (int u = 0; u < n; u++)
        for (auto [v, w] : adj[u]) indeg[v]++;
    queue<int> q;
    for (int i = 0; i < n; i++) if (indeg[i] == 0) q.push(i);
    vector<int> res;
    while (!q.empty()) {
        int u = q.front(); q.pop();
        res.push_back(u);
        for (auto [v, w] : adj[u]) {
            if (--indeg[v] == 0) q.push(v);
        }
    }
    return res;   // size < n 表示有环
}
```

#### 12.1.7 最短路径

| 算法 | 适用 | 边权 | 时间 | 思想 |
|---|---|---|---|---|
| BFS | 无权 | - | O(V+E) | 队列层序 |
| **Dijkstra** | 单源、非负权 | ≥ 0 | O((V+E) log V)（堆优化） | 贪心 |
| Bellman-Ford | 单源、可负权 | 任意 | O(VE) | 动态规划 |
| **Floyd-Warshall** | 全对 | 任意 | O(V³) | 动态规划 |
| SPFA | 单源、负权 | 任意 | 平均 O(E) | Bellman-Ford 队列优化 |

**(1) Dijkstra（重点）**
- **限制**：边权非负
- **步骤**：
  1. `dist[s] = 0`，其他 `= ∞`
  2. 选 `dist` 最小的未访问顶点 `u`
  3. 遍历 `u` 的邻居 `v`，尝试松弛：`dist[v] = min(dist[v], dist[u] + w(u,v))`
  4. 标记 `u` 已访问
  5. 重复直到所有顶点访问完
- **堆优化**：用 priority_queue 存 `(dist, v)`，O((V+E) log V)
```cpp
vector<long long> dist(n, LLONG_MAX);
dist[s] = 0;
priority_queue<pair<long long,int>, vector<pair<long long,int>>, greater<>> pq;
pq.push({0, s});
vector<bool> visited(n, false);
while (!pq.empty()) {
    auto [d, u] = pq.top(); pq.pop();
    if (visited[u]) continue;
    visited[u] = true;
    for (auto [v, w] : adj[u]) {
        if (dist[u] + w < dist[v]) {
            dist[v] = dist[u] + w;
            pq.push({dist[v], v});
        }
    }
}
```

**(2) Bellman-Ford**
- **可检测负环**
- **步骤**：松弛 V-1 轮所有边
- **时间**：O(VE)
- **优化**：第 i 轮只考虑上一轮被更新的节点的出边（SPFA）

**(3) Floyd-Warshall**
- **全对最短路**
- **状态**：`dist[i][j]` = i 到 j 当前最短距离
- **转移**：`dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j])` for k = 0..V-1
- **时间**：O(V³)
- **空间**：O(V²)
- **可检测负环**：`dist[i][i] < 0` 则有负环
```cpp
for (int k = 0; k < n; k++)
    for (int i = 0; i < n; i++)
        for (int j = 0; j < n; j++)
            dist[i][j] = min(dist[i][j], dist[i][k] + dist[k][j]);
```

#### 12.1.8 最小生成树（MST）

- **适用**：连通**无向加权**图
- **性质**：
  - MST 有 V-1 条边
  - 不唯一（边权相同时）
  - **切割性质**：跨任意切割的最短边必在某个 MST 中
  - **环性质**：环上最长的边不在 MST 中

| 算法 | 思路 | 时间 | 适用 |
|---|---|---|---|
| **Prim** | 从一个顶点出发，长边扩展 | O(E log V) | 稠密图 |
| **Kruskal** | 按边权排序，并查集 | O(E log E) | 稀疏图 |

**(1) Prim**
- 维护"已加入 MST 的顶点集合"和"候选边"
- 每次选候选边中权值最小的，把对应顶点加入
- 用堆优化

**(2) Kruskal**
- 边按权值升序
- 依次考虑：若两端点不连通（并查集）→ 加入 MST
- 加到 V-1 条边为止
- **并查集（Union-Find）**：路径压缩 + 按秩合并，O(α(n))

```cpp
// Kruskal
struct Edge { int u, v, w; };
vector<Edge> edges;
sort(edges.begin(), edges.end(), [](auto& a, auto& b){ return a.w < b.w; });
vector<int> parent(n);
for (int i = 0; i < n; i++) parent[i] = i;
int find(int x) { return parent[x] == x ? x : parent[x] = find(parent[x]); }
int mst = 0, cnt = 0;
for (auto& e : edges) {
    int pu = find(e.u), pv = find(e.v);
    if (pu != pv) {
        parent[pu] = pv;
        mst += e.w;
        if (++cnt == n-1) break;
    }
}
```

#### 12.1.9 强连通分量（SCC）

- **定义**：极大强连通子图
- **Kosaraju 算法**：
  1. DFS 原图，按完成时间倒序
  2. 在反图上按这个顺序再 DFS，每个连通块就是一个 SCC
  3. 时间 O(V+E)
- **Tarjan 算法**：单次 DFS，用栈和 low 标记

#### 12.1.10 关键路径（AOE 网）

- **AOE 网**：边表示活动、点表示事件
- **关键路径**：从源点到汇点的最长路径（决定工程最短工期）
- **关键活动**：关键路径上的活动（无时差）

#### 12.1.11 图的应用

| 场景 | 算法 |
|---|---|
| 工程调度（依赖关系） | 拓扑排序 |
| 地图导航 | Dijkstra / Floyd |

| 推荐系统 | 图嵌入 / 随机游走 |

- 邻接矩阵 / 邻接表的转换
