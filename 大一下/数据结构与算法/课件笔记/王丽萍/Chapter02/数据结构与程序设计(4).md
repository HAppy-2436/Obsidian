### Slide 12

```
目录BracketMat下例程

栈的应用—数制转换

将十进制数N转换为r进制的数，其转换方法利用辗转相除法：以N=3467，r=8为例转换方法如下：
     N         	N / 8 （整除）   	N % 8（求余）
     3467      	433                    	3
     433        	54                      	1
     54          	6                       	6
     6            	0                       	6
所以：（3467）10  =（6613）8

算法思想如下：当N>=0时重复1，2
1. 若 N≠0，则将N % r 压入栈s中 ，执行2;若N=0，将栈s的内容依次出栈，算法结束。
2. 用N / r 代替 N
```

### Slide 13

```
#include <iostream.h>
#include <stack>
using namespace std;  //Must be added.
void main()
{
   int n;
   int r;
   stack<int> numbers;  //  declares and initializes a stack of numbers
   cout << "输入要转换的十进制数N： " << endl;
   cin  >> n;
   cout << "输入要转换的进制数r：" << endl;
   cin  >> r;
   while(n){
	  numbers.push(n%r);
	  n=n/r;
   }
   while (!numbers.empty()) {
      cout << numbers.top() << " ";
      numbers.pop();
   }
   cout << endl;
}
```
