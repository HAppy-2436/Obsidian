### Slide 13

```
bool user_says_yes()
{
   int c;
   bool initial_response = true;
   do {  //  Loop until an appropriate input is received.
      if (initial_response)
         cout << “ (y,n)? ” << flush;  //清除缓冲区
      else
         cout << "Respond with either y or n: " << flush;
      do { //  Ignore white space.
         c = cin.get();   //得到一个字符
      } while (c == '\n' || c ==' ' || c == '\t');
      initial_response = false;
   } while (c != 'y' && c != 'Y' && c != 'n' && c != 'N');
   return (c == 'y' || c == 'Y');
}
```

1. To improve your program, review the logic. Don't optimize

4. Keep your functions short; rarely should any function be more

6. Verify the intricate（复杂的） parts of your algorithm.
