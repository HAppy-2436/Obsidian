# 本章导读

> [!info] 章节定位
> 本章是「数据结构与算法基础篇」的导读，帮你建立整个章节的学习地图，明确应该按什么顺序、什么深度来学习后面 50 个章节。

## 📚 学习目标

- 了解本章在整门课程中的定位：基础数据结构 + 排序算法
- 理解「基本数据结构」与「高级数据结构」的边界
- 把握本章代码示例的多种语言风格与「以 Java/C++/C/Go/Python/JavaScript 六语言对照」的特点
- 知道如何借助本站的「算法可视化面板」理解复杂操作
- 明确学习本章需要的预备知识与预期产出

## 🎯 一句话总结

> 本章用 50 个章节系统讲清常见数据结构（数组、链表、队列、栈、哈希表、二叉堆、二叉树、图等）的**原理 + 代码实现 + 适用场景**，并完整覆盖十大排序算法；学完后能精准判断**该用什么数据结构**以及**它的复杂度为什么是这样**。

## 🔗 前置知识

本章是整个基础篇的起点，**没有前置章节**。但你最好具备：

- 任意一门编程语言的基本语法（推荐 C++ / Java / Python）
- 「变量」「循环」「函数」「对象引用」等基本概念
- 知道「时间」「空间」是什么，不需要先学复杂度（下一章会教）

### 3. 为什么必须理解底层原理

数据结构的价值不在于「我会用 `vector`」或「我会调 `list.remove(x)`」，而在于：

1. **能选对工具**：同样要在队尾插入数据，`vector` 摊销 O(1) 但偶尔 O(n) 卡顿，双链表是稳定 O(1)。没理解原理就只能凭感觉写。
2. **能算对复杂度**：面试时如果说不清 `unordered_map` 平均 O(1) 来自「哈希 + 链表 + 负载因子」的设计，会被一眼看穿。
3. **能调对 bug**：链表写挂的常见原因（指针没断开、空指针、虚拟头结点缺失）全都藏在原理里。
4. **能举一反三**：学会了动态数组，环形数组、循环队列的代码立刻就能写出来；学会了单链表，双向链表、红黑树都只是「多一条 prev 指针」。

### 4. 可视化面板的使用

本章对**稍微复杂**的数据结构操作都配有**算法可视化面板**，可视化代码是 JavaScript 写的，但变量命名和执行流和正文代码保持一致，看不懂 JS 也能跟上动画。

> [!tip] 学习建议
> 看正文 → 看代码 → **打开可视化面板手动点几下**（addLast / addFirst / removeFirst 等）→ 回到正文确认自己理解对了。

### 5. 多种语言代码的实现取舍

正文给出的多语言代码（Java/C++/C/Go/Python/JavaScript）只确保**正确性和可读性**，**不做语言层面的极致优化**。

| 关注点 | 本章态度 |
| --- | --- |
| 代码正确性 | ✅ 严格保证 |
| 命名与可读性 | ✅ 优先 |
| 时间复杂度结论 | ✅ 严格分析 |
| 标准库级别的边界优化 | ❌ 留给读者自行查标准库源码 |
| 极端内存布局优化 | ❌ 不在教学范围 |

> [!warning] 注意
> 后续面试/工程中如果对**性能/内存**有极致要求（如高频交易、嵌入式），请去读对应语言标准库的源码（`std::vector`、`java.util.ArrayList` 扩容策略等），而不是把本章代码当「最优实现」。

### 7. 本章不覆盖什么

诚实地说，本章不覆盖以下内容（避免初学者误以为学完本章就什么都会）：

- **图论高级算法**：如网络流、强连通分量、Tarjan 等，会在后续「图算法」章节展开。
- **字符串算法**：KMP、Z 函数、后缀数组等，会在「字符串」章节展开。
- **动态规划 / 贪心 / 回溯**：这些是「算法范式」不是「数据结构」，属于另一条主线。
- **语言级标准库源码细节**：见上文 5。

## 📊 复杂度一览

> 本章是导读，不引入具体 API 复杂度。整体复杂度在后续每个数据结构的章节给出。

| 数据结构 | 查 | 改 | 增（尾） | 增（中） | 删（尾） | 删（中） |
| --- | --- | --- | --- | --- | --- | --- |
| 静态数组 | O(1) | O(1) | O(1) | O(N) | O(1) | O(N) |
| 动态数组 | O(1) | O(1) | 摊销 O(1) | O(N) | 摊销 O(1) | O(N) |
| 单链表 | O(N) | O(N) | O(N) 无尾指针 / O(1) 有尾指针 | O(N) | O(N) | O(N) |
| 双链表 | O(N) | O(N) | O(1) | O(N) | O(1) | O(N) |
| 环形数组 | O(1) | O(1) | O(1) | — | O(1) | — |

> 完整版对比见 [[02-array-basic|数组原理]]、[[05-linkedlist-basic|链表原理]]。

## 🛠️ 应用场景

本章不是工具 API 章节，但有几个宏观应用场景值得在导读里点出：

- **笔试/面试**：本节对应的代码实现题（LeetCode 707、622、641 等）几乎覆盖了所有一线公司面试题的数据结构基础部分。
- **竞赛入门**：先掌握本章，再去啃动态规划、图论，效率最高。
- **工程实践**：标准库容器的选型、Redis 底层跳表、LevelDB 布隆过滤器、Linux 内核链表等，都需要本章知识。
- **跨语言迁移**：C++ `std::vector` / Java `ArrayList` / Python `list` / Go `slice` 全部对应「动态数组」，学会本章一个，其他立刻能上手。
