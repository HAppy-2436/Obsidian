## 第零部分 · 算法复杂度体系（地基）

> **学这门课的第一个误解**：以为「时间复杂度」=「跑得多快」。**它是「随着数据规模 n 增长，运行次数的增长趋势」**，常数项和低阶项都忽略。

### 复杂度速查表

| 表达式 | 名称 | 典型场景 | 课件出处 |
| --- | --- | --- | --- |
| O(1) | 常数 | 哈希直接寻址、数组按下标 | 课件（一2）算法复杂度 |
| O(log n) | 对数 | 二分查找、堆插入、平衡树查找 | 课件（四）贪心分治 |
| O(n) | 线性 | 顺序扫描、Hash 录入 | 课件（一2） |
| O(n log n) | 线性对数 | 快排 / 归并 / 堆排 | 课件（一2） / labuladong §40-46 |
| O(n²) | 平方 | 冒泡 / 选择 / 插入 | 课件（一2） |
| O(2ⁿ) / O(n!) | 指数 / 阶乘 | 子集枚举、全排列 | 课件（一）基础 |

### 三个判断口诀

1. **「看最深层 for 循环」**：时间复杂度 = 最大嵌套层数（初学者可记）
2. **「看申请了多少空间」**：空间复杂度 = 额外数组 / 递归栈大小
3. **「最坏情况优先」**：同一题最优/平均/最坏可能不同（如快排最坏 O(n²)）

### 经典对比案例（课件原题）

> 给 n 个整数按从大到小输出前 m 大数（n, m ≤ 10⁶）
>
> - **冒泡排序**：O(n²)，n=10⁶ 时计算 3×10¹¹ 次 → 必超时
> - **快速排序 STL**：`sort()`，O(n log n) → 2000 万次，可过
> - **Hash 标记**：O(n) → 100 万次，最快

> **核心结论**：当 n ≤ 10⁸ 时可"暴力"；n > 10⁸ 必须用 O(n log n) 或更优。

## 第一部分 · 基础数据结构

### 1.1 数组（顺序表）

- **是什么**：连续内存、按下标 O(1) 随机访问
- **关键操作**：
  - 访问 / 修改：`O(1)`
  - 头部插入 / 删除：`O(n)`（要搬动后续元素）
  - 尾部插入（动态数组摊还）：`O(1)`
- **典型场景**：哈希表底层、堆、字符串、位图
- **陷阱**：动态数组扩容是「倍增策略」，单次最坏 O(n)，但摊还 O(1)
- **C++**：使用 `vector<T>` / `array<T, n>` / `bitset`

### 1.2 链表（链式存储）

- **是什么**：动态申请节点，节点含 `data` + `next`（双向链表再加 `prev`）
- **关键操作**（以课件「士兵队列」代码为准）：
  - 头插 / 头删：`O(1)`
  - 查找第 i 个节点：`O(n)`（必须从头走）
  - 中间插入 / 删除（已知前驱）：`O(1)`
  - 求链表长度：`O(n)`
- **三种形态**：
  - **单链表**：`struct Node { T data; Node* next; }`
  - **双向链表**：再加 `Node* prev`，插入/删除需改 4 个指针
  - **循环链表**：尾节点 `next` 指向头节点（→ 约瑟夫环问题的天然结构）
- **STL 对应**：`list<T>`（双向链表）、`forward_list<T>`（单向，C++11）
- **vs vector 选型**（课件原话）：
  - 插入删除频繁 + 随机访问少 → `list`
  - 随机访问频繁 + 插入删除少 → `vector`

### 1.3 栈（Stack）

- **是什么**：LIFO（后进先出），只允许在栈顶操作
- **关键操作**：`push` / `pop` / `top` 均 `O(1)`
- **典型场景**：
  - 括号匹配、表达式求值（前缀 / 中缀 / 后缀）
  - 递归 → 手动维护调用栈
  - DFS 非递归实现
  - 单调栈（接雨水、柱状图最大矩形）
- **C++**：`stack<T>` 配 `push/pop/top/empty`

### 1.4 队列（Queue）

- **是什么**：FIFO（先进先出）
- **关键操作**：`push`(back) / `pop`(front) / `front` / `back` 均 `O(1)`
- **两种实现**：
  - **循环队列**：数组 + 头尾指针，判满用 `(rear+1) % cap == front`
  - **链队列**：链表 + front / rear 两个指针
- **典型场景**：
  - **BFS**（广度优先遍历）
  - **SPFA**（最短路的队列优化，本质是 BFS 思想）
  - 任务调度、缓冲区
- **C++**：`queue<T>` 配 `push/pop/front/back/empty`；优先队列用 `priority_queue<T>`

### 1.5 双端队列（Deque）

- 头尾都可 `push` / `pop`，是栈 + 队列的超集
- 典型场景：滑动窗口最大值、单调队列

## 第二部分 · 递归与分治

### 2.1 递归三要素

1. **递归终止条件（边界）**
2. **递归表达式**（大问题 → 小问题）
3. **返回值定义**

```cpp
// 斐波那契（课件原题，O(2^n) 朴素版）
int f[1000001];
f[1] = f[2] = 1;
for (int i = 3; i <= n; ++i)
    f[i] = (f[i-1] + f[i-2]) % 1000000007;
```

### 2.2 分治法（Divide & Conquer）

> **两个特征**（课件《四》贪心分治）：
> 1. **平衡子问题**：子问题规模大致相同
> 2. **独立子问题**：子问题之间不重叠（**这是和 DP 的根本区别**）

**三步骤**：
- **分解（Divide）**：把问题切成 k 个子问题
- **解决（Conquer）**：递归解决子问题
- **合并（Combine）**：把子问题结果合起来

**典型应用**：归并排序、快速排序、汉诺塔、大整数乘法、Strassen 矩阵乘法

### 2.3 八皇后 / 递归回溯（课件原题）

```cpp
// 伪代码：在第 k 行选一列放皇后，列、主对角线、副对角线都不冲突
void dfs(int k) {
    if (k == n) { ans++; return; }  // 边界
    for (int j = 0; j < n; ++j) {
        if (col[j] || diag1[k+j] || diag2[k-j+n]) continue;
        col[j] = diag1[k+j] = diag2[k-j+n] = true;
        dfs(k+1);
        col[j] = diag1[k+j] = diag2[k-j+n] = false;  // 回溯
    }
}
```

## 第三部分 · 排序算法全家福

> **两个维度区分排序**：
> 1. **时间复杂度**（O(n²) vs O(n log n) vs 线性）
> 2. **稳定性**：相等元素排序后相对位置不变 → 「稳定排序」

### 3.2 关键代码模板

#### 归并排序（课件《四》+ labuladong §44，"妙用后序位置"）

```cpp
void mergeSort(int a[], int left, int right) {
    if (left >= right) return;
    int mid = left + (right - left) / 2;   // 防溢出
    mergeSort(a, left, mid);
    mergeSort(a, mid+1, right);
    merge(a, left, mid, right);
}
// 合并 a[left..mid] 与 a[mid+1..right] 到临时数组
// 递归合并后，左半 + 右半都已有序，单次 O(n)
```

**升级用法：求逆序对数（课件原题，hdu 4911）**

```cpp
long long cnt = 0;  // 逆序对数，可能超过 int
void merge(int a[], int l, int m, int r) {
    // ... 同上
    if (a[i] > a[j]) {
        cnt += m - i + 1;  // a[i..m] 都比 a[j] 大
        // ...
    }
}
// 最终结果：max(0, cnt - k)  // k 是允许交换次数
```

#### 快速排序（课件《四》+ labuladong §45）

```cpp
void qsort(int a[], int l, int r) {
    if (l >= r) return;
    int i = l, j = r, mid = a[(l+r)/2];
    while (i <= j) {
        while (a[i] < mid) i++;
        while (a[j] > mid) j--;
        if (i <= j) swap(a[i++], a[j--]);
    }
    qsort(a, l, j);
    qsort(a, i, r);
}
```

**快速选择**（求第 k 大）→ 平均 O(n)：
```cpp
int quickSelect(int a[], int l, int r, int k) {
    if (l == r) return a[l];
    int p = partition(a, l, r);     // 同 qsort 划分
    if (k <= p - l) return quickSelect(a, l, p, k);
    return quickSelect(a, p+1, r, k - (p - l + 1));
}
```

### 3.3 选择排序的稳定性陷阱

- 选择排序每次从未排序区**交换**最小值到前面 → 相等元素会换位
- 归并排序 / 冒泡排序 / 插入排序 **稳定**（只在 `>` 时交换，不在 `>=` 时交换）

## 第四部分 · 查找

### 4.1 顺序查找

- 时间 `O(n)`，无序也能用

### 4.2 二分查找（必须熟）

**前提**：有序数组 / 满足单调性的解空间

**两种风格**：
- 找**精确值**：`while (l <= r)`，用闭区间
- 找**左边界 / 右边界**：`while (l < r)`，用开区间（labuladong 风格）

```cpp
// 标准左闭右闭
int bsearch(int a[], int n, int target) {
    int l = 0, r = n - 1;
    while (l <= r) {
        int mid = l + (r - l) / 2;  // 防溢出
        if (a[mid] == target) return mid;
        else if (a[mid] < target) l = mid + 1;
        else r = mid - 1;
    }
    return -1;
}

// 找第一个 >= target 的位置（左边界）
int lowerBound(int a[], int n, int target) {
    int l = 0, r = n;  // [l, r) 半开区间
    while (l < r) {
        int mid = l + (r - l) / 2;
        if (a[mid] < target) l = mid + 1;
        else r = mid;     // 收缩右边界
    }
    return l;  // l == r == 第一个 >= target 的位置
}
```

**LIS 优化版**（课件《四》DP）+ 二分：

```cpp
// 最长上升子序列，O(n log n)
int ans[MAXN], len = 1;
ans[1] = a[1];
for (int i = 2; i <= n; ++i) {
    if (a[i] > ans[len]) ans[++len] = a[i];
    else {
        int pos = lower_bound(ans+1, ans+len+1, a[i]) - ans;
        ans[pos] = a[i];
    }
}
cout << len << endl;
```

### 4.3 哈希表（Hash）

- **思想**：以空间换时间 → 课件原题「找前 m 大数」用 hash 做到 O(n)
- **关键操作**：插入 / 查找 / 删除 平均 O(1)，最坏 O(n)
- **冲突处理**（课件 chapter09 Hash）：
  - **链地址法（拉链法）**：每个桶挂一个链表
  - **开放定址法**：冲突时探测下一个空位
    - 线性探测：`H + 1, H + 2, ...`
    - 二次探测：`H + 1², H + 2², ...`
    - 双重哈希
- **C++**：
  - `unordered_map<K, V>` / `unordered_set<K>` → 哈希实现，无序
  - `map<K, V>` / `set<K>` → 红黑树实现，**有序**，O(log n)

## 第五部分 · 树

### 5.1 树的基础术语（课件 ch10-01）

| 术语 | 定义 |
| --- | --- |
| 根（Root） | 没有父节点的节点 |
| 父 / 子 / 兄弟 | 父子关系 / 同父节点 |
| 叶子（External node） | 无子节点 |
| 度（Degree） | 节点的子节点数 |
| 深度（Depth） | 从根到该节点的边数 |
| 高度（Height） | 树中节点最大深度 |

**核心等式**（必考）：设 n₀ / n₁ / n₂ 分别为 0/1/2 度节点数，则 **n₀ = n₂ + 1**

### 5.2 二叉树（Binary Tree）

**定义**：每个节点至多 2 个子节点（递归：空 或 根 + 左子树 + 右子树）

**两种特殊形态**：
- **满二叉树**：深度 k，结点数 2^(k+1) - 1
- **完全二叉树**：除最下层右侧外都满 → **数组存储最优**（课件原话）
  - 节点 i 的左孩子：`2i+1`
  - 节点 i 的右孩子：`2i+2`
  - 节点 i 的父节点：`(i-1)/2`（堆排序的关键！）

**三种遍历**（必背手写）：

| 遍历 | 顺序 | 递归伪代码 |
| --- | --- | --- |
| **前序**（Preorder / NLR） | 根 → 左 → 右 | `visit(root); preorder(left); preorder(right);` |
| **中序**（Inorder / LNR） | 左 → 根 → 右 | `inorder(left); visit(root); inorder(right);` |
| **后序**（Postorder / LRN） | 左 → 右 → 根 | `postorder(left); postorder(right); visit(root);` |

> **labuladong 框架型理解**（§25-26）：
> - 前序位置 = "刚进入这个节点"
> - 中序位置 = "左子树处理完，即将处理右子树"
> - 后序位置 = "左右子树都处理完，即将离开"
> - **妙用前序**：快速排序的 partition（构造左右子区间）
> - **妙用后序**：归并排序的 merge（左右已排序，合并）

**例**：表达式树 `+ a * b - c d / e f`
- 前序：`- + a * b - c d / e f`
- 中序：`a + b * c - d - e / f`（依赖括号还原）
- 后序：`a b c d - * + e f / -`

**层序遍历（BFS）**：

```cpp
queue<TreeNode*> q;
q.push(root);
while (!q.empty()) {
    TreeNode* p = q.front(); q.pop();
    visit(p);
    if (p->left)  q.push(p->left);
    if (p->right) q.push(p->right);
}
```

### 5.3 二叉搜索树（BST）

**性质**：左子树 < 根 < 右子树（中序遍历必为升序）

| 操作 | 平均 | 最坏 |
| --- | --- | --- |
| 查找 | O(log n) | O(n)（退化成链表） |
| 插入 | O(log n) | O(n) |
| 删除 | O(log n) | O(n)（最复杂：3 种情况） |

**删除 3 种情况**（课件原话）：
1. 叶子节点：直接删
2. 只有一个子树：子树顶替
3. 有两个子树：用**前驱**（左子树最右）或**后继**（右子树最左）替代

### 5.4 AVL 树（平衡二叉搜索树）

> **动机**：BST 退化成链表 → 引入"平衡因子"

**定义**：左右子树高度差 ≤ 1（Height Balance）

**平衡因子 BF**：`BF = h(左) - h(右)`，取值为 `-1, 0, 1`

**四种失衡**（LL / RR / LR / RL）：
- **LL**：在失衡节点的**左孩子的左子树**插入 → **右旋**（单旋）
- **RR**：在失衡节点的**右孩子的右子树**插入 → **左旋**（单旋）
- **LR**：在失衡节点的**左孩子的右子树**插入 → **先左旋后右旋**（双旋）
- **RL**：在失衡节点的**右孩子的左子树**插入 → **先右旋后左旋**（双旋）

**代码骨架**（课件 ch10-03）：

```cpp
void rotate_left(Node* &sub_root) {
    Node* r = sub_root->right;
    sub_root->right = r->left;
    r->left = sub_root;
    sub_root = r;
}
void rotate_right(Node* &sub_root) { /* 对称 */ }

// 插入时自底向上检查 BF
// 若某节点 BF = ±2，按上述 4 种情况旋转
```

**复杂度**：查找 / 插入 / 删除 **O(log n)**，最坏情况不退化

### 5.5 伸展树（Splay Tree）

- **思想**：最近访问的节点 → 旋转到根（自调整）
- **操作**：zig / zig-zig / zig-zag
- **均摊复杂度 O(log n)**，但单次最坏 O(n)
- 适合**访问局部性**强的场景

### 5.6 多路树与 B / B+ 树（课件 ch11）

**动机**：二叉树深度大 → 磁盘 I/O 次数多 → 引入多路树

**B 树性质**（m 阶）：
- 每个节点**最多 m 个子节点**
- 除根外，每个节点**至少 ⌈m/2⌉ 个子节点**
- 节点内关键字数 = 子节点数 - 1
- 所有叶子在同一层

**B+ 树 vs B 树**：
- B+ 树**数据只在叶子**，内部节点只存索引
- B+ 树**叶子形成链表** → 范围查询高效
- 数据库索引普遍用 B+ 树

**2-3 树 / 2-3-4 树**：B 树的特例（m=3 / m=4）

## 第六部分 · 高级树

### 6.1 堆（Heap）

**定义**：完全二叉树 + 父节点 ≥（或 ≤）子节点的值

- **大顶堆**：父 ≥ 子 → 根最大
- **小顶堆**：父 ≤ 子 → 根最小

**数组存储**：
- 父 i → 左孩子 `2i+1`，右孩子 `2i+2`
- 子 i → 父 `(i-1)/2`

**核心操作**（O(log n)）：
- `sift_up(i)`：上浮（插入时用）
- `sift_down(i)`：下沉（删除 / 建堆时用）

**应用**：
1. **堆排序**（课件《九》）：建堆 O(n) + 反复取顶 O(n log n) → 总 O(n log n)，**原地**
2. **优先队列**（`priority_queue`）：每次取最大/最小

```cpp
// 堆排序
void heapSort(vector<int>& a) {
    int n = a.size();
    // 建堆
    for (int i = n/2 - 1; i >= 0; --i) siftDown(a, i, n);
    // 排序
    for (int i = n - 1; i > 0; --i) {
        swap(a[0], a[i]);
        siftDown(a, 0, i);
    }
}
```

### 6.2 字典树（Trie / 前缀树）

- **应用**：前缀匹配、词频统计、自动补全
- **结构**：每个节点有 26（或 52 / 128）个孩子的指针
- **复杂度**：插入 / 查找 **O(L)**，L 是字符串长度

### 6.3 并查集（Union-Find / DSU）

- **操作**：
  - `find(x)`：找 x 的根（带**路径压缩**后近 O(1)）
  - `union(x, y)`：合并两个集合（带**按秩合并**后近 O(1)）

```cpp
int fa[MAXN];
int find(int x) { return fa[x] == x ? x : fa[x] = find(fa[x]); }
void unite(int x, int y) {
    x = find(x); y = find(y);
    if (x != y) fa[x] = y;
}
```

- **应用**：最小生成树（Kruskal）、连通分量、朋友圈问题

### 6.4 其他高级结构（了解）

- **线段树**：区间查询 / 修改，O(log n)
- **树状数组（Fenwick）**：前缀和，O(log n)，代码更短
- **红黑树**：`map` / `set` 的底层，平衡通过颜色约束保证

## 第七部分 · 图

### 7.1 图的基础（课件 ch12-01）

**定义**：`G = (V, E)`，V 是顶点集，E 是边集

**分类**：
- **有向图 vs 无向图**：边是否有方向
- **加权图**：边有权值（距离 / 成本）
- **简单路径**：顶点不重复
- **环（Cycle）**：起点 = 终点的路径
- **连通图**：任意两顶点有路径

**度的等式**（必考）：`Σ 顶点的度 = 2 × 边数`

### 7.2 图的表示

| 表示 | 空间 | 判 u-v 相邻 | 找 v 所有邻居 | 适合 |
| --- | --- | --- | --- | --- |
| **邻接矩阵** | O(V²) | O(1) | O(V) | 稠密图 |
| **邻接表** | O(V+E) | O(degree) | O(degree) | 稀疏图 |

```cpp
// 邻接表（链式前向星，课件《七》最短路）
struct Edge { int v, w, next; } e[MAXM];
int p[MAXN], eid = 0;
void insert(int u, int v, int w) {
    e[eid] = {v, w, p[u]};
    p[u] = eid++;
}
```

### 7.3 图的遍历

#### DFS（深度优先搜索）—— 栈

```cpp
void dfs(int u) {
    visited[u] = true;
    for (int v : adj[u])
        if (!visited[v]) dfs(v);
}
```

#### BFS（广度优先搜索）—— 队列

```cpp
void bfs(int s) {
    queue<int> q;
    q.push(s);
    visited[s] = true;
    while (!q.empty()) {
        int u = q.front(); q.pop();
        for (int v : adj[u]) {
            if (!visited[v]) {
                visited[v] = true;
                q.push(v);
            }
        }
    }
}
```

**DFS vs BFS**：
- DFS：找一条可行解、拓扑排序、连通分量
- BFS：最短路（无权图）、层次遍历、最小步数

### 7.4 拓扑排序

- **场景**：DAG（有向无环图）上，将顶点排成线性序列使每条 u→v 边，u 在 v 之前
- **算法**：Kahn 算法，每次取**入度为 0** 的顶点
- **检测环**：拓扑排序后节点数 < 总数 → 有环

### 7.5 最短路算法（课件《七》核心）

| 算法 | 类型 | 边权 | 复杂度 | 备注 |
| --- | --- | --- | --- | --- |
| **BFS** | 单源 | 无权 | O(V+E) | 队列 |
| **Dijkstra** | 单源 | 非负 | O(V²) 朴素 / O((V+E) log V) 堆优化 | **必考** |
| **SPFA** | 单源 | 可负 | O(kE) 平均 | Bellman-Ford 队列优化 |
| **Bellman-Ford** | 单源 | 可负 | O(VE) | 可判负环 |
| **Floyd** | 多源 | 可负 | **O(V³)** | DP 思想，三重循环 |

#### Dijkstra（核心代码，课件《七》）

```cpp
// 朴素版 O(V²)
bool dijkstra(int s) {
    memset(dist, 0x3f, sizeof(dist));
    dist[s] = 0;
    for (int i = 0; i < n; ++i) {
        int u = -1, min_w = INF;
        for (int j = 0; j < n; ++j)
            if (!vst[j] && dist[j] < min_w) { min_w = dist[j]; u = j; }
        if (u == -1) return false;  // 不可达
        vst[u] = true;
        for (int v : adj[u])           // 松弛
            if (dist[u] + w(u,v) < dist[v])
                dist[v] = dist[u] + w(u,v);
    }
    return true;
}
```

**关键**：「松弛」操作 = `dist[v] = min(dist[v], dist[u] + w)`，所有最短路算法的核心

#### Floyd（多源，3 行）

```cpp
for (int k = 0; k < n; ++k)             // 中间节点
    for (int i = 0; i < n; ++i)
        for (int j = 0; j < n; ++j)
            g[i][j] = min(g[i][j], g[i][k] + g[k][j]);
```

#### 差分约束系统

- 形如 `x_j - x_i ≤ k` 的不等式组
- 连边 `<i, j>` 权值 k，求**最短路**（不存在负环则有解）
- 应用：小明银行卡、排队问题

## 第八部分 · 算法范式

### 8.1 贪心算法

- **思想**：每一步取**当前**最优，不回头
- **两个特征**（课件《四》）：
  1. **最优子结构**：问题的最优解包含子问题的最优解
  2. **贪心选择性质**：局部最优能推出全局最优

**经典例题**：
- 活动安排（按**结束时间**最早排序）
- 区间覆盖（按左端点排序，每次选右端点最大的）
- Huffman 编码（合并频次最小的两棵树，O(n log n)）
- Dijkstra（每步选 dist 最小的点）

**反例**：钱币问题 1/2/5 → 贪心有效；1/2/4/5/6 付 9 → 贪心无效（要 6+2+1，DP 才是 5+4）

### 8.2 动态规划（DP）

> **三大前提**（课件《四》）：
> 1. **最优化原理**：最优策略的子策略也是最优
> 2. **无后效性**：某阶段状态确定后，后续不受之前影响
> 3. **子问题重叠**：否则直接分治就行

**四步走**：
1. **定义状态**：`dp[i][j]` 是什么
2. **写转移方程**：大问题 → 小问题
3. **初始化**：边界条件
4. **遍历顺序**：保证用到的小问题已算过

#### 0-1 背包（课件《四》）

```cpp
// 状态：dp[i][j] = 前 i 个物品、容量 j 下的最大价值
// 转移：
//   dp[i][j] = dp[i-1][j]                                    // 不放第 i 个
//   dp[i][j] = max(dp[i][j], dp[i-1][j-w[i]] + v[i])        // 放第 i 个
for (int i = 1; i <= N; ++i)
    for (int j = 0; j <= V; ++j) {
        dp[i][j] = dp[i-1][j];
        if (j >= w[i]) dp[i][j] = max(dp[i][j], dp[i-1][j-w[i]] + v[i]);
    }
```

**空间优化**（一维滚动数组）：

```cpp
vector<int> dp(V+1, 0);
for (int i = 1; i <= N; ++i)
    for (int j = V; j >= w[i]; --j)  // ⚠️ 逆序！
        dp[j] = max(dp[j], dp[j-w[i]] + v[i]);
```

#### 完全背包（每种物品无限）

- 0-1 背包的"**正序**"版本（每个物品可重复选）：

```cpp
for (int i = 1; i <= N; ++i)
    for (int j = w[i]; j <= V; ++j)  // ✅ 正序
        dp[j] = max(dp[j], dp[j-w[i]] + v[i]);
```

#### 多重背包（每种物品有限 c_i 个）

- 朴素：拆成 c_i 个 0-1 物品，O(N·C·V)
- **二进制拆分优化**：把 c_i 拆成 1, 2, 4, ..., 2^k, 余数 → log c_i 个物品

#### LIS（最长上升子序列）

- O(n²) DP：`dp[i] = max{dp[j] + 1 | j < i, a[j] < a[i]}`
- **O(n log n) 优化**：维护 ans[] 数组 + 二分（见 §4.2）

#### LCS（最长公共子序列）

```cpp
// dp[i][j] = a 前 i 字符 与 b 前 j 字符的 LCS 长度
for (int i = 1; i <= n; ++i)
    for (int j = 1; j <= m; ++j) {
        if (a[i] == b[j]) dp[i][j] = dp[i-1][j-1] + 1;
        else dp[i][j] = max(dp[i-1][j], dp[i][j-1]);
    }
```

**回文串**（最少添加字符数）= `n - LCS(s, reverse(s))`

#### 状压 DP（课件《四》§4.7）

- 用**位掩码**表示集合 → 状态压缩为整数
- 适合 n ≤ 20 的子集枚举
- 例：n×m 网格最多选多少互不相邻格子
  - `dp[i][state]` = 前 i 行，第 i 行状态为 state 时的最大选择数
  - `(now & prev) == 0` 表示两行不冲突
  - `(now | flag) == flag` 表示选择合法
  - 时间 `O(n × 2^m × 2^m)`

### 8.3 字符串算法：KMP

- **问题**：在文本 T 中找模式 P 出现位置
- **朴素 O(n·m)** vs **KMP O(n+m)**
- **核心**：next[] 数组 = P 的**最长相等前后缀**长度
  - `next[i]` = 模式串前 i+1 个字符中，最长"前缀 = 后缀"的长度
  - 失配时模式串不回到开头，只回退到 `next[i]`

```cpp
// next 数组构造
next[0] = 0;
for (int i = 1, j = 0; i < m; ++i) {
    while (j > 0 && p[i] != p[j]) j = next[j-1];
    if (p[i] == p[j]) ++j;
    next[i] = j;
}
// KMP 匹配
for (int i = 0, j = 0; i < n; ++i) {
    while (j > 0 && t[i] != p[j]) j = next[j-1];
    if (t[i] == p[j]) ++j;
    if (j == m) { /* 找到了！i - m + 1 */ j = next[j-1]; }
}
```

#### 题目 10（易）— 链表反转

手写单链表反转函数。输入：1→2→3→4→5→NULL，输出：5→4→3→2→1→NULL

**考点**：链表基本操作（迭代 / 递归两种都要会）
**答案**：

```cpp
Node* reverse(Node* head) {
    Node* prev = NULL, *cur = head;
    while (cur) {
        Node* nxt = cur->next;
        cur->next = prev;
        prev = cur;
        cur = nxt;
    }
    return prev;
}
```

**提示**：课件《二2》链表 list 有原题

#### 题目 11（易）— 二分查找

在严格升序数组中找 target 第一次出现的位置，不存在返回 -1。

**考点**：左边界二分
**答案**：

```cpp
int lowerBound(int a[], int n, int target) {
    int l = 0, r = n;
    while (l < r) {
        int mid = l + (r - l) / 2;
        if (a[mid] < target) l = mid + 1;
        else r = mid;
    }
    return (l < n && a[l] == target) ? l : -1;
}
```

```cpp
    adj[v].push_back({u, w});

// (2) BFS 判连通
bool isConnected() {

            if (!vis[v]) { vis[v] = true; q.push(v); ++cnt; }

    return cnt == n;
}
```

### 附：易错点 Top 10（来自课件）

1. **二分边界**：`while (l <= r)` vs `while (l < r)` 用错
2. **DP 顺序**：0-1 背包必须**逆序**，完全背包必须**正序**
3. **Dijkstra 不能有负权边**（但 SPFA 可以）
4. **AVL 旋转**：左左右右分不清 → 抓住"新节点在失衡节点的哪一侧"
5. **快速排序不稳定**：partition 交换会打乱相等元素
6. **归并排序空间 O(n)**：需要临时数组
7. **DFS 用栈，BFS 用队列**：不要混
8. **邻接表空间 O(V+E)** 不是 O(V²)，但需要正确初始化 `memset(p, -1, ...)`
9. **KMP 的 next 数组**：是"最长前缀后缀"，**不是**"最大匹配长度"
10. **复杂度判断**：看的是**最坏情况**（快排 O(n²)），不是平均情况

### T1 · 链表快慢指针（判环 + 找环入口）

```cpp
// 返回环入口节点；无环返回 NULL
Node* detectCycle(Node* head) {
    Node *slow = head, *fast = head;
    while (fast && fast->next) {
        slow = slow->next;
        fast = fast->next->next;
        if (slow == fast) {                  // 有环
            Node* p = head;
            while (p != slow) { p = p->next; slow = slow->next; }
            return p;                        // 入口
        }
    }
    return NULL;                             // 无环
}
```
**原理**：相遇后，`head` 和 `slow` 同速走必在入口相遇（数学：`a + (n+1)b + nc = 2(a+b) ⇒ a = c`）。

### T5 · 堆排序（原地，O(n log n)）

```cpp
void heapify(vector<int>& a, int n, int i) {
    for (int largest = i; ;) {
        int l = 2*i+1, r = 2*i+2;
        if (l < n && a[l] > a[largest]) largest = l;
        if (r < n && a[r] > a[largest]) largest = r;
        if (largest == i) break;
        swap(a[i], a[largest]); i = largest;
    }
}
void heapSort(vector<int>& a) {
    int n = a.size();
    for (int i = n/2 - 1; i >= 0; --i) heapify(a, n, i);  // 建堆
    for (int i = n-1; i > 0; --i) { swap(a[0], a[i]); heapify(a, i, 0); }
}
```

### T6 · 归并排序（O(n log n) 稳定）

```cpp
void mergeSort(vector<int>& a, int l, int r) {
    if (l >= r) return;
    int m = (l + r) / 2;
    mergeSort(a, l, m); mergeSort(a, m+1, r);
    // merge
    vector<int> tmp(r-l+1); int i=l, j=m+1, k=0;
    while (i<=m && j<=r) tmp[k++] = a[i]<a[j] ? a[i++] : a[j++];
    while (i<=m) tmp[k++] = a[i++];
    while (j<=r) tmp[k++] = a[j++];
    for (int p=0; p<k; ++p) a[l+p] = tmp[p];
}
```

### T7 · Kruskal 最小生成树（并查集 + 边排序）

```cpp
struct DSU { int fa[MAXN]; void init(int n){ iota(fa, fa+n+1, 0); }
    int find(int x){ return fa[x]==x?x:fa[x]=find(fa[x]); }
    bool unite(int x,int y){ x=find(x); y=find(y); if(x==y) return false; fa[x]=y; return true; }
};
int kruskal(int n, vector<Edge>& e) {
    sort(e.begin(), e.end(), [](auto&a, auto&b){return a.w<b.w;});
    DSU d; d.init(n);
    int ans = 0, cnt = 0;
    for (auto& ed : e) if (d.unite(ed.u, ed.v)) { ans += ed.w; if (++cnt == n-1) break; }
    return cnt == n-1 ? ans : -1;       // -1 表示不连通
}
```

### T8 · Prim 最小生成树（稠密图朴素 O(V²)）

```cpp
int prim(int n, vector<vector<int>>& g) {
    vector<int> dist(n, INF); vector<bool> used(n, false);
    dist[0] = 0; int ans = 0;
    for (int i = 0; i < n; ++i) {
        int u = -1;
        for (int j = 0; j < n; ++j) if (!used[j] && (u==-1||dist[j]<dist[u])) u = j;
        if (dist[u] == INF) return -1;
        used[u] = true; ans += dist[u];
        for (int v = 0; v < n; ++v) if (!used[v]) dist[v] = min(dist[v], g[u][v]);
    }
    return ans;
}
```

```cpp
int editDist(string a, string b) {
    int n = a.size(), m = b.size();
    vector<vector<int>> dp(n+1, vector<int>(m+1, 0));
    for (int i = 0; i <= n; ++i) dp[i][0] = i;
    for (int j = 0; j <= m; ++j) dp[0][j] = j;
    for (int i = 1; i <= n; ++i)
        for (int j = 1; j <= m; ++j)
            dp[i][j] = a[i-1]==b[j-1] ? dp[i-1][j-1] :
                       1 + min({dp[i-1][j], dp[i][j-1], dp[i-1][j-1]});
    return dp[n][m];
}
```

### 12.1 顺序表（Sequence List）

```cpp
const int MAXSIZE = 1000;
struct SeqList {
    int data[MAXSIZE];
    int length;            // 当前长度
    void init()            { length = 0; }
    bool insert(int i, int x) {            // 在 i 位置插入 x（下标从 1 开始）
        if (i < 1 || i > length + 1 || length >= MAXSIZE) return false;
        for (int j = length; j >= i; --j) data[j] = data[j-1];
        data[i-1] = x; ++length; return true;
    }
    bool erase(int i) {                   // 删除 i 位置
        if (i < 1 || i > length) return false;
        for (int j = i; j < length; ++j) data[j-1] = data[j];
        --length; return true;
    }
    int  find(int x) {                    // 返回首次出现的下标（1 起），失败 0
        for (int i = 0; i < length; ++i) if (data[i] == x) return i+1;
        return 0;
    }
};
```
**复杂度**：插入/删除 O(n)（平均移动 n/2），查找 O(n)，按下标访问 O(1)
**易错点**：插入位置 i 的范围是 `[1, length+1]`，不是 `[0, length]`

### 12.2 单链表（带虚拟头节点）

```cpp
struct LNode {
    int data;
    LNode* next;
    LNode(int x = 0, LNode* n = NULL) : data(x), next(n) {}
};
struct LinkList {
    LNode* head;                          // 虚拟头节点（不存数据）
    void init()            { head = new LNode(); head->next = NULL; }
    // 头插：O(1)
    void pushFront(int x)  { head->next = new LNode(x, head->next); }
    // 尾插：O(n)，可优化为带尾指针的 O(1)
    void pushBack(int x) {
        LNode* p = head;
        while (p->next) p = p->next;
        p->next = new LNode(x);
    }
    // 按下标插入（0 起）：O(n)
    bool insert(int i, int x) {
        LNode* p = head;
        for (int k = 0; k < i && p; ++k) p = p->next;
        if (!p) return false;
        p->next = new LNode(x, p->next); return true;
    }
    // 按下标删除：O(n)，注意要 free
    bool erase(int i) {
        LNode* p = head;
        for (int k = 0; k < i && p->next; ++k) p = p->next;
        if (!p || !p->next) return false;
        LNode* q = p->next; p->next = q->next; delete q; return true;
    }
    // 反转：迭代 O(n)
    void reverse() {
        LNode *prev = NULL, *cur = head->next;
        while (cur) {
            LNode* nxt = cur->next;
            cur->next = prev; prev = cur; cur = nxt;
        }
        head->next = prev;
    }
    void print() {
        for (LNode* p = head->next; p; p = p->next) cout << p->data << " ";
        cout << endl;
    }
};
```
**复杂度**：头插/头删 O(1)，尾插 O(n)，按下标访问/查找/插入/删除 O(n)，反转 O(n)
**易错点**：
1. 反转后**头节点变了**（`head->next` 才是真正的首元）
2. 删节点要 `delete`，否则内存泄漏
3. 判空看 `head->next == NULL`，不是 `head == NULL`

### 12.4 顺序栈（基于数组）

```cpp
const int MAXSTACK = 1000;
struct SeqStack {
    int data[MAXSTACK];
    int top;                              // 栈顶下标（指向栈顶元素）
    void init()             { top = -1; }
    bool empty()            { return top == -1; }
    bool push(int x) {
        if (top >= MAXSTACK - 1) return false;
        data[++top] = x; return true;
    }
    bool pop(int& x) {
        if (empty()) return false;
        x = data[top--]; return true;
    }
    bool peek(int& x) {
        if (empty()) return false;
        x = data[top]; return true;
    }
};
```
**变体**：`top = 0` 指向**栈顶的下一个位置**（`push` 先存再 `top++`），两种风格都行，**别混**

### 12.6 顺序队列（循环队列，解决假溢出）

```cpp
const int MAXQUEUE = 1000;
struct CircQueue {
    int data[MAXQUEUE];
    int front, rear;                      // front 指向队首，rear 指向**队尾的下一个**
    void init()             { front = rear = 0; }
    bool empty()            { return front == rear; }
    bool full()             { return (rear + 1) % MAXQUEUE == front; }
    bool enqueue(int x) {
        if (full()) return false;
        data[rear] = x; rear = (rear + 1) % MAXQUEUE; return true;
    }
    bool dequeue(int& x) {
        if (empty()) return false;
        x = data[front]; front = (front + 1) % MAXQUEUE; return true;
    }
    int size()              { return (rear - front + MAXQUEUE) % MAXQUEUE; }
};
```
**关键点**：空与满都是 `front == rear`，区分靠**浪费一个单元**（满时 `rear+1 == front`）
**易错点**：`(rear + 1) % MAX` 别写成 `rear = (rear+1) % MAX` 之前 `data[rear] = x` 漏了

### 12.9 二叉搜索树（BST）

```cpp
struct BST {
    TreeNode* root;
    void init() { root = NULL; }
    // 插入：O(h)
    TreeNode* insert(TreeNode* r, int x) {
        if (!r) return new TreeNode(x);
        if (x < r->val) r->left  = insert(r->left, x);
        else if (x > r->val) r->right = insert(r->right, x);
        return r;                          // 重复值不插入
    }
    // 查找：O(h)
    TreeNode* find(TreeNode* r, int x) {
        if (!r || r->val == x) return r;
        return x < r->val ? find(r->left, x) : find(r->right, x);
    }
    // 找最小/最大：O(h)
    TreeNode* minNode(TreeNode* r) { while (r->left) r = r->left; return r; }
    TreeNode* maxNode(TreeNode* r) { while (r->right) r = r->right; return r; }
    // 删除：O(h)，分三种情况
    TreeNode* erase(TreeNode* r, int x) {
        if (!r) return NULL;
        if (x < r->val) r->left  = erase(r->left, x);
        else if (x > r->val) r->right = erase(r->right, x);
        else {
            if (!r->left)  { TreeNode* t = r->right; delete r; return t; }
            if (!r->right) { TreeNode* t = r->left;  delete r; return t; }
            // 两个孩子：用**后继**（右子树最小）替代
            TreeNode* t = minNode(r->right);
            r->val = t->val;
            r->right = erase(r->right, t->val);
        }
        return r;
    }
    // 中序遍历验证 BST
    void inOrder(TreeNode* r) {
        if (!r) return;
        inOrder(r->left);
        cout << r->val << " ";
        inOrder(r->right);
    }
};
```
**复杂度**：平均 O(log n)，最坏 O(n)（退化成链表）
**易错点**：
1. 删除节点有**两个孩子**时，必须用**后继/前驱**替代，不能直接 `delete`
2. 中序遍历结果应**升序**

### 12.10 AVL 树（自平衡 BST）

```cpp
struct AVLNode {
    int val, height;
    AVLNode *left, *right;
    AVLNode(int v) : val(v), height(1), left(NULL), right(NULL) {}
};
int height(AVLNode* n) { return n ? n->height : 0; }
int balance(AVLNode* n) { return n ? height(n->left) - height(n->right) : 0; }
void updateHeight(AVLNode* n) { n->height = 1 + max(height(n->left), height(n->right)); }

// 右旋（LL 型）
AVLNode* rotateRight(AVLNode* y) {
    AVLNode* x = y->left;
    AVLNode* T2 = x->right;
    x->right = y; y->left = T2;
    updateHeight(y); updateHeight(x);
    return x;                             // 新的根
}
// 左旋（RR 型）
AVLNode* rotateLeft(AVLNode* x) {
    AVLNode* y = x->right;
    AVLNode* T2 = y->left;
    y->left = x; x->right = T2;
    updateHeight(x); updateHeight(y);
    return y;
}
// 插入：O(log n)
AVLNode* avlInsert(AVLNode* n, int x) {
    if (!n) return new AVLNode(x);
    if (x < n->val) n->left  = avlInsert(n->left, x);
    else if (x > n->val) n->right = avlInsert(n->right, x);
    else return n;
    updateHeight(n);
    int bf = balance(n);
    // LL
    if (bf > 1 && x < n->left->val)  return rotateRight(n);
    // RR
    if (bf < -1 && x > n->right->val) return rotateLeft(n);
    // LR：左孩子左旋，再整体右旋
    if (bf > 1 && x > n->left->val) {
        n->left = rotateLeft(n->left);
        return rotateRight(n);
    }
    // RL：右孩子右旋，再整体左旋
    if (bf < -1 && x < n->right->val) {
        n->right = rotateRight(n->right);
        return rotateLeft(n);
    }
    return n;
}
```
**四种失衡的口诀**：「**新节点在左孩子的左子树 → LL → 右旋**」，**左孩子的右子树 → LR → 先左后右**
**复杂度**：插入/查找/删除 O(log n)，保证树高 ≤ 1.44 log n

### 12.11 堆（最大堆，数组实现）

```cpp
const int MAXHEAP = 1000;
struct MaxHeap {
    int data[MAXHEAP + 1];                // 1 起，方便计算左右孩子
    int sz;
    void init() { sz = 0; }
    // 上浮：插入后用，O(log n)
    void siftUp(int i) {
        while (i > 1 && data[i] > data[i/2]) {
            swap(data[i], data[i/2]); i /= 2;
        }
    }
    // 下沉：删除 / 建堆用，O(log n)
    void siftDown(int i) {
        while (2*i <= sz) {
            int j = 2*i;
            if (j+1 <= sz && data[j+1] > data[j]) ++j;  // 找更大的孩子
            if (data[i] >= data[j]) break;
            swap(data[i], data[j]); i = j;
        }
    }
    void push(int x) {
        if (sz >= MAXHEAP) return;
        data[++sz] = x; siftUp(sz);
    }
    int pop() {                            // 返回最大值
        int top = data[1];
        data[1] = data[sz--]; siftDown(1);
        return top;
    }
    // Floyd 建堆：O(n)
    void build(int a[], int n) {
        sz = n;
        for (int i = 0; i < n; ++i) data[i+1] = a[i];
        for (int i = sz/2; i >= 1; --i) siftDown(i);
    }
};
```
**复杂度**：插入/删除 O(log n)，建堆 O(n)，取顶 O(1)
**易错点**：1 起 vs 0 起，左右孩子下标公式 `2i`/`2i+1` 不同

### 12.12 并查集（DSU / Union-Find）

```cpp
struct DSU {
    int fa[MAXN], rnk[MAXN];              // rnk 近似树高
    void init(int n) { for (int i = 1; i <= n; ++i) { fa[i] = i; rnk[i] = 0; } }
    int find(int x) {                     // 路径压缩
        return fa[x] == x ? x : fa[x] = find(fa[x]);
    }
    bool unite(int x, int y) {            // 按秩合并
        x = find(x); y = find(y);
        if (x == y) return false;
        if (rnk[x] < rnk[y]) swap(x, y);
        fa[y] = x;
        if (rnk[x] == rnk[y]) ++rnk[x];
        return true;
    }
    bool connected(int x, int y) { return find(x) == find(y); }
};
```
**复杂度**：单次操作均摊 O(α(n))，α 是反阿克曼函数（≈ 1）
**易错点**：
1. `unite` 里**先 find 再比较**（直接比较原值是错的）
2. 路径压缩递归版会爆栈？迭代版：
   ```cpp
   int find(int x) {
       int r = x;
       while (fa[r] != r) r = fa[r];
       while (fa[x] != r) { int t = fa[x]; fa[x] = r; x = t; }
       return r;
   }
   ```

### 12.13 Trie 树（字典树 / 前缀树）

```cpp
const int ALPHA = 26;
struct TrieNode {
    TrieNode* ch[ALPHA];
    bool isEnd;                           // 是否为某单词的结尾
    TrieNode() { memset(ch, 0, sizeof(ch)); isEnd = false; }
};
struct Trie {
    TrieNode* root;
    void init() { root = new TrieNode(); }
    void insert(const string& s) {
        TrieNode* p = root;
        for (char c : s) {
            int idx = c - 'a';
            if (!p->ch[idx]) p->ch[idx] = new TrieNode();
            p = p->ch[idx];
        }
        p->isEnd = true;
    }
    bool search(const string& s) {        // 完整单词
        TrieNode* p = root;
        for (char c : s) {
            int i = c - 'a';
            if (!p->ch[i]) return false;
            p = p->ch[i];
        }
        return p->isEnd;
    }
    bool startsWith(const string& s) {    // 前缀
        TrieNode* p = root;
        for (char c : s) {
            int i = c - 'a';
            if (!p->ch[i]) return false;
            p = p->ch[i];
        }
        return true;
    }
};
```
**复杂度**：插入/查找 O(L)，L = 字符串长度
**应用**：单词查找、自动补全、IP 路由（前缀匹配）

### 12.14 Huffman 树（最优二叉树 / 编码）

```cpp
#include <queue>
struct HNode {
    int w;                                // 权值
    HNode *l, *r;
    HNode(int _w) : w(_w), l(NULL), r(NULL) {}
};
struct cmp { bool operator()(HNode* a, HNode* b) const { return a->w > b->w; } };
HNode* buildHuffman(vector<int>& weights) {
    priority_queue<HNode*, vector<HNode*>, cmp> pq;
    for (int w : weights) pq.push(new HNode(w));
    while (pq.size() > 1) {
        HNode* a = pq.top(); pq.pop();
        HNode* b = pq.top(); pq.pop();
        HNode* p = new HNode(a->w + b->w);
        p->l = a; p->r = b;
        pq.push(p);
    }
    return pq.top();                      // 根
}
// 输出编码（左 0 右 1）
void encode(HNode* r, string code, unordered_map<int, string>& mp) {
    if (!r) return;
    if (!r->l && !r->r) { mp[r->w] = code; return; }
    encode(r->l, code + "0", mp);
    encode(r->r, code + "1", mp);
}
```
**性质**：
- 权值越大的叶子，编码越短
- **任何字符编码都不是其他字符编码的前缀**（前缀码，无歧义）
- 总 WPL（带权路径长度）最小
**应用**：数据压缩（zip、png）、哈夫曼编码

### 12.15 邻接表（链式前向星 · 边的存储）

```cpp
const int MAXN = 1000, MAXM = 10000;
struct Edge { int to, w, next; } e[MAXM];

void addEdge(int u, int v, int w) {

}
```
