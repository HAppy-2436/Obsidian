## 2. 核心知识点清单

### 2.1 总论与史观

- **核心命题（定义）**：AI 生于数学，成于计算机 — 人类数学发展的底层推动力是对智能的追求。
- **引述（人物-引文）**：克莱因《古今数学思想》序言："课本中字斟句酌的叙述，未能表现出创造过程中的斗争、挫折…叙述数学家如何跌跤，如何在迷雾中摸索前进，会使搞研究工作的我们鼓起勇气。"
- **观点（事件）**：以古知今 — "一门科学的历史是那门科学中最宝贵的一部分"，"科学能给我们知识，历史能给我们智慧"。
- **数学推动人类历史（事件-列表）**：达芬奇绘画（15–16 世纪）、哥白尼日心说（16 世纪）、牛顿万有引力（17 世纪）、马尔萨斯人口论（18 世纪）、巴赫 12 平均律（18 世纪）、孟德尔遗传学（19 世纪）、达尔文进化论（19 世纪）、晶体结构（19 世纪）、无线电波（19 世纪）、巴贝奇机械计算机（19 世纪）、爱因斯坦相对论（20 世纪初）、DNA 双螺旋（20 世纪）、冯·诺依曼二进制与现代计算机（20 世纪）、图灵机（20 世纪）。
- **名言（人物-引文）**：林肯借"凡直角都相等"推出"为什么人不能相等呢" — 用数学公理论证政治平等。

### 2.2 数学史分期（定义）

- **数学形成时期**：约 30 万年（B.C.6 之前）。
- **初等数学时期**：约 2000 年（B.C.6 — 17 世纪）。
- **近代数学时期**：17–18 世纪。
- **现代数学时期**：1820 年至今。
- **总特征（定义）**：数学史呈"指数级加速" — 跨越仅 200 年，数学迎来指数级加速演进；从 30 万年慢跑 → 2000 年长跑 → 200 年狂飙。
- **初等数学细分**：
  - 古希腊数学（B.C.6 — 6 世纪）— 今天中小学教育内容的主体来源；
  - 中世纪东方数学（3 — 15 世纪）；
  - 欧洲文艺复兴时期（15 — 16 世纪）— 解析几何等开始进入中学。

### 2.3 数学形成时期 · 河谷文明

- **河谷文明四大发源地（事件）**：埃及、美索不达米亚、中国、印度。
- **数概念的形成（定义-年代）**：30 万年以前开始建立自然数概念。
- **计数演进路径（定义）**：手指 → 石子 → 结绳 → 刻痕 → 抽象数字（物质替代 → 信息存储 → 符号诞生 → 剥离物质属性的纯"数"）。
- **进位制（定义）**：
  - 10 进制（埃及 / 中国 / 希腊 / 印度）— 源于双手；
  - 20 进制（玛雅 / 法国）— 手脚并用；
  - 60 进制（古巴比伦）— 高级进位制，一手 12 指节 × 另一手 5 指 = 60。
- **印度数学的世界贡献（事件）**：0 的引入（印度人对世界文明的贡献）。
- **算术化方法（公式-例子）**：印度补数速算法 — 6×7 = 10−4 × 10−3 → 10(10−4−3)+4×3 → 30+12 = 42；以及 111111×999999 = 111111×(10⁶−1) = 111110888889。
- **法国数字的反人类特性（例子）**：80 = quatre-vingts（4×20），97 = quatre-vingt-dix-sept（4×20+10+7）— 听算 / 说算需做分解与重组，口算困难。
- **几何起源（定义）**：
  - 埃及几何 — "测地学"（徐光启译"几何"为 geo 音译）；
  - 中国几何 — 农耕、天文驱动，特征：算法化、代数化（如《九章算术》）；
  - 印度几何 — 宗教驱动，引入了伟大的"0"与数字符号；
  - 希腊几何 — 哲学思辨驱动，走向纯粹逻辑与抽象证明。

### 2.4 初等数学时期 · 古希腊数学（黄金前期）

- **常量数学（定义）**：最基本、最简单的成果，构成现代中学数学核心基础；主要成果：欧氏几何及延续。
- **初等代数学（定义）**：
  - "用字母表示数"是代数的基础；
  - 研究对象演进：自然数 → 有理数 → 无理数 → 虚数；
  - 初等代数以引进符号和未知数为特征，基本内容是方程求解。
- **三大几何问题（事件）**：尺规作图 — 化圆为方、倍立方体、三等分角。
- **早期代表人物与贡献**：
  - **泰勒斯（人物）**：测量金字塔塔高，引入相似形、全等形，被誉为"论证数学的开端"。
  - **毕达哥拉斯（人物）**：《几何原本》前两卷内容、正多面体作图、黄金分割；提出"万物皆(有理)数"并发现无理数。
  - **柏拉图学派 / 亚里斯多德学派（人物）**：亚里斯多德指出点、线、面、体等原始概念；定义公理、公设；创立独立逻辑学，提出矛盾律和排中律，成为间接证明的核心；雅典时期推动论证数学发展。
  - **形式逻辑的方法论意义（定义）**：为欧几里得演绎几何体系奠定方法论基础。

### 2.5 初等数学时期 · 古希腊数学（黄金时代）

- **欧几里得（人物）**：
  - 《几何原本》：历史上第一个数学公理体系，共 5 条公理、5 条公设、119 个定义、465 个命题；
  - 公理体系三大要求：独立性（每一公理不能由其它公理推出）、相容性（公理系统内不存在矛盾）、完备性（每一命题在公理系统内必可判定）。
- **阿基米德（人物，B.C.287 — B.C.212）**：古希腊哲学家、数学家、力学家、天文学家，与牛顿、高斯并称世界三大数学家；公元前两世纪用"穷竭法"，计算内接正 96 边形和外接正 96 边形，得 π 在 223/71 与 22/7 之间，取平均值 ≈ 3.141851。罗马士兵杀死了阿基米德，标志光辉时代结束；"务实的罗马人取代了爱好理论的希腊人…罗马人讲求实效，他们没有改进祖先的知识。"
- **阿波罗尼奥斯（人物）**：创立完美的圆锥曲线理论，从对顶锥得到所有圆锥曲线（圆、椭圆、双曲线、抛物线），是希腊演绎几何的最高成就。
- **刘徽对照（人物-例子）**：中国刘徽的割圆术，直至 3072 边形，得 π = 3927/1250 = 3.1416。

### 2.6 演绎推理的力量 · 埃拉托色尼测地球

- **埃拉托色尼测地球周长（事件）**：
  - 时间：2200 年前；方法：夏至日正午亚历山大城天顶与太阳夹角 7.2°；
  - 关键假设（定义）：地球是完美球体 + 太阳光视为绝对平行射线 + 同经线不同纬度正午时刻影子最短同时到达；
  - 计算结果：地球周长 ≈ 5000 × 50 × 157.5 = 3.969 万公里 ≈ 现代赤道周长 4.0076 万公里，精确度达 99% 以上；
  - 引述：清华大学吴国盛 — "真是了不起，这是希腊理性科学的伟大胜利。"
- **跨文明对比 ·《周髀算经》（事件）**：
  - 八尺高表杆 → 夏至正午周城日影 16 寸，向南一千里影长 15 寸，向北一千里影长 17 寸；
  - 极限推断：向南 16000 里日影归零；
  - 中式思维路径：经验归纳 → 代数规则极限推断；与希腊的演绎几何形成对比；毛泽东《七律二首·送瘟神》"坐地日行八万里，巡天遥看一千河"。
- **小地球的悲剧与哥伦布（事件）**：
  - 埃拉托色尼的精确地球因"太大都是海洋"被时代搁置；
  - 托勒密"小地球"模型（约 3 万公里）盛行 1000 多年；
  - 1492 年哥伦布带 3 艘帆船、87 名水手与托勒密《地理学》出海，正因深信地球极小才敢横渡大洋，最终发现美洲；
  - 观点：正确的数学结果并不总是立刻被接受；极其巨大的错误可能导致极其伟大的发现。

### 2.7 公理 vs 公设（定义）

- **公理（Axioms）**：适合于一切科学；任何学科中不需证明的基本原理；就"数"而言。
- **公设（Postulates）**：几何所特有的；几何学里不需要证明的基本原理；就"图形"而言。
- **欧氏几何五条公设**：
  1. 任意两个点可以通过一条直线连接；
  2. 任意线段能无限延伸成一条直线；
  3. 给定任意线段，可以以其一个端点为圆心、该线段为半径作一个圆；
  4. 彼此能够重合的物体是全等的；
  5. 若两条直线都与第三条直线相交，并且在同一边的内角之和小于两个直角，则这两条直线在这一边必定相交。
- **欧氏几何五条公理**：
  1. 等于同量的量彼此相等；
  2. 等量加等量，其和仍相等；
  3. 等量减等量，其差仍相等；
  4. 所有直角都全等；
  5. 整体大于部分（无穷集合情形下该公理待讨论，如自然数与整数的势相等）。
- **与软件工程的类比（定义）**：公理独立性 ≈ 模块化解耦；相容性 ≈ 无 Bug 运行；完备性 ≈ 算法确定性输出。

### 2.8 古希腊数学后期 · 丢番图与费马

- **丢番图（人物）**：《算术》中提出将一个已知平方数分为两个平方数，即 $x^2 + y^2 = z^2$（勾股数的丢番图形式）。
- **费马猜想大定理（定义）**：$x^n + y^n = z^n$（$n \ge 2$）没有正整数解。
- **怀尔斯（人物-事件）**：1994 年英国数学家怀尔斯证明费马猜想大定理。

### 2.9 古希腊数学的总体贡献

- **认识论贡献（定义）**：认识到数学在人类文明中的作用 — 毕达哥拉斯"万物皆数"、"自然数是万物之母"；"如果你想认识周围的世界，就必须找出事物中的数。一旦数的结构被抓住，你就能控制整个世界。"
- **方法论贡献（定义）**：奠定演绎推理的基础 — 放弃归纳、追求逻辑必然。
- **对西方文明的辐射（事件-列表）**：哲学、逻辑学、宗教教义、政治学、经济学、绘画、音乐、建筑、文学风格；大部分数学家同时是哲学家、物理学家、天文学家、音乐学家等。
- **古希腊"四艺"（定义）**：数的绝对理论—算术；静止的量—几何；运动的量—天文；数的应用—音乐 — 弦长之比为小整数比（1:2 / 2:3 / 3:4）时和谐悦耳。
- **中国"四艺"对照（事件）**：琴棋书画。

### 2.10 希腊式跃迁

- **定义**：从算"术"到数"学" — 从实用到抽象；从感性世界进入理性世界；将数和形抽象化，脱离具体事物而获得一般性。
- **观点（公式-引论）**：科学结论常需实验和归纳并可能被全盘否定（燃素 v.s. 勾股定理）；数学结论数千年都成立。演绎推理是数学惊人力量的源泉，是数学与其他科学的分水岭。

### 2.11 古希腊文明的革命性突破（小结）

- **驱动力**：数学与对人类的终极关怀连接；
- **特点**：从算"术"到数"学"，使数学从感性世界进入理性世界，将数与形抽象化；
- **成果**：逐步形成理论与证明之间逻辑关系的纯粹数学。
- **对比其他文明**：大多数文明驱动力为生产需要（农耕、天文、建筑），创造了实用的数学知识与极佳的计算技巧；局限是始终停留在应用数学领域和经验法则，未发展出纯粹、绝对的数学 — 路径是"发现现象 → 做成工具 → 满足实用"；古希腊是"发现现象 → 寻找规律 → 建立体系" — "多想了好几步"。
- **总结句**：大家都摸到了同一个真理的大门，只有古希腊人推门进去，把它变成了科学的起点。

### 2.12 罗马以后 · 黑暗中世纪

- **事件**：千余年的停滞 — 黑暗中世纪（公元 476 年 — 1453 年），直到迎来文艺复兴。

## 4. 章节内部逻辑

本章以"**AI 生于数学**"为顶层命题，论证分四步推进：
①**史观铺垫**：用克莱因序言和"以古知今"论断数学史的方法论价值，确立"史是智慧之源"的态度；
②**宏观分期**：以 30 万年 / 2000 年 / 200 年的"指数级加速"勾勒数学史的四阶段（萌芽—初等—近代—现代），本节聚焦前两段；
③**微观光谱**：从河谷文明（埃及/巴比伦/中国/印度）的实用算术与进制切入，对照中国《周髀算经》与希腊演绎法，凸显"经验归纳 vs 演绎推理"两条路径分野；
④**聚焦古希腊**：以泰勒斯（论证开端）→ 毕达哥拉斯（万物皆数）→ 欧几里得（公理体系）/阿基米德/阿波罗尼奥斯（黄金时代三大宗师）→ 丢番图/费马/怀尔斯（后期延伸）为主线，借埃拉托色尼测地球这一戏剧性案例展示演绎推理的力量，并补以"小地球—哥伦布—美洲"的故事证明数学史与社会史的纠缠。最终落到"古希腊把门推开了，其他文明只摸到了门"这一价值判断，为下一节"近代数学与现代计算机发明"埋下伏笔。
