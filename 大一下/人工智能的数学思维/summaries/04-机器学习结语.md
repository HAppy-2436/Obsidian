##2.核心知识点清单

### A.机器学习三要素（贯穿全章，结语再次点题）

- **模型**（定义）：线性模型 `f(x;θ) = wᵀx + b` 与广义线性模型 `f(x;θ) = wᵀφ(x) + b`，其中 φ(x) 为非线性基函数，是神经网络的等价表达。（公式）
- **学习准则**（定义）：损失函数 `L(y, f(x))`，常见5 种——0-1损失、绝对损失、平方损失（会放大离群点）、对数损失（似然函数）、指数损失（AdaBoost 用）。（公式）
- **优化**（定义）：把"降低训练误差"转为关于参数 θ 的优化问题；典型方法是最小二乘与梯度下降法。（公式）
- **期望风险最小化 →经验风险最小化**：用训练集平均损失 `R_emp(θ) = (1/N) Σ L(yⁱ, f(xⁱ;θ))`近似期望风险 `R(f) = E[L]`。（公式）

### B.监督学习的主要分支

- **分类问题**：`Y = {1,2, …, C}`。（定义）
- **回归问题**：`Y = ℝ`。（定义）
- **图生成文**：`y₁, y₂, …, yₜ`（自然语言文本序列）。（定义）

### C. 支持向量机 SVM总结（p.118–123，结语前回望）

- **硬间隔 SVM vs软间隔 SVM**：硬间隔要求"间隔内无误分类点"，软间隔允许间隔内有误分。（定义）
- **核函数高维映射**：`z = φ(x)` 将低维线性不可分数据映射到高维线性可分空间。（公式）
- **SVM 方法的局限**：训练时间复杂度 `O(n²) ~ O(n³)`，无法直接输出概率，对噪声敏感。2010 年后被深度学习取代。（事实）

### D.神经元与人工神经网络（p.124–130）

- **M-P 模型**：1943 年 McCulloch & Pitts提出的形式化神经元模型，step 函数作为激活函数，只能输出0/1，不连续不可导。（人物 +事件）
- **感知机**（1958, Rosenblatt）：首个具有学习能力的神经网络模型。（人物 +事件）
- **反向传播算法 BP**（1986, Rumelhart/Hinton/Williams 发表在 Nature）：用链式求导高效计算多层网络梯度。**Paul Werbos1970 年博士论文首次提出 BP思想**（"原创者"），1986 三人成为"推广者"。（人物 +事件）
- **梯度消失问题**：Sigmoid饱和区导数趋近0，深层网络 BP难以训练——是 ReLU 等激活函数被提出的直接原因。（原因）
- **CNN 的生物学灵感**：1962 年 David Hubel & Torsten Wiesel 在猫视觉皮层发现"局部感受野 + 分级处理"，获1981 年诺贝尔医学奖；1989 年 Yann LeCun 提出 LeNet-5，首次商用 BP+CNN（美国银行支票手写数字识别）。（人物 +事件）
- **CNN三大特性**：局部连接、权重共享、汇聚。（定义）

### E. 结语核心（p.131–146）

- **AI产业冲击三问**："改变什么 /替代什么 /解放什么"——企业架构系统性重组、AI劳动者承担专业工作、人类集中精力创新。（公式化命题）
- **首先被 AI颠覆的产业**：软件行业。"每一个不想被替代的人，都将加速替代过程。主动拥抱是唯一选择。"（引文）
- **AI时代属于谁**（人物群像）：
 -面向过程 vs面向成果；
 -主动拥抱 AI的人；
 -时刻空杯、时刻拥抱变化的人（"经验是资产，但不该是包袱"）；
 -关注价值、洞察需求、交付成果的人。
 （人物画像）
- **人类是怎么变聪明的：效法自然**：机器学习 = 一路借鉴、效仿——从数学、物理学、脑科学、心理学、经济学……，沿着 NN → DNN → CNN/RNN → LLM → Agent AI → AGI朝着"万能逼近"的目标演进。（事件 /比喻）
- **AI行业的主旋律是一场军备竞赛**（2025–2026）：
 - OpenAI 关停 Sora、转手发布 ChatGPT Images2.0；
 -行业共识："模型更大、数据更多、trick 更复杂，就是进步"——"越大越好，越复杂越强"；
 - **杨立昆最新论文**：15M 参数 +1块 GPU，打平1.24 亿张图片预训练的巨型模型，规划速度快48 倍。（人物 +事件）
- **AI攻克数学难题的时间线（DeepMind → 外行+大模型）**：
 -2022.10 AlphaTensor：改进50 年矩阵乘法记录；
 -2024.01 AlphaGeometry：解出25/42 道 IMO几何题（IMO银牌水平）；
 -2024.07 AlphaGeometry2（IMO 金牌水平28/42）；
 -2025.07 AlphaProof + Gemini Deep Think4.6：解出35/42（IMO2024 金牌）；
 -2026.03 Claude Opus ChatGPT Pro：破解30 年图论难题；
 -2026.0423 岁外行：攻克60 年数学猜想。
 （事件 + 时间点）
- **认知层级**（例子）：
 - 表层（推理思维，逻辑/数学/语言）：~10,000 年；
 - 中层（感知系统，视觉/听觉/触觉）：~5 亿年；
 -底层（运动本能，平衡/协调/抓取）：~10 亿年。
 （比喻 /例子）
- **我们大的认知盲区**：可能对自己真正的能力视而不见，却对最薄的表皮沾沾自喜。"你以为很简单的事其实很难，你以为很难的事其实很简单。一个两岁小孩伸手拿到一块饼干，这个动作背后站着10 亿年的进化智慧。"（引文）
- **东方"心" vs西方"脑"**（文化对照）：人工神经网络模仿人脑，有没有可能模拟"心"？——思、想、意、念、怒、急、忘、忍…。（人物 /思想实验）
- **注意力机制——王阳明 vs渲染管线**（例子）：
 - 王阳明："你未看此花时，此花与汝同归于寂；你来看此花时，则此花颜色一时明白起来。"（引文）
 - 计算机视觉渲染3招：①视锥剔除 Frustum Culling；②细节层次渲染 LOD；③注视点渲染 Foveated Rendering。（例子）
