## 🔢必背公式 / 对照表

###贝叶斯公式
```
P(A|B) = P(B|A) · P(A) / P(B)
 = 后验 =似然 × 先验 /证据
```

###期望风险 →经验风险
```
R(f) = E[L]              ←期望风险（理论）
R_emp(θ) = (1/N) Σ Lᵢ ←经验风险（实际，用训练集平均近似）
```

## 📐问答 /论述题答题框架（4步走）

```
①锚定时间 +人物（一句话点题）
② 给原因（技术瓶颈 OR 工程条件未成熟）
③ 用"表示 /概率 /优化 /搜索 /反馈闭环"四要素之一解释思想变化
④收尾用金句：一切回到四要素
```

**示范**（AI第二次低谷）：
>①1987 年专家系统进入第二次低谷。②原因：知识获取瓶颈 +维护成本高 +鲁棒性差。③思想转向：从"手工规则"（知识工程）转向"自动学习"（统计学习）。④ 从四要素看，**"表示"从规则符号转为概率分布，"优化"从专家调试转为梯度下降** —— 这是 AI第三次浪潮的伏笔。

## 🧠神经网络关键节点速记

- **1943 M-P 模型** —— McCulloch & Pitts形式化神经元
- **1958感知机** —— Rosenblatt首个有学习能力的网络
- **1970 BP原创者** —— Werbos博士论文
- **1986 BP推广** —— Rumelhart / Hinton / Williams（Nature）
- **1962 Hubel-Wiesel** ——猫视觉皮层 →1981诺奖 → **1989 LeNet-5**（LeCun，CNN商用，BP+CNN）

**梯度消失**：Sigmoid饱和区导数 →0 →深层难训 → **ReLU出现**
