## 3.智能简史 · 数学史脉络【源自03.1-第一部分 +03.1-第二部分】

### 3.1史观与总论

- **核心命题**：AI 生于数学，成于计算机——人类数学发展的底层推动力是对智能的追求。来源：`03.1-数学史-第一部分.md`。
- **克莱因《古今数学思想》序言**："叙述数学家如何跌跤，如何在迷雾中摸索前进，会使搞研究工作的我们鼓起勇气。"来源：`03.1-数学史-第一部分.md`、`03.1-数学史-第二部分.md`。
- **"以古知今"**：一门科学的历史是那门科学中最宝贵的部分；科学给我们知识，历史给我们智慧。来源：`03.1-数学史-第一部分.md`。
- **数学推动人类历史清单**：达芬奇（15–16 世纪）、哥白尼日心说（16 世纪）、牛顿万有引力（17 世纪）、马尔萨斯（18 世纪）、巴赫12 平均律（18 世纪）、孟德尔遗传学（19 世纪）、达尔文进化论（19 世纪）、晶体结构（19 世纪）、无线电波（19 世纪）、巴贝奇机械计算机（19 世纪）、爱因斯坦相对论（20 世纪初）、DNA 双螺旋（20 世纪）、冯·诺依曼二进制与现代计算机（20 世纪）、图灵机（20 世纪）。来源：`03.1-数学史-第一部分.md`、`03.1-数学史-第二部分.md`。

### 3.2 数学史分期

- **数学形成时期**：约30 万年（B.C.6之前）。来源：`03.1-数学史-第一部分.md`。
- **初等数学时期**：约2000 年（B.C.6 —17 世纪）；细分 = 古希腊（B.C.6–6 世纪）+ 中世纪东方（3–15 世纪）+欧洲文艺复兴（15–16 世纪）。来源：`03.1-数学史-第一部分.md`。
- **近代数学时期**：17–18 世纪。来源：`03.1-数学史-第一部分.md`。
- **现代数学时期**：1820至今。来源：`03.1-数学史-第一部分.md`。
- **总特征**：数学史呈"指数级加速"——30 万年慢跑 →2000 年长跑 →200 年狂飙。来源：`03.1-数学史-第一部分.md`。

### 3.3 数学形成时期 ·河谷文明

- **河谷文明四大发源地**：埃及、美索不达米亚、中国、印度。来源：`03.1-数学史-第一部分.md`。
- **计数演进路径**：手指 → 石子 → 结绳 →刻痕 →抽象数字。来源：`03.1-数学史-第一部分.md`。
- **进位制**：10进制（埃及/中国/希腊/印度）、20进制（玛雅/法国）、60进制（古巴比伦）。来源：`03.1-数学史-第一部分.md`。
- **印度数学的世界贡献**：0 的引入。来源：`03.1-数学史-第一部分.md`。
- **印度补数速算法**：6×7 =10−4 ×10−3 →42；111111×999999 =111110888889。来源：`03.1-数学史-第一部分.md`。
- **几何起源对照**：埃及（测地学）、中国（算法化/代数化）、印度（宗教驱动+0）、希腊（哲学思辨→抽象证明）。来源：`03.1-数学史-第一部分.md`。

### 3.4 初等数学 · 古希腊

- **泰勒斯**：测量金字塔塔高，引入相似形/全等形，"论证数学的开端"。来源：`03.1-数学史-第一部分.md`。
- **毕达哥拉斯**：正多面体作图、黄金分割；"万物皆（有理）数"；发现无理数。来源：`03.1-数学史-第一部分.md`、`03.1-数学史-第二部分.md`。
- **柏拉图学派 /亚里斯多德学派**：定义原始概念、公理/公设；创立独立逻辑学，提出矛盾律与排中律。来源：`03.1-数学史-第一部分.md`。
- **欧几里得《几何原本》**：历史上第一个数学公理体系——5 公理 +5 公设 +119 定义 +465命题。来源：`03.1-数学史-第一部分.md`、`03.1-数学史-第二部分.md`。
- **公理体系三大要求**：独立性（每一公理不能由其它公理推出）、相容性（系统内不存在矛盾）、完备性（每一命题必可判定）。来源：`03.1-数学史-第一部分.md`。
- **公理 vs 公设**：公理 =适用于一切科学（关于"数"）；公设 =几何所特有（关于"图形"）。来源：`03.1-数学史-第一部分.md`。
- **欧氏五条公设**：两点连直线 / 线段无限延伸 / 以线段为半径作圆 /彼此重合全等 /第五公设（平行线）。来源：`03.1-数学史-第一部分.md`、`03.1-数学史-第二部分.md`。
- **欧氏五条公理**：等量相等 / 等量加等量 / 等量减等量 / 所有直角都全等 /整体大于部分。来源：`03.1-数学史-第一部分.md`。
- **公理体系 ↔ 软件工程类比**：独立性 ≈ 模块化解耦；相容性 ≈ 无 Bug 运行；完备性 ≈ 算法确定性输出。来源：`03.1-数学史-第一部分.md`。
- **阿基米德（B.C.287–212）**：穷竭法 → π 在223/71 与22/7 之间（取平均 ≈3.141851）；与牛顿、高斯并称世界三大数学家。来源：`03.1-数学史-第一部分.md`。
- **阿波罗尼奥斯**：圆锥曲线理论（圆/椭圆/双曲线/抛物线）。来源：`03.1-数学史-第一部分.md`。
- **刘徽割圆术**：3072边形 → π =3927/1250 =3.1416。来源：`03.1-数学史-第一部分.md`。
- **埃拉托色尼测地球周长**：2200 年前；夏至正午亚历山大城7.2°；得周长 ≈39,690 km，与现代赤道40,076 km 仅差386 km（精确度 >99%）。来源：`03.1-数学史-第一部分.md`。
- **《周髀算经》对照**：八尺高表杆→夏至正午周城日影16寸，向南一千里影长15寸，向北17寸；中式经验归纳 vs希腊演绎几何。来源：`03.1-数学史-第一部分.md`。
- **小地球的悲剧与哥伦布**：埃拉托色尼的"精确地球"被搁置；托勒密"小地球"（约3 万公里）盛行1000多年；1492 年哥伦布横渡大洋，最终发现美洲。来源：`03.1-数学史-第一部分.md`。
- **三大几何问题**：尺规作图——化圆为方、倍立方体、三等分角。来源：`03.1-数学史-第一部分.md`。
- **丢番图**：勾股数的丢番图形式 $x^2+y^2=z^2$。来源：`03.1-数学史-第一部分.md`。
- **费马猜想大定理**：$x^n+y^n=z^n$（$n \ge2$）没有正整数解。来源：`03.1-数学史-第一部分.md`。
- **怀尔斯 (1994)**：证明费马猜想大定理。来源：`03.1-数学史-第一部分.md`。
- **古希腊"四艺"**：算术 /几何 / 天文 /音乐（弦长之比为小整数比1:2 /2:3 /3:4 时和谐悦耳）。来源：`03.1-数学史-第一部分.md`。
- **希腊式跃迁**：从算"术"到数"学"——从实用到抽象；从感性世界进入理性世界。来源：`03.1-数学史-第一部分.md`。

### 3.5近代数学 ·变量进入数学

- **近代数学**：16–17 世纪大工业和科学革命要求研究"运动"和"变化"，由此产生变量与函数。来源：`03.1-数学史-第二部分.md`。
- **笛卡尔1637《几何学》**：奠定解析几何基础，"变量进入了数学，运动进入了数学"；催生极限与微积分。来源：`03.1-数学史-第二部分.md`。
- **微积分四类核心问题**：速度与加速度 /曲线的切线 / 最大值与最小值 / 求积问题。来源：`03.1-数学史-第二部分.md`。
- **牛顿（从力学出发）+莱布尼茨（从几何切入）**：17 世纪后半叶几乎同时创立微积分。来源：`03.1-数学史-第二部分.md`。
- **射影几何（17 世纪）**：绘画中的透视（如《最后的晚餐》有/无透视对比）。来源：`03.1-数学史-第二部分.md`。
- **近代变量数学新分支**：级数理论、微分方程论、微分几何、复变函数论。来源：`03.1-数学史-第二部分.md`。
- **概率论（17 世纪产生）**：研究大量"随机"现象的规律，偶然性中的必然性。来源：`03.1-数学史-第二部分.md`。
- **N=2 二体问题（伯努利,1710）**：天体的运行轨迹是某种圆锥曲线。来源：`03.1-数学史-第二部分.md`。
- **"宇宙和谐"链条**：哥白尼日心说 → 开普勒三大定律 →牛顿万有引力 →行星椭圆轨道。来源：`03.1-数学史-第二部分.md`。
- **阿波罗8 号返航**（1968）："我想现在是牛顿在驾驶。"来源：`03.1-数学史-第二部分.md`。

### 3.6 现代数学 ·抽象代数与非欧几何

- **现代代数质变（19 世纪）**：考察比数具有更普遍性质的"量"——群概念诞生 →抽象代数。来源：`03.1-数学史-第二部分.md`。
- **伽罗瓦**：抽象代数"研究超越具体数值的'结构'与'规则'"。来源：`03.1-数学史-第二部分.md`。
- **第五公设等价命题**：过直线外一点有且仅有一条直线与已知直线平行。来源：`03.1-数学史-第二部分.md`。
- **罗巴切夫斯基（19 世纪20 年代，俄国喀山大学）**：以矛盾命题代替第五公设，建立罗氏几何。来源：`03.1-数学史-第二部分.md`。
- **罗氏几何**：过直线外一点至少有两条直线与已知直线平行；三角形内角和 < 两直角。来源：`03.1-数学史-第二部分.md`。
- **黎曼（B.1826 年）**：创立黎曼几何；过直线外一点不存在直线与已知直线平行；三角形内角和 > 两直角。来源：`03.1-数学史-第二部分.md`。
- **三种几何对照**：欧氏（平坦 /日常工程）、罗氏（双曲马鞍面 /原子核量子微观）、黎曼（球面 /广义相对论时空弯曲）。来源：`03.1-数学史-第二部分.md`。
- **"改变一条假设，重塑一个宇宙"**：三种几何的核心差异口号。来源：`03.1-数学史-第二部分.md`。
- **大圆航线（北京→纽约经北极）**：黎曼几何球面世界里两点间的最短距离是大圆的弧线。来源：`03.1-数学史-第二部分.md`。

### 3.7 三次数学危机

- **第一次危机（B.C.5 世纪）**：希伯索斯发现 √2 不是有理数；摧毁"万物皆整数之比"信仰；欧多克索斯（B.C.370）通过给"比例"下新定义解决。来源：`03.1-数学史-第二部分.md`。
- **第二次危机（17–19 世纪）**：微积分的无穷小悖论；贝克莱（1734）抨击 ∆x 为"逝去量的鬼魂"；芝诺"阿喀琉斯追乌龟"悖论；19 世纪20 年代极限理论取代无穷小量。来源：`03.1-数学史-第二部分.md`。
- **第三次危机（19–20 世纪）**：康托尔集合论悖论动摇整个数学的基础；至今未完善解决，仅有 ZF / NBG 公理系统打补丁。来源：`03.1-数学史-第一部分.md`、`03.1-数学史-第二部分.md`。
- **康托尔**：19 世纪下半叶创立集合论——现代数学的基石。来源：`03.1-数学史-第二部分.md`。
- **庞加莱（1900）**："数学的严格性今天可以说是严格实现了。"（危机爆发前夕的乐观）。来源：`03.1-数学史-第二部分.md`。
- **开尔文（同期）**："在已经基本建成的科学大厦中，后辈物理学家只能做一些零碎的修补工作了。"来源：`03.1-数学史-第二部分.md`。
- **罗素悖论（1902）**：$A = \{ X \mid X \notin X \}$。来源：`03.1-数学史-第二部分.md`。
- **说谎者悖论**："我正在说的这句话是假话"——若真则假，若假则真。来源：`03.1-数学史-第二部分.md`。
- **理发师悖论**（罗素悖论通俗版）："我只给不给自己刮脸的人刮脸"——理发师是否应给自己刮脸？来源：`03.1-数学史-第二部分.md`。
- **自指三艺术化呈现**：巴赫《蟹行卡农》（听觉闭环）、艾舍尔版画（视觉闭环）、哥德尔（逻辑闭环）。来源：`03.1-数学史-第二部分.md`。
- **"自指是通向智能的关键"**：本课程数学→智能衔接的核心金句。来源：`03.1-数学史-第二部分.md`。
- **ZF / NBG 公理系统**：对"所有集合的集合"打补丁的两条思路。来源：`03.1-数学史-第二部分.md`。
- **"哥德尔不完备性定理即将到来"**：第三次危机的预告——为3.2章节铺垫。来源：`03.1-数学史-第二部分.md`。

##4. 现代数学与计算机的发明【源自03.2-现代数学-计算机】

###4.1三大流派

- **第三次数学危机（19 世纪末）**：集合论悖论动摇数学根基。来源：`03.2-现代数学-计算机.md`。
- **逻辑主义（Logicism）**：戴德金、弗雷格、罗素——"数学 =逻辑"。来源：`03.2-现代数学-计算机.md`。
- **直觉主义（Intuitionism）**：布劳威尔、克罗尼克——"数学 = 心智构造"；反对实无穷、不承认排中律、不接受非构造的存在性证明。来源：`03.2-现代数学-计算机.md`。
- **形式主义（Formalism）**：希尔伯特——"数学 = 无矛盾的符号游戏"；希尔伯特纲领提出"用有限方法证明整个数学的一致性"。来源：`03.2-现代数学-计算机.md`。

###4.2希尔伯特与纲领

- **大卫·希尔伯特（1862–1943）**：德国数学家，哥廷根学派领袖，"数学界的无冕之王"；横跨几何/数论/代数/分析/逻辑/数学物理，最后一位数学全才。来源：`03.2-现代数学-计算机.md`。
- **学术里程碑**：1899《几何基础》、Hilbert空间、代数数论与类域论、物理公理化。来源：`03.2-现代数学-计算机.md`。
- **希尔伯特23 问题（1900.8.8巴黎 ICM）**：分四组——1–6 数学基础 /7–12 数论 /13–18 代数几何 /19–23 数学分析；半数至今未解。来源：`03.2-现代数学-计算机.md`。
- **希尔伯特纲领（1928）三大目标**：完备性 / 相容性 / 可判定性（Entscheidungsproblem）。来源：`03.2-现代数学-计算机.md`。
- **希尔伯特第十问题**：是否存在通用算法，判断任意丢番图方程有无整数解？来源：`03.2-现代数学-计算机.md`。
- **马季亚谢维奇（1970）**：希尔伯特第十问题的否定答案——不存在通用算法。来源：`03.2-现代数学-计算机.md`。
- **希尔伯特墓碑箴言**："我们必须知道，我们必将知道。"来源：`03.2-现代数学-计算机.md`。

###4.3哥德尔不完备定理

- **库尔特·哥德尔（1906–1978）**：奥地利裔美国数学家/逻辑学家；爱因斯坦誉其为"自亚里士多德以来最伟大的逻辑学家"；《时代》20 世纪百位思想家中数学家排名第一。来源：`03.2-现代数学-计算机.md`。
- **哥德尔不完备定理（1931）**：发表于《论数学原理及其相关系统的形式不可判定命题》。来源：`03.2-现代数学-计算机.md`。
- **第一定理（否定完备性）**：任何包含自然数的相容形式系统 F 中，存在不可判定命题 S。来源：`03.2-现代数学-计算机.md`。
- **第二定理（否定相容性）**：任何包含自然数系的相容形式系统 F，其相容性不能在 F 中被证明。来源：`03.2-现代数学-计算机.md`。
- **"思维—语言—实在"三层压缩**：思维是客观实在的有损压缩，语言是思维的有损压缩。来源：`03.2-现代数学-计算机.md`。
- **划时代意义**：①摧毁"数学可完全公理化"的信念；②打破古典数学绝对完美；③驱动科学疆界拓宽。来源：`03.2-现代数学-计算机.md`。

###4.4 图灵与图灵机

- **艾伦·图灵（1912–1954）**：英国数学家、计算机科学之父、AI 之父。来源：`03.2-现代数学-计算机.md`。
- **图灵机五要素**：无限长纸带（存储）+读写头 +规则表（程序）+状态寄存器 + 输出打印。来源：`03.2-现代数学-计算机.md`。
- **人类计算模型化**：草稿纸 =纸带；笔 =读写头；计算过程 =规则表；记忆/进位 =状态。来源：`03.2-现代数学-计算机.md`。
- **停机问题（Halt problem）**：判断"任意程序对给定输入能否在有限时间内停止运行"——不可判定。来源：`03.2-现代数学-计算机.md`。
- **邱奇-图灵论题**：Alonzo Church（1936）用 λ演算否定判定问题；图灵同年到普林斯顿跟 Church读博——殊途同归。来源：`03.2-现代数学-计算机.md`。
- **图灵论文反响（1936）**：因哥德尔不完备定理抢风头、Church 先发、图灵初出茅庐、用"机器"解数理逻辑过于新颖，反应冷淡——普林斯顿演讲听众不足10 人。来源：`03.2-现代数学-计算机.md`。
- **可计算理论**：研究什么样的数学函数可有效计算；分类——递归/迭代/穷举/贪心；评价维度——时间复杂度、空间复杂度；引入 P vs NP 问题、自动机理论。来源：`03.2-现代数学-计算机.md`。

###4.5冯·诺依曼与现代计算机

- **冯·诺依曼（1903–1957）**：美籍匈牙利数学家、物理学家、计算机科学家；"现代计算机之父""博弈论之父"。来源：`03.2-现代数学-计算机.md`。
- **天才事迹**：6 岁心算8 位数除法、8 岁掌握微积分、精通7 国语言、能逐字默写整本书。来源：`03.2-现代数学-计算机.md`。
- **Colossus（巨人,1943，英国图灵团队）**：破译 Enigma密码，将破译时间从几周缩短到几小时，助攻诺曼底登陆。来源：`03.2-现代数学-计算机.md`。
- **ENIAC（1946，美国冯·诺依曼团队）**：研制原子弹需超几十亿次运算的电子计算机，为氢弹研制立功。来源：`03.2-现代数学-计算机.md`。
- **图灵通用机**：能完成任何目的——现代计算机反映"抽象—理论—设计"三阶段过程。来源：`03.2-现代数学-计算机.md`。
- **战后格局**：战后英国衰退 → 图灵孤僻 vs协作需求 →思考机器智能；战后美国未受战争破坏 →顶级科学家涌入 → ENIAC推动电子工程大发展 →普林斯顿高研院 → MIT/IBM →仙童/微软/苹果/英特尔 → PC → AI。来源：`03.2-现代数学-计算机.md`。
- **图灵测试（Turing Test,1950）**：行为主义哲学转向——通过客观行为测试判断机器是否具备智能；测试者通过文本与人和机器盲测对话，无法区分则机器通过。来源：`03.2-现代数学-计算机.md`。

###4.6 控制论与图灵-维纳分歧

- **诺伯特·维纳（1894–1964）**：应用数学家、控制论创始人。来源：`03.2-现代数学-计算机.md`。
- **控制论（Cybernetics,1948）**：《控制论——关于在动物和机器中控制和通讯的科学》；人机统一性——人和机器都通过反馈实现目的。来源：`03.2-现代数学-计算机.md`。
- **维纳《人有人的用处》（1950）**：自动机器与人均由"感知装置 + 信息传递装置 +行动装置"构成独立系统。来源：`03.2-现代数学-计算机.md`。
- **图灵 vs维纳哲学分歧**：①图灵 ≈ 功能主义（着眼机器内部结构与运行方式）；维纳 ≈行为主义（着眼机器-环境的输入输出与反馈调节）。②图灵设想一次性成型；维纳设想尝试-测试-调整、反复迭代。来源：`03.2-现代数学-计算机.md`。
- **维纳思想的成功应用**：①神经网络反向传播算法——非构造性证明思想的具体化；②强化学习五步法。来源：`03.2-现代数学-计算机.md`。

##5.人工智能发展史【源自03.3-AI发展史】

###5.1学科起源与奠基人物

###5.3第一次热潮 (1956-1974)：符号主义早期

- **1954 年**：西蒙 &纽厄尔 Logic Theorist 程序独立证明《数学原理》第2 章38 条定理。来源：`03.3-AI发展史.md`。
- **1958 年**：王浩在 IBM704 上用5 分钟证明《数学原理》命题演算220 条定理。来源：`03.3-AI发展史.md`。

- **1956 年**：Selfridge第一个字符识别程序，开创"模式识别"新领域。来源：`03.3-AI发展史.md`。
- **1957 年**：西蒙 &纽厄尔研究通用问题求解器 GPS。来源：`03.3-AI发展史.md`。
- **1956 年**：萨缪尔跳棋程序（机器学习先驱），1962 年击败美国州冠军。来源：`03.3-AI发展史.md`。

- **乐观预言**：西蒙1956"10 年内计算机成为国际象棋世界冠军"；1966"20 年内机器能完成人能做到的一切"。来源：`03.3-AI发展史.md`。

###5.4第一次低谷 (1974-1980)

- **1965 年**：机器定理证明遇瓶颈——连续函数之和的简单证明也要推数十万步。来源：`03.3-AI发展史.md`。
- **机器翻译失败案例**："The spirit is willing but the flesh is weak" 经英→俄→英 → "The wine is good but the meet is spoiled"。来源：`03.3-AI发展史.md`。
- **1974 年起**：政府与大学削减 AI经费，AI 进入第一次寒冬。来源：`03.3-AI发展史.md`。

###5.5第二次热潮 (1980-1987)：专家系统

- **Edward Feigenbaum**："专家系统之父"；传统 AI过于强调通用求解，忽视领域知识。来源：`03.3-AI发展史.md`。
- **范式转向**：从"通用求解（弱领域约束）"转向"知识驱动（强领域知识）"；目标从"能求解"转向"在特定领域可用、可靠"。来源：`03.3-AI发展史.md`。
- **典型专家系统**：DENDRAL（1968 化学分析，世界首个成功专家系统）；XCON（DEC工业配置）；MYCIN（1970s医学诊断）；CASNET/PUFF（呼吸病）。来源：`03.3-AI发展史.md`。
- **知识工程（Knowledge Engineering）**：由专家系统兴起催生的软件产业新分支。来源：`03.3-AI发展史.md`。
- **知识获取瓶颈**：规则库依赖人工访谈专家，扩展成本高、迭代慢。来源：`03.3-AI发展史.md`。

###5.6第二次低谷 (1987-2012)：分化期

- **AI六大方向**（1987 后形成）：计算机视觉、自然语言处理、认知与推理、机器学习、机器人学、博弈与伦理。来源：`03.3-AI发展史.md`。
- **生物智能延伸**：脑机接口、电子眼/耳/鼻/舌、假肢、基因编辑等"换器官、修复神经"路线。来源：`03.3-AI发展史.md`。

###5.7联结主义学派与神经网络

- **20 世纪80 年代**：机器学习由边缘分支成为关注焦点，知识开始自下而上涌现。来源：`03.3-AI发展史.md`。
- **三大学派（80-90 年代）**：符号学派、联结学派、行为学派；后 AI关注点迅速偏向联结学派。来源：`03.3-AI发展史.md`。
- **联结学派（Connectionism）**：起源40 年代，关注物理学与神经科学；用概率矩阵与加权神经元动态识别、分类、归纳。来源：`03.3-AI发展史.md`。
- **2000 年**：大多数科学家对深度学习失去兴趣；Hinton坚持冷门领域，2009 年获意外成功。来源：`03.3-AI发展史.md`。

###5.8深度学习突破 (2012-2018)

- **2009 年 Hinton DNN**：语音识别重大突破，错误率降25%。来源：`03.3-AI发展史.md`。
- **2012 年**：谷歌 &斯坦福多层神经网络 ImageNet 实验（2 万类别、1400 万张图）；人脸识别率81.7% vs传统71.0%。来源：`03.3-AI发展史.md`。
- **2012 年**：Hinton团队成果应用于 Apple Siri，iPhone5 热卖；AlexNet ImageNet夺冠。来源：`03.3-AI发展史.md`。
- **2014 年**：深层 CNN普及，视觉精度提升。来源：`03.3-AI发展史.md`。
- **2015 年**：微软图像识别错误率3.57% <人类5.1%。来源：`03.3-AI发展史.md`。
- **2016 年**：微软语音识别词错率5.9%达专业速记员水平；同年 AlphaGo4:1战胜李世石。来源：`03.3-AI发展史.md`。
- **AlphaGo 系列**：2015.10 AlphaGo-Fan5:0樊麾；2016.3 AlphaGo-Lee；2017.1 AlphaGo-Master3:0柯洁；2017.10 AlphaGo-Zero 从零自学，3 天100:0胜 Lee，40 天超所有前代。来源：`03.3-AI发展史.md`。
- **AlphaZero**：通用棋类 AI（围棋/国际象棋/将棋一套算法通吃），24 小时训练碾压各领域冠军 AI。来源：`03.3-AI发展史.md`。

###5.9 AlphaZero衍生后代

- **AlphaTensor (2022)**：突破 Strassen50 年纪录，发现新4×4矩阵乘法算法；提升70+ 种矩阵计算速度。来源：`03.3-AI发展史.md`。
- **AlphaGeometry (2024)**：奥数级几何定理证明 AI，30 道 IMO几何题做对25 道，达 IMO 金牌水平。来源：`03.3-AI发展史.md`。
- **AlphaDev**：用类 AlphaZero 方法优化排序、哈希等底层算法。来源：`03.3-AI发展史.md`。
- **AlphaProof**：形式化数学证明，配合 AlphaGeometry。来源：`03.3-AI发展史.md`。
- **MuZero**：通用规划模型，无需规则即可自学并超越 AlphaZero。来源：`03.3-AI发展史.md`。

###5.10 AI 长足进步三大原因

- **数据**：海量数据 +频繁人机互动（移动互联网 =2010 年代）。来源：`03.3-AI发展史.md`。
- **算法**：概率统计方法的大发展 + 工程学方法。来源：`03.3-AI发展史.md`。
- **计算能力**：AI芯片——GPU、FPGA、ASIC、TPU、类脑芯片。来源：`03.3-AI发展史.md`。

###5.11 时间轴小结

- **1900**（数学/工程基础准备）→ **1956**达特茅斯（AI 元年）→ **1956-1974**第一次热潮 → **1974-1980**第一次低谷 → **1980-1987**第二次热潮（专家系统）→ **1987-2012**第二次低谷（分化）→ **2012**深度学习崛起 →2018+预训练大模型/LLM。来源：`03.3-AI发展史.md`。

###5.12五大 AI 学派

- **符号主义 (Symbolicism)**：认知即计算，起源2300 年前，关注哲学/逻辑学/心理学；逆向演绎；算法：规则与决策树；代表：专家系统。来源：`03.3-AI发展史.md`。
- **联结主义 (Connectionism)**：认知即网络；用概率矩阵和加权神经元；算法：深度学习、神经网络、反向传播。来源：`03.3-AI发展史.md`。
- **行为主义 (Actionism)**：感知-行动；起源40 年代；遗传学、进化生物学、控制论基础；算法：遗传算法、蚁群算法、强化学习。来源：`03.3-AI发展史.md`。
- **贝叶斯学派**：统计学与概率推理；算法：朴素贝叶斯、马尔科夫过程。来源：`03.3-AI发展史.md`。
- **类推学派**：关注心理学和数学优化；按约束条件优化函数；算法：支持向量机 (SVM)。来源：`03.3-AI发展史.md`。

###5.13符号主义的本质障碍

- **优点**：可解释性、推理过程与人类基本一致、鲁棒性强。来源：`03.3-AI发展史.md`。
- **局限性**：①需要丰富的知识（人类知识与经验难以准确描述）；②就事论事（应用领域狭窄）；③难以处理不确定性；④需对问题抽象出精确数学意义上的解析式模型。来源：`03.3-AI发展史.md`。
- **条件问题 (Qualification Problem)**：不可能枚举一个智能行为的所有先决条件。来源：`03.3-AI发展史.md`。
- **分枝问题 (Ramification Problem)**：不可能枚举一个智能行为的所有隐形结果。来源：`03.3-AI发展史.md`。

###5.14统计学习与贝叶斯理论

- **统计学四个里程碑**：①1995 Vapnik & Chervonenkis（VC维）；②Tikhonov, Ivanov, Philips（正则化原则处理病态问题）；③1999 Parzen, Rosenblatt, Chentsov（非参数统计）；④Kolmogorov, Solmonoff（算法复杂度与归纳推理的关系）。来源：`03.3-AI发展史.md`。
- **统计学习理论 (2000)**：模型必须与问题匹配，过简→欠拟合，过繁→过拟合。来源：`03.3-AI发展史.md`。
- **贝叶斯公式**：$P(A|B) = P(B|A) \times P(A) / P(B)$——后验 =似然 × 先验 /证据。来源：`03.3-AI发展史.md`。
- **贝叶斯定理式学习**：不是从零开始，而是在已有知识的基础上，根据新证据不断更新信念。来源：`03.3-AI发展史.md`。

###5.15深度学习的优缺点

- **优点**：不需要领域知识自动学习；能处理大数据；应对开放环境；处理"知其然不知其所以然"的问题；性能高、可扩展。来源：`03.3-AI发展史.md`。
- **局限性**：黑箱（数学理论支持不足）；泛化能力差（过度依赖训练数据，同分布假设）；缺乏推理与因果表达能力。来源：`03.3-AI发展史.md`。
- **深度学习幻觉**：对图片加噪声后 DNN仍以99.99%置信度识别为"2"——本质是分类而非识别。来源：`03.3-AI发展史.md`。

###5.16概率统计方法的根本困难

- **大数据时代的五大转变**：①部分与全部；②多胜于好；③数量与质量；④关联与因果（correlation > causation）；⑤只知"做什么"，不问"为什么"。来源：`03.3-AI发展史.md`。
- **Judea Pearl**（2011 图灵奖，"贝叶斯网络之父"）：《机器学习的理论障碍》（2017）——纯统计机器学习存在理论限制，反事实推理是突破方向。来源：`03.3-AI发展史.md`。
- **反事实 (Counterfactual)**："如果当初没有吃感冒药，现在感冒会不会好？"——纯统计方法无法证明感冒药效果，因其缺乏反事实层级信息。来源：`03.3-AI发展史.md`。

###5.17模拟大脑与脑计划

- **德国海德尔堡大学硬件模拟大脑**：10 亿神经元 +10¹³突触（人脑1/10）。来源：`03.3-AI发展史.md`。
- **瑞士洛桑理工 + IBM蓝色大脑计划**：逆向工程开发虚拟大脑。来源：`03.3-AI发展史.md`。
- **脑计划**：美国 (2013)、欧盟人类大脑计划 (2013)、日本脑/思维计划 (2013)、中国脑计划 (2021)。来源：`03.3-AI发展史.md`。

###5.18行为学派与昆虫机器人

- **Rodney Brooks（MIT）**：研制的机器昆虫不需要大脑干预，凭四肢和关节协调适应环境。来源：`03.3-AI发展史.md`。
- **Boston Dynamics**：体现行为学派"感知-行动"理念的现代机器人代表。来源：`03.3-AI发展史.md`。

###5.19人类计算 (Human Computation)

- **Luis von Ahn（CMU）**：2007 年开发 reCAPTCHA，开创"人类计算"研究方向。来源：`03.3-AI发展史.md`。
- **reCAPTCHA 工作原理**：选 OCR 不能识别的词→变形→生成"控制词"→让用户识别→3 用户相同答案即采纳；精度99.1% (vs 原83.5%)；每天平均校对160 本书。来源：`03.3-AI发展史.md`。
- **ESP Game**：把图像识别打包为游戏，让用户通过竞争给图片贴标签。来源：`03.3-AI发展史.md`。
- **Verbosity**：双人合作游戏，通过句子模板 (`... is a kind of ...`、`... is used for ...`)收集常识事实；267玩家1 周输入7871事实，精度85%。来源：`03.3-AI发展史.md`。

###5.20关键金句

- **1956达特茅斯会议宣言**："学习的每一方面或智力的任何其他功能，原则上都可以准确地描述，并由机器模拟……"。来源：`03.3-AI发展史.md`。
- **"AI 长足进步 = 数据 + 算法 + 计算能力"**。来源：`03.3-AI发展史.md`。
- **Judea Pearl**："目前的机器学习系统几乎完全以纯粹的统计模式运行，这对它们的表现提出了严格的理论限制。"来源：`03.3-AI发展史.md`。

##6.机器学习的核心议题【源自04-机器学习结语】

###6.1机器学习三要素

- **模型**（定义）：线性模型 `f(x;θ) = wᵀx + b` 与广义线性模型 `f(x;θ) = wᵀφ(x) + b`，其中 φ(x) 为非线性基函数，是神经网络的等价表达。来源：`04-机器学习结语.md`。
- **学习准则**（定义）：损失函数 `L(y, f(x))`，常见5 种——0-1损失、绝对损失、平方损失（会放大离群点）、对数损失（似然函数）、指数损失（AdaBoost 用）。来源：`04-机器学习结语.md`。
- **优化**（定义）：把"降低训练误差"转为关于参数 θ 的优化问题；典型方法是最小二乘与梯度下降法。来源：`04-机器学习结语.md`。
- **期望风险最小化 →经验风险最小化**：用训练集平均损失 `R_emp(θ) = (1/N) Σ L(yⁱ, f(xⁱ;θ))`近似期望风险 `R(f) = E[L]`。来源：`04-机器学习结语.md`。

###6.2监督学习的主要分支

- **分类问题**：`Y = {1,2, …, C}`。来源：`04-机器学习结语.md`。
- **回归问题**：`Y = ℝ`。来源：`04-机器学习结语.md`。
- **图生成文**：`y₁, y₂, …, yₜ`（自然语言文本序列）。来源：`04-机器学习结语.md`。

###6.3 支持向量机 SVM

- **硬间隔 SVM vs软间隔 SVM**：硬间隔要求"间隔内无误分类点"，软间隔允许间隔内有误分。来源：`04-机器学习结语.md`。
- **核函数高维映射**：`z = φ(x)` 将低维线性不可分数据映射到高维线性可分空间。来源：`04-机器学习结语.md`。
- **SVM 方法的局限**：训练时间复杂度 `O(n²) ~ O(n³)`，无法直接输出概率，对噪声敏感；2010 年后被深度学习取代。来源：`04-机器学习结语.md`。

###6.4神经元与人工神经网络

- **M-P 模型**：1943 年 McCulloch & Pitts提出的形式化神经元模型，step 函数作为激活函数，只能输出0/1，不连续不可导。来源：`04-机器学习结语.md`。
- **感知机**（1958, Rosenblatt）：首个具有学习能力的神经网络模型。来源：`04-机器学习结语.md`。
- **反向传播算法 BP**（1986, Rumelhart/Hinton/Williams 发表在 Nature）：用链式求导高效计算多层网络梯度。来源：`04-机器学习结语.md`。
- **Paul Werbos1970 年博士论文**：首次提出 BP思想（"原创者"），1986 三人成为"推广者"。来源：`04-机器学习结语.md`。
- **梯度消失问题**：Sigmoid饱和区导数趋近0，深层网络 BP难以训练——是 ReLU 等激活函数被提出的直接原因。来源：`04-机器学习结语.md`。
- **CNN 的生物学灵感**：1962 年 David Hubel & Torsten Wiesel 在猫视觉皮层发现"局部感受野 + 分级处理"，获1981 年诺贝尔医学奖。来源：`04-机器学习结语.md`。
- **LeNet-5（1989, Yann LeCun）**：首次商用 BP+CNN（美国银行支票手写数字识别）。来源：`04-机器学习结语.md`。
- **CNN三大特性**：局部连接、权重共享、汇聚。来源：`04-机器学习结语.md`。

###6.5 结语核心

- **AI产业冲击三问**："改变什么 /替代什么 /解放什么"。来源：`04-机器学习结语.md`。
- **首先被 AI颠覆的产业**：软件行业——"每一个不想被替代的人，都将加速替代过程。主动拥抱是唯一选择。"来源：`04-机器学习结语.md`。
- **AI时代属于谁**：①面向过程 vs面向成果；②主动拥抱 AI的人；③时刻空杯、时刻拥抱变化的人（"经验是资产，但不该是包袱"）；④关注价值、洞察需求、交付成果的人。来源：`04-机器学习结语.md`。
- **"人类是怎么变聪明的：效法自然"**：机器学习 = 一路借鉴、效仿——NN → DNN → CNN/RNN → LLM → Agent AI → AGI朝着"万能逼近"的目标演进。来源：`04-机器学习结语.md`。
- **AI行业的主旋律是一场军备竞赛**（2025–2026）：OpenAI 关停 Sora、转手发布 ChatGPT Images2.0；行业共识"越大越好"；**杨立昆最新论文**：15M 参数 +1块 GPU，打平1.24 亿张图片预训练的巨型模型，规划速度快48 倍。来源：`04-机器学习结语.md`。
- **AI攻克数学难题时间线（DeepMind）**：2022.10 AlphaTensor →2024.01 AlphaGeometry (IMO银牌) →2024.07 AlphaGeometry2 (IMO 金牌) →2025.07 AlphaProof + Gemini Deep Think4.6 (IMO 金牌35/42) →2026.03 Claude Opus ChatGPT Pro →2026.0423 岁外行攻克60 年猜想。来源：`04-机器学习结语.md`。
- **认知层级（例子）**：表层（推理思维 ~10,000 年）+ 中层（感知系统 ~5 亿年）+底层（运动本能 ~10 亿年）。来源：`04-机器学习结语.md`。
- **我们大的认知盲区**："你以为很简单的事其实很难，你以为很难的事其实很简单。一个两岁小孩伸手拿到一块饼干，这个动作背后站着10 亿年的进化智慧。"来源：`04-机器学习结语.md`。
- **东方"心" vs西方"脑"**：人工神经网络模仿人脑，有没有可能模拟"心"？来源：`04-机器学习结语.md`。
- **注意力机制——王阳明 vs渲染管线**：王阳明"你未看此花时，此花与汝同归于寂；你来看此花时，则此花颜色一时明白起来"；计算机视觉渲染3招：视锥剔除 Frustum Culling /细节层次渲染 LOD /注视点渲染 Foveated Rendering。来源：`04-机器学习结语.md`。
- **未来不押注单一技术**："你生活在1985 年，你可能会押注专家系统；你生活在2013 年，你可能会押注深度学习。站在2026 年，你会押什么？"来源：`04-机器学习结语.md`。
- **结语金句（数学思维统合）**："无论是符号主义、连接主义、行动主义、FCN、CNN、RNN、Transformer、LLM、Agent AI、……从数学思维看，一切都可以回到表示、概率、优化、搜索与反馈闭环……"来源：`04-机器学习结语.md`。
- **期末考题型提示**：闭卷；题型 = 选择 +填空 +问答；问答题会有综合型思考——"三次浪潮为何兴起、为何遇冷，以及每次转向背后的思想变化"。来源：`04-机器学习结语.md`。
