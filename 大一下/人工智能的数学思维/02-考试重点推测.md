## A. 高优先级（必考 / 综合型 / 占分 ≥40%）

### A1. AI 三次浪潮为何兴起、为何遇冷、每次转向背后的思想变化（问答题 /论述题，约15–20分）

- **考察形式猜测**：**问答题 /论述题**——结语已直接预告"问答题会有综合型思考"。
- **具体考点**：
1. **第一次热潮 (1956-1974)兴起**：符号主义范式 +早期乐观预言（西蒙1956/1966）+ Logic Theorist、四色猜想等里程碑；来源：`03.3-AI发展史.md`。
2. **第一次低谷 (1974-1980)遇冷**：机器定理证明遇瓶颈（1965）、机器翻译失败案例（英→俄→英"The spirit is willing..."变"The wine is good..."）、政府与大学削减 AI经费；来源：`03.3-AI发展史.md`。
3. **第二次热潮 (1980-1987)复兴**：Feigenbaum范式转向——从"通用求解（弱领域约束）"转向"知识驱动（强领域知识）"；DENDRAL/MYCIN/XCON 等专家系统；知识工程兴起；来源：`03.3-AI发展史.md`。
4. **第二次低谷 (1987-2012) 分化**：专家系统的"知识获取瓶颈"+维护成本高 +鲁棒性差；六大学科方向分化（视觉/NLP/认知/ML/机器人/博弈）；来源：`03.3-AI发展史.md`。
5. **2012+深度学习崛起**：Hinton DNN语音突破（2009）、AlexNet ImageNet夺冠（2012）、Apple Siri商用 →数据+算法+算力三要素成熟；来源：`03.3-AI发展史.md`、`04-机器学习结语.md`。
6. **"每次转向背后的思想变化"**：①符号→统计（归纳演绎 →概率统计）；②知识工程→学习（手工规则 → 自动学习）；③ 大数据→小样本反思（鹦鹉 vs乌鸦）；④关联→因果（Pearl 反事实）；⑤算力军备→效率反思（杨立昆15M 参数论文）；来源：`02-引言-AI现状.md`、`03.3-AI发展史.md`、`04-机器学习结语.md`。
- **为什么重要**：这是结语金句级问题，覆盖整课程的纵向主线；问答题会直接拿来当综合题。
- **建议复习策略**：
 - **记时间轴**："1900→1956→1965→1974→1980→1987→2009→2012→2016→2017→2022→2024→2026"——每个节点至少记1 个事件 +1 个思想转向。
 - **记每次低谷的根本原因**（"能力边界暴露 +成本过高"）；每次热潮的必备条件（"新范式 +工程条件成熟"）。
 - **记"五大学派"的对应算法**：符号=专家系统；联结=神经网络/反向传播；行为=强化学习；贝叶斯=贝叶斯网络；类推=SVM。
 - 用 **结语金句**："从数学思维看，一切都可以回到表示、概率、优化、搜索与反馈闭环"——任何"思想变化"都能映射到四要素变化之一。

### A2.莫拉维克悖论 + "鹦鹉 vs乌鸦"（问答题 /简答题，约5–10分）

- **考察形式猜测**：**简答题 /问答题**——这两组对照是课件的"金句双子星"，且与 AI现状核心痛点直接对应。
- **具体考点**：
1. **莫拉维克悖论原文**："对于机器而言，人类困难的问题是简单的，人类简单的问题是困难的。"（莫拉维克,1988,《智力后裔》）；来源：`02-引言-AI现状.md`。
2. **莫拉维克三层脑进化结构**：底层运动本能（~10 亿年）+中层感知系统（~5 亿年）+表层推理思维（~10,000 年）；"推理只是人类思维最薄的一层表皮"；来源：`02-引言-AI现状.md`、`04-机器学习结语.md`。
3. **"鹦鹉 vs乌鸦"对照表**：大数据 vs 小样本；核电站级能耗 vs0.1–0.2瓦；概率映射 vs 因果推理；垂直领域有效 vs自主感知-决策-执行；来源：`02-引言-AI现状.md`。
4. **"乌鸦吃坚果"5步推理链**：砸→扔马路→评估风险→观察红绿灯→终极方案——小样本、零容错、生存级因果推理；来源：`02-引言-AI现状.md`。
- **为什么重要**：直接对应 AI现状的核心痛点（"概率映射 vs 因果推理""大数据 vs 小样本""高能耗 vs 低能耗"），是后续"为何需要数学思维"的论证锚点。
- **建议复习策略**：把"鹦鹉 vs乌鸦"四维对照表当成一张图默写；记住莫拉维克三层进化结构（"10亿/5亿/1万年"）作为反直觉论据；把"乌鸦吃坚果5步"作为小样本推理的具体案例背下来。

### A3. 数学史 → AI哲学主线（问答题 /论述题，约10–15分）

- **考察形式猜测**：**问答题 /论述题**——这部分是第3 章整章的"暗线"，跨度大、可综合性强。
- **具体考点**：
1. **"AI 生于数学，成于计算机"** 的核心命题（克莱因序言 +跨文明对比）；来源：`03.1-数学史-第一部分.md`。
2. **希腊式跃迁**：从算"术"到数"学"——从实用到抽象；从感性世界进入理性世界；其他文明路径"发现现象→做成工具→满足实用"，希腊路径"发现现象→寻找规律→建立体系"；来源：`03.1-数学史-第一部分.md`。
3. **公理体系三大要求**：独立性（公理不能由其它公理推出）、相容性（系统内不存在矛盾）、完备性（每一命题必可判定）；与软件工程类比（模块化解耦 /无 Bug 运行 /算法确定性输出）；来源：`03.1-数学史-第一部分.md`。
4. **"自指是通向智能的关键"**：巴赫《蟹行卡农》（听觉闭环）+艾舍尔版画（视觉闭环）+哥德尔（逻辑闭环）；来源：`03.1-数学史-第二部分.md`。
5. **三种几何对照**：欧氏（平坦 /日常）/罗氏（双曲 /量子微观）/黎曼（球面 /广义相对论）；"改变一条假设，重塑一个宇宙"；来源：`03.1-数学史-第二部分.md`。
6. **三次数学危机的方法论结论**：第一次危机→"直觉和经验靠不住，推理证明才可靠"；第二次危机→极限理论取代无穷小；第三次危机→ZF/NBG 打补丁 +哥德尔不完备性；来源：`03.1-数学史-第二部分.md`。
- **为什么重要**：第3 章的哲学与方法论基础，所有"AI思想变化"的源头。
- **建议复习策略**：把"数学史 → AI简史"看成一根连续的"接力棒"链条，每一段都要能说出"为何 A段产出的是为后续 B段准备的"；背诵克莱因序言 /希腊式跃迁 /三种几何对照三个金句。

### A4. 现代数学危机 → 计算机发明 → AI诞生（问答题 /论述题，约10–15分）

- **考察形式猜测**：**问答题 /论述题**——3.2章节本身是"核心枢纽"，课件篇幅占比大且承接3.1 与3.3。
- **具体考点**：
1. **三大流派**（逻辑/直觉/形式）：分别主张"数学=逻辑"、"数学=心智构造"、"数学=无矛盾的符号游戏"；代表人物（戴德金/弗雷格/罗素、布劳威尔/克罗尼克、希尔伯特）；来源：`03.2-现代数学-计算机.md`。
2. **希尔伯特纲领（1928）三大目标**：完备性 / 相容性 / 可判定性（Entscheidungsproblem）；后被哥德尔（不完备定理）+ 图灵/Church（否定判定问题）逐一击破；来源：`03.2-现代数学-计算机.md`。
3. **哥德尔不完备定理（1931）**：①第一定理（否定完备性）：任何包含自然数的相容形式系统 F 中，存在不可判定命题 S；②第二定理（否定相容性）：相容性不能在 F内部证明；来源：`03.2-现代数学-计算机.md`。
4. **图灵机五要素**：无限长纸带（存储）+读写头 +规则表（程序）+状态寄存器 + 输出打印；停机问题不可判定；来源：`03.2-现代数学-计算机.md`。
5. **冯·诺依曼体系**：Colossus（1943,英国图灵）+ ENIAC（1946,美国冯·诺依曼）；共同特点"打孔卡输入 +真空管计算 +体积庞大 + 二战功勋卓著；只为专门目的设计，不能储存程序"；改进方向：图灵通用机；来源：`03.2-现代数学-计算机.md`。
6. **图灵 vs维纳的 AI路径分歧**：①图灵 ≈ 功能主义（着眼机器内部结构）；维纳 ≈行为主义（着眼输入输出与反馈调节）；②图灵设想一次性成型；维纳设想尝试-测试-调整、反复迭代；维纳思想成功应用于神经网络反向传播 +强化学习；来源：`03.2-现代数学-计算机.md`。
7. **图灵测试（1950）**：行为主义哲学转向——通过客观行为测试判断机器是否具备智能；与图灵三步法（智能的表述→设计→测试）相呼应；来源：`03.2-现代数学-计算机.md`。
- **为什么重要**：直接对接第3 章整章的因果链——"为什么 AI必然出现"。
- **建议复习策略**：记住"三大流派 →希尔伯特 →哥德尔 → 图灵 →冯·诺依曼 →图灵测试 →维纳控制论"的顺序链；背诵希尔伯特墓碑箴言"我们必须知道，我们必将知道"；背诵图灵测试的原文表述。

### A5.贝叶斯公式 +统计学四里程碑 + Pearl 反事实批判（简答题 / 计算题 /论述题，约5–10分）

- **考察形式猜测**：**简答题 / 计算题 /论述题**——贝叶斯公式可考计算，统计学四里程碑可考简答，Pearl批判可考论述。
- **具体考点**：
1. **贝叶斯公式**：$P(A|B) = P(B|A) \times P(A) / P(B)$——"后验 =似然 × 先验 /证据"；来源：`03.3-AI发展史.md`。
2. **贝叶斯定理式学习定义**：不是从零开始，而是在已有知识的基础上，根据新证据不断更新信念；来源：`03.3-AI发展史.md`。
3. **统计学四个里程碑**（多由俄罗斯数学家建立）：①1995 Vapnik & Chervonenkis（VC维，函数空间大数定律）；②Tikhonov, Ivanov, Philips（正则化原则处理病态问题）；③1999 Parzen, Rosenblatt, Chentsov（非参数统计）；④Kolmogorov, Solmonoff（算法复杂度与归纳推理的关系）；来源：`03.3-AI发展史.md`。
4. **统计学习理论 (2000)**：模型必须与问题匹配——过简→欠拟合，过繁→过拟合；来源：`03.3-AI发展史.md`。
5. **Judea Pearl批判（2011 图灵奖，"贝叶斯网络之父"）**：2017 年报告《机器学习的理论障碍》——纯统计机器学习存在理论限制；反事实推理是突破方向；来源：`03.3-AI发展史.md`。
6. **反事实（Counterfactual）**："如果当初没有吃感冒药，现在感冒会不会好？"——纯统计方法无法证明感冒药效果，因其缺乏反事实层级信息；来源：`03.3-AI发展史.md`。
7. **大数据时代的五大转变**：①部分与全部（from some to all）；②多胜于好（more trumps better）；③数量与质量；④关联与因果（correlation > causation）；⑤只知"做什么"，不问"为什么"；来源：`03.3-AI发展史.md`。
- **为什么重要**：贝叶斯公式是统计学习 /概率图模型的核心工具；Pearl 反事实是 AI 第三代（因果 AI）的核心方向。
- **建议复习策略**：把贝叶斯公式背熟——能解释每个符号的含义；记住统计学四里程碑的年份 +人物 + 内容对应表；把 Pearl 的"反事实批判"当作"当前 AI 的根本局限"来论述。

## B. 中优先级（必考 /覆盖度广 / 占分约30–40%）

### B1. Tom Mitchell1998机器学习定义 + E/T/P 三元组（填空题 / 选择题 /简答题，约3–5分）

- **考察形式猜测**：**填空题 / 选择题 /简答题**——这是经典 ML定义的"开宗明义"原文，几乎所有 ML教材都有。
- **具体考点**：原文"A computer program is said to learn from experience E with respect to some task T and some performance measure P, if its performance on T, as measured by P, improves with experience E."；E/T/P 三元组对应关系；来源：`02-引言-AI现状.md`。
- **为什么重要**：是机器学习范式的最经典表述，可能原文填空或英文缩写。
- **建议复习策略**：把原文背诵，对 E/T/P三个字母的含义默写。

### B2. 五大学派对照（符号/联结/行为/贝叶斯/类推）（选择题 /简答题，约3–5分）

- **考察形式猜测**：**选择题 /简答题**——五大学派是课件末段的核心框架。
- **具体考点**：
 - **符号主义**：认知即计算，起源2300 年前，逆向演绎，算法=规则与决策树，代表=专家系统；
 - **联结主义**：认知即网络，算法=深度学习/神经网络/反向传播；
 - **行为主义**：感知-行动，算法=遗传算法/蚁群算法/强化学习；
 - **贝叶斯学派**：统计学与概率推理，算法=朴素贝叶斯/马尔科夫过程；
 - **类推学派**：心理学和数学优化，算法=支持向量机 SVM；
 来源：`03.3-AI发展史.md`。
- **为什么重要**：是后续 SVM（类推）/神经网络（联结）/强化学习（行为）/贝叶斯网络（贝叶斯）等专题的导航图。
- **建议复习策略**：记成一张二维表——"学派 ×起源 × 代表算法 × 代表人物"，默写3遍以上。

### B3.1956达特茅斯会议六位奠基人 + 时间（选择题 /填空题，约2–3分）

- **考察形式猜测**：**选择题 /填空题**——AI学科元年必须识记。
- **具体考点**：McCarthy（1971 图灵奖，发起人）/ Minsky（1969 图灵奖，AI与认知学）/ Shannon（信息论创始人）/ Newell（1975 图灵奖）/ Simon（诺贝尔经济学奖 +1975 图灵奖）/ Rochester（IBM 计算机设计者）；1956 年8 月达特茅斯夏季研讨班；"Artificial Intelligence"名称的提出；来源：`03.3-AI发展史.md`。
- **为什么重要**：AI学科元年标志性事件，几乎所有 AI教材都会提及。
- **建议复习策略**：把六位奠基人名字 +各自头衔默写；记住1956.8 这个时间点。

### B4.符号主义 vs联结主义 vs深度学习的优缺点对照（简答题 /论述题，约5–8分）

- **考察形式猜测**：**简答题 /论述题**——每个学派都有详尽的"优缺点"清单，是论述题常见出题点。
- **具体考点**：
1. **符号主义优点**：可解释性、推理过程与人类基本一致、鲁棒性强；**局限性**：①需要丰富知识；②就事论事应用领域狭窄；③难以处理不确定性；④需对问题抽象出精确数学意义上的解析式模型；**两大本质障碍**：条件问题（Qualification Problem）、分枝问题（Ramification Problem）；来源：`03.3-AI发展史.md`。
2. **深度学习优点**：不需要领域知识自动学习、能处理大数据、应对开放环境、处理"知其然不知其所以然"的问题、性能高可扩展；**局限性**：黑箱（数学理论支持不足）、泛化能力差（过度依赖训练数据，同分布假设）、缺乏推理与因果表达能力；**典型失败案例**：对图片加噪声后 DNN仍以99.99%置信度识别为"2"——本质是分类而非识别；来源：`03.3-AI发展史.md`。
- **为什么重要**：理解"为什么 AI 需要走向第三代（因果 AI）"的论证。
- **建议复习策略**：把符号主义 +深度学习的优缺点各自列成对照表背诵；把"条件问题"和"分枝问题"作为"符号主义为何衰落"的两大原因记住。

### B5.欧氏五条公理 + 五条公设 + 公理体系三大要求（填空题 /简答题，约3–5分）

- **考察形式猜测**：**填空题 /简答题**——3.1 数学史核心基础内容。
- **具体考点**：
 - **欧氏五条公设**：①任意两个点可以通过一条直线连接；②任意线段能无限延伸成一条直线；③给定任意线段，可以其一个端点为圆心、该线段为半径作一个圆；④所有直角都全等；⑤若两条直线都与第三条直线相交，并且在同一边的内角之和小于两个直角，则这两条直线在这一边必定相交。
 - **欧氏五条公理**：①等于同量的量彼此相等；②等量加等量，其和仍相等；③等量减等量，其差仍相等；④所有直角都全等；⑤整体大于部分。
 - **公理体系三大要求**：独立性 /相容性 /完备性。
 - **公理 vs 公设**：公理 =适用于一切科学（关于"数"）；公设 =几何所特有（关于"图形"）。
 来源：`03.1-数学史-第一部分.md`、`03.1-数学史-第二部分.md`。
- **为什么重要**：数学史章节的最基础内容 +后续公理化方法的核心。
- **建议复习策略**：把五条公设 +五条公理当成"11条默写清单"；公理体系三性记忆口诀——"独相完"（独立性/相容性/完备性）。
