# 举报渠道汇总（联系方式、推荐指数、操作要点）

> 按"投递顺序"而非"重要性"组织。**先内部、再外部；先正式渠道、再舆论；先小范围、再扩大**。
> ⚠️ 所有材料投递前请先脱敏（详见 `12_隐私保护与证据保全指南.md`）。

### 学校内部投递建议

- **不要只发一处**：至少同时抄送"学院院长 + 校长信箱 + 教务处 + 学生工作部 + 纪委"中的 3—4 处，让信息在不同部门流转。
- **不要附个人身份证号 / 学号**：教育部是建议"实名举报"以提升受理概率，但你可以**只披露到学院、年级、专业层次**，不暴露唯一标识。
- **如担心被倒查，先从校长信箱投起**：校长办公室和学院属于不同部门，比单一从学院起步更稳。

## E. 自媒体平台（影响力放大、风险也最大）

> ⚠️ **风险提示**：
> 1. 平台用户账号如绑手机号可能泄露给平台进而被反向追踪；
> 2. 部分高校对在校生发布批评学校内容有过激反应；
> 3. 建议用**专门的小号、新邮箱、非本人常用手机号**操作自媒体。

| 平台 | 投放策略 | 自媒体适配文案 | 备注 |
|---|---|---|---|
| 微博 | 热搜型 #河南大学临时改校历# + 转评赞互动 | 短平快、附校历对比图 | 国内舆论放大主力 |
| 小红书 | "我在河大被……" 长图文 / 视频 | 适合曲高和寡但长尾 | "大学避雷"类话题流量大 |
| 抖音 | 15—60 秒短视频（校历 + 邮件/PDF + 吐槽） | 视觉冲击 + BGM | 24 小时冲热门的可能 |
| B 站 | 3—8 分钟视频（深度梳理 + 时间线） | 适合中长视频讲解 | 与知乎配合最强 |
| 知乎 | "如何评价 XX" 问题 + 长文回答 | 适合做事实陈列 | 知乎曝光有助于舆论沉淀 |

### 自媒体发布原则

- **证据截图打码**：你自己的微信号、头像、姓名不能出现；其他同学的头像可保留（公开群信息）。
- **不要煽情、不要造谣**：把所有判断建立在"事实 + 法条 + 程序"之上，不情绪化以免被删帖。
- **附上你已走过的"内部救济"流程**：证明你已穷尽校内渠道，避免被质疑"为什么直接上网"。
- **评论区互动要克制**：不要主动攻击具体老师姓名，只谈事、不针对人。

## F. 法律途径（远期保底）

| 渠道 | 联系方式 | 适用情形 |
|---|---|---|
| 12348 法律援助热线 | 12348 | 咨询维权可行性与程序 |
| 12345 政务服务 | 12345 | 通用投诉入口 |
| 河南省学生联合会 / 共青团河南省委 | 通过校团委转达 | 学生权益协调 |
| 郑州市 / 开封市两级法院 立案 | 民事诉讼，主张"退还学费中未提供教学服务部分"或"违约损失" | 极少数情况下的最后手段 |
| 北京大学 / 清华大学 法学院公益诉讼中心 | 通过媒体或法律援助机构对接 | 探索性地诉讼维权 |

### 法律途径的实用性

- 学生基于校历有合理信赖利益，单方面提前一周确实可能构成对教学合同的单方变更。
- 但走法律途径**周期长、精力投入大**，不建议作为首选项。
- 现实路径：**先用舆论/上级主管施压让学校主动让步 → 再考虑诉讼**。
