## 五、相关法条（直接可用）

### 教育部令第 41 号《普通高等学校学生管理规定》

- **第六条（五）**：学生对学校与学生权益相关事务享有**知情权、参与权、表达权和监督权**；
- **第六条（六）**：对学校、教职员工侵犯其合法权益的行为，提出**申诉**或者依法提起**诉讼**；
- **第六十五条**：学生认为学校及其工作人员违反本规定，侵害其合法权益的，可向**省级教育行政部门投诉**；教育主管部门应**责令改正**，发现违法违纪应当**追究有关责任人的责任**。

### 各高校通行调（停）课管理规定

> 凡校历已明确规定的时间不得提前或延迟；特殊情况须经书面报告、教务处审核、主管院长同意后重新发布；**未按上述程序履行的，视为教学事故**。

### 《信访工作条例》（2022 年施行）

> 任何组织和个人有权向各级国家机关反映情况，提出建议、意见或者投诉请求。信访工作机构对收到的信访事项应当登记、转送、交办、督办。
