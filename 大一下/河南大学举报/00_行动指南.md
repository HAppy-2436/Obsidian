# 立刻可用的行动指南（D+1 · 7月1日）

> **打开这份的时候，其它都不用看。**
> 你下面要做的，就是按数字走。每一步就是一个 "**打开文件 → 复制 → 粘贴**" 的动作。

---

## 在你开始之前（一次性准备 · 10 分钟）

- [ ] 一台**干净设备**（二手手机 / 借的电脑 / 没装本人实名 APP 的设备）
- [ ] **新办一个手机号**（备用小号，去营业厅 / 网上 App 申请）
- [ ] **新注册一个邮箱**（Gmail / Outlook，备用邮箱，**不要用本人实名过的**）
- [ ] 干净设备**关 GPS / 关 Wi-Fi 定位**
- [ ] 干净设备**只装**：浏览器 + 一个微信（备用号）

## 7 步行动 · 一对一映射

### ✅ 第 1 步：给校长写信

- **打开文件**：`邮件A_校长信箱_可直接粘贴.txt`
- **Ctrl+A 全选 · Ctrl+C 复制**
- **目标邮箱**：`hdxb@henu.edu.cn`
- **新邮件正文区**：Ctrl+V 粘贴
- **添加 3 个附件**：
  - `01_校历原定时间.jpg`
  - `02_学院小学期安排.pdf`
  - `03_临时通知.jpg`
- **点击发送 · 截屏回执**

### ✅ 第 2 步：给校办公室 / 纪委写信

- **打开文件**：`邮件B_校办公室纪委_可直接粘贴.txt`
- **Ctrl+A · Ctrl+C**
- **目标邮箱**：`xbmsk@henu.edu.cn`
- **邮件标题**：上面文件自带，直接粘贴
- **正文粘贴 · 加 3 个附件 · 发送 · 截屏**

### ✅ 第 3 步：给教育部写信

- **打开文件**：`邮件C_教育部举报_可直接粘贴.txt`
- **Ctrl+A · Ctrl+C**
- **目标邮箱**：`12391@moe.edu.cn`
- **正文粘贴 · 加 3 个附件 · 发送 · 截屏**

### ✅ 第 4 步：去国家信访局网投

- **打开浏览器** → http://www.gjxfj.gov.cn
- **首页 → "网上信访大厅" → "网上投诉请求"**
- **注册账号**：用备用小号（收到的短信验证）
- **登录后**：点击"投诉请求" → 选择类别"教育" → "高等教育"
- **打开文件**：`邮件D_通用在线表单版_可直接粘贴.txt`
- **删除最前面"致XX"那一行**（因为这是面对面"致 XXX"的口吻，表单不需要）
- **Ctrl+A · Ctrl+C · 回到网站表单 · Ctrl+V 粘贴到正文区**
- **上传 3 个附件**
- **提交 · 截屏保存查询码**

### ✅ 第 5 步：去教育部在线投诉

- **打开浏览器** → https://hudong.moe.gov.cn/siju_jbz/gxsyreportform.jsp
- **必填项**：姓名填"H 同学" / 联系邮箱填备用邮箱 / 电话填备用小号
- **类别**：选"教育教学"或"高校治理"
- **打开文件**：`邮件D_通用在线表单版_可直接粘贴.txt`
- **删除最前面"致XX"那一行**
- **Ctrl+A · Ctrl+C · 在网站正文区 Ctrl+V 粘贴**
- **上传 3 个附件**
- **提交 · 截屏回执**

### ✅ 第 6 步：去人民网领导留言板

- **打开浏览器** → http://liuyan.people.com.cn
- **注册账号**（用备用邮箱）
- **进入"留言" → 给"河南省委书记"或"河南省长"**
- **打开文件**：`邮件D_通用在线表单版_可直接粘贴.txt`
- **删除最前面"致XX"那一行**
- **Ctrl+A · Ctrl+C · 在留言框 Ctrl+V 粘贴**
- **上传 3 个附件**
- **提交 · 截屏保留留言编号**

### ✅ 第 7 步：发微博（专号）

- **新手机号注册微博号**（这一步在你干净设备上做）
- **不要上传本人头像**
- **撰写一段主爆料（你可以直接参考 `09_自媒体爆款文案.md` 中的微博部分，但只用前 200 字就够）**
- **必须 @ 的账号**：`@河南大学` `@河南省教育厅` `@河南日报` `@大河报` `@澎湃新闻`
- **加话题**：`#河南大学音乐学院临时改校历#`
- **点击发布**

| 时间 | 预期 |
|---|---|
| 当天 / 24 小时 | 学院或辅导员可能开始找你或年级群了解情况 |
| 1—3 个工作日 | 国家信访局 / 教育部 / 教育厅受理，截图回执，校长信箱给一些"已收到"回复 |

| 10+ 工作日 | 如学校继续推诿，按文件 `11_行动时间表.md` 进入第二阶段 |

## 你哪天感觉不对劲就立刻停

- 老师 / 辅导员**任何人**让你去办公室"聊聊"——必须带 2 人以上的陪同，全程录音。
- 学院 / 学校**任何人**让你"撤回材料"作为交换条件——**拒绝**。
- 看到个人信息**被泄露**给学院 / 其他人——**立刻截图**作为证据，提交给河南省纪委 / 公安。
- 任何时候感到**不安全**——**停手**，找我重新评估。
