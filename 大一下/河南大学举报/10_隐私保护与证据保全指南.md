# 隐私保护与证据保全指南（重要！先看完再投递）

---

## 一、底线原则

> **你今天做的所有"举报"动作，明天可能都会被学校复盘。**
> 所以——
> 1. 所有材料**只用一次性的备用身份**（新手机号 + 新邮箱 + 代称）；
> 2. 所有材料**不出现唯一指向个人的内容**（姓名、学号、班级、宿舍号、家庭背景）；
> 3. 一旦决定行动，**先固化证据，再发声**——证据有时效性，**先截图后投诉**。

### 2.2 截图的处理标准

#### A. 群通知截图

- **必须保留**：群名、发布时间、发送人昵称、消息内容、PDF 缩略图（如果有）。
- **必须打码**：你自己的微信头像 + 昵称 + 微信号二维码 + 个人签名档。
- **可保留**：其他同学的头像与昵称（因为是公开群内信息）。
- **特别注意**：有些群消息"撤回"了还能在个人会话中找到——如果希望引用某条撤回的消息，必须确认其真实存在再使用。

#### B. 学院 PDF

- 提交完整 PDF（不要只截屏部分）。
- 在 PDF 外不要加任何注水印（避免被认定为伪造）。
- 如果文件中**有二维码或网址（百度网盘链接）**，保留它是合法且合理的——群友也有权下载。

#### C. 个人订单截图

- **绝对不要**发到任何第三方。
- 仅在**向校长信箱 / 教育厅信访 / 学校纪委**提出"经济损失补偿诉求"时，**模糊掉订单号、姓名、身份证号、银行信息后**提交。
- 提交个人订单给**媒体时也可以模糊**，但在**自媒体上不要发任何具体金额**——用"数百至数千元"区间表达。

### 2.3 微信聊天记录的特殊处理

⚠️ 微信聊天记录**易丢、易被清理**。建议：

1. **立即**用微信自带的"聊天记录迁移"功能备份到电脑。
2. 同时在电脑上**截图关键对话**（不重要的内容也截一下，避免被指"选择性提交"）。
3. **不要**主动删除任何聊天记录。
4. **不要**改动群里任何消息（包括撤回）。

### 2.4 网盘内容的特殊处理

- 截图时如果群里有"百度网盘链接"，**不要保存到自己网盘**，避免与原文件混淆。
- 备份文件最好**改个名**（如 `M-音乐学院2026年小学期-20260629.pdf`）——这样在你自己云盘里不会和其他资料混。

## 三、隐私保护（投递侧）

### 3.1 必备"三件套"

| 项目 | 推荐 |
|---|---|
| 备用手机号 | 新办一个**完全空白的号码**（不要从你的实名身份衍生的副号） |
| 备用邮箱 | **新注册** Gmail / Outlook / 163 邮箱，**不能绑你常手机** |
| 备用支付 | 必要时预留一个**单独的支付方式**（处理快递等场景） |

> 也可以用 **Google Voice / TextNow（境外）** 或 **阿里小号 / 中国移动和多号** 做过渡。
> 关键：**不要让这个备用联系方式能反过来关联到你本人**。

### 3.2 不同投递渠道的个人信息暴露风险

| 渠道 | 暴露风险 | 建议 |
|---|---|---|
| 校长信箱 | 中（OA 系统能查收件人身份） | 用备用邮箱；身份描述到"学院 + 年级"层级 |
| 教务处窗口 | 高（面对面） | 不建议只用口头反映；带书面的；让对方现场打印摘要签字 |
| 河南省教育厅信访 | 低（按信访工作条例，对反映人信息保密） | 备用小号 |
| 国家信访局 | 低 | 备用小号 |
| 教育部监督举报 | 低 | 备用小号；如接受实名，建议只用学院 + 年级层次 |
| 媒体 | 极高（媒体方可能需核实身份） | 一般用"代称 + 学生干部身份"；不接受面访就用电话 |
| 自媒体 | 极高（账号绑定设备可能被追踪） | **专门小号 + 干净设备 + 关位置权限** |

### 3.3 提交材料中的"代称"使用建议

- **不要用任何能反向追踪的代称**（如"河南大学南门第 7 家奶茶店常客"）；
- 使用**完全虚构的姓名 + 学院 + 年级**（如"H 同学 + 音乐学院 + 2025 级 + 本科"）；
- 学院 / 年级 / 211 人这些是公开事实，**不构成个人识别**；
- 但**不要**写出具体班级（如"音乐表演 1 班"），因为学院内 1 班可能就十几人，能被反推。

### 3.4 媒体沟通中的隐私边界

| 媒体行为 | 是否同意 |
|---|---|
| 媒体要求面访 | 同意前**签订书面保密协议**，并**自行审稿**确认不暴露你的姓名 / 学校详细地址 / 班级 |
| 媒体要求录音 / 录像 | 在不暴露身份的前提下同意；让媒体后期用变声 + 头像模糊处理 |
| 媒体把你的 QQ/微信给线人 | 拒绝。除非你自己向线人告知 |
| 媒体采访同学 | 让同学自己决定；不必替同学做主 |
| 媒体要求提交原始材料 | 提供**打码版**；同时声明只能用于报道，不能外传 |

## 四、可能被"找上门"的场景预案

### 4.1 辅导员找你"聊聊"

**场景**：学院通过辅导员侧面了解"是谁在反映"。
**应对**：
- "我就是反映人代表全班的大群，看学院怎么处理。"
- 不要主动承认任何举报"主导"。
- **不要**单独去学院办公室——最好有人陪同。
- **不要**承认"我就是发微博 / 知乎 / 小红书那个人"。

### 4.2 学院找你"约谈"

**场景**：学院约你单独谈。
**应对**：
1. **必须有陪同**（建议 2 名同学同行，最好包括班委）；
2. **提前约定**："我希望有同学一起，而且希望能有书面记录。"
3. **不要签署任何文件**——除非你完全看清每一条；
4. **不要接受任何带有"你不再向上反映"含义的承诺**——可以接受事实性的"学院会补偿/调整"，但**不接受**封口性质的条件。
5. **全程自己录音**（在不违反当地法律、且对方知情的前提下）。

### 4.3 同学劝你"算了"

**场景**：其他同学怕事，劝你别搞了。
**应对**：
- 不要因为几个声音放弃；
- 你可以做的是**降低你的参与度、加大材料分发**——让学校的注意力放在"问题本身"，而不是"谁在反映"。

### 4.4 学院通过家长联系你

**场景**：学院通过辅导员联系家长。
**应对**：
- 提前和家里说清楚原委，避免家长在不知情下被惊吓；
- 家庭是你天然的"后备安全网"——不要隐瞒。

### 4.5 学校威胁/警告处分

**场景**：学校以"违反校规"威胁。
**应对**：
- 检视你过去所有动作的合规性；
- 只要你**没**违背《高等教育法》《信访工作条例》《教育法》，学校无法以"散布谣言""扰乱公共秩序"等名义处理；
- 反之，**学校因学生正常反映问题而处分学生**，可向省级教育行政部门投诉。

### 4.6 媒体反而联系学院，你被学院"反向追责"

**场景**：学院反向追责媒体来源。
**应对**：
- 这是正常博弈——学院无权查阅媒体线索；
- 媒体有职业操守，**不会**主动交出"线人是谁"；
- 如学院通过新闻稿报道中暴露的内容反推"只有这位同学可能有动机"，**不要主动承认**——让证据说话。

## 五、删除与回滚策略

### 5.1 如果你想中止行动

可以**保留所有材料但停止投递**。每个渠道都没有"承诺了必须完成"的义务。**任何时候你想停，把材料归档，未来需要再拿出来用**。

### 5.2 如果事态走向不利于你

不要主动删除证据——**那才是你最大的保护**。事件过去之后，**所有材料长期保存**（5—10 年）。

### 5.3 如果你担心"打击报复"

**记住**：所有正规渠道都接受匿名举报；自媒体可以用专号。
**核心**：让证据在不暴露身份的前提下，替你说话。
