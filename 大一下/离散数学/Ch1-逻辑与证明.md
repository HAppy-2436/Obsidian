## 1.1 命题逻辑

### 核心概念

- **命题**：能唯一判断真假的陈述句
- **非命题**：疑问句、祈使句、感叹句、悖论、含未定义变量的句子
  - 例：`"x + 1 = 2"` 不是命题（要看 x）；`"这朵花真美"` 不是命题（带主观）

### 5 个联结词 + 真值表

| 符号 | 名称 | 读法 | 真值规则 |
|---|---|---|---|
| ¬p | 否定（NOT） | 非 p | 0→1, 1→0 |
| p ∧ q | 合取（AND） | p 且 q | 都 1 才 1 |
| p ∨ q | 析取（OR） | p 或 q | 都 0 才 0 |
| p ⊕ q | 异或（XOR） | p 异或 q | 相同为 0，不同为 1 |
| p → q | 条件 | 若 p 则 q | **只有 p=1, q=0 时为 0** |
| p ↔ q | 双条件 | p 当且仅当 q | 真值相同才 1 |

### 位运算（计算机表示）

- 位 AND = 合取（∧）
- 位 OR = 析取（∨）
- 位 XOR = 异或（⊕）

> [!tip] 速记
> 条件 p→q 反直觉：**前件真后件假才假**，其他三种情况全真（"承诺"逻辑：许诺没兑现才违约）。

### 典型题型

1. 判断句子是否是命题
2. 给命题，写联结词表达式
3. 画真值表
4. 用位运算求字符串结果

### 易错点

- 异或用 ⊕ 别和普通 OR 混
- 条件命题 "如果... 那么..." 默认是蕴含 p→q，不是"如果... 只有..."

## 1.3 命题等价

### 核心概念

- **永真式** (tautology)：所有赋值下真值都为真（T）
- **矛盾式** (contradiction)：所有赋值下真值都为假（F）
- **可满足式** (contingency)：既非永真也非矛盾
- **逻辑等价** p ≡ q：p ↔ q 是永真式

### 重要等价律（必背）

| 名称 | 公式 |
|---|---|
| 同一律 | p ∧ T ≡ p, p ∨ F ≡ p |
| 支配律 | p ∨ T ≡ T, p ∧ F ≡ F |
| 幂等律 | p ∨ p ≡ p, p ∧ p ≡ p |
| 双重否定 | ¬(¬p) ≡ p |
| 交换律 | p ∨ q ≡ q ∨ p |
| 结合律 | (p ∨ q) ∨ r ≡ p ∨ (q ∨ r) |
| 分配律 | p ∨ (q ∧ r) ≡ (p ∨ q) ∧ (p ∨ r) |
| 德摩根律 | ¬(p ∧ q) ≡ ¬p ∨ ¬q |
| 吸收律 | p ∨ (p ∧ q) ≡ p |
| 否定律 | p ∨ ¬p ≡ T, p ∧ ¬p ≡ F |

### 关键等价（转换核心）

- **条件 ↔ 析取**：p → q ≡ ¬p ∨ q
- **条件否定**：¬(p → q) ≡ p ∧ ¬q
- **双条件 ↔ 复合**：p ↔ q ≡ (p → q) ∧ (q → p) ≡ (p ∧ q) ∨ (¬p ∧ ¬q)
- **双条件否定**：¬(p ↔ q) ≡ p ⊕ q
- **异或**：p ⊕ q ≡ (p ∨ q) ∧ ¬(p ∧ q) ≡ (p ∧ ¬q) ∨ (¬p ∧ q)

### 证明等价的方法

1. 真值表法：列所有赋值，对比两列真值
2. 等价变换法：用等价律逐步替换
3. 真值表 + 等价律混用

### 易错点

- 条件 ↔ 析取转换是**单向蕴含**的逆：p → q 不等价于 ¬q → ¬p（这个是逆否命题，但**只等价于** ¬p ∨ q）
- ¬(p → q) 是 p ∧ ¬q，不是 ¬p → ¬q

## 1.4 谓词与量词

### 核心概念

- **谓词** P(x)：含变量的命题函数
- **论域** (domain)：变量取值范围
- **命题**：指定具体变量值后才是命题

### 两种量词

| 量词 | 记号 | 含义 |
|---|---|---|
| 全称量词 | ∀x P(x) | 论域内**所有** x 使 P(x) 为真 |
| 存在量词 | ∃x P(x) | 论域内**存在**某个 x 使 P(x) 为真 |

### 量词的德摩根律（必考）

$$\neg \forall x\, P(x) \equiv \exists x\, \neg P(x)$$
$$\neg \exists x\, P(x) \equiv \forall x\, \neg P(x)$$

> [!tip] 速记
> 否定量词 = 量词互换（∀↔∃）+ 谓词否定

### 限制论域写法

- ∀x ∈ S P(x) ≡ ∀x (x ∈ S → P(x))
- ∃x ∈ S P(x) ≡ ∃x (x ∈ S ∧ P(x))

> **注意区别**：∀ 用蕴含 →，∃ 用合取 ∧

### 典型题型

1. 把自然语言翻译成谓词逻辑（含量词）
2. 量词否定
3. 论域非空集时判断真假

### 易错点

- 空论域时 ∀x P(x) 为真（空真），∃x P(x) 为假
- 限制论域的 ∧ 和 → 别搞混

## 1.5 嵌套量词

### 量词顺序

| 同类量词 | 可交换 |
|---|---|
| ∀x∀y P(x,y) ≡ ∀y∀x P(x,y) | ✅ |
| ∃x∃y P(x,y) ≡ ∃y∃x P(x,y) | ✅ |
| ∀x∃y P(x,y) ≡ ∃y∀x P(x,y) | ❌ |

> [!warning] 重点
> ∀∃ 与 ∃∀ 顺序不同则**含义不同**：
> - ∀x ∃y (x < y)：每个数都有比自己大的数（真）
> - ∃y ∀x (x < y)：存在一个最大的数（假）

### 嵌套量词的否定

逐层取反，∀ ↔ ∃ 互换：

$$\neg \forall x \exists y\, P(x,y) \equiv \exists x \forall y\, \neg P(x,y)$$

### 典型题型

1. 翻译嵌套量词语句（特别是"对所有...存在..."与"存在...对所有..."）
2. 取反多层量词
3. 给出反例

## 1.6 推理规则（**必考大题**）

### 命题逻辑 8 条规则

| 名称 | 形式 | 记忆 |
|---|---|---|
| **Modus Ponens (MP)** | p → q, p ⊢ q | 假言推理 |
| **Modus Tollens (MT)** | p → q, ¬q ⊢ ¬p | 拒取式 |
| **Hypothetical Syllogism (HS)** | p → q, q → r ⊢ p → r | 假言三段论 |
| **Disjunctive Syllogism (DS)** | p ∨ q, ¬p ⊢ q | 析取三段论 |
| **Addition (Add)** | p ⊢ p ∨ q | 附加 |
| **Simplification (Simp)** | p ∧ q ⊢ p | 化简 |
| **Conjunction (Conj)** | p, q ⊢ p ∧ q | 合取 |
| **Resolution (Res)** | p ∨ q, ¬p ∨ r ⊢ q ∨ r | 消解 |

### 谓词逻辑 4 条规则

| 名称 | 形式 | 备注 |
|---|---|---|
| **Universal Instantiation (UI)** | ∀x P(x) ⊢ P(c) | c 是任意元素 |
| **Universal Generalization (UG)** | P(c) for arbitrary c ⊢ ∀x P(x) | c 必须任意 |
| **Existential Instantiation (EI)** | ∃x P(x) ⊢ P(c) for some specific c | c 是某个特定元素 |
| **Existential Generalization (EG)** | P(c) ⊢ ∃x P(x) | c 是某个元素 |

### 证明结构

每一步写出**前提 → 结论 + 引用规则 + 行号**：

```
1. p → q         前提
2. ¬q            前提
3. ¬p            1, 2, MT
4. ...
```

### 典型证明题型

1. 用规则证明论证有效（直接给规则名）
2. 用规则证明（用条件等价、置换）
3. 反证法：要证 p，反设 ¬p，推出矛盾
4. 分情况证明：用 p ∨ ¬p 分两种情况

### 易错点

- **UI 和 EI 区别**：UI 得到的 P(c) 中 c 可复用，EI 得到的 c 是某个特定元素，**不能用于 UG**
- **UG 限制**：c 必须是真的任意，不能是被假设为存在的
- **MT vs 取逆**：MT 是合法规则，p → q, ¬q ⊢ ¬p；但 p → q, ¬p ⊢ ¬q 不合法（这是肯定前件谬误）
