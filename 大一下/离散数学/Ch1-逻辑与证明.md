## 1.1 命题逻辑（基础，作业量大但不难）

### 真值表（必背）

| 符号 | 名称 | 真值规则 |
|---|---|---|
| ¬p | 否定 | 0→1, 1→0 |
| p ∧ q | 合取 | 都 1 才 1 |
| p ∨ q | 析取 | 都 0 才 0 |
| p ⊕ q | 异或 | 相异为 1 |
| **p → q** | 条件 | **p=1, q=0 时为 0；其他都为 1** |
| p ↔ q | 双条件 | 真值相同才 1 |

> [!tip] 速记
> 条件 p→q："**只有 p 真 q 假才假**" —— 这是其他 3 种情况都为真的"承诺"逻辑

### 位运算（作业会考）

- 位 AND = ∧, OR = ∨, XOR = ⊕
- 字符串按位运算，从左到右

### 反复考的题型

1. **翻译自然语言为符号**（作业 10, 12, 14）：用 p, q 表示原命题
2. **画真值表**（作业 32-38）：列出所有 2ⁿ 行赋值
3. **判断等价**（作业 32-38 子问）：两列真值相同则等价
4. **位运算求值**（作业 48）：把命题当 bit string 计算

## 1.3 命题等价

### 必背等价律

| 名称 | 公式 |
|---|---|
| 同一律 | p∧T ≡ p, p∨F ≡ p |
| 支配律 | p∨T ≡ T, p∧F ≡ F |
| 幂等律 | p∨p ≡ p, p∧p ≡ p |
| 双重否定 | ¬¬p ≡ p |
| 交换/结合 | p∨q ≡ q∨p, (p∨q)∨r ≡ p∨(q∨r) |
| 分配律 | p∨(q∧r) ≡ (p∨q)∧(p∨r) |
| 德摩根律 | ¬(p∧q) ≡ ¬p∨¬q |
| 吸收律 | p∨(p∧q) ≡ p |
| 否定律 | p∨¬p ≡ T, p∧¬p ≡ F |

### 必背转换（反复考）

| 转换 | 公式 |
|---|---|
| **条件 → 析取** | p → q ≡ **¬p ∨ q** |
| **条件否定** | ¬(p → q) ≡ **p ∧ ¬q** |
| **双条件 → 复合** | p ↔ q ≡ (p∧q)∨(¬p∧¬q) |
| **双条件否定** | ¬(p ↔ q) ≡ **p ⊕ q** |
| **异或** | p ⊕ q ≡ (p∧¬q)∨(¬p∧q) |

> [!warning] 易错
> - ¬(p→q) ≡ p∧¬q（**不是** ¬p→¬q）
> - 逆否等价：p→q ≡ ¬q→¬p（**正确**），但 p→q ≡ q→p（**错**，除非 p↔q）

### 反复考的题型

1. **真值表证明等价**（作业 4, 12）：列真值表，对比两列
2. **用等价律证明**（作业 4 关联律，5 分配律）：从左边逐步变换到右边

## 1.4 谓词与量词

### 反复考的核心

#### 量词德摩根律（必背，必考）

$$\neg \forall x\, P(x) \equiv \exists x\, \neg P(x)$$
$$\neg \exists x\, P(x) \equiv \forall x\, \neg P(x)$$

#### 限制论域（必考）

- ∀x ∈ S P(x) ≡ ∀x (x ∈ S **→** P(x))
- ∃x ∈ S P(x) ≡ ∃x (x ∈ S **∧** P(x))

> **∀ 用 →，∃ 用 ∧**——这是反复错的点

#### 空论域

- 空论域时 ∀x P(x) 为真（**空真**）
- 空论域时 ∃x P(x) 为假

### 反复考的题型

1. **量词翻译**（作业 16, 18, 20, 30）："所有...都..." "有些..." "没有..."
2. **量词否定**（作业 38, 40）：逐层取反，∀↔∃ 互换
3. **限制论域翻译**（作业 12, 14, 16）：∀ 用 →，∃ 用 ∧

## 1.5 嵌套量词

### 核心：量词顺序（必考）

| 同类 | 可交换？ |
|---|---|
| ∀x∀y P(x,y) ≡ ∀y∀x P(x,y) | ✅ |
| ∃x∃y P(x,y) ≡ ∃y∃x P(x,y) | ✅ |
| **∀x∃y P(x,y) ≡ ∃y∀x P(x,y)** | ❌ |

> [!warning] 易错
> ∀∃ 与 ∃∀ **含义不同**：
> - ∀x ∃y (x < y)：每个数都有比自己大的数（**真**）
> - ∃y ∀x (x < y)：存在最大数（**假**）

### 嵌套量词的否定（必考）

逐层取反：
$$\neg \forall x \exists y\, P(x,y) \equiv \exists x \forall y\, \neg P(x,y)$$

### 反复考的题型

1. **量词翻译**（作业 26, 28, 30）："每个学生都有某门课..." "有些学生每门课..."
2. **量词否定**（作业 40）：多量词嵌套的否定
3. **找反例**（作业 24, 38）：∀∃ 假，给一个 ∀x 不存在 y 的反例

## 1.6 推理规则（**核心必考大题**）

### 必须背到肌肉记忆的 8 条命题规则

| 规则名 | 形式 | 必背要点 |
|---|---|---|
| **MP** (Modus Ponens) | p→q, p ⊢ q | 最常用 |
| **MT** (Modus Tollens) | p→q, ¬q ⊢ ¬p | 拒取式 |
| **HS** (Hypothetical Syllogism) | p→q, q→r ⊢ p→r | 假言三段论 |
| **DS** (Disjunctive Syllogism) | p∨q, ¬p ⊢ q | 析取三段论 |
| **Add** (Addition) | p ⊢ p∨q | 加东西 |
| **Simp** (Simplification) | p∧q ⊢ p | 简化 |
| **Conj** (Conjunction) | p, q ⊢ p∧q | 合起来 |
| **Res** (Resolution) | p∨q, ¬p∨r ⊢ q∨r | 消解 |

> [!tip] 速记
> **MP/MT/HS** 处理蕴含
> **DS** 处理析取
> **Add/Simp/Conj** 处理合取
> **Res** 处理两个析取

### 必须会用的 4 条谓词规则

| 规则名 | 形式 | 关键限制 |
|---|---|---|
| **UI** | ∀x P(x) ⊢ P(c) | c 是**任意**元素（可复用） |
| **UG** | P(c) ⊢ ∀x P(x) | c 必须**真的任意**（不能是假设存在的） |
| **EI** | ∃x P(x) ⊢ P(c) | c 是**某个特定**元素（**不能用于 UG**） |
| **EG** | P(c) ⊢ ∃x P(x) | c 是某个元素 |

> [!warning] 反复错点
> - EI 推出的 c **不能用于 UG**（因为 c 是"某个特定"而不是"任意"）
> - UG 时 c 不能是前面假设存在的（如 ∃x 推出 c 后想用 UG）

### 反复出现的题型 1：用规则证明论证有效

**题型模板**（作业 12, 28, 30）：

```
前提:  P1, P2, ..., Pn
结论:  Q
求:    用规则从 P1..Pn 推出 Q
```

**标准解法步骤**（作业 12 是典型）：

1. **把条件命题转换**：p→q ≡ ¬p∨q（这是作业反复考的突破口）
2. **从结论反推**：结论需要什么前件？逆推回去
3. **从已知前件开始**：能用 MP/MT/HS 的就先用
4. **每步标注规则名 + 行号**

**作业 12 完整解法参考**（这是必考模式）：

```
前提: (p∧t)→(r∨s),  q→(u∧t),  u→p,  ¬s
结论: q→r

证明:
1. (p∧t)→(r∨s)              前提
2. q→(u∧t)                   前提
3. u→p                       前提
4. ¬s                        前提
5. 假设 q                    （用 Exercise 11 的方法，对 q→r 反证或前件假设）
6. u∧t                       2,5,MP
7. u                         6,Simp
8. t                         6,Simp
9. p                         3,7,MP
10. p∧t                      8,9,Conj
11. r∨s                      1,10,MP
12. r                        11,4,DS    ← 关键：s 排除掉，留 r
13. ∴ q→r                    5-12 条件证明
```

> [!tip] 关键技巧
> - **DS（消解三段论）**：有 p∨q 和 ¬p 时，能推出 q
> - **MP 反着用**：要证 q 时找 p→q 和 p

### 反复出现的题型 2：找论证形式 + 判断有效性

**题型模板**（作业 1, 2, 9, 10）：

```
前提 1: "若 Socrates 是人, 则 Socrates 会死"
前提 2: "Socrates 是人"
结论:   "Socrates 会死"
```

**方法**：

1. 用 p, q, r 替换具体陈述
2. 写出论证形式（前提们，∴ 结论）
3. 用真值表法判断有效性：找**所有前提真但结论假**的赋值
   - 找不到 → valid（有效）
   - 找到 → invalid（无效），反例就是该赋值

**典型 valid 论证**（必背）：

- 假言推理：p→q, p, ∴ q
- 拒取式：p→q, ¬q, ∴ ¬p
- 假言三段论：p→q, q→r, ∴ p→r
- 析取三段论：p∨q, ¬p, ∴ q

**典型 invalid 论证**（作业 5, 6 反复考）：

- 肯定后件：p→q, q, ∴ p（**谬误**）
- 否定前件：p→q, ¬p, ∴ ¬q（**谬误**）

### 反复出现的题型 3：指出论证错误 / 找反例

**典型错误**（作业 17, 18, 24 反复考）：

- **EI→UG 错误**：从 ∃x P(x) 用 EI 得到 P(c)，再 UG 得 ∀x P(x) —— **错**，c 是某个特定的
- **逆否混淆**：p→q, ¬p, ∴ ¬q —— **否前谬误**
- **空论域问题**：c 不知道是否真存在

**作业 17 典型解**（反复考的"指出错误"）：

```
前提: ∃x H(x)
结论: H(Lola)

错误: ∃x H(x) 只能推出 ∃x H(x) → H(c) (某个 c)，不能推出 H(Lola)
     "某个快乐的人存在" ≠ "Lola 快乐"
```

### 反复出现的题型 4：谓词规则证明

**典型模式**（作业 13, 14, 26）：

```
前提: ∀x (P(x)→Q(x)),  ∀x (Q(x)→R(x))
结论: ∀x (P(x)→R(x))

证明:
1. ∀x(P(x)→Q(x))         前提
2. ∀x(Q(x)→R(x))         前提
3. P(c)→Q(c)              1, UI
4. Q(c)→R(c)              2, UI
5. P(c)→R(c)              3,4, HS
6. ∀x(P(x)→R(x))         5, UG    ← c 是任意的（没在前面假设存在）
```

### 反复出现的题型 5：用消解证

**作业 30 典型**：

```
前提: "Allen 是坏男孩 ∨ Hillary 是好女孩"        (A∨H)
      "Allen 是好男孩 ∨ David 开心"               (¬A∨D)
结论: "Hillary 是好女孩 ∨ David 开心"             (H∨D)

证明:
1. A∨H                      前提
2. ¬A∨D                     前提
3. H∨D                      1,2, Res  ← 消解
```
