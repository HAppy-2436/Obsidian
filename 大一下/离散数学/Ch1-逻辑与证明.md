## 1.1 命题逻辑

### 核心概念

- **命题**：能唯一判断真假的**陈述句**
- **非命题**：疑问句、祈使句、感叹句、悖论、含未定义变量

### 5 个联结词

| 符号 | 名称 | 真值规则 |
|---|---|---|
| ¬p | 否定 | 0→1, 1→0 |
| p ∧ q | 合取 | 都 1 才 1 |
| p ∨ q | 析取 | 都 0 才 0 |
| p ⊕ q | 异或 | 相异为 1 |
| p → q | 条件 | **只有 p=1, q=0 时为 0** |
| p ↔ q | 双条件 | 真值相同才 1 |

> [!tip] 速记
> 条件 p→q 反直觉：**前件真后件假才假**，其他三种情况全真（"承诺"逻辑）

### 位运算

位 AND = ∧, OR = ∨, XOR = ⊕（计算机底层的逻辑电路）

## 1.3 命题等价

### 核心概念

- **永真式** (tautology)：所有赋值下真
- **矛盾式** (contradiction)：所有赋值下假
- **可满足式** (contingency)：既非永真也非矛盾
- **逻辑等价** p ≡ q：p ↔ q 是永真式

### 必背等价律

| 名称 | 公式 |
|---|---|
| 同一律 | p∧T ≡ p, p∨F ≡ p |
| 支配律 | p∨T ≡ T, p∧F ≡ F |
| 幂等律 | p∨p ≡ p, p∧p ≡ p |
| 双重否定 | ¬¬p ≡ p |
| 交换/结合 | p∨q ≡ q∨p, (p∨q)∨r ≡ p∨(q∨r) |
| 分配律 | p∨(q∧r) ≡ (p∨q)∧(p∨r) |
| 德摩根律 | ¬(p∧q) ≡ ¬p∨¬q |
| 吸收律 | p∨(p∧q) ≡ p |
| 否定律 | p∨¬p ≡ T, p∧¬p ≡ F |

### 关键等价（转换核心）

- **条件 ↔ 析取**：p → q ≡ ¬p ∨ q
- **条件否定**：¬(p → q) ≡ p ∧ ¬q
- **双条件 ↔ 复合**：p ↔ q ≡ (p∧q)∨(¬p∧¬q)
- **双条件否定**：¬(p ↔ q) ≡ p ⊕ q
- **异或**：p ⊕ q ≡ (p∨q)∧¬(p∧q) ≡ (p∧¬q)∨(¬p∧q)

> [!warning] 易错
> ¬(p→q) ≡ p∧¬q（不是 ¬p→¬q）
> p→q 不等价于逆否 ¬q→¬p（这个等价于 ¬p∨q）

## 1.4 谓词与量词

### 核心概念

- **谓词** P(x)：含变量的命题函数
- **论域** (domain)：变量取值范围
- **量词**：
  - ∀x P(x)：论域内**所有** x 使 P(x) 为真
  - ∃x P(x)：论域内**存在**某个 x 使 P(x) 为真

### 量词的德摩根律（必考）

$$\neg \forall x\, P(x) \equiv \exists x\, \neg P(x)$$
$$\neg \exists x\, P(x) \equiv \forall x\, \neg P(x)$$

> [!tip] 速记
> 否定量词 = **量词互换（∀↔∃）+ 谓词否定**

### 限制论域

- ∀x ∈ S P(x) ≡ ∀x (x ∈ S **→** P(x))
- ∃x ∈ S P(x) ≡ ∃x (x ∈ S **∧** P(x))

> **注意区别**：∀ 用蕴含 →，∃ 用合取 ∧

### 空论域

- 空论域时 ∀x P(x) 为真（**空真**）
- 空论域时 ∃x P(x) 为假

## 1.5 嵌套量词

### 量词顺序

| 同类 | 可交换？ |
|---|---|
| ∀x∀y P(x,y) ≡ ∀y∀x P(x,y) | ✅ |
| ∃x∃y P(x,y) ≡ ∃y∃x P(x,y) | ✅ |
| **∀x∃y P(x,y) ≡ ∃y∀x P(x,y)** | ❌ |

> [!warning] 重点
> ∀∃ 与 ∃∀ 顺序不同则**含义不同**：
> - ∀x ∃y (x < y)：每个数都有比自己大的数（**真**）
> - ∃y ∀x (x < y)：存在一个最大的数（**假**）

### 嵌套量词的否定

逐层取反，∀ ↔ ∃ 互换：

$$\neg \forall x \exists y\, P(x,y) \equiv \exists x \forall y\, \neg P(x,y)$$

## 1.6 推理规则（**必考大题**）

### 命题逻辑 8 条

| 名称 | 形式 | 记忆 |
|---|---|---|
| **Modus Ponens (MP)** | p → q, p ⊢ q | 假言推理 |
| **Modus Tollens (MT)** | p → q, ¬q ⊢ ¬p | 拒取式 |
| **Hypothetical Syllogism (HS)** | p → q, q → r ⊢ p → r | 假言三段论 |
| **Disjunctive Syllogism (DS)** | p ∨ q, ¬p ⊢ q | 析取三段论 |
| **Addition (Add)** | p ⊢ p ∨ q | 附加 |
| **Simplification (Simp)** | p ∧ q ⊢ p | 化简 |
| **Conjunction (Conj)** | p, q ⊢ p ∧ q | 合取 |
| **Resolution (Res)** | p ∨ q, ¬p ∨ r ⊢ q ∨ r | 消解 |

### 谓词逻辑 4 条

| 名称 | 形式 | 备注 |
|---|---|---|
| **Universal Instantiation (UI)** | ∀x P(x) ⊢ P(c) | c 是**任意**元素 |
| **Universal Generalization (UG)** | P(c) for arbitrary c ⊢ ∀x P(x) | c 必须**真的任意** |
| **Existential Instantiation (EI)** | ∃x P(x) ⊢ P(c) for some specific c | c 是**某个特定**元素 |
| **Existential Generalization (EG)** | P(c) ⊢ ∃x P(x) | c 是某个元素 |

### 证明格式

```
1. p → q        前提
2. ¬q           前提
3. ¬p           1, 2, MT
4. ...
```

### 典型证明题

1. 用规则证明论证有效（直接给规则名）
2. 找论证形式并判断有效性
3. 用条件等价、置换后证明
4. 反证法（要证 p，反设 ¬p，推出矛盾）
5. 分情况证明（p ∨ ¬p 分两种情况）

> [!warning] 易错
> - **UI 和 EI 区别**：UI 的 c 可复用，EI 的 c **不能用于 UG**
> - **UG 限制**：c 必须真的任意，不能是假设存在的
> - **MT vs 肯定前件谬误**：p→q, ¬q ⊢ ¬p ✅；p→q, ¬p ⊢ ¬q ❌
