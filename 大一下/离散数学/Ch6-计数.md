## 6.2 鸽巢原理

### 基本鸽巢原理

n 个物体放入 k 个盒子，若 n > k，则**至少有一个盒子 ≥ 2 个物体**

### 广义鸽巢原理

n 个物体放入 k 个盒子，则**至少有一个盒子 ≥ ⌈n/k⌉ 个物体**

> [!tip] 速记
> "鸽巢"思想：物体比盒子多 → 必有盒子被塞多个

## 6.3 排列与组合

### 排列 P(n, r)

从 n 个**不同**元素中**有序**取 r 个：

$$P(n, r) = n(n-1)(n-2)\cdots(n-r+1) = \frac{n!}{(n-r)!}$$

### 组合 C(n, r)

从 n 个**不同**元素中**无序**取 r 个：

$$C(n, r) = \binom{n}{r} = \frac{n!}{r!(n-r)!}$$

### 关键性质

- $\binom{n}{r} = \binom{n}{n-r}$
- $\binom{n}{0} = \binom{n}{n} = 1$

## 6.4 二项式系数与恒等式

### Pascal 恒等式

$$\binom{n}{k} = \binom{n-1}{k-1} + \binom{n-1}{k}$$

**Pascal 三角**：每行两端 1，中间每个数 = 上一行两肩之和

### 二项式定理

$$(x+y)^n = \sum_{k=0}^{n} \binom{n}{k} x^{n-k} y^k = \sum_{k=0}^{n} \binom{n}{k} x^k y^{n-k}$$

### Vandermonde 恒等式

$$\binom{m+n}{r} = \sum_{k=0}^{r} \binom{m}{r-k}\binom{n}{k}$$

### 常用求和

$$\sum_{k=0}^{n} \binom{n}{k} = 2^n$$
$$\sum_{k=0}^{n} (-1)^k \binom{n}{k} = 0$$

## 6.5 广义排列与组合（**必考**）

### 可重排列

从 n 类元素中**有重复**地取 r 个（每类可选多个）：

$$n^r$$

### 可重组合

从 n 类元素中**无序**取 r 个（每类可重复，允许为空）：

$$\binom{n+r-1}{r}$$

> [!warning] 易错
> 可重组合 vs 普通组合：
> - 普通组合：每类最多选 1 个
> - 可重组合：每类可多选，是**"组合+重复"**的总数

### 不可区分对象的排列

n 个元素分成 k 类，每类分别有 n₁, n₂, ..., nₖ 个（n₁+n₂+...+nₖ = n）：

$$\frac{n!}{n_1!\, n_2!\, \cdots\, n_k!}$$
