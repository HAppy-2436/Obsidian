## 6.1 计数基础（**必考**）

### 4 大原理判定（作业反复考）

| 原理 | 判定关键字 | 公式 |
|---|---|---|
| **加法** | "或"（互斥任务） | \|A∪B\| = \|A\|+\|B\| |
| **乘法** | "和""又"（独立步骤） | \|A×B\| = \|A\|·\|B\| |
| **减法** | "不超过" "非" "补" | \|A\| = \|U\| - \|Ā\| |
| **除法** | "等价类" "每组大小相同" | 计数 = 总数 / 每等价类大小 |

### 反复考的题型

#### 题型 1：加法 + 乘法原理组合

**作业 24 典型**：3 男 5 女，选 1 男 1 女

```
总选法 = 3 × 5 = 15
（乘法：先选男再选女）
```

**作业 30 典型**：从 4 个男孩 3 个女孩选 1 个做代表，但某条件排除

```
总 = 4+3 = 7
或：总 - 不合格 = 总 - 排除
```

#### 题型 2：减法原理（"至少"题）

**作业 36 典型**：1000 以内可被 7 或 11 整除的正整数

```
|A∪B| = |A| + |B| - |A∩B|
     = ⌊1000/7⌋ + ⌊1000/11⌋ - ⌊1000/77⌋
     = 142 + 90 - 12 = 220
```

> [!tip] 速记
> "或"用加法，"和"用乘法，"至少"用减法

#### 题型 3：树形图

枚举所有可能（作业 16 简单情况用）

## 6.2 鸽巢原理

### 两个版本

| 版本 | 公式 | 含义 |
|---|---|---|
| **基本鸽巢** | n 物体放 k 盒，n > k → 必有一盒 ≥ 2 | 物体比盒子多 |

## 6.3 排列与组合

### 必背公式

$$P(n, r) = n(n-1)(n-2)\cdots(n-r+1) = \frac{n!}{(n-r)!}$$

$$\binom{n}{r} = \frac{n!}{r!(n-r)!}$$

### 反复考的判定

| 题型 | 选 P 还是 C |
|---|---|
| 排队、座位、密码（**有顺序**） | **P(n,r)** |
| 选人、选菜、选委员（**无顺序**） | **C(n,r)** |

## 6.4 二项式系数

### 必背

#### Pascal 恒等式

$$\binom{n}{k} = \binom{n-1}{k-1} + \binom{n-1}{k}$$

#### 二项式定理

$$(x+y)^n = \sum_{k=0}^{n} \binom{n}{k} x^{n-k} y^k$$

> [!warning] 易错
> - $\binom{n}{k}$ 中 x 的指数是 n-k（不是 k）
> - 展开 (x+y)ⁿ 时，y 的指数从 0 到 n 递增

#### 常用求和

$$\sum_{k=0}^{n} \binom{n}{k} = 2^n$$
$$\sum_{k=0}^{n} (-1)^k \binom{n}{k} = 0$$

### 反复考的题型

1. **Pascal 恒等式证明**（作业 8）：用阶乘展开
2. **二项式展开特定项**（作业 14, 24）：找 x^a y^b 项
3. **恒等式证明**（作业 26, 28, 32）：用 Pascal 或 Vandermonde

## 6.5 广义排列与组合（**核心必考**）

### 3 个核心公式

| 类型 | 公式 | 关键判别 |
|---|---|---|
| **可重排列** | n^r | 元素可重复，**有序** |
| **可重组合** | $\binom{n+r-1}{r}$ | 元素可重复，**无序** |
| **不可区分对象排列** | $\frac{n!}{n_1!\, n_2!\, \cdots\, n_k!}$ | n 个对象分 k 类，nᵢ 不可区分 |

### 反复考的题型（这是 6.5 全部重点）

#### 题型 1：可重组合（作业反复考，**必考大题**）

**公式**：从 n 类元素中**无序**取 r 个（每类可重复）

$$\binom{n+r-1}{r}$$

**作业 6 典型**：从 4 种水果买 6 个，有多少种买法（数量不限）？

```
= C(4+6-1, 6) = C(9, 6) = 84
```

> [!tip] 速记
> "**星星-竖线**"法：6 个星星 + 3 条竖线 = 9 个位置选 3 条放竖线 = C(9,3)

#### 题型 2：不可区分对象排列（**必考大题**）

**公式**：n 个对象分成 k 类（nᵢ 个不可区分）：

$$\frac{n!}{n_1!\, n_2!\, \cdots\, n_k!}$$

**作业 8 典型**：8 个硬币，3 正 5 反，排列成一行

```
= 8! / (3! × 5!) = 56
```

**作业 32 典型**：单词 MISSISSIPPI 的排列数

```
M:1, I:4, S:4, P:2 → 11!/(1!4!4!2!) = 11!/(1×24×24×2) = 34650
```

> [!warning] 易错
> - 不可区分 = 字母重复
> - 必须把每类数量都算上，分母是各类阶乘的乘积

#### 题型 3：可重排列

**公式**：n^r

**作业 14 典型**：3 位二进制字符串（每位可 0 或 1）数量

```
= 2³ = 8
```

## 6.6 排列与组合的生成

### 反复考的算法

#### 字典序生成排列

从小到大生成所有 n! 个排列：
- 找最右的 aᵢ < aᵢ₊₁ 位置
- 找 aᵢ 右边的最小比 aᵢ 大的元素
- 交换，反转右半

#### 最小变化生成组合

每次只改变一个元素的位置
