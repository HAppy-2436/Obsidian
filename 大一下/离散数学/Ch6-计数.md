# C1 计数原理

## 核心概念

- **乘法原理**：依次选，每步有 nᵢ 种选择 → 总数 = ∏ nᵢ
- **加法原理**：互斥分类，每类 nᵢ 种 → 总数 = ∑ nᵢ
- **减法原理** = 加法原理的变形（用补集）

## 深入讲解

### 三个原理的判定

**乘法 vs 加法**（最容易混）：
- **乘法**：一个完整任务**分多步**，每步必做 → 用乘法
- **加法**：一个完整任务**分多类**（互斥），只做一类 → 用加法

**判断技巧**：
```
"先做 A 再做 B" → 乘法（顺序）
"要么 A 要么 B" → 加法（分类）
"既不是 A 也不是 B" → 减法（用补集）
```

### 实战例子

**例 1**：从 5 男 3 女中选 1 男 1 女 = 5 × 3 = 15（乘法）

**例 2**：从 5 男 3 女中选 1 人（男或女）= 5 + 3 = 8（加法）

**例 3**：从 5 男 3 女中选 1 人**不是男的** = 8 - 5 = 3（减法 = 全 - 男）

### 树形图 vs 公式

**小问题**（n ≤ 10）：树形图最直观

**大问题**：必须用公式，否则会算错或算漏

### 两个原理的组合

**实战**：选 1 班 1 人，再从另一个班选 1 人
- 班 A 有 m 人，班 B 有 n 人
- 选法 = m·n（乘法）
- 选 1 人来自 A 或 B = m+n（加法）

### 容斥原理（高级加法）

对于**重叠**的情况：
$$|A \cup B| = |A| + |B| - |A \cap B|$$

```
例：选会英语或法语的人
A = 会英语, B = 会法语
|A ∪ B| = |A| + |B| - |A ∩ B|
```

## 典型例题

### 例 1.1：原理判定

**题**：从上海到北京有 3 种交通方式（飞机/高铁/火车），从北京到天津有 4 种方式（汽车/火车/大巴/出租）。上海→北京→天津有多少种方式？

**解**：3 × 4 = **12 种**（乘法，每步必做）

### 例 1.2：混合原理

**题**：学校有 5 个篮球社员，6 个足球社员（无重叠）。选 1 人去比赛。
- a) 选 1 个篮球社员或 1 个足球社员
- b) 选 1 个篮球社员和 1 个足球社员（不同人）

**解**：
- a) 5 + 6 = **11**（加法，分类）
- b) 5 × 6 = **30**（乘法，先选篮球再选足球）

### 例 1.3：容斥

**题**：30 人中，会英语 18 人，会法语 10 人，两种都会 5 人。至少会一种的有多少人？

**解**：|英语 ∪ 法语| = 18 + 10 - 5 = **23 人**

## 易错点总结

- ❌ 乘法/加法混淆（关键是看"分步"还是"分类"）
- ❌ 互斥假设错（A ∪ B 时，A 和 B 可能不互斥，要用容斥）
- ❌ 减法用错（必须保证"全集 - 减去的就是补集"）

# C2 鸽巢原理

## 核心概念

- **鸽巢原理（Pigeonhole）**：n+1 个物体放入 n 个盒子，**至少有 1 个盒子有 ≥ 2 个物体**
- **广义**：k 个物体放入 n 个盒子，至少有 1 个盒子 ≥ ⌈k/n⌉ 个物体
- **加强版**：N 个物体放入 k 个盒子，至少有 1 个盒子 ≥ ⌈N/k⌉

## 深入讲解

### 鸽巢原理的"长相"

看到题目有"必定存在""至少有"→ 鸽巢原理。

**典型模式**：
- N 个东西放到 k 个抽屉
- 至少有两个在同一抽屉（当 N > k）
- 某抽屉至少有 ⌈N/k⌉ 个（一般情况）

### 实战例子

**例 1**：13 个人中，至少有 2 个人生日在同一月份（12 个月）。
- 13 个人，12 个月，13 > 12 → 至少 2 人同月 ✓

**例 2**：100 人，至少有几人在同一属相？
- 12 属相，⌈100/12⌉ = 9
- 至少有 9 人同属相 ✓

**例 3**：n+1 个 ≤ 2n 的正整数，至少有 2 个互素。
- 把 1..2n 分成 n 组 {2k-1, 2k}（每组两个连续整数必互素）
- n+1 个数，n 组，至少 2 个数在同一组 → 互素 ✓

### 鸽巢原理证明思路

**证明模板**（反证）：
```
【反证】设每个盒子最多 1 个物体
则总物体数 ≤ n
但实际有 n+1 个 → 矛盾
∴ 必有盒子 ≥ 2 个 ✓
```

### 应用场景

**数论**：
- n+1 个 ≤ 2n 的整数，必有 2 个数其中一个整除另一个
- n 个整数，必有 2 个数模 n 同余

**组合**：
- 边长 2 的等边三角形顶点涂色 2 色，单色三角形必有
- 给定 N 个点，必有 2 点距离 ≤ √2

**字符串**：
- 二进制串长度 ≥ n+1，必有 2 个相同子串（鸽巢）

### "找配对" vs "找 n 个"

**找配对**（鸽巢基础）：n+1 → 2 个
**找 n 个**（广义）：k 个 → n 个
- 例：5 顶帽子给 3 人，必有 1 人 ≥ 2 顶；20 顶帽子给 3 人，必有 1 人 ≥ 7 顶

## 典型例题

### 例 2.1：基础鸽巢

**题**：13 个人中，至少有几人生日在同一月份？

**解**：12 个月，13 人，⌈13/12⌉ = 2 → 至少有 **2 人** 同月

### 例 2.2：广义鸽巢

**题**：100 只鸽子飞回 10 个鸽舍，至少有几只在同一鸽舍？

**解**：⌈100/10⌉ = **10 只**

### 例 2.3：进阶应用

**题**：从 1 到 200 中选 101 个数，必有 1 个数是另一个的倍数。

**解**：
```
每个数写成 2^k × m 形式，其中 m 是奇数，m ≤ 200
m ∈ {1, 3, 5, ..., 199} 共 100 个奇数
101 个数 → 至少有 ⌈101/100⌉ = 2 个数有相同奇数部分
这两个数：2^a × m 和 2^b × m（a < b）
后者 = 2^(b-a) × (2^a × m) = 2^(b-a) × 前者
∴ 后者是前者的倍数 ✓
```

### 例 2.4：综合应用

**题**：证明任意 5 个整数中，必有 3 个数之和能被 3 整除。

**解**：
```
每个整数模 3 只有 3 种余数：0, 1, 2
5 个数，3 种余数 → 必有不少于 ⌈5/3⌉ = 2 个数同余

情形1：3 个数都同余 r
→ 3r = 0 (mod 3) → 3 个数之和 ≡ 0 ✓

情形2：2 个数同余 r
还剩 3 个数分到其他余数
继续分：若剩下的同余，则变情形 1
若不同，则其他 2 个余数都被用过

设 5 个数模 3 余数为 a₁ ≤ a₂ ≤ ... ≤ a₅（已排序）
抽屉：0, 1, 2

实际可用如下思路：
任意 5 个数中，必有 3 个同余或都不同余
若都不同余（覆盖 0/1/2），则 0+1+2 = 3 ≡ 0 (mod 3) ✓
若有两个同余，第三个任意，都同余的那三个加和被 3 整除 ✓

∴ 必有 3 个数之和被 3 整除 ✓
```

> **实战模板**：找模数（3），鸽巢原理（5 个数 3 类），分类讨论。

## 易错点总结

- ❌ 鸽巢用反（"至少 n 个"要用 ⌈k/n⌉ 而不是 ⌊k/n⌋）
- ❌ 应用题没识别"鸽巢模式"（看到"必有""至少"要立刻想到）
- ❌ 鸽子数 vs 盒子数搞反

- 6.2.8, 6.2.10（广义鸽巢）
- 6.2.15, 6.2.18（数论应用）
- 6.2.20（组合应用）

# C3 排列组合

## 核心概念

| 符号 | 名称 | 公式 | 含义 |
|---|---|---|---|
| P(n, k) | 排列 | n!/(n-k)! | n 选 k，**有序** |
| C(n, k) | 组合 | n!/(k!(n-k)!) | n 选 k，**无序** |

**关系**：P(n, k) = C(n, k) · k!

## 深入讲解

### 排列 vs 组合的核心判定

**排列**：选出的元素**有顺序**（谁在前谁在后不同）
- "前 3 名"、"密码"、"排队"

**组合**：选出的元素**无顺序**（只关心选了谁）
- "选 3 个代表"、"3 种水果"

**口诀**："**有顺无组，无顺有组**"（错！应该是：**有顺排列，无顺组合**）

### C(n, k) 的关键性质

**对称性**：C(n, k) = C(n, n-k)

**Pascal 恒等式**：C(n+1, k) = C(n, k-1) + C(n, k)

**两个和**：
$$\sum_{k=0}^{n} C(n, k) = 2^n$$
$$\sum_{k=0}^{n} (-1)^k C(n, k) = 0 \quad (n \geq 1)$$

### 排列组合的常见题型

**题型 1：直接套公式**
- 5 人选 3 人排队 → P(5, 3) = 60
- 5 人选 3 人当代表 → C(5, 3) = 10

**题型 2：分类讨论**

例：从 5 男 3 女中选 3 人，至少 2 女
- 情形1：3 女 → C(3,3)·C(5,0) = 1
- 情形2：2 女 1 男 → C(3,2)·C(5,1) = 15
- 总：1 + 15 = 16

**题型 3：分步乘法**

例：5 人选 3 人当班长、副班长、学习委员
- 选 1 班长：5 种
- 选 1 副班长：4 种
- 选 1 学委：3 种
- 总：5 × 4 × 3 = 60 = P(5, 3)

**题型 4：含禁条件**

例：5 人中 3 男 2 女，3 人排队，要求女生不相邻
- 方法 1：先排男生 → 3!，再插空 → 2 个空位选 2 个位置 → P(2, 2) = 2
- 总：3! × 2 = 12

### 排列组合的"分母消除"技巧

**问题**：很多题需要除以某个数（去重），容易漏。

**典型**：
- 5 人排成一圈 → 5!/5 = 4!（除以旋转等价）
- 字母 ANAGRAM 排列 → 7!/(2!2!)（除以重复字母）
- 选 3 人当代表，再选 1 人当主席 → C(n,3)·3（不能写成 C(n,3)·n，因为选完后谁当主席不影响）

## 典型例题

### 例 3.1：直接套公式

**题**：从 26 字母中选 3 个组成密码（不重复），共多少种？

**解**：P(26, 3) = 26·25·24 = **15,600**

### 例 3.2：分类讨论（经典）

**题**：从 6 男 4 女中选 4 人，至少 2 男 2 女（每种性别内无序）。

**解**：
```
情形1：2 男 2 女：C(6,2)·C(4,2) = 15·6 = 90
情形2：3 男 1 女：C(6,3)·C(4,1) = 20·4 = 80
总：90 + 80 = 170
```

### 例 3.3：插空法（女生不相邻）

**题**：3 男 3 女排成一排，要求女生互不相邻。

**解**：
```
先排 3 个男生：3! = 6 种
女生要插到男生的"空"里

男生产生的"位"：
_M_M_M_

有 4 个空位（两端 + 中间），但**只考虑中间 2 个空**就够

实际：正确插空法
_M_M_M_  4 个空位
选 3 个空位给女生：C(4,3) = 4
3 个女生内部排列：3! = 6
总：6 × 4 × 6 = 144
```

### 例 3.4：分母消除（圆排列）

**题**：5 人围圆桌就座，共多少种？

**解**：5!/5 = **24**（旋转等价）

### 例 3.5：分组去重

**题**：单词 "MISSISSIPPI" 的字母可以组成多少种排列？

**解**：
- 字母数：M(1) I(4) S(4) P(2) 共 11 个
- 11! / (1! 4! 4! 2!) = **34,650**

## 易错点总结

- ❌ 排列组合公式记混（P 还是 C）
- ❌ 分类讨论漏情形
- ❌ 圆排列忘了除以 n
- ❌ 字母重排没考虑重复字母
- ❌ 插空法数错空位数（一般是 n+1，但有时两端不算）

# C4 二项式定理

## 核心概念

**二项式定理**：
$$(x+y)^n = \sum_{k=0}^{n} C(n, k) x^k y^{n-k}$$

**关键性质**：
- C(n, k) = C(n, n-k)（对称）
- C(n, k) 构成杨辉三角
- 帕斯卡恒等式：C(n+1, k) = C(n, k-1) + C(n, k)

## 深入讲解

### 二项式系数的应用

**应用 1：展开某项**

$(x+y)^n$ 中 $x^k y^{n-k}$ 的系数是 C(n, k)。

**应用 2：求特定项**

例：$(2x + 3)^5$ 的 $x^2$ 项系数
- $x^2$ 项：取 2 次 $(2x)$，3 次 3
- 系数：C(5,2)·2²·3³ = 10·4·27 = 1080

**应用 3：求和**

$$\sum_{k=0}^{n} C(n, k) = 2^n \quad (\text{令 } x=y=1)$$
$$\sum_{k=0}^{n} C(n, k) (-1)^k = 0 \quad (\text{令 } x=-1, y=1)$$

### Pascal 三角

```
n=0:                    1
n=1:                  1   1
n=2:                1   2   1
n=3:              1   3   3   1
n=4:            1   4   6   4   1
n=5:          1   5  10  10   5   1
```

**规律**：每个数 = 上方两数之和。

### 二项式系数的性质

**对称性**：C(n, k) = C(n, n-k)
**单峰性**：从 C(n, 0) 单增到 C(n, ⌊n/2⌋)，再单减
**总和**：∑ = 2ⁿ
**奇偶和**：奇数位之和 = 偶数位之和 = 2^(n-1)

## 典型例题

### 例 4.1：求特定项

**题**：$(x + 2)^7$ 中 $x^3$ 项的系数。

**解**：
- $x^3$ 项：选 3 次 $x$，4 次 2
- 系数：C(7,3)·2⁴ = 35·16 = **560**

### 例 4.2：求特定项（变量在括号内）

**题**：$(2x - 1)^{10}$ 中 $x^7$ 项的系数。

**解**：
- $x^7$ 项：选 7 次 $(2x)$，3 次 $(-1)$
- 系数：C(10,7)·2⁷·(-1)³ = 120·128·(-1) = **-15,360**

### 例 4.3：求和

**题**：求 $\sum_{k=0}^{5} C(5, k) 3^k$

**解**：
- 令二项式定理 $(1+3)^5 = \sum C(5,k) 3^k$
- = 4⁵ = **1024**

### 例 4.4：杨辉三角

**题**：C(7, 3) = ?

**解**：
```
n=0:            1
n=1:          1   1
n=2:        1   2   1
n=3:      1   3   3   1
n=4:    1   4   6   4   1
n=5:  1   5  10  10   5   1
n=6: 1   6  15  20  15   6   1
n=7: 1   7  21  35  35  21   7   1
```
C(7, 3) = **35**

## 易错点总结

- ❌ 二项式定理符号搞错（(-1)^k 的影响）
- ❌ 求项时漏常数（系数包含 C(n,k) × 常数部分）
- ❌ 求和题忘了套二项式

## 配套作业（精选）

- 6.4.3, 6.4.5（求特定项）
- 6.4.10, 6.4.12（求和）
- 6.4.15（杨辉三角性质）

# C5 **广义组合 / 分配问题（必考大题）**

## 核心概念

**三大类分配问题**：

| 类型 | 含义 | 公式 |
|---|---|---|
| n 个不同球 → k 个不同盒（每盒 1 个）| 不允许空 | C(n, k)·k! = P(n, k) |
| n 个不同球 → k 个相同盒 | 盒可空 | **第二类 Stirling 数** S(n, k) |
| n 个相同球 → k 个不同盒 | 盒可空 | **隔板法** C(n+k-1, k-1) |
| n 个相同球 → k 个相同盒 | 盒可空 | **分拆数** p(n, k) |

**广义组合**核心：

$$\binom{n+k_1+k_2+\cdots+k_m-1}{k_1, k_2, \ldots, k_m} = \frac{(n+k_1+\cdots+k_m-1)!}{k_1! k_2! \cdots k_m!}$$

## 深入讲解

### 三大分配的判定流程

**第一步：球相同还是不同？**
- 球相同：找"球""水果"等
- 球不同：找"人""编号""任务"等

**第二步：盒相同还是不同？**
- 盒不同：编号盒、人、位置
- 盒相同：组、堆、不可区分容器

**第三步：允许空盒吗？**

### 公式 1：n 球不同 → k 盒不同（**允许空**）

**直接公式**：kⁿ（每个球独立选盒）

**理解**：n 个球，每个有 k 种选法 → k·k·...·k = kⁿ

**例**：3 个不同球 → 5 个不同盒（允许空）= 5³ = 125

### 公式 2：n 球相同 → k 盒不同（**允许空** = 隔板法）

**公式**：C(n+k-1, k-1) 或 C(n+k-1, n)

**理解**：把 n 个球排成一排，有 n-1 个间隔。要分成 k 段，需要 k-1 个隔板。
- n 个球 + (k-1) 个隔板 = n+k-1 个位置
- 选 k-1 个位置放隔板 = C(n+k-1, k-1)

**例**：10 个相同球 → 3 个不同盒（允许空）
- C(10+3-1, 3-1) = C(12, 2) = **66**

**不允许空**：先给每盒 1 个球（去掉 k 个），再允许空
- C(n-1, k-1)

### 公式 3：n 球相同 → k 盒相同（**允许空** = 分拆数）

**分拆数 p(n, k)**：把 n 拆成 k 个非负整数的方案数（顺序不计）

**特点**：没有简单公式，只能枚举或递推。

**递推公式**：
$$p(n, k) = p(n-1, k-1) + p(n-k, k)$$

**例**：p(5, 2) = ?
- 5 = 0+5, 1+4, 2+3（顺序不计）
- 共 **3 种**

### 公式 4：n 球不同 → k 盒相同（**允许空** = 第二类 Stirling 数）

**S(n, k)**：把 n 个不同元素分成 k 个非空子集的方案数

**递推公式**：
$$S(n, k) = S(n-1, k-1) + k·S(n-1, k)$$

**解释**：
- 元素 n 自己成一组 → S(n-1, k-1)
- 元素 n 加入某组（k 种选择） → k·S(n-1, k)

**小值表**：
```
n\k | 1  2  3  4  5
1   | 1  0  0  0  0
2   | 1  1  0  0  0
3   | 1  3  1  0  0
4   | 1  7  6  1  0
5   | 1  15 25 10 1
```

**例**：S(3, 2) = 3（{1}|{2,3}, {2}|{1,3}, {3}|{1,2}）

### 实际问题的转化

**看到实际问题 → 转化为分配**：

| 问题 | 球 | 盒 | 允许空？|
|---|---|---|---|
| 10 张选票分给 3 人 | 相同 | 不同 | 否（每人至少 1 张）|
| 3 个任务分给 5 人 | 不同 | 不同 | 是 |
| 5 种水果装 3 盘 | 不同 | 相同 | 是 |
| 把 6 个相同球放 3 个相同盒 | 相同 | 相同 | 是 |

### 实战判定口诀

**球不同，盒不同**：kⁿ（允许空）或 P(n, k)（不允许空）

**球相同，盒不同**：隔板法 C(n+k-1, k-1)

**球相同，盒相同**：分拆数（最难，要枚举）

**球不同，盒相同**：第二类 Stirling 数（考试一般不给公式，要用递推或枚举）

## 典型例题

### 例 5.1：球同盒不同（隔板法，**必考**）

**题**：把 10 个相同的球放入 3 个不同的盒子，**每个盒子至少 1 个球**，多少种方法？

**解**：
```
先给每个盒子 1 个球（去掉 3 个），剩 7 个球
7 个球 + 2 个隔板 = 9 个位置
选 2 个位置放隔板 = C(9, 2) = 36
∴ 36 种
```

> **实战技巧**：每个盒子至少 1 个 → 先各放 1 个，剩下的允许空。

### 例 5.2：球不同盒不同（kⁿ）

**题**：3 个不同的人发 5 封不同的邮件，每人至少 1 封，多少种？

**解**：
```
k=3 盒（人）, n=5 球（邮件），球不同，盒不同
但要求每盒 ≥ 1
```
**容斥**：
- 总：3⁵ = 243
- 减去至少 1 盒空的：
  - 1 盒空（选哪个：3 种）：2⁵ = 32 → 3·32 = 96
  - 但 2 盒空重复算了，要加回来：3·1⁵ = 3
- 答案：243 - 96 + 3 = **150**

### 例 5.3：球同盒同（分拆数）

**题**：把 6 个相同的球放入 3 个相同的盒子（允许空），多少种？

**解**：
```
枚举：找所有 a ≤ b ≤ c，a+b+c=6
(0,0,6), (0,1,5), (0,2,4), (0,3,3),
(1,1,4), (1,2,3),
(2,2,2)

共 **7 种**
```

### 例 5.4：球不同盒同（Stirling）

**题**：把 {1,2,3,4} 分成 2 个非空子集（子集无序），多少种？

**解**：S(4, 2) = **7**
- {1}|{2,3,4}
- {2}|{1,3,4}
- {3}|{1,2,4}
- {4}|{1,2,3}
- {1,2}|{3,4}
- {1,3}|{2,4}
- {1,4}|{2,3}

### 例 5.5：综合判定（**必考**）

**题**：10 本不同的书分给 3 个学生，每名学生至少 1 本。
- a) 学生相同
- b) 学生不同

**解**：
- a) 学生相同 → 球不同盒相同不允许空 → S(10, 1) + S(10, 2) + S(10, 3)
- b) 学生不同 → 球不同盒不同不允许空 → 满射计数

**b) 的算法**：用容斥
- 总：3¹⁰
- 减（至少 1 空）：C(3,1)·2¹⁰
- 加（至少 2 空）：C(3,2)·1¹⁰
- 答案：3¹⁰ - 3·2¹⁰ + 3·1¹⁰ = 59049 - 3072 + 3 = **55980**

### 例 5.6：多项式系数

**题**：$(x_1 + x_2 + x_3 + x_4)^{10}$ 中 $x_1^3 x_2^4 x_3^2 x_4^1$ 的系数。

**解**：
- 系数 = 10! / (3! 4! 2! 1!) = **12600**

> **多项式系数公式**：n! / (k₁! k₂! ... kₘ!) 其中 ∑kᵢ = n

## 易错点总结

- ❌ 球/盒相同与否判定错
- ❌ 隔板法忘了"先各放 1 个"处理不允许空
- ❌ kⁿ 用反（应该是球不同盒不同的情况）
- ❌ 多项式系数公式记错（漏阶乘）
- ❌ 容斥时漏加回"加回项"

# C6 生成函数（入门）

## 核心概念

**生成函数**：
$$G(x) = a_0 + a_1 x + a_2 x^2 + \cdots = \sum_{n=0}^{\infty} a_n x^n$$

**对应关系**：
- $(1+x)^n$ → 二项式系数
- $\frac{1}{1-x}$ → 全是 1
- $\frac{1}{(1-x)^2}$ → n+1

## 深入讲解

### 常见生成函数（必背）

| G(x) | 系数 aₙ |
|---|---|
| 1 | [1, 0, 0, 0, ...] |
| $\frac{1}{1-x}$ | 全是 1 |
| $\frac{1}{1-ax}$ | $a^n$ |
| $\frac{1}{(1-x)^2}$ | $n+1$ |
| $\frac{1}{(1-x)^k}$ | C(n+k-1, n) |
| $(1+x)^n$ | C(n, k) |
| $(1+x)^n / (1-x)$ | 求和类 |

### 生成函数的运算

**乘法**：两个生成函数相乘 → 对应系数是卷积

**求导**：$G'(x) = \sum n a_n x^{n-1}$ → 系数变为 n·aₙ

### 实际应用：解递推

**例**：$a_n = a_{n-1} + n$, $a_0 = 0$

用生成函数：$A(x) = \sum a_n x^n$
- $A(x) - xA(x) = \sum n x^n = x/(1-x)^2$
- $A(x) = x/[(1-x)^3]$
- 查表得：$a_n = n(n+1)/2$

### 组合解释

**例**：把 n 个相同球分到 k 个不同盒（允许空）= C(n+k-1, n)

**生成函数证明**：
$$\frac{1}{(1-x)^k} = \sum_{n=0}^{\infty} C(n+k-1, n) x^n$$

第 k 个盒"贡献" 1/(1-x)，k 个盒独立贡献相乘 → 总生成函数。

## 典型例题

### 例 6.1：写出生成函数

**题**：$a_n = n$，写出 $a_n$ 的生成函数。

**解**：
$$G(x) = \sum_{n=0}^{\infty} n x^n = \frac{x}{(1-x)^2}$$

### 例 6.2：用生成函数解递推

**题**：$a_n = 2 a_{n-1}$, $a_0 = 1$。求 $a_n$。

**解**：
- $A(x) = \sum a_n x^n$
- $A(x) - 2xA(x) = a_0 = 1$
- $A(x) = 1/(1-2x)$
- 查表（用 $\frac{1}{1-ax} = \sum a^n x^n$）：$a_n = 2^n$

## 易错点总结

- ❌ 生成函数公式记混
- ❌ 卷积运算搞反
