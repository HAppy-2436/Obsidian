## 6.1 计数基础

### 三个核心原理（**必背**）

| 原理 | 条件 | 公式 |
|---|---|---|
| **加法原理** | 任务分 n 类，**互斥** | 总数 = ∑ᵢ \|Aᵢ\| |
| **乘法原理** | 任务分 n 步，**独立** | 总数 = ∏ᵢ \|Aᵢ\| |
| **减法原理** | \|A ∪ B\| = \|A\| + \|B\| - \|A ∩ B\| | 用于 A、B 不互斥 |

### 加法 vs 乘法判定（**典型题**）

> [!tip] 关键判别
> - 看是"**或**"（分类）还是"**且**"（分步）
> - 加法：完成方式**互不重叠**（一个人选语文班或数学班）
> - 乘法：完成方式**按步骤**（先选上衣再选裤子）

### 除法原理

任务可分成 k 类，每类方法数相同，总数 = k × 每类方法数

### 树形图

当 n 较小（n ≤ 3）时，画树形图直观计数

### 典型题型

1. 决策树分析
2. 加法/乘法原理选择与计算
3. 用减法原理处理不互斥
4. 字符串计数（用加法/乘法）

### 易错点

- 加法原理：每种方法只能归**一类**（互斥）
- 乘法原理：每一步选法**独立**
- "或"≠"且"：做题时看题目用词

## 6.2 鸽巢原理

### 基本鸽巢原理

> 把 n+1 个物件放入 n 个盒子，至少一个盒子有 ≥ 2 个物件

**一般形式**：n 个物件放 k 个盒子，至少一个盒子有 ≥ ⌈n/k⌉ 个物件

### 应用模式

- "证明至少两个..."→ 鸽巢
- "存在子集..."→ 鸽巢
- 反证 + 鸽巢：假设每盒 ≤ 某值，推出矛盾

### 常见套路

| 题型 | 鸽巢设置 |
|---|---|
| 生日问题 | 366 人 → 至少两人同生日 |
| 同余 | n 个数 → 至少两个模 n 同余 |
| 袜子 | 抽屉里袜子数 < 颜色数 |
| 子集和 | n+1 个 ≤ 2n 的正整数 → 某子集和为 2n |
| 十进制表示 | 任意 n+1 个数 → 某两个末位相同 |

### 典型题型

1. 直接应用鸽巢（盒子数 / 物件数）
2. 找合适的"鸽子"和"巢"
3. 证明某个特定子集存在

### 易错点

- ⌈n/k⌉ 不是 ⌊n/k⌋
- 鸽巢原理只能保证"至少"，不能给出具体位置

## 6.3 排列与组合

### 排列 P(n, r)

从 n 个不同元素**有序**选 r 个：

$$P(n, r) = n \cdot (n-1) \cdot \ldots \cdot (n-r+1) = \frac{n!}{(n-r)!}$$

### 组合 C(n, r)

从 n 个不同元素**无序**选 r 个：

$$C(n, r) = \binom{n}{r} = \frac{n!}{r!(n-r)!}$$

### 关系式（**必背**）

$$C(n, r) = C(n, n-r)$$

$$C(n, r) = \frac{n}{r} C(n-1, r-1)$$

$$C(n, r) = C(n-1, r-1) + C(n-1, r) \quad \text{(Pascal 恒等式)}$$

### 帕斯卡三角

```
        1
       1 1
      1 2 1
     1 3 3 1
    1 4 6 4 1
   1 5 10 10 5 1
```

### 典型题型

1. 求排列/组合数
2. 用 C(n, r) = C(n, n-r) 简化
3. 用递推关系（如 C(n, r) = C(n-1, r-1) + C(n-1, r)）
4. 证明组合恒等式

### 易错点

- 排列 vs 组合：**看顺序重要与否**
- 0! = 1（重要约定）
- C(n, 0) = C(n, n) = 1

## 6.4 二项式系数与恒等式

### Pascal 恒等式

$$\binom{n+1}{k} = \binom{n}{k-1} + \binom{n}{k}$$

> [!tip] 意义
> 第 n+1 行的第 k 个数 = 第 n 行第 k-1 + 第 n 行第 k 个数之和

### 二项式定理（**必背**）

$$(x + y)^n = \sum_{k=0}^{n} \binom{n}{k} x^{n-k} y^k = \sum_{k=0}^{n} \binom{n}{k} x^k y^{n-k}$$

### Vandermonde 恒等式

$$\binom{m+n}{r} = \sum_{k=0}^{r} \binom{m}{r-k} \binom{n}{k}$$

### 常用推论

- $\binom{n}{k} = \frac{n}{k}\binom{n-1}{k-1}$
- $\binom{n}{k} = \frac{n-k+1}{k}\binom{n}{k-1}$
- $k\binom{n}{k} = n\binom{n-1}{k-1}$（吸收恒等式）

### 常用求和

- $\sum_{k=0}^{n} \binom{n}{k} = 2^n$
- $\sum_{k=0}^{n} (-1)^k \binom{n}{k} = 0$ (n > 0)

### 典型题型

1. 二项式展开求系数
2. 用二项式定理证明恒等式
3. Pascal / Vandermonde 恒等式的应用
4. 用 C(n, r) 的递推生成帕斯卡三角

### 易错点

- $\binom{n}{k} = \binom{n}{n-k}$ 别忘
- 帕斯卡恒等式：$n+1$ 行不是 $n$ 行

## 6.5 广义排列与组合（**必考大题**）

### 三类问题

| 问题 | 公式 | 关键词 |
|---|---|---|
| **可重排列** | $n^r$ | 元素可重复使用，每次选 n 个之一，选 r 次 |
| **可重组合** | $\binom{n+r-1}{r}$ | 元素可重复，从 n 类选 r 个（无序） |
| **不可区分对象排列** | $\frac{n!}{n_1! n_2! \cdots n_k!}$ | 多类对象，每类不可区分 |

### 可重组合公式的两种理解

- n 类对象选 r 个（无序、每类可重复）= $\binom{n+r-1}{r}$
- 把 r 个相同球放入 n 个不同盒子（可空）= $\binom{n+r-1}{r}$

### 经典问题套路

**问题 → 转化为哪种？**

1. **密码位数、路径选择**：可重排列 → n^r
2. **方程非负整数解**：可重组合 → $\binom{n+r-1}{r}$
3. **单词重排（含重复字母）**：不可区分对象 → n! / (n₁! n₂!...)
4. **相同物品分组**：不可区分对象
5. **二项式系数**：直接 C(n, r)

### 典型题型

1. 求方程 x₁ + x₂ + ... + xₖ = n 的非负整数解数
2. 把 n 个相同球放 k 个不同盒子
3. 字母（含重复）排列数
4. 多面骰子掷多次的点数组合
5. 路径数（在网格中从 (0,0) 到 (m, n)）

### 易错点

- 不可区分对象公式：分母是每种对象的**数量阶乘**
- 可重组合 ≠ 组合 + 重复：C(5, 2) = 10 是普通组合，C(5+2-1, 2) = 15 是可重组合
- 路径数：m 步右 + n 步上 = $\binom{m+n}{m}$ 步

## 6.6 生成排列与组合

### 字典序

按字典顺序生成所有排列（最小→最大）

### Johnson-Trotter 算法

通过**相邻对换**生成下一个排列：

1. 找最大可移动箭头（指向可交换方向的）
2. 交换该箭头方向上的相邻元素
3. 反转所有大于该箭头的箭头方向

### 生成 r-组合的算法

**方法 1**：用二进制表示，r 个 1 表示选了 r 个

**方法 2**：lexicographic 序生成
- 从 {1,2,...,r} 开始
- 找最右可加 1 的元素 aᵢ
- aᵢ ← aᵢ + 1
- aⱼ ← aⱼ₋₁ + 1（j > i）

### 典型题型

1. 给定当前排列/组合，写下一个
2. 描述算法的步骤
3. 计算特定排列/组合在字典序中的位置

### 易错点

- Johnson-Trotter 用箭头表示移动方向，不是元素值大小
- 二进制法：位置 = 1 表示选，从右往左对应第 k 个元素
