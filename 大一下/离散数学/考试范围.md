## 各章考点框架

### Ch.1 Logic and Proofs

- **1.1 Propositional Logic**：命题、5 个联结词的真值表（¬ ∧ ∨ ⊕ → ↔）、位运算（bitwise AND / OR / XOR）
- **1.3 Propositional Equivalences**：永真 / 永假 / 可满足式、逻辑等价、重要等价律（De Morgan、分配、吸收、双重否定、条件 ↔ 析取、双条件）、证明两个式子等价
- **1.4 Predicates and Quantifiers**：谓词、全称量词 ∀、存在量词 ∃、量词的 De Morgan 否定、限制论域写法
- **1.5 Nested Quantifiers**：量词顺序不可交换、嵌套量词的翻译和否定
- **1.6 Rules of Inference**：命题逻辑 8 条推理规则（MP / MT / HS / DS / Add / Simp / Conj / Res）+ 谓词逻辑 4 条（UI / UG / EI / EG）、证明论证有效性

### Ch.2 Basic Structures

- **2.1 Sets**：集合表示（列举法 / 谓词法）、子集 / 真子集 / 空集 / 幂集、笛卡尔积、用量词表达集合命题
- **2.2 Set Operations**：交 / 并 / 补 / 差 / 对称差、Venn 图、集合恒等式
- **2.3 Functions**：定义域 / 值域 / 像、单射 / 满射 / 双射、逆函数、复合函数、⌊ ⌋ ⌈ ⌉ 上下取整
- **2.4 Sequences and Summations**：序列、Σ 求和、Σ 求和公式、递推关系初步
- **2.5 Cardinality of Sets**：可数 / 不可数、Hilbert 大饭店、Cantor 对角线证法
- **2.6 Matrices**：矩阵加减乘、转置、布尔矩阵（join / meet / Boolean product）

### Ch.6 Counting

- **6.1 The Basics of Counting**：加法原理、乘法原理、减法原理（容斥 2 集）、除法原理、树形图
- **6.2 Pigeonhole Principle**：基本 + 广义
- **6.3 Permutations and Combinations**：P(n,r)、C(n,r)、二项式系数
- **6.4 Binomial Coefficients and Identities**：Pascal 恒等式、二项式定理、Vandermonde 恒等式
- **6.5 Generalized**：可重排列 P(n+r−1,r)、可重组合、不可区分对象的排列
- **6.6 Generating**：字典序生成排列、生成组合的算法

### Ch.8 Advanced Counting Techniques

- **8.1 Recurrence Relations**：递推建模（汉诺塔 / 斐波那契）、初始条件
- **8.2 Solving Linear Recurrence Relations**：齐次线性递推的特征方程、非齐次线性递推
- **8.3 Divide-and-Conquer**：主定理（Master Theorem）—— 判断三种情况
- **8.4 Generating Functions**：生成函数定义、用生成函数解递推、计数问题
- **8.5 Inclusion-Exclusion**：PIE（容斥原理）
- **8.6 Applications of Inclusion-Exclusion**：满射计数、错位排列（derangements）

### Ch.9 Relations

- **9.1 Relations and Their Properties**：二元关系、自反 / 对称 / 反对称 / 传递、关系复合
- **9.3 Representing Relations**：0-1 关系矩阵、关系有向图、矩阵运算 ↔ 关系运算
- **9.4 Closures of Relations**：自反 / 对称 / 传递闭包、Warshall 算法
- **9.5 Equivalence Relations**：等价关系、等价类、集合划分（一对一对应）
- **9.6 Partial Orderings**：偏序、Hasse 图、极大 / 极小 / 最大 / 最小元素、格（lattice）

### Ch.10 Graphs

- **10.1 Graphs and Graph Models**：无向 / 有向图、顶点 / 边 / 度、完全图 K_n / 圈图 C_n / 轮图 W_n / 二部图 K_{m,n}
- **10.2 Graph Terminology**：邻接 / 关联、孤立点 / 悬挂点、子图 / 并图
- **10.3 Representing Graphs**：邻接矩阵、邻接表、关联矩阵；图同构判定、不变量（度序列、连通分量数等）
- **10.4 Connectivity**：路径 / 回路、连通图 / 连通分量、有向图的强连通、用邻接矩阵幂计两点间路径数
- **10.5 Euler and Hamilton Paths**：Euler 路径 / 回路（必要 + 充分条件）、Hamilton 路径 / 回路（必要条件 + Dirac / Ore 充分条件）、Gray 码
- **10.7 Planar Graphs**：Euler 公式 V − E + F = 2（连通平面图）、Kuratowski 定理（K₅ 和 K₃,₃）
- **10.8 Graph Coloring**：色数 χ、四色定理、平面图着色
