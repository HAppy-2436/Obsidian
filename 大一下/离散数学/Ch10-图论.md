## 10.5 Euler 与 Hamilton 路径（**必考大题**）

### Euler 路径/回路

**Euler 回路**：经过**每条边恰好一次**的回路
**Euler 路径**：经过每条边恰好一次的路径

#### 判定定理（**必背**）

| 类型 | 无向图 | 有向图 |
|---|---|---|
| **Euler 回路** | 连通 + **所有顶点度数为偶数** | 弱连通 + 每个顶点入度 = 出度 |
| **Euler 路径**（非回路） | 连通 + **恰好 2 个奇度顶点**（路径端点） | 弱连通 + 1 点入度=出度+1, 1 点入度=出度-1, 其他入=出 |

> [!tip] 速记
> **所有偶度 → Euler 回路**
> **2 个奇度 → Euler 路径**（从奇度点开始）
> **多于 2 个奇度 → 无 Euler 路径**

### Hamilton 路径/回路

**Hamilton 回路**：经过**每个顶点恰好一次**的回路

> [!warning] Hamilton 判定：**没有简单的充要条件**！判定是 NP-完全问题

#### 必要条件（**必背**）

| 条件 | 描述 |
|---|---|
| **鸽巢** | 删 k 个顶点，剩下连通分量数 ≤ k |
| **度条件** | deg(u) + deg(v) ≥ n-1（每对 u≠v） |

#### 充分条件（**必背**）

| 充分条件 | 含义 |
|---|---|
| **Dirac 定理** | n≥3 且每个顶点度 ≥ n/2 → Hamilton 回路 |
| **Ore 定理** | n≥3 且每对不邻接顶点 deg(u)+deg(v) ≥ n → Hamilton 回路 |

### Gray 码

**作业 47 反复考**：长度为 n 的 Gray 码，递归构造 G(n) = 0G(n-1) 拼接 1G(n-1)ᵣ

## 10.7 平面图（**必考**）

### 反复考的核心

#### Euler 公式（**必背**）

连通平面图：

$$
V - E + F = 2
$$

其中 V=顶点数, E=边数, F=面数（包括外面那个）

> [!warning] 反复错点
> - 必须**连通**
> - F 包括外面那个无限面
> - 不连通：V - E + F = C + 1（C 是连通分量数）

#### 推论（必背）

| 推论 | 公式 |
|---|---|
| **简单平面图** | E ≤ 2V - 5（V ≥ 3 且无三角形） |
| **二部平面图** | E ≤ 2V - 4（V ≥ 3） |
| **每个面至少 k 条边** | kF ≤ 2E |

### Kuratowski 定理

**K₅ 和 K₃,₃ 是非平面图**。一个图平面 ↔ 不含 K₅ 或 K₃,₃ 的细分。

## 10.8 图着色（**必考大题**）

### 反复考的核心

#### 色数 χ(G)

χ(G) = 给 G 顶点着色使相邻顶点异色所需的最少颜色数

#### 反复考的定理

| 定理 | 内容 |
|---|---|
| **四色定理** | 任何平面图 χ ≤ 4 |
| **χ ≥ ω** | ω 是最大团的大小 |
| **χ ≤ Δ+1** | Δ 是最大度（贪心上界） |

#### 反复考的题型

**典型公式**：

```
Kₙ: χ = n（每点都相邻）
C_n: χ = 2 (n 偶) 或 3 (n 奇)
K_{m,n}: χ = 2（只有 2 个分块）
W_n: χ = 3 (n 偶) 或 4 (n 奇)
```

**反复考的题型 1**：求 χ(G)

```
策略：
  1. 找最大完全子图 Kₖ → χ ≥ k
  2. 贪心着色
  3. 调整直到用最少颜色
```

**反复考的题型 2**：证明 χ(G) = 3

```
1. χ(G) ≥ 3（有三角形）
2. 构造 3 着色方案 → χ(G) ≤ 3
3. 结论 χ(G) = 3
```
