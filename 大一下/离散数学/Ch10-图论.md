## 10.1 图与图模型

### 基本定义

- **图** G = (V, E)：V 顶点集，E 边集
- **无向图**：边是无序对 {u, v}
- **有向图**：边是有序对 (u, v)，也叫弧

### 度

| 类型 | 定义 |
|---|---|
| **度** deg(v) | 与 v 关联的边数（无向图）|
| **出度** deg⁺(v) | 从 v 出发的边数 |
| **入度** deg⁻(v) | 到达 v 的边数 |

### 握手引理（**必背**）

$$\sum_{v \in V} \deg(v) = 2|E| \quad \text{(无向图)}$$

$$\sum_{v \in V} \deg^+(v) = \sum_{v \in V} \deg^-(v) = |E| \quad \text{(有向图)}$$

### 奇度顶点个数为偶数（推论）

无向图中奇度顶点的个数必为偶数

### 简单图 vs 多重图

- **简单图**：无自环、无多重边
- **多重图**：可有平行边

### 常用特殊图（**必背**）

| 图 | 记号 | 顶点数 | 边数 |
|---|---|---|---|
| **完全图** | K_n | n | n(n-1)/2 |
| **圈图** | C_n | n | n |
| **轮图** | W_n | n+1 | 2n |
| **完全二部图** | K_{m,n} | m+n | mn |
| **n 立方体图** | Q_n | 2ⁿ | n·2^{n-1} |

### 二部图（**重要**）

- 顶点分成两个不相交集合，所有边连接两集合
- **判定**：G 是二部图 ⇔ G 无奇圈
- **完全二部图** K_{m,n}：两个集合大小为 m 和 n

### 典型题型

1. 给场景建模（画图）
2. 计算 deg(v)、|E|
3. 用握手引理求边数或顶点数
4. 识别完全图、圈图、轮图、二部图

### 易错点

- 自环计度 **2**（无向图）
- 完全图 K_n 边数 = C(n, 2)
- W_n = C_{n-1} + 中心顶点（不是 C_n + 中心）

## 10.2 图的术语

### 邻接与关联

- **邻接**：两顶点被边连接（adjacent）
- **关联**：边与顶点连接（incident）

### 特殊顶点

| 类型 | 条件 |
|---|---|
| **孤立点** | deg(v) = 0 |
| **悬挂点** | deg(v) = 1 |
| **偶度顶点** | deg(v) 为偶数 |
| **奇度顶点** | deg(v) 为奇数 |

### 子图相关

- **子图** $G' \subseteq G$：$V' \subseteq V$，$E' \subseteq E$
- **真子图**：$G' \subset G$
- **生成子图** (spanning)：$V' = V$，边是 E 的子集
- **并图** $G_1 \cup G_2$：$(V_1 \cup V_2, E_1 \cup E_2)$

### 邻域与闭邻域

- **邻域** $N(v)$：与 v 相邻的所有顶点
- **闭邻域** $\bar N(v) = N(v) \cup \{v\}$

### 典型题型

1. 识别孤立点、悬挂点
2. 求邻域
3. 画子图、并图

### 易错点

- 子图必须满足：子图的边集 ⊆ 原图边集，且每条边的端点 ⊆ 顶点集
- 并图可能产生新的孤立点（连接外部的边没了）

## 10.3 图的表示与同构（**必考**）

### 邻接矩阵 A

$$A_{ij} = \begin{cases} \text{边数}(v_i, v_j) & \text{多重图} \\ 1 & \text{简单图有边}(v_i, v_j) \\ 0 & \text{否则} \end{cases}$$

**性质**：
- 无向图：A 对称
- 行和 = deg(vᵢ)
- 列和 = deg(vᵢ)
- **$A^n$ 的 (i,j) 元素 = 从 $v_i$ 到 $v_j$ 长度 n 的路径数**

### 邻接表

每个顶点一个链表，记录相邻顶点

### 关联矩阵

- 行 = 顶点，列 = 边
- $M_{ij} = 1$ if 顶点 i 在边 j 上

### 图同构（**必考**）

**同构**：存在双射 $f: V_1 \to V_2$ 使 $(u, v) \in E_1$ ⇔ $(f(u), f(v)) \in E_2$

**判定同构**：
1. 顶点数、边数相等
2. 度序列相同
3. 找匹配的双射
4. 验证邻接关系保留

**判定不同构（不变量）**：
- 顶点数或边数不同
- 度序列不同
- 度的多重集不同
- 连通分量数不同
- 圈数不同

### 典型题型

1. 画邻接矩阵
2. 用 $A^n$ 求路径数
3. 判定两个图是否同构
4. 给出不同构的证据

### 易错点

- 同构是**存在性**，要构造双射
- 度序列相同是必要不充分
- $A^n$ 计路径数（不是简单路径）

## 10.4 连通性

### 路径与回路

| 类型 | 定义 |
|---|---|
| **路径** | $v_0 - v_1 - \cdots - v_n$（边首尾相连）|
| **长度** | 边数 |
| **简单路径** | 顶点不重复 |
| **回路** | $v_0 = v_n$ 的路径 |
| **简单回路** | 顶点不重复（除起终点）|

### 连通性

| 概念 | 定义 |
|---|---|
| **连通** | 任意两顶点间有路径 |
| **连通分量** | 极大连通子图 |
| **割点** | 删除后连通分量数增加的顶点 |
| **割边** | 删除后连通分量数增加的边 |

### 有向图连通性

| 概念 | 定义 |
|---|---|
| **强连通** | 任意 u, v 都有 u→v 和 v→u 的路径 |
| **弱连通** | 忽略方向后无向图连通 |
| **强连通分量** | 极大强连通子图 |

### 路径计数（**必考**）

$$\text{从 } v_i \text{ 到 } v_j \text{ 长度 } n \text{ 的路径数} = (A^n)_{ij}$$

其中 A 是邻接矩阵

### 典型题型

1. 判断连通性
2. 求连通分量数
3. 求割点、割边
4. 用邻接矩阵幂求特定长度的路径数

### 易错点

- 路径 vs 简单路径：路径可重复顶点，简单路径不能
- $A^n$ 计**路径**数，不是简单路径数
- 有向图：强连通 ≠ 弱连通

## 10.5 Euler 与 Hamilton 路径（**必考大题**）

### Euler 路径 / 回路（**核心**）

- **Euler 路径**：经过**每条边**恰好一次的路径
- **Euler 回路**：起点 = 终点的 Euler 路径

### 判定定理（**必背**）

| 图类型 | Euler 回路条件 | Euler 路径条件 |
|---|---|---|
| 无向图（连通）| **所有顶点度为偶数** | **恰 2 个奇度顶点**（起终点）|
| 有向图（弱连通）| 每个顶点入度 = 出度 | 恰 1 个顶点出度 - 入度 = 1（起点），恰 1 个入度 - 出度 = 1（终点），其他入 = 出 |

### Euler 路径构造（Fleury 算法 / Hierholzer）

- Fleury：避桥原则
- Hierholzer：逐步合并回路

### Hamilton 路径 / 回路（**必考**）

- **Hamilton 路径**：经过**每个顶点**恰好一次的路径
- **Hamilton 回路**：起终点相同的 Hamilton 路径
- **判定**比 Euler 难很多（没有简单的充要条件）

### Hamilton 必要条件（**必背**）

> **删 k 个顶点，若剩余图连通分量数 > k，则 G 无 Hamilton 回路**

### Hamilton 充分条件

| 定理 | 条件 | 结论 |
|---|---|---|
| **Dirac 定理** | n ≥ 3，∀v, deg(v) ≥ n/2 | G 有 Hamilton 回路 |
| **Ore 定理** | n ≥ 3，∀不相邻 u,v，deg(u)+deg(v) ≥ n | G 有 Hamilton 回路 |

### Gray 码（应用）

- 2ⁿ 个 n 位二进制串，相邻串恰差一位
- 第一个和最后一个也差一位
- 是 Hamilton 回路在 n 立方体图 $Q_n$ 上的实例

### 典型题型

1. 判定 Euler 路径/回路存在
2. 给 Euler 路径/回路的例子
3. 证明 Hamilton 回路不存在（用必要条件）
4. 应用 Dirac / Ore 定理
5. 找 Gray 码

### 易错点

- **Euler 看的是度**，**Hamilton 看的是顶点**
- Hamilton 没有简单的充要条件，只有充分条件和必要条件
- Dirac 条件是 $\geq n/2$，不是 $> n/2$

## 10.7 平面图（**必考**）

### 定义

**平面图**：可以画在平面上，**边不相交**（除顶点处）

### Euler 公式（**核心**）

> 对于**连通**平面图：
> $$V - E + F = 2$$

其中 V 顶点数，E 边数，**F 面数**（含外部面）

### 推论（**必背**）

对于**连通平面简单图**（无自环无重边）：

| 条件 | 上界 |
|---|---|
| 无限制 | $E \leq 3V - 6$ |
| 无三角形（无 3-圈）| $E \leq 2V - 4$ |

### 应用（判非平面图）

- **K₅**：E = 10 > 3(5) - 6 = 9 → 非平面
- **K₃,₃**：E = 9 > 2(6) - 4 = 8（无奇圈 → 无 3 圈）→ 非平面

### Kuratowski 定理

> G 是平面图 ⇔ G 不含 K₅ 或 K₃,₃ 的**细分**（同胚子图）

### 平面图判定的典型套路

1. **检查** $|E| > 3|V| - 6$ → 非平面
2. **寻找** K₅ 或 K₃,₃ 的同胚子图 → 非平面
3. 否则可能平面，**尝试画**无交叉

### 典型题型

1. 用 Euler 公式求面数 F = 2 - V + E
2. 验证不等式判定非平面
3. 找 K₅/K₃,₃ 证明非平面
4. 画平面图（重画边不交叉）

### 易错点

- Euler 公式**要求连通**；不连通要用 V - E + F = 1 + C（C = 连通分量数）
- 不等式 $E \leq 3V - 6$ 是**必要**不充分
- K₅ 有三角形，所以要用 $E \leq 3V-6$；K₃,₃ 是二部图无 3-圈，用 $E \leq 2V-4$

## 10.8 图着色（**必考**）

### 基本定义

- **顶点着色**：给顶点分配颜色，**邻接顶点不同色**
- **色数** χ(G)：最少需要的颜色数
- **k-可着色**：可用 k 种颜色正常着色

### 关键不等式

$$\chi(G) \geq \omega(G) \quad (\text{色数} \geq \text{团数})$$

### 常见图的色数

| 图 | χ(G) |
|---|---|
| K_n | n |
| C_n（n 偶）| 2 |
| C_n（n 奇）| 3 |
| 树 | 2 |
| 二部图 | 2 |
| 平面图 | ≤ 4（四色定理）|

### 四色定理（**经典**）

> 任何平面图都可 4-着色：χ(G) ≤ 4

### 贪心着色算法

按顺序给顶点分配最小可用颜色，色数 ≤ 1 + 最大度

$$\chi(G) \leq \Delta(G) + 1 \quad (\Delta = \text{最大度})$$

### 边着色（简单提及）

- **边着色**：相邻边不同色
- **边色数** χ'(G)
- Vizing 定理：$\Delta(G) \leq \chi'(G) \leq \Delta(G) + 1$

### 典型应用

| 应用 | 模型 |
|---|---|
| 地图着色 | 国家 = 顶点，相邻国家 = 边 |
| 排课表 | 课程 = 顶点，时间冲突 = 边 |
| 频率分配 | 发射塔 = 顶点，距离近 = 边 |
| 寄存器分配 | 变量 = 顶点，同时活跃 = 边 |

### 典型题型

1. 给图求色数 χ(G)（画着色）
2. 给图求团数 ω(G)（找最大完全子图）
3. 用四色定理上界
4. 应用：地图着色、排课
5. 贪心着色的颜色数上界

### 易错点

- χ(G) ≥ ω(G)，但可能 χ > ω（**团数是色数的下界**）
- 平面图**不**意味着 χ = 4，可能 χ = 2 或 3
- χ ≤ 4 是**充分条件**（一定能 4-着色），不是 χ = 4
