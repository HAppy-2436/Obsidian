# G1 图基础 + 握手定理

## 核心概念

**图 G = (V, E)**：V 顶点集，E 边集。
- **无向图**：边无方向 {u,v}
- **有向图**：边有方向 (u,v)
- **简单图**：无自环、无重边
- **多重图**：可有重边（多重边数 ≥ 2）
- **加权图**：边上带权

**度**：
- deg(v) = 与 v 关联的边数（自环算 2）
- 无向图：deg(v) ≥ 0
- 有向图：出度 deg⁺(v) + 入度 deg⁻(v)

**握手定理**（必背）：

$$\sum_{v \in V} \deg(v) = 2|E|$$

> **推论**：度数为奇数的顶点个数必为偶数。

**有向版**：

$$\sum_v \deg^+(v) = \sum_v \deg^-(v) = |E|$$

## 反复考的题型

### 题型 1：求度数和边数
套握手定理。

### 题型 2：构造特定图（按 V, E 列表）
照单画图。

### 题型 3：握手定理反推
已知度数求边数，或已知边数反推度数。

# G2 特殊图类型

## 核心概念

| 类型 | 记号 | 顶点数 | 边数 | 度数 |
|---|---|---|---|---|
| **完全图** | Kₙ | n | n(n−1)/2 | 每个顶点 n−1 |
| **圈** | Cₙ (n≥3) | n | n | 每个顶点 2 |
| **轮** | Wₙ | n+1 | 2n | 中心 n，其他 3 |
| **n-立方体** | Qₙ | 2ⁿ | n·2ⁿ⁻¹ | 每个顶点 n |
| **完全二部图** | Kₘ,ₙ | m+n | mn | 一侧 m，另一侧 n |

**补图** Ḡ：与 G 共享顶点，但边是 K_n 中 G 没有的边。

**二部图**：顶点可分成两部分，所有边跨部分。

**判定二部图**：图无奇圈 ⟺ 是二部图。

直接套公式。

### 题型 3：判断二部图
BFS 染色：相邻顶点异色 → 二部图；有奇圈 → 不是二部图。

### 题型 4：求补图
把 G 没有的边画出来。

# G3 图的表示 + 同构

## 核心概念

**邻接矩阵** A：A[i][j] = 1 当且仅当 (vᵢ, vⱼ) ∈ E
- 无向图：A 对称
- 有向图：A 不一定对称
- A²[i][j] = vᵢ 到 vⱼ 长度为 2 的路径数

**关联矩阵** M：M[i][j] = 顶点 i 是否在边 j 上

**邻接表**：每个顶点存其邻居列表

**同构（isomorphism）**：存在双射 f: V(G) → V(H) 使 (u,v) ∈ E(G) ⟺ (f(u), f(v)) ∈ E(H)

**同构的必要条件**（不变量）：
- 顶点数相同
- 边数相同
- 度数序列相同（多重集）
- 圈数相同

**充分性**：找到具体对应关系 + 验证所有边对应。

## 反复考的题型

### 题型 1：写邻接矩阵
按顶点顺序填 0/1。

### 题型 2：邻接表表示
按每个顶点列邻居。

### 题型 3：判断同构

**步骤**：
1. 检查不变量（顶点数、边数、度数序列）
2. 不变量不同 → 不同构
3. 不变量相同 → 尝试构造双射，验证边对应关系

# G4 连通性 + DFS/BFS

## 核心概念

**连通**：无向图中任意两顶点间有路径。

**连通分量**：极大连通子图。

**强连通（有向图）**：任意两顶点间双向有路径。

**强连通分量**：极大强连通子图。

**DFS（深度优先搜索）**：从一个顶点沿路径走到尽头，回溯，换边。

**BFS（广度优先搜索）**：逐层访问，先访问所有邻居再下一层。

**生成树**：连通图的极小连通子图（n-1 条边）。
- 连通图必有生成树
- n 顶点连通图至少 n-1 条边

**割点**：删除后图不连通的顶点。

## 反复考的题型

### 题型 1：找连通分量
BFS/DFS 遍历。

### 题型 2：求强连通分量（有向图）
Kosaraju 算法（两次 DFS）。

### 题型 3：DFS / BFS 树
按算法遍历，记录访问顺序与生成边。

### 题型 4：证明连通性条件
套定义。

# G5 Euler 路径/回路（**必考**）

## 核心概念

**Euler 路径**：经过**每条边恰好一次**的路径。
**Euler 回路**：起点 = 终点的 Euler 路径。

**Euler 定理（必背）**：

| 图类型 | 有 Euler 回路条件 | 有 Euler 路径（无回路）条件 |
|---|---|---|
| **无向图** | 连通 + **每个顶点度数为偶** | 连通 + **恰 2 个奇度顶点** |
| **有向图** | 连通 + 每个顶点入度 = 出度 | 连通 + 恰 1 个入度比出度大 1（终点）+ 恰 1 个出度比入度大 1（起点）|

**Euler 路径存在性**（必背）：

> **定理**：无向连通图有 Euler 回路 ⟺ 0 个奇度顶点；有 Euler 路径 ⟺ 恰 2 个奇度顶点。

## 反复考的题型

### 题型 1：判断 Euler 回路/路径存在
- 数奇度顶点个数
- 0 个 → Euler 回路
- 2 个 → Euler 路径（起点/终点为奇度顶点）
- 其他 → 都没有

### 题型 2：具体构造 Euler 路径
Fleury 算法（避免断桥）或 Hierholzer 算法（拼接圈）。

### 题型 3：Königsberg 七桥问题
经典案例：七桥图有 4 个奇度顶点（C, D 各一），无 Euler 路径。

### 题型 4：特殊图 Euler 判定

| 图 | 有 Euler 回路？|
|---|---|
| Kₙ | n 为奇数（每个顶点 n-1 是偶数）|
| Cₙ | 总是 |
| Wₙ (n ≥ 3) | 总是 |
| Qₙ | 总是 |
| Kₘ,ₙ | m = n 时（每个顶点度 n 或 m 都偶？不一定，要 m=n 且都偶）|

具体：
- Kₘ,ₙ 有 Euler 回路 ⟺ m = n 且 m, n 都偶
- Kₘ,ₙ 有 Euler 路径 ⟺ |m − n| = 1 且较小者为偶

# G6 Hamilton 路径/回路（**必考**）

## 核心概念

**Hamilton 路径**：经过**每个顶点恰好一次**的路径。
**Hamilton 回路**：起点 = 终点的 Hamilton 路径。

**与 Euler 的区别**：
- Euler：经过每条边一次（边不重复）
- Hamilton：经过每个顶点一次（点不重复）

**判定 Hamilton 路径无简单充要条件**，但有几个充分条件：

| 定理 | 条件 | 结论 |
|---|---|---|
| **Dirac 定理** | 简单图，n ≥ 3，**每个顶点 deg ≥ n/2** | 有 Hamilton 回路 |
| **Ore 定理** | 简单图，n ≥ 3，**任意两不相邻顶点 u, v 满足 deg(u) + deg(v) ≥ n** | 有 Hamilton 回路 |

**必要条件**：
- 有 Hamilton 回路的图：删任意 k 顶点，图最多有 k 个连通分量
- 二部图有 Hamilton 回路 → 两部分顶点数相等

**特殊图 Hamilton 判定**：

| 图 | 有 Hamilton 回路条件 |
|---|---|
| Kₙ (n ≥ 3) | 总是 |
| Cₙ | 总是 |
| Wₙ | 总是 |
| Qₙ | n ≥ 2（Q₂ = C₄ 有）|
| Kₘ,ₙ | m = n（必要）|

## 反复考的题型

### 题型 1：判断 Hamilton 回路存在
- 套 Dirac / Ore 定理
- 找必要条件（如奇数顶点二部图不能有 Hamilton 回路）

### 题型 2：构造 Hamilton 路径
尝试逐顶点路径。

### 题型 3：Dirac / Ore 验证
计算度数和 → 套条件。

### 题型 4：图示图判定
通常需要结合 Dirac/Ore 定理与必要条件。

# G7 平面图 + Euler 公式（**必考**）

## 核心概念

**平面图**：可画在平面上使边不交叉的图。

**面**：平面图将平面分成的区域。

**Euler 公式**（**必背**）：

$$V - E + F = 2$$

其中 V = 顶点数，E = 边数，F = 面数（含外部面）。

**Kuratowski 定理**：

> 图是平面 ⟺ 不含 K₅ 或 K₃,₃ 的"细分"（subdivision）

**平面图边数上界**：
- 一般：E ≤ 2V − 4（V ≥ 3, 无三角形）
- 二部：E ≤ 2V − 4
- 含三角形：E ≤ 3V − 6

**非平面图判定**：
- 含 K₅ 或 K₃,₃ 作为子图 → 非平面
- E > 3V − 6 → 非平面（充分）
- E > 2V − 4 且二部 → 非平面

## 反复考的题型

### 题型 1：用 Euler 公式求 V/E/F

```
8 个顶点，每个度 3
→ E = 8·3/2 = 12
→ F = 2 - V + E = 2 - 8 + 12 = 6
∴ 分成 6 个区域
```

### 题型 2：判断平面性
- E > 3V − 6 → 非平面（一般图）
- E > 2V − 4 → 非平面（二部图）
- 含 K₅/K₃,₃ 子图 → 非平面

### 题型 3：求顶点数 / 边数
已知 V, F 求 E 或反推。

**通用公式**：
- 单连通：V − E + F = 2
- k 个连通分量：V − E + F = 1 + k

# G8 图着色 + 色数（**必考**）

## 核心概念

**图着色**：给每个顶点涂色，相邻顶点不同色。

**色数 χ(G)**：所需最少颜色数。

**着色的关键事实**：
- χ(G) = 1 ⟺ G 无边
- χ(G) = 2 ⟺ G 是二部图（无奇圈）
- 含奇圈 → χ(G) ≥ 3
- χ(Kₙ) = n

**色数上界**（贪心）：χ(G) ≤ Δ(G) + 1，其中 Δ(G) 是最大度。
**Brooks 定理**：G 连通且**不是完全图也不是奇圈**，则 χ(G) ≤ Δ(G)。

**面着色（地图着色）**：
- 四色定理：每个平面图可 4 着色

**边着色**：给边涂色，相邻边不同色。
- 边色数 χ'(G) = G 是 Kₙ 时 n（n 奇），n−1（n 偶）
- χ'(Cₙ) = 2 或 3
- χ'(Wₙ) = n（n 奇），n−1（n 偶）

## 反复考的题型

### 题型 1：求色数
- 找最大完全子图 Kₘ，χ ≥ m
- 找二部划分 → χ = 2
- 否则试 3 色，看是否可

### 题型 2：构造着色
按阶着色（贪心）。

### 题型 3：色数判定

| 图 | χ(G) |
|---|---|
| Kₙ | n |
| Cₙ | 2（n 偶），3（n 奇）|
| Wₙ | 4（n 偶），3（n 奇）|
| Qₙ | 2 |

### 题型 4：边着色
- Kₙ 边色数 = n（n 奇）或 n−1（n 偶）
- 间隔分配颜色

### 题型 5：考试安排问题
每门课是一个顶点，共用学生的课之间有边 → 着色 → 时间段数 = 色数
