## 10.1 图与图模型（**基础**）

### 基本术语

- **顶点（vertex）** V(G)
- **边（edge）** E(G)
- **无向图** vs **有向图**
- **简单图** vs **多重图**（无平行边 vs 有平行边）
- **邻接** (adjacent)：两点之间有边
- **关联** (incident)：边连到点
- **环** (loop)：连接同一顶点的边
- **度** deg(v)：v 上的边数

### 反复考的度数公式（**必考**）

**握手定理**（无向图）：

$$\sum_{v \in V} \text{deg}(v) = 2|E|$$

> [!warning] 推论
> - **度数为奇数的顶点个数必为偶数**
> - 推论用于判断：奇数度顶点 > 2 个 → 不可能有 Euler 回路

### 反复考的题型

#### 题型 1：求度和的奇偶性（作业 3-9 反复考）

```
作业 4 典型：25 人，每人至少 1 个朋友
   总度 ≥ 25 → 总边 ≥ 12.5 → 至少 13 条边
```

#### 题型 2：握手定理求边数

```
作业 6 典型：图有 6 个顶点，每个度为 3
   总度 = 18 → |E| = 9
```

## 10.2 图的术语

### 关键术语

| 术语 | 定义 |
|---|---|
| **子图** (subgraph) | V' ⊆ V, E' ⊆ E |
| **生成子图** (spanning) | V' = V |
| **完全图** K_n | 每对顶点都有边（n 顶点） |
| **圈图** C_n | n 个顶点的环 |
| **轮图** W_n | C_n + 中心点连所有外点 |
| **二部图** K_{m,n} | m+n 顶点，m 与 n 之间所有连 |

### 反复考的：握手定理应用

**作业 6 典型**：图有 5 个顶点，每个度为偶数，最少多少条边？

```
最少：每点度 0 → 0 条边
但要每个度都是偶数，且边数 = 总度/2
```

## 10.3 图的表示 + 同构（**必考**）

### 三种表示

| 表示 | 适用 |
|---|---|
| **邻接矩阵** A[i][j] = 1 iff (vᵢ,vⱼ) ∈ E | 密集图 |
| **邻接表** 每个顶点列出其邻居 | 稀疏图 |
| **关联矩阵** M[i][j] = 1 iff vᵢ 在 eⱼ 上 | 边-顶点关系 |

### 图同构判定（**必考大题**）

**定义**：存在双射 f: V₁ → V₂ 使 (u,v) ∈ E₁ iff (f(u), f(v)) ∈ E₂

**反复考的判定方法**（作业 30-46 反复考）：

1. **不变量检查**（必要条件）：
   - 顶点数 = 顶点数
   - 边数 = 边数
   - 度序列（多重集）= 度序列
2. **同构检验**：
   - 邻接矩阵是否可通过行列重排相同
3. **细查对应**：
   - 邻接关系逐对验证

> [!tip] 同构判定速记
> - 度序列不同 → 不同构（一步否决）
> - 同度序列但图不连通 → 可能不同构
> - 二部图 + 同分块大小 → 可能同构

### 反复考的：邻接矩阵幂（**必考**）

**Aᵏ[i][j]** = 从 vᵢ 到 vⱼ 长度为 k 的路径数

**作业 14 典型**：邻接矩阵 A，求 A² 含义

```
A²[i][j] = 长度为 2 的路径数（vᵢ → vk → vⱼ）
```

## 10.4 连通性

### 关键概念

| 概念 | 定义 |
|---|---|
| **路径** (path) | 一串相邻的边 |
| **回路** (circuit) | 起点 = 终点的路径 |
| **简单路径** | 顶点不重复 |
| **连通图** | 任意两点都有路径 |
| **连通分量** | 最大连通子图 |
| **强连通**（有向图） | 任意两点互相可达 |

### 反复考的：邻接矩阵幂与路径数（**必考**）

```
A^k[i][j] = 从 vᵢ 到 vⱼ 长度为 k 的路径数
A + A² + ... + A^n = 任意长度路径数（最多 n-1 步）
```

> [!warning] 易错
> A^k[i][j] 是路径数，**包括非简单路径**（即可以重复顶点的路径）
> 想求简单路径数需要用其他方法

### 反复考的：强连通分量（**必考**）

**作业 24 典型**：画有向图，求强连通分量

```
方法：找 SCC 间的压缩图
强连通性：u→v 且 v→u
```

## 10.5 Euler 与 Hamilton 路径（**必考大题**）

### Euler 路径/回路

**Euler 回路**：经过**每条边恰好一次**的回路
**Euler 路径**：经过每条边恰好一次的路径

#### 判定定理（**必背**）

| 类型 | 无向图 | 有向图 |
|---|---|---|
| **Euler 回路** | 连通 + **所有顶点度数为偶数** | 弱连通 + 每个顶点入度 = 出度 |
| **Euler 路径**（非回路） | 连通 + **恰好 2 个奇度顶点**（路径端点） | 弱连通 + 1 点入度=出度+1, 1 点入度=出度-1, 其他入=出 |

> [!tip] 速记
> **所有偶度 → Euler 回路**
> **2 个奇度 → Euler 路径**（从奇度点开始）
> **多于 2 个奇度 → 无 Euler 路径**

#### 反复考的题型（作业 2, 4, 6, 8 反复考）

**作业 2 典型**：画图，判断是否有 Euler 回路/路径

```
数每个顶点的度
  0 个奇度 → Euler 回路
  2 个奇度 → Euler 路径
  4+ 个奇度 → 都没有
```

**作业 18 典型**：Euler 回路存在但 Hamilton 回路不存在

```
例：K₅ 删一条边
Euler：所有度为 3（奇数）→ 无 Euler 回路
```

### Hamilton 路径/回路

**Hamilton 回路**：经过**每个顶点恰好一次**的回路
**Hamilton 路径**：经过每个顶点恰好一次的路径

> [!warning] Hamilton 判定
> Hamilton 路径**没有简单的充要条件**！判定是 NP-完全问题
> 只有**必要条件**和**充分条件**

#### 必要条件（**必背**）

| 条件 | 描述 |
|---|---|
| **鸽巢** | 删 k 个顶点，剩下连通分量数 ≤ k |
| **度条件** | deg(u) + deg(v) ≥ n-1（每对 u≠v） |

> 这两个都是**必要**条件——满足不一定有 Hamilton 回路，不满足则一定没有

#### 充分条件（**必背**）

| 充分条件 | 含义 |
|---|---|
| **Dirac 定理** | n≥3 且每个顶点度 ≥ n/2 → Hamilton 回路 |
| **Ore 定理** | n≥3 且每对不邻接顶点 deg(u)+deg(v) ≥ n → Hamilton 回路 |

#### 反复考的题型（作业 28, 42, 44 反复考）

**作业 28 典型**：判断图是否有 Hamilton 路径/回路

```
先用鸽巢/度条件看必要条件
  不满足 → 没有 Hamilton 路径
满足 → 试试看能不能构造一条（不保证）
```

**作业 42, 44 典型**：用 Dirac/Ore 充分条件

```
作业 44 典型：10 顶点图，每个度 ≥ 5
  n=10, 每个度 ≥ 5 = n/2 → Dirac 条件满足 → Hamilton 回路
```

## 10.7 平面图（**必考**）

### 反复考的核心

#### Euler 公式（**必背**）

连通平面图：

$$V - E + F = 2$$

其中 V=顶点数, E=边数, F=面数（包括外面那个）

> [!warning] 反复错点
> - 必须**连通**
> - F 包括外面那个无限面
> - 不连通：V - E + F = C + 1（C 是连通分量数）

#### 推论（必背）

| 推论 | 公式 |
|---|---|
| **简单平面图** | E ≤ 2V - 5（V ≥ 3 且无三角形） |
| **二部平面图** | E ≤ 2V - 4（V ≥ 3） |
| **每个面至少 k 条边** | kF ≤ 2E |

#### 反复考的题型（作业 12, 13, 14, 16, 18 反复考）

**作业 12 典型**：连通平面图，8 个顶点，每个度 3

```
总度 = 24 → 2E = 24 → E = 12
Euler: V - E + F = 2 → 8 - 12 + F = 2 → F = 6
分 6 个区域
```

**作业 14 典型**：连通平面图，30 条边，20 个面

```
Euler: V - 30 + 20 = 2 → V = 12
```

**作业 16 典型**：连通二部平面图，证明 E ≤ 2V-4

```
二部图无奇圈，特别是无三角形
每个面至少 4 条边（因为无三角形）
2E ≥ 4F → F ≤ E/2
V - E + F = 2 → V - E + E/2 = 2 → V - E/2 = 2 → E = 2V - 4
```

### Kuratowski 定理

**K₅ 和 K₃,₃ 是非平面图**。一个图平面 ↔ 不含 K₅ 或 K₃,₃ 的细分。

**作业 23-25 反复考**：用 Kuratowski 判定平面性

## 10.8 图着色（**必考大题**）

### 反复考的核心

#### 色数 χ(G)

χ(G) = 给 G 顶点着色使相邻顶点异色所需的最少颜色数

#### 反复考的定理

| 定理 | 内容 |
|---|---|
| **四色定理** | 任何平面图 χ ≤ 4 |
| **χ ≥ ω** | ω 是最大团的大小（完全子图） |
| **χ ≤ Δ+1** | Δ 是最大度（贪心上界） |
| **χ ≥ 圈数 / 2**（偶圈 = 2，奇圈 = 3） | 圈着色 |

#### 反复考的题型（作业 5-11 反复考）

**作业 5 典型**：画图，求 χ(G)

```
策略：
  1. 找最大完全子图 Kₖ → χ ≥ k
  2. 贪心着色（按度排序）
  3. 调整直到用最少颜色
```

**作业 6 典型**：求 K₅, K₆, K₃,₃, C_n 的色数

```
Kₙ: χ = n（每点都相邻）
C_n: χ = 2 (n 偶) 或 3 (n 奇)
K_{m,n}: χ = 2（只有 2 个分块）
W_n: χ = 3 (n 偶) 或 4 (n 奇)
```

**作业 7 典型**：K₅ 是否平面 + 平面图 K₅ 的色数

```
K₅ 非平面（Kuratowski），但假设它平面则 χ = 5
但 K₅ 不平面，所以这个无意义
```

**作业 11 典型**：给定图着色，证明 χ(G) = 3

```
1. χ(G) ≥ 3（有三角形）
2. 构造 3 着色方案 → χ(G) ≤ 3
3. 结论 χ(G) = 3
```
