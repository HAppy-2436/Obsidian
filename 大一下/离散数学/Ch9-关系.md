## 9.1 关系与其性质（**基础**）

### 4 大性质（必背）

| 性质 | 定义 | 判定方法 |
|---|---|---|
| **自反** (reflexive) | ∀a∈A, (a,a) ∈ R | 矩阵对角线全是 1 |
| **对称** (symmetric) | (a,b)∈R → (b,a)∈R | 矩阵关于对角线对称 |
| **反对称** (antisymmetric) | (a,b)∈R ∧ (b,a)∈R → a=b | 对称位置不能同时为 1（除非 a=b） |
| **传递** (transitive) | (a,b)∈R ∧ (b,c)∈R → (a,c)∈R | 复合 R² ⊆ R |

> [!warning] 易错
> - **对称 vs 反对称**：一个关系可以既对称又反对称（如恒等关系）
> - 也可以既不对称也不反对称（如 < 关系）

### 反复考的题型

#### 题型 1：判断 4 大性质（作业 6, 10, 14, 26 等反复考）

**作业 14 典型**：A = {1,2,3,4,5} 上的关系 R = {(a,b) | a ≡ b (mod 3)}

```
自反: a ≡ a (mod 3) ✓
对称: a ≡ b → b ≡ a ✓
反对称: a ≡ b ∧ b ≡ a → a = b（不一定，可能是 a≠b 但 a≡b）✗
传递: a ≡ b ∧ b ≡ c → a ≡ c ✓
所以是等价关系（自反+对称+传递）
```

#### 题型 2：复合关系 R∘S（作业 30, 32, 34）

```
R∘S = {(a,c) | ∃b: (a,b)∈S ∧ (b,c)∈R}
```

> 注意：R∘S 中 S 先作用，R 后作用

#### 题型 3：用关系图判断（作业 42, 44, 46）

```
每个元素画一个圈
每个 (a,b)∈R 画 a→b 的箭头
自反：每个点都有自环
对称：每条边都有反向边
```

## 9.3 关系表示

### 三种表示

| 表示 | 适用 |
|---|---|
| **集合** (有序对列表) | 小集合 |
| **0-1 矩阵 Mᵢⱼ = 1 iff (aᵢ,aⱼ)∈R** | 中等规模 |
| **有向图**（点 + 箭头） | 中等规模，可视化 |

### 矩阵运算 ↔ 关系运算

| 关系运算 | 矩阵运算 |
|---|---|
| R ∪ S | M_R ∨ M_S |
| R ∩ S | M_R ∧ M_S |
| **R ∘ S**（复合） | **M_S × M_R**（注意顺序） |
| R⁻¹（逆） | M_R 转置 |
| R̄（补） | ~M_R（非） |

> [!warning] 易错
> 复合 R∘S 对应矩阵 M_S × M_R（**S 的矩阵在左边**）
> 矩阵乘法 cᵢⱼ = Σₖ aᵢₖ bₖⱼ
> R∘S 中 (a,c) ∈ R∘S 当且仅当 ∃b: (a,b)∈S ∧ (b,c)∈R

### 反复考的题型（作业 4, 8, 10, 12, 14 反复出）

**作业 4 典型**：两关系 R₁, R₂，求 R₁∪R₂, R₁∩R₂, R₂∘R₁, R₁∘R₁, R₁⊕R₂

```
矩阵按位运算
R₂∘R₁ = M_R₁ × M_R₂（注意 R₁ 的矩阵在左边）
```

## 9.4 关系的闭包（**必考**）

### 三大闭包

| 闭包 | 记号 | 构造方法 |
|---|---|---|
| **自反闭包** | r(R) | 矩阵对角线全置 1 |
| **对称闭包** | s(R) | 矩阵关于对角线求并（M ∨ Mᵀ） |
| **传递闭包** | t(R) | **Warshall 算法**（反复 M ∨ M²） |

### 反复考的：Warshall 算法（**必考大题**）

**算法步骤**：

设 M 是关系 R 的 0-1 矩阵

```
W₀ = M
For k = 1 to n:
    Wₖ[i][j] = 1 iff (Wₖ₋₁[i][j] = 1) OR (Wₖ₋₁[i][k] = 1 AND Wₖ₋₁[k][j] = 1)
Wₙ = 传递闭包 t(R)
```

**关键**：在第 k 步，把 i 经 k 到 j 的路径标为 1

> [!tip] 速记
> Wₖ 含义：考虑中间节点 ∈ {1, 2, ..., k} 时 i 到 j 的可达性

**作业 26 反复考**：用 Algorithm 1（Warshall）求传递闭包

### 反复考的：自反闭包 + 对称闭包（作业 2, 4, 6, 8 反复出）

**作业 2 典型**：R = {(a,b) | a ≠ b}，求自反闭包

```
r(R) = R ∪ {(a,a) | a ∈ A} = R ∪ I_A
其中 I_A = {(a,a) | a ∈ A}
```

**作业 6 典型**：关系图上画对称闭包

```
对每条 (a,b) 边加反向 (b,a)
```

### 反复考的：证明闭包不存在（作业 35）

```
性质 P 满足两个条件才有闭包：
  (i) 包含 R
  (ii) P-closed（保持 P）
"is not reflexive" → 没有闭包（因为包含 R 后可能仍不 reflexivity，但可加 I_A）
```

## 9.5 等价关系（**核心必考**）

### 反复考的核心

#### 题型 1：判断等价关系（作业 1, 11, 12 反复考）

**作业 1 典型**：判断 R 是否是等价关系

```
必须三性质都满足：
  ✓ 自反：∀a, (a,a)∈R
  ✓ 对称：(a,b)∈R → (b,a)∈R
  ✓ 传递：(a,b)∈R ∧ (b,c)∈R → (a,c)∈R
三个 ✓ → 等价关系
两个 ✓ 一个 ✗ → 指出缺哪个性质
```

**作业 12 典型**：R = {位串 (x,y) | 长度 ≥ 3 且前 3 位相同}

```
自反：x 前 3 位 = x 前 3 位 ✓
对称：(x,y) 前 3 位相同 → (y,x) 前 3 位相同 ✓
传递：(x,y) 前 3 位相同 ∧ (y,z) 前 3 位相同 → (x,z) 前 3 位相同 ✓
是等价关系
```

#### 题型 2：证明 R 是等价关系（作业 15, 16 反复考）

**作业 15 典型**：R = {((a,b),(c,d)) | a+d = b+c}

```
自反：(a,b) → (a,b)：a+b = b+a ✓
对称：(a,b)R(c,d) → a+d=b+c → c+b=d+a → (c,d)R(a,b) ✓
传递：(a,b)R(c,d) ∧ (c,d)R(e,f) → a+d=b+c ∧ c+f=d+e
       → a+d=b+c, c+f=d+e → a+d=b+c 和 c+f=d+e
       → a+d+d+e = b+c+c+f → a+e = b+f → (a,b)R(e,f) ✓
```

#### 题型 3：求等价类（作业 22, 24, 36, 40, 42, 48 反复考）

**作业 22 典型**：模 m 同余的等价类

```
R = {(a,b) | a ≡ b (mod m)}
[a]_m = {x ∈ ℤ | x ≡ a (mod m)}
     = {a + km | k ∈ ℤ}

m=2: [0]₂ = {..., -4, -2, 0, 2, 4, ...}（偶数）
     [1]₂ = {..., -3, -1, 1, 3, ...}（奇数）
```

**作业 40 典型**：R = {((a,b),(c,d)) | ad = bc} 求 [(1,2)]_R

```
[(1,2)]_R = {(c,d) | 1·d = 2·c} = {(c,d) | d = 2c}
          = {(c, 2c) | c ∈ ℤ⁺}
          = {(1,2), (2,4), (3,6), ...}
```

#### 题型 4：等价类 ↔ 划分（一对一对应，**必考**）

**核心定理**：

> 集合 A 上的等价关系 R 与 A 的划分一一对应：
> - 每个等价类 = 划分的一个块
> - 划分每个块 = 一个等价类

**作业 55-66 反复考**：给定划分，找对应等价关系；反之亦然

```
划分 {{1,2,3}, {4,5}}
等价关系 R = {(a,b) | a,b ∈ 同一块}
         = {(1,1),(1,2),(1,3),(2,1),(2,2),(2,3),(3,1),(3,2),(3,3),
            (4,4),(4,5),(5,4),(5,5)}
```

## 9.6 偏序（**必考**）

### 反复考的核心

#### 题型 1：判断偏序（自反 + 反对称 + 传递）

```
必须三性质都满足：
  ✓ 自反
  ✓ 反对称
  ✓ 传递
```

#### 题型 2：画 Hasse 图（**必考大题**）

**步骤**：

1. 列出所有元素作为顶点
2. 画覆盖关系（无中间元素的 aRb 关系）
3. 按层次排列：最小元在底部，最大元在顶部
4. 不画自环、不画传递边、不画箭头

**作业 2 典型**：(P({1,2}), ⊆) 的 Hasse 图

```
P({1,2}) = {∅, {1}, {2}, {1,2}}
⊆ 关系：
  ∅ ⊆ {1}, ∅ ⊆ {2}, ∅ ⊆ {1,2}
  {1} ⊆ {1,2}, {2} ⊆ {1,2}

Hasse 图：
  {1,2}
   / \
  {1} {2}
   \ /
    ∅
```

#### 题型 3：找极大/极小/最大/最小元素

| 概念 | 定义 |
|---|---|
| **最大元素** | ∀a∈A, a ≤ M |
| **最小元素** | ∀a∈A, m ≤ a |
| **极大元素** | 没有比它**更大**的元素 |
| **极小元素** | 没有比它**更小**的元素 |

> [!warning] 易错
> - **最大 ↔ 极大**：最大就是极大，但极大不一定是最大（可能有多个极大）
> - 一个偏序集**最大可能不存在**（如链 (Z, ≤) 没有最大）
> - 多个极大可能不共存（如反链）

**作业 16 典型**：A = {1,2,3,4,6,12} 上的整除关系 |，求所有极大/极小/最大/最小

```
12 = 1·2·2·3
画 Hasse 图：
       12
      / \
     6   4
     |   |
     3   2
      \ /
       1

最大：12（只有一个，所有元素都整除 12）
最小：1
极大：12, 6, 4
极小：1
```

#### 题型 4：格（lattice）

**定义**：每对元素都有最小上界（LUB）和最大下界（GLB）的偏序集

**判断方法**：

- 每对 a, b：找 a∨b（LUB）和 a∧b（GLB）
- 都存在 → 格

> [!tip] 速记
> 格的判定：每个"钻石"必须能合（任意两元素有交和并）
