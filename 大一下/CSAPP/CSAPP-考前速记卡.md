## 🎯 一、第一章 + 2.1 信息存储

| 必记点 | 答案 |
|--------|------|
| **汇编器**产生汇编代码（不是编译器！） | 编译器产生 |
| **二进制**输出的工具 | 汇编器、链接器 |
| 高速缓存位置 | **CPU 内**（不在主存） |
| `int **` 占字节数（64 位机） | **8** |
| 大端 `0xAC5A2EF3`，&x=0x2100，问 0x2102 | 高位字节在前 → **0x2E** |
| 小端，同上 | **0x5A** |
| x 最低字节不变，其他位置 1 | **`x \| 0xFF`** |
| `! 0x56` | **0x00**（任何非零都变 0） |

**口诀**：缓存属 CPU，内存条是 DRAM；小端低位低地址；置 1 用 OR。

## 🎯 二、第二章 整数 + 浮点

| 必记点 | 答案 |
|--------|------|
| 10 位**原码**最小 | `-(2^9 - 1) = -511` |
| 10 位**补码** TMin | `-2^9 = -512` |
| TMax 加法逆元 | **TMax 自身** |
| `U2T_8(180)` | `180 - 256 = -76` |
| `char x = -100; unsigned char ux = x` | ux = **156**（位模式不变） |
| `unsigned i = -1; i >= 0` | **true** |
| `-1 < 1U` | **false（0）** |
| `x * 52` 用移位 | `(x<<6) - (x<<4) + (x<<2)` |
| 负数除 128 的偏置 | **127** |
| 无符号加法溢出 | `x + y < x`（或 < y）|
| 8 位补码 33 截到 6 位 | **-31** |
| 补码运算律 | **只交换律** |
| `1e20 + (-1e20 + 1.25)` | **0.0**（不结合）|
| `f*f ≥ 0` 对 float | **总成立**（除 inf/NaN）|
| `i*i ≥ 0` 对 int | **不一定**（50000² 溢出）|
| 类型转换相等 | **只有 A 总为真**（`x==(int)(double)x`）|
| 向偶舍入 `11.011` | `11.10`（看保留位奇偶）|
| 11 位（1+6+4）非规格最大 | `2^-34 × 15` |
| 6 位（1+2+3）非规格最小 | `2^-4 = 1/16` |

**口诀**：原码最小 `-(2^(w-1)-1)`；补码最小 `-2^(w-1)`；浮点不结合；float 精度比 double 差。

## 🎯 三、第三章 机器级表示

| 必记点 | 答案 |
|--------|------|
| 函数返回值 | **%rax** |
| 函数参数 1~6 | `%rdi, %rsi, %rdx, %rcx, %r8, %r9` |
| 寻址公式 | `Imm + R[A] + s·R[B]` |
| `0x36(%rcx, %rdi, 2)` rcx=0x100 rdi=0x18 | **0x166** |
| leaq 行为 | **算地址不访存** |
| `leaq 0x51(%rdi, %rsi, 4), %rdx` rdi=0x310 rsi=0x06 | **0x379** |
| `xorq %r, %r` | **清零** |
| 乘 2 用 | **salq $1, %rdx** |
| 错误 MOV | **C** `movq %edx, (%rsp)`（32 位存 64 位） |
| `movq %rdx, (%rdi)` | **`*p = a`**（p 在 rdi, a 在 rdx） |
| 栈顶 0x0f0100，`pushq %rbx` 后 | **0x0f00f8**（减 8） |
| ZF 是 | **零标志** |
| `cmpl %edi, %esi; setbe %al` | **无符号 a <= b** |
| `cmp %rdi, (%rdx); jge .L3` | **`*p >= a` goto .L3**（注意是有符号 jge） |
| `cmpq $12, %rcx; jl .L3` | if **`k < 12`** |
| `testq %r, %r; jne` | if **`r != 0`** |
| while 翻译 | **guarded-do** |
| call 之前 %rip | **指向 call 自己** |
| call 压栈的 | **call 下一条** |
| call 之后 %rip | **目标函数入口** |
| `int A[6]` 第 i 个地址 | `a + 4i` |
| `char **C[12]` 第 k 个地址 | `c + 8k` |
| `movl 0x60(%rdi, %rsi, 256)` | `rect[i][24]`（rsi 是 i） |
| 结构体对齐规则 | 偏移 = 大小倍数；总大小 = 最大成员倍数 |
| union 字段位置 | **共享同一段内存** |
| 小端 union 解读 | **最低字节放最低位** |
| 按行读 2D 数组 | **空间局部性好** |

**口诀**：返回值 %rax；cmp src,dst → dst-src；set/jX 后缀 g/l 有符号、a/b 无符号；call %rip 套路。

## 🎯 四、第六章 存储器层次

| 必记点 | 答案 |
|--------|------|
| 速度排序 | **SRAM > DRAM > SSD > Flash** |
| 内存条是 | **DRAM 主存** |
| 高速缓存三类 | **直接映射（E=1）、组相联、全相联（S=1）** |
| 缓存参数公式 | `S = C/(B×E)`, `b = log₂B`, `s = log₂S`, `t = m-s-b` |
| m=16 C=1024 E=4 B=4 的 S | **64** |
| 同上 t | **8** |
| 三类不命中 | **冷 / 容量 / 冲突**（无"热"） |
| 步长 vs 空间局部性 | **步长小 → 好** |
| LRU 模拟 | 看组号 + 替换最久未用 |
| Amdahl | `1/((1-α)+α/k)` |
| 消毒 75% + 缝制 25%，整体加速 4/3 的消毒 | **1.50** |

**口诀**：缓存三要素 C/B/E；不命中三类无热；Amdahl 公式记牢。

## 🎯 五、第七章 链接

| 必记点 | 答案 |
|--------|------|
| `.text` | **程序代码** |
| `.data` | 已初始化全局/静态变量 |
| `.bss` | 未初始化静态变量（**占位不占空间**） |
| 符号表位置 | **`.symtab`** |
| 看 ELF 工具 | **`readelf`** |
| **局部变量**在符号表里吗 | **不在**（auto 局部变量运行时栈） |
| 已初始化全局是 | **强符号** |
| 弱符号位置 | **COMMON**（不是 .bss！）|
| 多强同名 | **链接错误** |
| 静态库循环依赖 | **命令行重复出现** |
| 重定位条目 offset | **节内偏移** |
| 重定位条目位置 | **`.rel.text`**（代码段）|
| `ja 4004d5` + 下一条 1 字节 | 第一条地址 **4004d3**（77 02 偏移 +2）|
| 静态库可执行文件依赖 .a 吗 | **不需要**（已拷进可执行）|
| `-fpic` 含义 | **位置无关代码** |
| 链接器打桩条件 | **能拿到可重定位 .o** |
| 动态链接器输出 | **可执行目标文件** |

**口诀**：.text 代码、.data 已初始化、.bss 占位；COMMON 弱符号；多强链接错；静态库循环要重复。

## 🎯 六、第八章 异常控制流

| 必记点 | 答案 |
|--------|------|
| 不属于 ECF | **强制类型转换** |
| 中断 | **异步** |
| 陷阱 | **有意触发（不是致命错误）** |
| 故障 | 返回**当前指令**（可重试）|
| 终止 | **从不返回** |
| 并发进程对（A1-4, B3-6, C5-9, D7-11）| **3 对**（A-B, B-C, C-D）|
| fork 输出 2 行数 | 画进程树，**4 行 2**（常见题）|
| shell 流程 | **fork → 子进程 execve** |
| `wait()` 行为 | **阻塞父进程** |
| 僵尸 vs 孤儿 | 僵尸：子结束父没 wait；孤儿：父没了 init 收养 |
| Ctrl-C / Ctrl-Z | SIGINT / SIGTSTP |
| `kill -9` 信号 | SIGKILL（**不可捕获**）|
| 杀进程组 | `/bin/kill -9 -pgrp` |
| setjmp 特点 | 调用一次返回**一次或多次** |
| signal + pause + exit handler | 输出 **2 行：Wait + Caught**，不会 End |

**口诀**：异常四类记"中陷故终"；fork 返回两次；僵尸孤儿别混；Ctrl-C→SIGINT；Ctrl-Z→SIGTSTP。

## 🎯 七、第九章 虚拟内存 + 动态内存分配

| 必记点 | 答案 |
|--------|------|
| 虚拟内存位置 | **硬盘**（机制 + 数据可放硬盘）|
| CPU 用什么地址 | **虚拟地址** |
| TLB 好处 | **省页表查找时间** |
| 地址翻译 PPO | **= VPO**（页内偏移一致）|
| 多级页表 VPN_i | **查第 i 级页表** |
| 多级页表 VPN_k | **直接拿 PPN** |
| 多级页表大小由 | **虚页数**决定（不是实页数）|
| 第一级页表 | **常驻主存** |
| 隐式分配器 | **Java new / Python GC** |
| 性能权衡 | **时间 vs 空间** |
| 隐式链表链接 | **块头部长度** |
| 放置策略 | **首次 / 下次 / 最佳**（无 Mark and Sweep）|
| 边界标记用途 | **便于和前一个空闲块合并** |
| 块标记位 | **最低位** |
| 提取标记位 | **`& 1`** |
| 显式空闲块最小 | **16 字节**（头 4 + 脚 4 + pred 4 + succ 4）|
| malloc 实际方案 | **分离适配** |

**口诀**：CPU 用虚拟地址；TLB 省页表时间；多级页表 VPN_i 查 i 级；隐式靠头部遍历；malloc 分离适配。
