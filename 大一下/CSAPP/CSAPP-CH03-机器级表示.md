# CH03 程序的机器级表示（x86-64）

> 一句话总结：**汇编是 C 语言的直接翻译**：操作数有立即数/寄存器/内存三种，访存靠**有效地址公式 `Imm + R[Ri] + s`**，条件跳转用**条件码 + set/jX**，调用用 `call/ret + %rsp` 维护栈帧。

> 相关：[[CSAPP-CH01-漫游与信息存储]]、[[CSAPP-CH06-存储器层次与高速缓存]]、[[CSAPP全部测验题目]]

## 1. 核心框架

```
C 源 ──▶ GCC 编译 ──▶ x86-64 汇编
                       │
        ┌──────────────┼──────────────┐
        │              │              │
   数据传送 mov    算术/逻辑 add/sub/...   控制流 jX / call / ret
        │              │              │
     寄存器/内存    设置条件码       修改 %rip / %rsp
```

## 2. 数据格式与寄存器

### 2.1 数据宽度后缀

| 后缀 | 大小 | C 类型 |
|------|------|--------|
| `b` | 1 字节 | char |
| `w` | 2 字节 | short |
| `l` | 4 字节 | int |
| `q` | 8 字节 | long / 指针 |

### 2.2 关键寄存器（记住代号）

| 寄存器 | 用途 |
|--------|------|
| `%rax` | **返回值** |
| `%rsp` | **栈指针**（指向栈顶） |
| `%rdi, %rsi, %rdx, %rcx, %r8, %r9` | 函数参数 1~6 |
| `%rbp` | 帧基址 |
| `%rbx, %r12~%r15` | 被调者保存 |

📝 **真题**：函数返回值最后放 **%rax**。

## 3. 寻址模式（必考）

### 3.1 操作数三种类型

- **立即数**：`$0x1234`
- **寄存器**：`%rax`
- **内存引用**：`Imm(%rA, %rB, s)` 表示 `M[Imm + R[%rA] + s·R[%rB]]`

### 3.2 有效地址公式

$$\text{EA} = \text{Imm} + R[\text{Ri}] + s$$

⚠️ **比例因子 s = 2 的幂**（1/2/4/8），对应元素大小。

📝 **真题**：操作数 `0x36(%rcx, %rdi, 2)`，rcx=0x100, rdi=0x18
→ `0x36 + 0x100 + 2×0x18 = 0x36 + 0x100 + 0x30 = 0x166`

### 3.3 leaq（地址算术）

`leaq Imm(%rA, %rB, s), %rd`：**把地址算出来存到寄存器**，不访存。

📝 **真题**：`leaq 0x51(%rdi, %rsi, 4), %rdx`，rdi=0x310, rsi=0x06
→ `0x51 + 0x310 + 4×0x06 = 0x51 + 0x310 + 0x18 = 0x379`

## 4. 数据传送指令

### 4.1 mov 限制（⚠️ 必考）

| 规则 | 例子 |
|------|------|
| 不能**内存→内存** | `mov (%rax), (%rbx)` ❌ |
| 源 / 目的大小要匹配 | `movw %si, %di` ✅（都是 16 位） |
| `movq %edx, (%rsp)` | ❌ **32 位值不能直接存到 64 位栈槽**（要 movslq 先零扩展） |
| `movb $22, -23(%rsi)` | ✅（立即数 → 内存） |

📝 **真题**：下列 MOV 错误的是 **C**（32 位存 64 位栈槽）。

### 4.2 mov ↔ C 代码

| 汇编 | C 代码 |
|------|--------|
| `movq %rdx, (%rdi)` | **`*p = a`**（p 在 %rdi，a 在 %rdx） |
| `movq (%rdi), %rax` | `a = *p` |
| `movq %rax, %rdi` | `p = a`（传参数） |

📝 **真题**：`movq %rdx, (%rdi)` 对应 `*p = a`（选项 C）。

## 5. 栈操作（push / pop）

### 5.1 pushq %rbx

```
%rsp -= 8                    // 栈顶下移
M[%rsp] ← %rbx               // 值写入新栈顶
```

### 5.2 popq %rax

```
%rax ← M[%rsp]               // 读栈顶
%rsp += 8                    // 栈顶上移
```

📝 **真题**：
- 栈顶 `0x0f0100`，`pushq %rbx` → 栈顶 = `0x0f00f8`（减 8）
- `popq %rax` 后 `%rsp += 8` → `0x158 → 0x160`

## 6. 算术逻辑指令

### 6.1 关键指令

| 指令 | 作用 |
|------|------|
| `xorq %r, %r` | **把 r 清零**（常用技巧） |
| `salq $1, %rdx` | 左移 1 位（**乘 2**） |
| `sarq $1, %rdx` | 算术右移 1 位（**除 2**） |
| `imulq` | 有符号乘法 |
| `leaq` | 算地址 |

⚠️ **`imulq $2` 只有一个操作数是错的**（imul 是双操作数或三操作数）。**乘 2 用左移**。

📝 **真题**：%rdx 乘以 2 = `salq $1, %rdx`（选 C）。

## 7. 条件码与控制流（重点）

### 7.1 条件码

| 条件码 | 含义 |
|--------|------|
| **CF** | 进位标志（无符号溢出） |
| **ZF** | **零标志** |
| **SF** | 符号标志（结果最高位） |
| **OF** | 溢出标志（有符号溢出） |

📝 **真题**：ZF = **零标志**（B）。

### 7.2 set 指令（按条件写字节）

| set 指令 | 条件 | 对应 C |
|---------|------|--------|
| `sete` | ZF | a == b |
| `setne` | !ZF | a != b |
| `setg` | SF==OF && !ZF | a > b（**有符号**） |
| `setl` | SF != OF | a < b（**有符号**） |
| `setge` | SF==OF | a >= b（**有符号**） |
| `setle` | SF!=OF \|\| ZF | a <= b（**有符号**） |
| `seta` | !CF && !ZF | a > b（**无符号**） |
| `setb` | CF | a < b（**无符号**） |
| `setbe` | CF \|\| ZF | a <= b（**无符号**） |
| `setae` | !CF | a >= b（**无符号**） |

📝 **真题**：`cmpl %edi, %esi` + `setbe %al` → **无符号** `esi <= edi` → 选 D（unsigned a <= b）。

### 7.3 cmp + jX（条件跳转）

⚠️ **cmp 是** `cmp src, dst` **→ 计算 dst - src**，不是 src - dst！

| 汇编 | C |
|------|---|
| `cmp %rdi, (%rdx)` | 计算 `*p - a`（%rdx 是 p，%rdi 是 a） |
| `jge .L3` | 跳 if `*p >= a` |

📝 **真题**：`cmp %rdi, (%rdx); jge .L3` → if `*p >= a` goto .L3（**选 D**，不是 `<=`！）。

### 7.4 jX 反推条件（⚠️ 注意有符号/无符号）

| 跳转 | 条件 |
|------|------|
| `je / jne` | == / != |
| `jg / jge / jl / jle` | > / >= / < / <=（**有符号**） |
| `ja / jae / jb / jbe` | > / >= / < / <=（**无符号**） |
| `js / jns` | 负 / 非负 |

### 7.5 cmpq 与条件码

- `cmpq $12, %rcx`：计算 `rcx - 12`
- `jl .L3`：跳 if `rcx - 12 < 0` → 即 `rcx < 12`

📝 **真题**：`cmpq $12, %rcx; jl .L3` → if **`k < 12`** then ...（选 A）。

### 7.6 testq + jne

`testq %rdx, %rdx` 做 **AND 不写回**，只设条件码。
- 结果 = 0 → ZF=1 → `jne` 不跳
- 结果 ≠ 0 → ZF=0 → `jne` 跳

📝 **真题**：`testq %rdx, %rdx; jne .L3` → if `rdx != 0` goto .L3。

## 9. 过程调用（call / ret）

### 9.1 call 指令执行步骤

```
callq addr
   │
   ├─ 1. %rip ← addr（PC 切换到目标）
   ├─ 2. %rsp -= 8
   └─ 3. M[%rsp] ← 返回地址（call 下一条指令）
```

### 9.2 %rip 的"先取后增"

⚠️ **x86 是先取指令后递增**，所以：
- 执行 `mov 0x5(%rdx), %rax` 时，`%rip = 310040`（指向 mov）
- 执行 `callq 310020` 时，**%rip 已经是 310045**（call 自己的地址）
- `callq` 压入栈的返回地址 = **310052**（call 下一条 add 的地址）
- 进入被调函数后，`%rip = 310020`（目标函数第一条指令）

📝 **真题**：
- call 之前 %rip = **310045**（指向 call 自己）
- call 压栈的 = **310052**（call 下一条）
- call 之后 %rip = **310020**（目标函数入口）

## 10. 数组与指针算术

### 10.1 一维数组

`int A[6]` 每个元素 4 字节，第 i 个元素地址 = `a + 4·i`。

`char **C[12]` 每个元素是指针（8 字节），第 k 个元素地址 = `c + 8·k`。

⚠️ **`sizeof(指针) = 8`**（64 位机），`sizeof(int) = 4`，别搞混。

📝 **真题**：
- `int A[6]`，第 i 个元素地址 = **`a + 4i`**
- `char **C[12]`，第 k 个元素地址 = **`c + 8k`**

### 10.2 二维数组访问（看题算偏移）

`int rect[32][32]`，rect[i][j] 地址 = `&rect[0][0] + i·32·4 + j·4` = `&rect[0][0] + 128·i + 4·j`

📝 **真题**：`movl 0x60(%rdi, %rsi, 256), %eax`，rdi=rect, rsi=i
→ 偏移 = `0x60 + 256·i = 96 + 256·i`
→ 应该是 `rect[i][j]`，j·4 = 96 → j=24
→ **`result = rect[i][24]`**（i 在 rsi）

⚠️ **看清楚哪个寄存器是哪个变量！** 题里 `i` 在 rsi，`rect` 在 rdi。

### 10.3 寄存器分配规则

- 局部变量**能放寄存器就放寄存器**（编译器优化）
- 必须存内存的：`&a` 取地址、`a` 作为参数传给函数
- **被调者保存寄存器**在过程调用前后必须恢复

📝 **真题**：`fun1` 中 `a2 = 456` 的 `&a2` 传给 `change`，所以 **`a2 必须存内存`**，但 `a1` 可以放寄存器（只传值）。

## 11. 结构体布局（高频易错）

### 11.1 三大规则

1. **成员按声明顺序排列**
2. **每个成员的偏移量是其大小的整数倍**
3. **结构体总大小是最大成员大小的整数倍**（对齐）

### 11.2 重排优化（最小化 padding）

- **大成员放前面**，小成员扎堆放后面
- 同样大小的成员顺序不重要

📝 **真题**（MEMO 结构）：`int(4) + char(1) + int(4) + char[21](21) + char*(8) + char(1)`
- 原始顺序会有 3 字节 padding 给 `flag`（4 字节对齐）、7 字节 padding 给 `ref` 后面（8 字节对齐）、7 字节 padding 给 `content` 后面（8 字节对齐）
- 最优：把大的放前面，`title` 21 字节放中间，**`content(8) + index(4) + ref(4) + flag(1)+title[21]+isPrivate(1)`**

### 11.3 嵌套结构体（必考）

`struct TCP`：短(2)+字符(1)+int(4)+三字符(3)+指针(8)

| 字段 | 偏移 | 大小 |
|------|------|------|
| dest_port (short) | 0 | 2 |
| offset (char) | 2 | 1 |
| **padding** | 3 | 1 |
| sn (int) | 4 | 4 |
| ACK | 8 | 1 |
| SYN | 9 | 1 |
| FIN | 10 | 1 |
| **padding** | 11 | 5 |
| next (指针) | 16 | 8 |

📝 **真题**：`msg->flag.SYN` 的偏移 = **9**（选 C）。
**结构体总大小 = 24**（下一个 8 字节对齐边界）。

## 12. union（共用体）

所有字段**共享同一段内存**，大小 = 最大字段大小。

📝 **真题**：
```
union { char c[8]; unsigned int i[2]; } dw;
dw.c[0]=0xd0, dw.c[1]=0xd1, ..., dw.c[7]=0xd7;
```
- 小端机器，i[0] = `M[0..3]` = `0xd0 + 0xd1·256 + 0xd2·256² + 0xd3·256³` = `0xd3d2d1d0`
- i[1] = `0xd7d6d5d4`

⚠️ **小端是低位在低地址**：i[0] 看 c[0..3] 的字节，但组成 int 时是 **最低字节放最低位**，所以是 `c[0]` 放到 int 最低 8 位。

## 13. 空间局部性

| 模式 | 局部性 |
|------|--------|
| **按行读 2D 数组** | 空间局部性**好**（连续地址） |
| **按列读 2D 数组** | 空间局部性**差**（跳步） |

📝 **真题**：按行读空间局部性更好（选 A）。

## 15. 速记口诀

> **返回值在 %rax，参数在 %rdi~%r9**
> **栈顶 push 减 8、pop 加 8**
> **寻址公式：Imm + R[A] + s·R[B]**
> **leaq 算地址不访存**
> **mov 不能内存对内存，32 位不能直接存 64 位栈槽**
> **乘 2 用 salq $1，不用 imulq**
> **ZF = 零标志；CF = 进位；SF = 符号；OF = 溢出**
> **cmp src, dst → dst - src**
> **set/jX 后缀：g/l 是有符号，a/b 是无符号**
> **while 翻译是 guarded-do**
> **call 之前 %rip 指向 call 自己；call 压栈是 call 下一条；call 之后 %rip 指向目标**
> **结构体对齐：偏移是大小倍数，总大小是最大成员倍数**
> **小端：低位字节在低地址，组成 int 时最低字节放最低位**
> **按行读 2D 数组空间局部性好**
