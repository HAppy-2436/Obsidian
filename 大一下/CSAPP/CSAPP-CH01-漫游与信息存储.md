# CH01+2.1 系统漫游与信息存储

> 一句话总结：**系统 = CPU + 主存 + I/O 设备**，通过总线连接；信息在内存里就是字节序列，规则由**字长、大小端、补码**决定。

> 相关：[[CSAPP-CH02-整数与浮点]]、[[CSAPP全部测验题目]]

---

## 1. 核心框架

### 1.1 编译系统四阶段（高频考）

```
源文件 .c ──预处理器──▶ .i ──编译器──▶ .s ──汇编器──▶ .o ──链接器──▶ 可执行
   (cpp)        (cc1)        (as)         (ld)
```

| 工具 | 输入 | 输出 | 是否二进制 |
|------|------|------|------------|
| **预处理器** | .c + .h | .i（文本） | ❌ |
| **编译器** | .i | .s（汇编文本） | ❌ |
| **汇编器** | .s | .o（可重定位目标文件） | ✅ |
| **链接器** | .o + .o + 库 | 可执行文件 | ✅ |

⚠️ **真题点**：汇编器、链接器的输出是**二进制**；汇编代码是**编译器**产生的（汇编器只翻译，不"写"汇编代码）。

### 1.2 存储器层次（速度 ↑ / 容量 ↓）

```
CPU寄存器 → L1/L2/L3高速缓存 → 主存(DRAM) → 本地SSD/磁盘 → 远程存储
   ↑                                                    ↑
 高速缓存属于 CPU，不属于主存！
```

⚠️ **真题点**：高速缓存是 **CPU 内部组件**（不在主存里），主存也叫内存条。

## 2. 进制与数制（速算口诀）

### 2.1 进制转换

- **十 → 二**：除 2 取余，**倒序**写 → `172 = 10101100`（128+32+8+4）
- **二 → 十六**：4 位一组，低位开始补零 → `100100100 = 0x124`
- **二 → 八**：3 位一组

### 2.2 N 进制乘法（手算）

例：六进制 `13 × 12`：
- 转十进制：`9 × 8 = 72`
- 转回六进制：`72 = 1·36 + 2·6 + 0 → 120₆`

⚠️ **别把答案看成十进制数！选项 A "156" 是十进制，六进制运算结果仍是六进制数字。**

## 3. 字长、大小端、寻址

### 3.1 字长

- 64 位机器：**指针 8 字节**，int 4 字节，long 8 字节
- `int **` 存的是指针 → **8 字节**

### 3.2 大端 vs 小端（必考）

| 模式 | 高位字节 | 低位字节 |
|------|----------|----------|
| **小端** | 高地址 | 低地址 |
| **大端** | 低地址 | 高地址 |

例：`int x = 0xAC5A2EF3`，&x = 0x2100

| 地址 | 2100 | 2101 | 2102 | 2103 |
|------|------|------|------|------|
| 小端 | F3   | 2E   | **5A** | AC |
| 大端 | AC   | 5A   | **2E** | F3 |

📝 **真题**：地址 0x2102 看是第几字节（从 0 开始 = 第 3 字节），小端答案 `5A`，大端答案 `2E`。

### 3.3 字符串字面量

字符串按**字符顺序**从低地址开始存（与 int 大小端无关）。
例：`s = "hello world..."`，`s = 0x2300` 大端 → 地址 0x2305 = 第 6 个字符 = `o`（h=0,e=1,l=2,l=3,o=4,空格=5）

⚠️ **字符串第几位字符直接数索引**，不要套 int 的字节序规则。

## 4. 位运算技巧（做题秒杀）

### 4.1 常用 mask

| 需求 | 表达式 |
|------|--------|
| **最低字节不变，其他位全 1** | `x \| 0xFF` |
| 只保留最低字节 | `x & 0xFF` |
| 把最低字节清零 | `x & ~0xFF` |
| 把所有位翻转 | `x ^ ~0` |

📝 **真题**：x 最低字节不变，其他位置 1 → `x | 0xFF`（答案 C）。

### 4.2 逻辑运算 vs 位运算（必考）

| 运算符 | 作用 | 结果 |
|--------|------|------|
| `!x` | 逻辑非 | **0 或 1** |
| `~x` | 按位取反 | 每一位都翻转 |
| `&&` / `\|\|` | 短路 | **0 或 1** |
| `&` / `\|` | 按位 | 逐位运算 |

⚠️ **任何非零值做 `!` 都得 0；任何非零值做 `~` 结果位都翻转。**
例：`! 0x56` = 0（因为 0x56 ≠ 0）。

## 6. 速记口诀

> **汇编器、链接器 → 二进制；预处理器、编译器 → 文本**
> **缓存属 CPU，内存条是 DRAM**
> **小端：低位在低地址；大端：高位在低地址**
> **置 1 用 OR，清 0 用 AND，取反用 XOR**
> **`!` 永远得 0 或 1，`~` 永远逐位翻转**
