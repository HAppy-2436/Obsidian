# 2.82
## A
不总为1
-TMin = Tmin
故若取x = Tmin, y = 1结果为0

## B
总是为1
(x + y) << 4 = (x + y) * 16
故显然左=右

## C
总是为1
-x = ~x + 1
~x + ~y + 1 = (-x - 1) + (-y - 1) + 1 = -x - y - 1 = ~(x + y)

# D
总是为1
转换为无符号数比特位不变，x - y = -(y - x)成立，
故(ux - uy) = -(unsigned)(y - x)

## E
总是为1
x > 0, x / 4为向零取整，此时((x >> 2) << 2) <= x
x < 0, x / 4为向负无穷取整，此时((x >> 2) << 2) <= x
